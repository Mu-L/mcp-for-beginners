# Ondersteuning

## Hoe problemen melden en hulp krijgen  

Dit project gebruikt GitHub Issues om bugs en featureverzoeken bij te houden. Zoek alstublieft eerst in de bestaande 
issues voordat u nieuwe issues aanmaakt om duplicaten te voorkomen. Voor nieuwe issues, dien uw bug of 
featureverzoek in als een nieuwe Issue.

Voor hulp en vragen over het gebruik van dit project, gelieve:
- Bekijk de [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) waar experts en mede-ontwikkelaars samenkomen
- Gebruik de [Model Context Protocol Community](https://modelcontextprotocol.io/community/) bronnen voor protocolgerelateerde vragen

## Microsoft Ondersteuningsbeleid  

Ondersteuning voor het Model Context Protocol (MCP) voor Beginners project is beperkt tot de hierboven vermelde bronnen.
