# Support

## Comment signaler des problèmes et obtenir de l'aide  

Ce projet utilise GitHub Issues pour suivre les bugs et les demandes de fonctionnalités. Veuillez rechercher les 
problèmes existants avant d’en créer de nouveaux afin d'éviter les doublons. Pour de nouveaux problèmes, 
signalez votre bug ou demande de fonctionnalité en créant une nouvelle Issue.

Pour obtenir de l'aide et poser des questions sur l'utilisation de ce projet, veuillez :
- Consulter le [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) où experts et développeurs se rassemblent
- Utiliser les ressources de la [Model Context Protocol Community](https://modelcontextprotocol.io/community/) pour les questions liées au protocole

## Politique de support Microsoft  

Le support pour le projet Model Context Protocol (MCP) for Beginners est limité aux ressources listées ci-dessus.
