# การสนับสนุน

## วิธีการรายงานปัญหาและรับความช่วยเหลือ  

โครงการนี้ใช้ GitHub Issues ในการติดตามบั๊กและคำขอฟีเจอร์ กรุณาค้นหา  
issues ที่มีอยู่ก่อนที่จะรายงาน issue ใหม่เพื่อป้องกันการซ้ำกัน สำหรับ issue ใหม่ ให้รายงานบั๊กหรือ  
คำขอฟีเจอร์ของคุณเป็น Issue ใหม่

สำหรับความช่วยเหลือและคำถามเกี่ยวกับการใช้โครงการนี้ กรุณา:
- ตรวจสอบที่ [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ซึ่งเป็นที่รวมตัวของผู้เชี่ยวชาญและนักพัฒนาร่วมกัน
- ใช้แหล่งข้อมูลของ [Model Context Protocol Community](https://modelcontextprotocol.io/community/) สำหรับคำถามที่เกี่ยวข้องกับโปรโตคอล

## นโยบายการสนับสนุนของ Microsoft  

การสนับสนุนสำหรับโครงการ Model Context Protocol (MCP) for Beginners จำกัดเฉพาะแหล่งข้อมูลที่กล่าวถึงข้างต้นเท่านั้น
