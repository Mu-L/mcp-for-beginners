# 🚀 เครื่องมือ MCP

## 🧠 ภาพรวมของเส้นทางการเรียนรู้การผนวกฐานข้อมูล MCP

ที่นี่คุณจะได้เรียนรู้วิธีใช้เครื่องมือที่ใช้ MCP ประสบการณ์ของคุณในการใช้เครื่องมือเหล่านี้จะดีขึ้นเมื่อคุณเชื่อมต่อเซิร์ฟเวอร์ MCP กับเครื่องมือเหล่านี้

## 🔗 ทรัพยากรอย่างเป็นทางการของ MCP

- 📘 [เอกสาร MCP](https://modelcontextprotocol.io/) – บทช่วยสอนและคู่มือสำหรับผู้ใช้โดยละเอียด
- 📜 [ข้อกำหนด MCP (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – สถาปัตยกรรมโปรโตคอลและข้อมูลอ้างอิงทางเทคนิค
- 🧑‍💻 [ที่เก็บ GitHub ของ MCP](https://github.com/modelcontextprotocol) – SDK แบบโอเพนซอร์ส เครื่องมือ และตัวอย่างโค้ด
- 🌐 [ชุมชน MCP](https://github.com/orgs/modelcontextprotocol/discussions) – เข้าร่วมการสนทนาและมีส่วนร่วมกับชุมชน
- 🔒 [OWASP MCP Top 10](https://microsoft.github.io/mcp-azure-security-guide/mcp/) – แนวทางปฏิบัติด้านความปลอดภัยที่ดีที่สุดและการลดความเสี่ยง
