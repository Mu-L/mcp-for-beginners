# เครื่องมือสร้างแผนการศึกษาพร้อม Chainlit & Microsoft Learn Docs MCP

## ข้อกำหนดเบื้องต้น

- Python 3.8 ขึ้นไป
- pip (ตัวจัดการแพ็กเกจของ Python)
- การเชื่อมต่ออินเทอร์เน็ตเพื่อเชื่อมต่อกับเซิร์ฟเวอร์ Microsoft Learn Docs MCP

## การติดตั้ง

1. โคลนรีโพซิทอรีนี้หรือดาวน์โหลดไฟล์โปรเจกต์
2. ติดตั้ง dependencies ที่จำเป็น:

   ```bash
   pip install -r requirements.txt
   ```

## การใช้งาน

### กรณีที่ 1: การสอบถามเอกสาร MCP แบบง่าย
ไคลเอนต์บรรทัดคำสั่งที่เชื่อมต่อกับเซิร์ฟเวอร์ Docs MCP ส่งคำถาม และแสดงผลลัพธ์

1. รันสคริปต์:
   ```bash
   python scenario1.py
   ```
2. พิมพ์คำถามเอกสารของคุณที่พรอมต์

### กรณีที่ 2: เครื่องมือสร้างแผนการศึกษา (แอปเว็บ Chainlit)
อินเทอร์เฟซแบบเว็บ (ใช้ Chainlit) ที่ให้ผู้ใช้สร้างแผนการศึกษาส่วนตัวเป็นรายสัปดาห์สำหรับหัวข้อทางเทคนิคใดก็ได้

1. เริ่มแอป Chainlit:
   ```bash
   chainlit run scenario2.py
   ```
2. เปิด URL ท้องถิ่นที่แสดงในเทอร์มินัลของคุณ (เช่น http://localhost:8000) ในเว็บเบราว์เซอร์
3. ในหน้าต่างแชท พิมพ์หัวข้อการศึกษาของคุณและจำนวนสัปดาห์ที่ต้องการศึกษา (เช่น "AI-900 certification, 8 weeks")
4. แอปจะตอบกลับด้วยแผนการศึกษารายสัปดาห์ พร้อมลิงก์ไปยังเอกสาร Microsoft Learn ที่เกี่ยวข้อง

**ตัวแปรสภาพแวดล้อมที่จำเป็น:**

สำหรับการใช้กรณีที่ 2 (แอปเว็บ Chainlit กับ Azure OpenAI) คุณต้องตั้งค่าตัวแปรสภาพแวดล้อมต่อไปนี้ในไฟล์ `.env` ภายใต้ไดเรกทอรี `python`:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

กรอกข้อมูลเหล่านี้ด้วยรายละเอียดทรัพยากร Azure OpenAI ของคุณก่อนรันแอป

> [!TIP]
> คุณสามารถปรับใช้โมเดลของคุณเองได้อย่างง่ายดายโดยใช้ [Microsoft Foundry](https://ai.azure.com/).

### กรณีที่ 3: เอกสารในโปรแกรมแก้ไขด้วย MCP Server ใน VS Code

แทนที่การสลับแท็บเบราว์เซอร์เพื่อค้นหาเอกสาร คุณสามารถนำ Microsoft Learn Docs เข้ามาใน VS Code ของคุณโดยตรงด้วยเซิร์ฟเวอร์ MCP ซึ่งช่วยให้คุณ:
- ค้นหาและอ่านเอกสารภายใน VS Code โดยไม่ต้องออกจากสภาพแวดล้อมการเขียนโค้ด
- อ้างอิงเอกสารและแทรกลิงก์โดยตรงใน README หรือไฟล์หลักสูตรของคุณ
- ใช้ GitHub Copilot และ MCP ร่วมกันเพื่อเวิร์กโฟลว์เอกสารด้วย AI ที่ราบรื่น

**กรณีการใช้งานตัวอย่าง:**
- เพิ่มลิงก์อ้างอิงใน README ได้อย่างรวดเร็วระหว่างเขียนเอกสารหลักสูตรหรือโปรเจกต์
- ใช้ Copilot สร้างโค้ด และ MCP ค้นหาและอ้างอิงเอกสารที่เกี่ยวข้องทันที
- โฟกัสในโปรแกรมแก้ไขและเพิ่มประสิทธิภาพการทำงาน

> [!IMPORTANT]
> ตรวจสอบให้แน่ใจว่าคุณมีการตั้งค่า [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) ที่ถูกต้องในพื้นที่ทำงานของคุณ (ตำแหน่งไฟล์คือ `.vscode/mcp.json`)

## ทำไมต้องใช้ Chainlit สำหรับกรณีที่ 2?

Chainlit คือเฟรมเวิร์กโอเพ่นซอร์สสมัยใหม่สำหรับสร้างแอปเว็บสนทนา ทำให้ง่ายต่อการสร้างอินเทอร์เฟซแชทที่เชื่อมต่อกับบริการแบ็กเอนด์เช่น Microsoft Learn Docs MCP server โปรเจกต์นี้ใช้ Chainlit เพื่อมอบวิธีที่เรียบง่ายและโต้ตอบได้ในการสร้างแผนการศึกษาที่ปรับตามรายบุคคลแบบเรียลไทม์ โดยใช้ Chainlit คุณสามารถสร้างและปรับใช้เครื่องมือแชทที่ช่วยเพิ่มประสิทธิภาพและการเรียนรู้ได้อย่างรวดเร็ว

## สิ่งที่แอปนี้ทำได้

แอปนี้ช่วยให้ผู้ใช้สร้างแผนการศึกษาส่วนตัวได้โดยง่ายเพียงแค่ป้อนหัวข้อและระยะเวลา แอปจะวิเคราะห์อินพุตของคุณ สอบถามเซิร์ฟเวอร์ Microsoft Learn Docs MCP เพื่อหาคอนเทนต์ที่เกี่ยวข้อง และจัดระเบียบผลลัพธ์เป็นแผนรายสัปดาห์ คำแนะนำของแต่ละสัปดาห์จะถูกแสดงในแชท ทำให้ง่ายต่อการติดตามและวางแผนความก้าวหน้า การผสานนี้รับประกันว่าคุณจะได้รับทรัพยากรการเรียนรู้ล่าสุดและเกี่ยวข้องที่สุดเสมอ

## ตัวอย่างคำค้น

ลองใช้คำค้นเหล่านี้ในหน้าต่างแชทเพื่อดูการตอบสนองของแอป:

- `AI-900 certification, 8 weeks`
- `Learn Azure Functions, 4 weeks`
- `Azure DevOps, 6 weeks`
- `Data engineering on Azure, 10 weeks`
- `Microsoft security fundamentals, 5 weeks`
- `Power Platform, 7 weeks`
- `Azure AI services, 12 weeks`
- `Cloud architecture, 9 weeks`

ตัวอย่างเหล่านี้แสดงความยืดหยุ่นของแอปสำหรับเป้าหมายการเรียนรู้และกรอบเวลาที่หลากหลาย
