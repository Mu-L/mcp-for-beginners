# الدعم

## كيفية تقديم المشكلات والحصول على المساعدة 

يستخدم هذا المشروع GitHub Issues لتعقب الأخطاء وطلبات الميزات. يرجى البحث في 
المشكلات الموجودة قبل تقديم مشكلات جديدة لتجنب التكرار. لتقديم مشكلات جديدة، قدم خطأك أو 
طلب الميزة كمشكلة جديدة.

للحصول على المساعدة والأسئلة حول استخدام هذا المشروع، يرجى:
- التحقق من [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) حيث يجتمع الخبراء والمطورون
- استخدام موارد [Model Context Protocol Community](https://modelcontextprotocol.io/community/) للأسئلة المتعلقة بالبروتوكول

## سياسة دعم مايكروسوفت 

الدعم لمشروع Model Context Protocol (MCP) للمبتدئين يقتصر على الموارد المدرجة أعلاه.
