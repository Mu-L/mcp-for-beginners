# AGENTS.md

## Преглед на проекта

**MCP за начинаещи** е образователна учебна програма с отворен код за изучаване на Model Context Protocol (MCP) - стандартизиран фреймуърк за взаимодействия между AI модели и клиентски приложения. Това хранилище предоставя изчерпателни учебни материали с практически примери за код на различни програмни езици.

### Основни технологии

- **Езици за програмиране**: C#, Java, JavaScript, TypeScript, Python, Rust
- **Фреймуъркове и SDK**: 
  - MCP SDK (`@modelcontextprotocol/sdk`)
  - Spring Boot (Java)
  - FastMCP (Python)
  - LangChain4j (Java)
- **Бази данни**: PostgreSQL с разширение pgvector
- **Облачни платформи**: Azure (Container Apps, OpenAI, Content Safety, Application Insights)
- **Инструменти за билдване**: npm, Maven, pip, Cargo
- **Документация**: Markdown с автоматизиран многоезичен превод (48+ езика)

### Архитектура

- **11 основни модула (00-11)**: Последователен учебен път от основи до напреднали теми
- **Практически лаборатории**: Практически упражнения с пълен код за решение на няколко езика
- **Примерни проекти**: Работещи реализации на MCP сървър и клиент
- **Система за превод**: Автоматизиран GitHub Actions работен поток за многоезична поддръжка
- **Изображения**: Централизирана директория с изображения с преведени версии

## Работен процес при разработка

### Структура на документацията

- **Модули 00-11**: Основно съдържание на учебната програма в последователен ред
- **translations/**: Езикови версии (автоматично генерирани, не редактирайте директно)
- **translated_images/**: Локализирани версии на изображения (автоматично генерирани)
- **images/**: Изходни изображения и диаграми

### Направяне на промени в документацията

1. Редактирайте само английските markdown файлове в коренните директории на модулите (00-11)
2. Актуализирайте изображенията в директория `images/` при нужда
3. GitHub Action co-op-translator автоматично генерира преводи
4. Преводите се възстановяват при всеки push към основния клон

### Работа с преводите

- **Автоматизиран превод**: GitHub Actions обработва всички преводи
- **НЕ редактирайте ръчно** файлове в директория `translations/`
- Метаданните за превод са включени във всеки преведен файл
- Поддържани езици: 48+ езика включително арабски, китайски, френски, немски, хинди, японски, корейски, португалски, руски, испански и други

## Насоки за стил на кода

### Стил на документацията

- Използвайте ясен език, подходящ за начинаещи
- Включвайте примери с код на различни езици, когато е приложимо
- Следвайте добрите практики при Markdown:
  - Използвайте ATX стил заглавия (`#` синтаксис)
  - Използвайте оградени блокове с идентификатори на езика
  - Включвайте описателен alt текст за изображения
  - Поддържайте разумна дължина на редовете (без твърда граница, но разумно)

### Стил на примерите с код

#### TypeScript/JavaScript
- Използвайте ES модули (`import`/`export`)
- Спазвайте строг режим на TypeScript
- Включвайте анотации за типове
- Целете ES2022

#### Python
- Следвайте PEP 8 стилови насоки
- Използвайте индикации за типове когато е подходящо
- Включвайте docstrings за функции и класове
- Използвайте модерни функции на Python (3.8+)

#### Java
- Следвайте конвенции на Spring Boot
- Използвайте функции на Java 21
- Спазвайте стандартна Maven проектна структура
- Включвайте Javadoc коментари
