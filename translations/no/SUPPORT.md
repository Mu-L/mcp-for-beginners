# Support

## Hvordan registrere problemer og få hjelp  

Dette prosjektet bruker GitHub Issues til å spore feil og funksjonsforespørsler. Vennligst søk gjennom eksisterende 
issues før du registrerer nye for å unngå duplikater. For nye issues, registrer feilen eller 
funksjonsforespørselen som en ny Issue.

For hjelp og spørsmål om bruk av dette prosjektet, vennligst:
- Sjekk [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) hvor eksperter og andre utviklere samles
- Bruk ressursene til [Model Context Protocol Community](https://modelcontextprotocol.io/community/) for spørsmål relatert til protokollen
