## 🏗️ Teknologistabel

### 🔌 Model Context Protocol (MCP)

MCP er **"USB-C for AI"** - en universell standard som kobler AI-modeller til eksterne verktøy og datakilder.

**✨ Nøkkelfunksjoner:**

- 🔄 **Standardisert integrasjon**: Universelt grensesnitt for AI-verktøytilkoblinger
- 🏛️ **Fleksibel arkitektur**: Lokale og eksterne servere via stdio/SSE transport
- 🧰 **Rikt økosystem**: Verktøy, prompt og ressurser i én protokoll
- 🔒 **Bedriftsklar**: Innebygd sikkerhet og pålitelighet

**🎯 Hvorfor MCP er viktig:**
Akkurat som USB-C eliminerte kabelkaos, fjerner MCP kompleksiteten med AI-integrasjoner. Én protokoll, uendelige muligheter.

### 🤖 Microsoft Foundry Toolkit Extension for VS Code

Microsofts flaggskip AI-utviklingsutvidelse som gjør VS Code til en AI-kraftpakke.

**🚀 Kjernemuligheter:**

- 📦 **Modellkatalog**: Tilgang til modeller fra Azure AI, GitHub, Hugging Face, Ollama
- ⚡ **Lokal inferens**: ONNX-optimalisert CPU/GPU/NPU utføring
- 🏗️ **Agentbygger**: Visuell AI-agentutvikling med MCP-integrasjon
- 🎭 **Multimodal**: Støtte for tekst, visjon og strukturert output

**💡 Utviklingsfordeler:**

- Null-konfig modelltildeling
- Visuell prompt-ingeniørkunst
- Sanntidstestplass
- Sømløs MCP-serverintegrasjon
