# चेनलिट आणि मायक्रोसॉफ्ट लर्न डॉक्ज MCP सह अभ्यास योजना जनरेटर

## आवश्यक पूर्वतयारी

- Python 3.8 किंवा त्याहून मोठा आवृत्ती
- pip (Python पॅकेज व्यवस्थापक)
- मायक्रोसॉफ्ट लर्न डॉक्ज MCP सर्व्हरशी कनेक्ट होण्यासाठी इंटरनेट कनेक्शन

## इन्स्टॉलेशन

1. हे रेपॉजिटरी क्लोन करा किंवा प्रोजेक्ट फायली डाउनलोड करा.
2. आवश्यक अवलंबने इंस्टॉल करा:

   ```bash
   pip install -r requirements.txt
   ```

## वापर

### परिस्थिती 1: साधा क्वेरी Docs MCP कडे
एक कमांड-लाइन क्लायंट जो Docs MCP सर्व्हरशी कनेक्ट होतो, क्वेरी पाठवतो आणि निकाल प्रिंट करतो.

1. स्क्रिप्ट चालवा:
   ```bash
   python scenario1.py
   ```
2. प्रॉम्प्टवर आपला दस्तऐवजीकरण प्रश्न टाका.

### परिस्थिती 2: अभ्यास योजना जनरेटर (Chainlit वेब अ‍ॅप)
एक वेब-आधारित इंटरफेस (Chainlit वापरून) जो वापरकर्त्यांना कोणत्याही तांत्रिक विषयासाठी वैयक्तिकृत, आठवड्याप्रमाणे अभ्यास योजना तयार करण्याची परवानगी देतो.

1. Chainlit अ‍ॅप सुरू करा:
   ```bash
   chainlit run scenario2.py
   ```
2. आपल्या टर्मिनलमध्ये दिलेला स्थानिक URL (उदा. http://localhost:8000) आपल्या ब्राउझरमध्ये उघडा.
3. चॅट विंडोमध्ये आपला अभ्यास विषय आणि किती आठवडे अभ्यास करायचे ते टाका (उदा. "AI-900 प्रमाणपत्र, 8 आठवडे").
4. अ‍ॅप आठवड्याप्रमाणे अभ्यास योजना उत्तर म्हणून देईल, ज्यात संबंधित मायक्रोसॉफ्ट लर्न दस्तऐवजीकरणासाठी दुवे असतील.

**आवश्यक पर्यावरण चल:**

परिस्थिती 2 (Azure OpenAI सह Chainlit वेब अ‍ॅप) वापरण्यासाठी, आपल्याला `python` निर्देशिकेत `.env` फाईलमध्ये खालील पर्यावरण चल सेट करावे लागतील:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

अ‍ॅप चालवण्यापूर्वी आपल्या Azure OpenAI संसाधन तपशीलांसह ही मूल्ये भरा.

> [!TIP]
> आपण [Microsoft Foundry](https://ai.azure.com/) वापरून आपले स्वतःचे मॉडेल सहजपणे तैनात करू शकता.

### परिस्थिती 3: VS Code मध्ये MCP सर्व्हरसह इन-एडिटर डॉक्स

ब्राउझर टॅब बदलण्याऐवजी दस्तऐवज शोधण्यासाठी, आपण Microsoft Learn Docs थेट आपल्या VS Code मध्ये MCP सर्व्हर वापरून आणू शकता. यामुळे:
- आपला कोडिंग वातावरण सोडल्याशिवाय VS Code मध्ये डॉक्स शोधा आणि वाचा.
- दस्तऐवज संदर्भ द्या आणि README किंवा कोर्स फायलींमध्ये दुवे थेट घाला.
- GitHub Copilot आणि MCP एकत्र वापरून एक अखंड, AI-संचालित डॉक्युमेंटेशन कार्यप्रवाह मिळवू शकता.

**उदाहरण वापर प्रकरणे:**
- कोर्स किंवा प्रोजेक्ट डॉक्युमेंटेशन लिहिताना README मध्ये त्वरीत संदर्भ दुवे जोडा.
- Copilot वापरून कोड तयार करा आणि MCP वापरून तत्काळ संबंधित डॉक्युमेंटेशन शोधा आणि उद्धृत करा.
- आपल्या एडिटरमध्ये लक्ष केंद्रित ठेवा आणि कार्यक्षमता वाढवा.

> [!IMPORTANT]
> आपल्या वर्कस्पेसमध्ये वैध [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) कॉन्फिगरेशन असल्याची खात्री करा (स्थिती `.vscode/mcp.json`).

## परिस्थिती 2 साठी Chainlit का?

Chainlit हे संवादी वेब अनुप्रयोग तयार करण्यासाठी एक आधुनिक खुल्या स्त्रोताचे फ्रेमवर्क आहे. हे मायक्रोसॉफ्ट लर्न डॉक्ज MCP सर्व्हर सारख्या बॅकएंड सेवांशी कनेक्ट करणारे चॅट-आधारित वापरकर्ता इंटरफेस सहज तयार करण्यास मदत करते. हा प्रोजेक्ट Chainlit वापरून वैयक्तिकृत अभ्यास योजना रिअल टाइम मध्ये तयार करण्याचा सोपा, संवादात्मक मार्ग प्रदान करतो. Chainlit चा वापर करून आपण जलदपणे चॅट-आधारित साधने तयार आणि तैनात करू शकता जी कार्यक्षमता आणि शिकवण सुधारतात.

## हे काय करते

हे अ‍ॅप वापरकर्त्यांना केवळ एक विषय आणि कालावधी टाकून वैयक्तिकृत अभ्यास योजना तयार करण्याची परवानगी देते. अ‍ॅप आपला इनपुट पार्स करतो, मायक्रोसॉफ्ट लर्न डॉक्ज MCP सर्व्हरला संबंधित सामग्रीसाठी क्वेरी पाठवतो आणि निकाल आठवड्याप्रमाणे रचनेत सजवतो. प्रत्येक आठवड्याचे शिफारसी चॅटमध्ये दाखवल्या जातात, ज्यामुळे प्रगतीचे अनुसरण करणे सोपे होते. एकत्रीकरणामुळे आपल्याला नेहमीच नवीनतम, सर्वात संबंधित शिकणारे स्रोत उपलब्ध होतात.
