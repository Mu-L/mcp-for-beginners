# समर्थन

## समस्या नोंदविण्याचा आणि मदत मिळविण्याचा मार्ग

हा प्रकल्प बग्स आणि वैशिष्ट्य विनंत्या ट्रॅक करण्यासाठी GitHub Issues वापरतो. कृपया नवीन समस्या नोंदविण्यापूर्वी विद्यमान 
समस्यांचा शोध घ्या जेणेकरून पुनरावृत्ती टाळता येतील. नवीन समस्यांसाठी, आपल्या बग किंवा 
वैशिष्ट्य विनंती नवीन इश्यू म्हणून नोंदवा.

या प्रकल्पाचा वापर कसा करायचा याबाबत मदत आणि प्रश्नांसाठी, कृपया:
- [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) तपासा जिथे तज्ञ आणि साथीदार विकसक एकत्र येतात
- प्रोटोकॉलशी संबंधित प्रश्नांसाठी [Model Context Protocol Community](https://modelcontextprotocol.io/community/) संसाधने वापरा

## Microsoft समर्थन धोरण

Model Context Protocol (MCP) for Beginners प्रकल्पासाठी समर्थन वरील सूचीबद्ध संसाधनांपुरते मर्यादित आहे.
