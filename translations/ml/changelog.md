# മാറ്റലുകൾ: MCP for Beginners പാഠ്യപദ്ധതി

മോഡൽ കോൺടൈക്സ്റ്റ് പ്രോട്ടോക്കോൾ (MCP) for Beginners പാഠ്യപദ്ധതിയിൽ വരുത്തിയ എല്ലാ പ്രധാന മാറ്റങ്ങൾ രേഖപ്പെടുത്തുന്നതിന് ഈ ഡോക്യുമെന്റ് സേവിക്കുന്നു. മാറ്റങ്ങൾ മറുവഴി ക്രമത്തിൽ രേഖപ്പെടുത്തിയിട്ടുണ്ട് (പുതിയ മാറ്റങ്ങൾ ആദ്യമായി).

## ജൂൺ 16, 2026

### MCP സ്പെസിഫിക്കേഷൻ സഹകരണം & സാംപ്പ് സാധുത പരിശോധന

നിലവിലെ **MCP Specification 2025-11-25**-നും പുതിയ ഔദ്യോഗിക SDKകൾക്കും യോജിച്ച് പാഠ്യപദ്ധതി സ്ഥിരീകരിച്ചു, ശേഷമുള്ള പഴക്കമുള്ള സ്പെസിഫിക്കേഷൻ റഫറൻസുകൾ തിരുത്തി, കോർ സാമ്പിൾ ബിൽഡ് ചെയ്ത് പ്രവർത്തിക്കുന്നതിനു സ്ഥിരീകരിച്ചു.

#### സ്പെസിഫിക്കേഷൻ പതിപ്പ് തിരുത്തലുകൾ (2025-06-18 / 2025-03-26 → 2025-11-25)

പഴയ സ്പെക് പതിപ്പിനെ നിലവിലെ/അത്യുന്നതായി പ്രഖ്യാപിച്ച ഇംഗ്ലീഷ് ഉള്ളടക്കം പുതുക്കി, canonical `modelcontextprotocol.io` സ്പെക് പാതകൾക്ക് ലിങ്കുകൾ പുതുക്കി:
- **05-AdvancedTopics/mcp-security/README.md**: "നിലവിലെ മാനദണ്ഡം" ബാനർ, പരിചയം, മുഖ്യ സുരക്ഷാ സിദ്ധാന്തങ്ങൾ, നിർബന്ധിത ആവശ്യകതകൾ, Microsoft Entra ID വിഭാഗം, റഫറൻസുകൾ & റിസോഴ്സസ് ലിങ്കുകൾ, ക്ലോസ് ചെയ്യൽ സുരക്ഷാ നോട്ടീസ് (8 റഫറൻസുകൾ) 2025-11-25-ന് അപ്ഡേറ്റ് ചെയ്തു
- **05-AdvancedTopics/mcp-transport/README.md**: അധിക റിസോഴ്സസ് സ്പെക് ലിങ്കും "നിലവിലെ മാനദണ്ഡം" ബാനറും 2025-11-25-ന് അപ്ഡേറ്റ് ചെയ്തു
- **05-AdvancedTopics/mcp-realtimesearch/README.md**: പഴയ `2025-03-26` സുരക്ഷാ-അൻഡ്-ട്രസ്റ്റ് ലിങ്ക് 2025-11-25 സുരക്ഷാ മികച്ച പ്രക്രിയകളുടെ പേജിലേക്ക് മാറ്റി
- **03-GettingStarted/14-sampling/README.md**: ഔദ്യോഗിക സാമ്പിളിംഗ് ഡോക്‌സ് ലിങ്ക് 2025-11-25-ന് അപ്ഡേറ്റ് ചെയ്തു
- **03-GettingStarted/05-stdio-server/README.md**: നിലവിലെ MCP സ്പെസിഫിക്കേഷൻ റഫറൻസ് പ്രലപ്പിച്ചു; അധിക റിസോഴ്സസ് സ്പെക് ലിങ്ക് 2025-11-25-ന് പുതുക്കി (ചരിത്ര SSE-പുറത്താക്കൽ കുറിപ്പുകൾ കൃത്യതയ്ക്കായി അകത്ത് ಇച്ചു)

#### നിലവിലെ SDKകളുമായി സാമ്പിൾ സാധുത പരിശോധന

- **TypeScript (03-GettingStarted/01-first-server/solution/typescript)**: `npm install`-ൽ `@modelcontextprotocol/sdk@1.29.0`‌ സഫലമായി; `tsc --noEmit` ടൈപ്പ് എററുകൾ കൂടാതെ കടന്നു — നിലവിലുള്ള `McpServer`/`StdioServerTransport` APIകൾ സാധുവായിരിക്കുന്നു
- **Python (03-GettingStarted/01-first-server/solution/python)**: സജീവ `.venv`-ൽ `mcp[cli]` (1.27.2) ഉപയോഗിച്ച് പരിശോധന നടത്തി; `py_compile` പാസ്സായി, `FastMCP.list_tools()` ശരിയായ `add` & `subtract` ടൂളുകൾ തിരിച്ചുകൊടുത്തു
- എല്ലാ സാമ്പിൾ `@modelcontextprotocol/sdk` പതിപ്പ് വൈകല്യങ്ങൾ (`>=1.26.0` / `^1.26.0` / `^1.27.0`) ഇപ്പോഴത്തെ `1.29.0`-ന്റെ പരിധിയിൽ ലയനരീതിയായും പ്രയാസമില്ലാതെ പരിഹരിച്ചു

#### ഡിപ്പൻഡൻസി പിനിന്റെ ക്രമീകരണം (പതിപ്പിലെ ഇടവേളകൾ അടയ്ക്കൽ)

പഴങ്കാല SDK പിനുകളെ പുതുക്കി MCP റിലീസ് പിന്തുടരുന്ന ഓരോ സാമ്പിൾക്കും റെപ്പോവിന്റെ പൊതുവായ സമാനനിർദ്ദേശം പാലിച്ച്:
- **03-GettingStarted/05-stdio-server/solution/typescript/package.json**: `@modelcontextprotocol/sdk` `^1.8.0` → `>=1.26.0` പിന് മാറ്റി, പഴങ്കാല `"updated for MCP 2025-06-18"` പാക്കേജ് വിവരണം `"aligned with MCP Specification 2025-11-25"` ആയി പുതുക്കി
- **10-StreamliningAIWorkflows.../lab3/code/weather_mcp/pyproject.toml** & **lab4/code/github_mcp_server/pyproject.toml**: കൃത്യമായ പിനായ `mcp==1.23.0` → `mcp>=1.26.0` മാറ്റി; രണ്ട് `uv.lock` ഫയലുകളും പുതുക്കി, ഉഭയതും `mcp 1.27.2`-നോടൊപ്പം സമന്വയം ഉറപ്പാക്കി

#### പാഠ്യപദ്ധതി ഗ്യാപ് വിശകലനം — പുതിയ സ്പെക് ഫീച്ചർ പരിധി

MCP 2025-11-25-ൽ അവതരിപ്പിച്ച/വ്യാപ്തമായ എല്ലാ പ്രാഥമികാംശവും പാഠ്യപദ്ധതിയിൽ ഉൾപ്പെടുത്തി, ഉള്ളടക്കം ഇടവേളകളില്ലെന്ന് ഉറപ്പിച്ചു:
- **Sampling**: പാഠം 03-GettingStarted/14-sampling കൂടാതെ 05-AdvancedTopics/mcp-sampling
- **Elicitation (URL മോഡ് ഉൾപ്പെടെ)**: 01-CoreConcepts & 05-AdvancedTopics/mcp-protocol-features-ൽ രേഖപ്പെടുത്തിയിരിക്കുന്നു
- **Roots**: 00-Introduction, 01-CoreConcepts, 05-AdvancedTopics/mcp-root-contexts-ൽ രേഖപ്പെടുത്തിയിരിക്കുന്നു
- **ടാസ്കുകൾ (പരീക്ഷണാധിഷ്ഠിത, ദീർഘകാല ഓപ്പറേഷൻസ്)**: 01-CoreConcepts & 05-AdvancedTopics/mcp-protocol-features-ൽ രേഖപ്പെടുത്തിയിരിക്കുന്നു
- **ടൂൾ അനോട്ടേഷനുകൾ** (`readOnlyHint` / `destructiveHint`): 01-CoreConcepts & 05-AdvancedTopics/mcp-protocol-features-ൽ രേഖപ്പെടുത്തിയിരിക്കുന്നു

### സുരക്ഷ ശക്തീകരണം & ഡിപ്പൻഡൻസി ദുര്‍ബലത പരിഹാരം

എല്ലാ ഡിപ്പൻഡൻസി മാനിഫെസ്റ്റ് റദ്ദാക്കലുകളും സാമ്പിൾ സോഴ്‌സ് കോഡും സമഗ്ര സുരക്ഷ പരിശോധന നടത്തി, റിപ്പോർട്ട് ചെയ്ത എല്ലാ npm അലേർട്ടുകളും ഒരൊറ്റ കോഡ് തലത്തിൽ കണ്ടെത്തലും പരിഹരിച്ചു. പരിഹാരത്തിന് ശേഷം, `npm audit` എല്ലായ്‌പ്പോഴും **0 ദുര്‍ബലതകൾ** കാണിച്ചു.

#### npm ഡിപ്പൻഡൻസി ദുര്‍ബലതകൾ (പരവതാനി) — പരിഹരിച്ചു

15 സമർപ്പിച്ച `package-lock.json` ഫയലുകൾ എല്ലാം പരിശോധന ചെയ്തു. ദുര്‍ബലതകൾ MCP ഇൻസ്പെക്ടർ ഡെവ് ടൂൽ, OpenAI ക്ലയന്റ്, MCP SDK മൂലം കടന്നവ മാത്രമായിരുന്നു; ഇപ്പോൾ സാമ്പിളുകൾക്ക് സംവേചനമില്ലാതെ പരിഹരിച്ചു:
- **10-StreamliningAIWorkflows.../lab4/code/github_mcp_server/inspector** & **lab3/code/weather_mcp/inspector**: `@modelcontextprotocol/inspector` (`0.16.6` / `0.14.1` → `0.22.0`) പരിഷ്ക്കരിച്ചു, ബണ്ട് ചെയ്ത `ajv`, `brace-expansion`, `diff`, `path-to-regexp`, `ws` അലേർട്ടുകൾ നീക്കം ചെയ്തു. npm `overrides` ഉപയോഗിച്ച് `shell-quote@1.8.4` പാച്ചുചെയ്ത പതിപ്പ് നിർബന്ധിച്ചു, `concurrently`-യുടെ പ്രധാന അലേർട്ട് നീക്കം ചെയ്തു; ലോക്ക് ഫയലുകൾ പുനഃരജനനം നടത്തി (ഇപ്പോൾ 0 ദുര്‍ബലതകൾ)
- **03-GettingStarted/samples/typescript**: `npm audit fix` പരിധിയിലുള്ള `qs` പാട്ച് പതിപ്പിലേക്ക് അപ്ഡേറ്റ് ചെയ്തു
- **03-GettingStarted/samples/javascript**: `npm audit fix` പരിധിയിലുള്ള `hono` പാട്ച് പതിപ്പിലേക്ക് അപ്ഡേറ്റ് ചെയ്തു

#### കോർ ആശയ നിർവചനം വർദ്ധിപ്പിക്കൽ (01-CoreConcepts/)
- **പുതിയ ക്ലയന്റ് പ്രിമിറ്റീവ് - Roots**: പേര്‍ഫയൽ സിസ്റ്റം അതിരുകൾ, ആക്‌സസ് അനുവാദങ്ങൾ മനസിലാക്കാൻ സർവറുകൾക്ക് സഹായിക്കുന്ന സമഗ്ര ഡോക്യുമെന്റേഷൻ ചേർത്തു
- **ടൂൾ അനോട്ടേഷനുകൾ**: ടൂൾ പെരുമാറ്റ അനോട്ടേഷനുകൾ (`readOnlyHint`, `destructiveHint`) കൊടുത്ത് മികച്ച ടൂൾ എക്സിക്യൂഷൻ നിബന്ധനകൾ ഉൾപ്പെടുത്തി
- **സാമ്പ്ലിംഗിൽ ടൂൾ വിളിക്കല്‍**: മതിപ്പുള്ള ടൂൾ ഉപയോഗത്തിനായി സാമ്പ്ലിംഗ് ഡോക്യുമെന്റേഷനിൽ `tools` & `toolChoice` പാരാമീറ്ററുകൾ ചേർത്തു
- **URL മോഡ് എലിസിറ്റേഷൻ**: സർവർ-ആരംഭിച്ച വെബ് ഇടപെടലിനായി URL അടിസ്ഥാനത്തിലുള്ള എലിസിറ്റേഷൻ ഡോക്യുമെന്റേഷൻ ചേർത്തു
- **തൊഴിലുകൾ (പ്രായോഗിക പരീക്ഷണം)**: ദീർഘകാല എക്സിക്യൂഷൻ റാപ്പറുകളും deferred ഫലം സമാഹരണത്തിനുള്ള experimental tasks ഫീച്ചർ ചേർത്തു
- **ഐക്കണുകളുടെ പിന്തുണ**: ടൂളുകൾ, റിസോഴ്സുകൾ, റിസോഴ്സ് ടെംപ്ലേറ്റുകൾ, പ്രോംപ്റ്റുകളിൽ ഐക്കണുകളും metadata ആയി ചേർക്കാവുന്ന വിധം രേഖപ്പെടുത്തി

#### സ്പെസിഫിക്കേഷൻ അനുസരണം പരിശോധന
- **പ്രോട്ടോക്കോൾ വേർഷൻ**: എല്ലാ ഡോക്യുമെന്റും MCP സ്പെസിഫിക്കേഷൻ 2025-11-25 ഇപ്പോഴുള്ള പതിപ്പിനെ അനുസരിക്കുന്നു എന്ന് സ്ഥിരീകരിച്ചു
- **ആർക്കിടെക്ചർ യോജിപ്പ്**: രണ്ട്-ലെയർ ആർക്കിടെക്ചർ (ഡാറ്റ ലെയർ + ട്രാൻസ്പോർട്ട് ലെയർ) ശരിയായി രേഖപ്പെടുത്തിയിട്ടുണ്ട്

## ഒക്ടോബർ 6, 2025

### ആരംഭികാനുള്ള സംവരണം – മികച്ച സെർവർ ഉപയോഗവും ലളിത ഓതന്റിക്കേഷൻ

#### ഉയർന്ന തല സെർവർ ഉപയോഗം (03-GettingStarted/10-advanced)
- **പുതിയ അദ്ധ്യായം ചേർത്തു**: സാധാരണയും ലോ-ലെവൽ MCP സെർവർ ആർക്കിടെക്ചറുകളും ഉൾക്കൊള്ളുന്ന സമഗ്ര മാർഗ്ഗനിർദേശം അവതരിപ്പിച്ചു.
  - **സാധാരണ വേഴ്സസ് ലോ-ലെവൽ സെർവർ**: ഇരുവിധ സമീപനങ്ങൾക്കുള്ള Python, TypeScript വഴി വിശദമായ താരതമ്യം കൂടാതെ കോഡ് ഉദാഹരണങ്ങൾ.
  - **ഹാൻഡ്‌ലർ അടിസ്ഥാനമാക്കിയുള്ള ഡിസൈൻ**: സ്കെയിലബിളും ഫ്ലെക്സിബിളുമായി സെർവറുകൾ നിർമ്മിക്കാൻ ഹാൻഡ്‌ലർ അടിസ്ഥാനപ്പെട്ട ടൂൾ/റീസോഴ്‌സ്/പ്രോംപ്റ്റ് മാനേജ്മെന്റിന്റെ വ്യാഖ്യാനം.
  - **പ്രായോഗിക പാറ്റേറ്ണുകൾ**: ആധുനിക സവിശേഷതകൾക്കും ആർക്കിടെക്ചറിനും low-level സെർവർ പാറ്റേർണുകൾ ഉപയോഗപ്രദമായ യാഥാർത്ഥ്യ സാന്ദർഭങ്ങൾ.

#### ലളിതമായ ഓഡന്റിക്കേഷൻ (03-GettingStarted/11-simple-auth)
- **പുതിയ അധ്യായം ചേർത്തു**: MCP സെർവറുകളിൽ ലളിതമായ ഓതന്റിക്കേഷൻ നടപ്പിലാക്കാനുള്ള പടിവാതിൽ മാർഗ്ഗനിർദ്ദേശം.
  - **ഓതന്റിക്കേഷൻ ആശയങ്ങൾ**: ഓതന്റിക്കേഷൻ vs. അഥോറൈസേഷൻ, ക്രെഡൻഷ്യൽ കൈകാര്യം വ്യക്തമാക്കൽ.
  - **അടിസ്ഥാന ഓത് നടപ്പാക്കൽ**: Python (Starlette)യും TypeScript (Express)ഉം ഉള്ള മിഡിൽവെയർ അടിസ്ഥാനമാക്കിയുള്ള ഓതന്റിക്കേഷൻ പാറ്റേറ്ണുകൾ, കോഡ് സാമ്പിളുകളോടുകൂടി.
  - **അഡ്വാൻസ്ഡ് സെക്യൂരിറ്റി ദിശയിലേക്ക് പുരോഗതി**: ലളിതമായ ഓതിൽ നിന്ന് ആരംഭിച്ച് OAuth 2.1, RBAC എന്നിവയിലേക്ക് മുന്നേറ്റത്തിന് മാർഗ്ഗനിർദ്ദേശം, അഡ്വാൻസ്ഡ് സെക്യൂരിറ്റി മോഡ്യൂളുകളിലേക്കുള്ള റഫറൻസുകൾ.

ഈ ചേർക്കലുകൾ MCP സെർവറുകളുടെ കൂടുതൽ മെച്ചപ്പെട്ട, സുരക്ഷിതവും ഫ്ലെക്സിബിളുമായ നടപ്പാക്കലുകൾക്ക് പ്രായോഗികമായ മാർഗ്ഗനിർദ്ദേശം നൽകുന്നു, അടിസ്ഥാന ആശയങ്ങളിൽ നിന്ന് ആധുനിക പ്രൊഡക്ഷൻ പാറ്റേർണുകൾക്ക് ഇടയൊരുക്കുന്നു.

  - **എന്റർപ്രൈസ് സെക്യൂരിറ്റി ഇന്റഗ്രേഷൻ**: ആസ്യൂർ એપ്ലിക്കേഷൻ ഇൻസൈറ്റ്സ് മോണിറ്ററിംഗ്, ഭീഷണി കണ്ടെത്തൽ പൈപ്പ്ലൈനുകൾ, സാമ്പിള്‍ ചെയിൻ സുരക്ഷ
  - **ഇംപ്ലിമെന്റേഷൻ ചെക്ലിസ്റ്റ്**: മൈക്രോസോഫ്റ്റ് സെക്യൂരിറ്റി ഈകോസിസ്റ്റത്തിന്റെ നേട്ടങ്ങളോടുകൂടിയ നിർബന്ധമായും ശുപാർശ ചെയ്തതുമായ സുരക്ഷാ നിയന്ത്രണങ്ങൾ വ്യക്തമായി

### ഡോക്യുമെന്റേഷൻ ഗുണമേന്മയും സ്റ്റാൻഡേർഡുകളുമായി പൊരുത്തവും
- **സ്പെസിഫിക്കേഷൻ റഫറൻസുകൾ**: എളുപ്പമാക്കപ്പെട്ട MCP സ്പെസിഫിക്കേഷൻ 2025-06-18-ന്‍റെ പുനഃപരിശോധനയുള്‍പ്പെടെ എല്ലാ റഫറൻസുകളും പുതുക്കി
- **മൈക്രോസോഫ്റ്റ് സെക്യൂരിറ്റി ഈകോസിസ്റ്റം**: എല്ലാ സെക്യൂരിറ്റി ഡോക്യുമെന്റേഷനിലും സംയോജിത മാർഗ്ഗനിർദ്ദേശം മെച്ചപ്പെടുത്തി
- **പ്രായോഗിക અમല്**: എന്റര്‌പ്രൈസ് പാറ്റേണുകളോടെ .NET, ജാവ, പൈതൺ എന്നിവയിൽ വിശദമായ കോഡ് ഉദാഹരണങ്ങൾ ചേർത്തു
- **വനരസ്രോതസ്സിന്റെ ഓർഗനൈസേഷൻ**: ഔദ്യോഗിക ഡോക്യുമെന്റേഷൻ, സുരക്ഷാ സ്റ്റാൻഡേർഡ്സ്, തന്മാത്രാ മാർഗ്ഗനിർദേശങ്ങളുടെ സമഗ്രവർഗ്ഗീകരണം
- **വിജ്വൽ സൂചനകൾ**: നിർബന്ധമായ ആവശ്യങ്ങൾക്കും ശുപാർശചെയ്ത രീതികൾക്കും വ്യക്തമായ അടയാളപ്പെടുത്തലുകൾ


#### കോർ ആശയങ്ങൾ (01-CoreConcepts/) - സമ്പൂർണ്ണ ആധുനികീകരണം
- **പ്രോട്ടോക്കോൾ വേർഷൻ അപ്ഡേറ്റ്**: MCP സ്പെസിഫിക്കേഷൻ 2025-06-18-ന്റെ തീയതിയുമായി യോജിച്ചവർഗ്ഗീകരണം (YYYY-MM-DD ഫോർമാറ്റ്)
- **സൗരഭ്യം പുതുക്കൽ**: ഹോസ്റ്റുകൾ, ക്ലയന്റ്, സെർവറുകൾ എന്നിവയുടെ MCP നൂതന ആർക്കിടെക്ചർ പാറ്റേണുകൾ പ്രതിഫലിപ്പിക്കുന്ന മായുള്ള വിശദീകരണങ്ങൾ മെച്ചപ്പെടുത്തി
  - ഹോസ്റ്റുകൾ ഇപ്പോള്‍ എഐ ആപ്ലിക്കേഷനുകളായി സ്‌പഷ്ടമായും നിർവചിച്ചിട്ടുണ്ട്, ഒന്നിലധികം MCP ക്ലയന്റ് കണക്ഷനുകൾ കോഷിണീകരിക്കുന്നു
  - ക്ലയന്റുകൾ പ്രോട്ടോക്കോൾ കണക്ടറുകളായി വിവരണം, ഒരോ സെർവറുമായുള്ള ഒറ്റത്തവണ ബന്ധം നിലനിർത്തുന്നു
  - സെർവറുകൾ പ്രാദേശികവും ദൂരസ്ഥതയുമുള്ള വിന്യാസ السينാരികളോടൊപ്പം മെച്ചപ്പെടുത്തി
- **പ്രിമിറ്റീവ് പുനർസംഘടന**: സെർവർ, ക്ലയന്റ് പ്രിമിറ്റീവുകളുടെ സമ്പൂർണ്ണ നവീകരണം
  - സെർവർ പ്രിമിറ്റീവ്‌സ്: റിസോഴ്‌സസ് (ഡേറ്റാ സ്രോതസ്സുകൾ), പ്രൊംപ്റ്റ്സ് (ടെംപ്ലേറ്റുകൾ), ടൂൾസ് (പ്രവർത്തനക്ഷമമായ ഫംഗ്ഷനുകൾ), വിശദീകരണങ്ങളും ഉദാഹരണങ്ങളുമായി
  - ക്ലയന്റ് പ്രിമിറ്റീവ്‌സ്: സാമ്പ്ലിംഗ് (LLM പൂർത്തീകരണങ്ങൾ), ആദായം (ഉപയോക്തൃ ഇൻപുട്ട്), ലോക്കിംഗ് (ഡീബഗ്/മോണിറ്ററിംഗ്)
  - നിലവിലുള്ള കണ്ടെത്തൽ (`*/list`), പുനരുദ്ധാരണം (`*/get`), പ്രവർത്തനം (`*/call`) മെത്തഡുകൾ അപ്ഡേറ്റ് ചെയ്തു
- **പ്രോട്ടോക്കോൾ ആർക്കിടെക്ചർ**: രണ്ട് നിലകളുള്ള ആർക്കിടെക്ചർ മോഡൽ അവതരിപ്പിച്ചു
  - ഡേറ്റാ ലെയർ: JSON-RPC 2.0 അടിസ്ഥാനവും ലൈഫ്സൈക്കിള്‍ മാനേജ്മെന്റും പ്രിമിറ്റീവുകളും
  - ട്രാൻസ്പോർട്ട് ലെയർ: STDIO (പ്രാദേശിക) ഒപ്പം SSE ഉൾപ്പെട്ട സ്റ്റ്രീമബിൾ HTTP (ദൂരസ്ഥ) ട്രാൻസ്പോർട് മെക്കാനിസങ്ങൾ
- **സെക്യൂരിറ്റി ഫ്രെയിംവർക്ക്**: വ്യക്തമായ ഉപയോക്തൃ സമ്മതം, ഡേറ്റാ സ്വകാര്യത സംരക്ഷണം, ഉപകരണ പ്രവർത്തനസുരക്ഷ, ട്രാൻസ്പോർട്ട് ലെയർ സുരക്ഷ എന്നീ സുതാര്യ സുരക്ഷാ സിദ്ധാന്തങ്ങൾ
- **സംവാദ മാതൃകകൾ**: പ്രോട്ടോക്കോൾ സന്ദേശങ്ങൾ ആരംഭം, കണ്ടെത്തൽ, പ്രവർത്തനം, അറിയിപ്പ് പ്രവാഹങ്ങൾ കാണിക്കുന്ന തരത്തിൽ പുതുക്കി
- **കോഡ് ഉദാഹരണങ്ങൾ**: MCP SDK പാറ്റേണുകൾ പ്രതിഫലിപ്പിക്കുന്ന ബഹുഭാഷാ (.NET, ജാവ, പൈതൺ, ജാവാസ്ക്രിപ്റ്റ്) ഉദാഹരണങ്ങൾ പുതുക്കി

#### സെക്യുരിറ്റി (02-Security/) - സമഗ്ര സുരക്ഷാ വിപ്ലവം  
- **സ്റ്റാൻഡേർഡ് പൊരുത്തം**: MCP സ്പെസിഫിക്കേഷൻ 2025-06-18 എല്ലാ സുരക്ഷാ ആവശ്യകതകളുമായി പൂർണ്ണ പൊരുത്തം
- **ഓതന്റിക്കേഷൻ പരിണാമം**: സ്വന്തംOauth സെർവറുകളിൽ നിന്ന് മൈക്രോസോഫ്റ്റ് എൻട്രഏ ഐഡി ചേർന്നെക്സ്റ്റേൺ ഐഡൻറിറ്റി പ്രൊവൈഡർ ഡെലിഗേഷനായി രേഖപ്പെടുത്തി
- **AI-പ്രത്യേക ഭീഷണി വിശകലനം**: ആധുനിക AI ആക്രമണ മാർഗ്ഗങ്ങൾ മെച്ചപ്പെടുത്തിയ വിശദീകരണം
  - മുനഞ്ഞുണ്ടാക്കൽ അറ്റാക്കുകൾക്ക് വാസ്തവ ഉദാഹരണങ്ങളോടെ വിശദ വിവരണം
  - ടൂൾ വിഷംകുത്തൽ മെക്കാനിസങ്ങൾ, "റഗ് പുൾ" ആക്രമണ മാതൃകകൾ
  - കോൺടെക്സ്റ്റ് വിൻഡോ വിഷംകുത്തൽ, മോഡൽ കാണ്ഫ്യൂഷൻ ആക്രമണങ്ങൾ
- **മൈക്രോസോഫ്റ്റ് എഐ സെക്യൂരിറ്റി സൊല്യൂഷൻസ്**: മൈക്രോസോഫ്റ്റ് സുരക്ഷാ ഈകോസിസ്റ്റത്തിന്റെ സമഗ്ര അവതരണം
  - AI പ്രൊംപ്റ്റ് ഷിൽഡുകൾ: ഉന്നത ഡിറ്റക്ഷൻ, സ്പോട്ട്‌ലൈറ്റിംഗ്, ഡെലിമിറ്റർ സാങ്കേതിക വിദ്യകൾ
  - ആസ്യൂർ കോൺറ്റന്റ് സേഫ്റ്റി ഇന്റഗ്രേഷൻ പാറ്റേണുകൾ
  - ഗിറ്റ്‌ഹബ് അഡ്വാൻസ്ഡ് സെക്യൂരിറ്റി സപ്ലൈ ചെയിൻ സംരക്ഷണം
- **ഉന്നത ഭീഷണി നിവാരണങ്ങൾ**: വിശദമായ സുരക്ഷാ നിയന്ത്രണങ്ങൾ
  - MCP-പ്രത്യേക സെഷൻ ഹൈജാക്കിംഗ് ആക്രമണങ്ങൾക്കും ക്രിപ്‌ടോഗ്രാഫിക് സെഷൻ ഐഡി ആവശ്യകതകൾക്കും
  - MCP പ്രോക്സി സീനാരികളിലെ ആശയക്കുഴപ്പം പ്രശ്നങ്ങൾ വ്യക്തമായ സമ്മത നിയമങ്ങളോടെ
  - ടോക്കൺ പാസ്‌ത്രൂ ദുർബലതകൾ, നിർബന്ധിത ശരിയായ പരിശോധന നിയന്ത്രണങ്ങൾ
- **സപ്ലൈ ചെയിൻ സുരക്ഷ**: ഫൗണ്ടേഷൻ മോഡലുകൾ, എംബഡ്ഡിംഗ് സർവിസുകൾ, കോൺടെക്സ്റ്റ് പ്രൊവൈഡറുകൾ, മൂന്നാം കക്ഷി APIകൾ ഉൾപ്പെടെ AI സപ്ലൈ ചെയിൻിന്റെ വിപുലമായ പരിതസ്ഥിതി
- **ഫൗണ്ടേഷൻ സുരക്ഷ**: എന്റർപ്രൈസ് സുരക്ഷാ പാറ്റേണുകൾ, സീറോ ട്രസ്റ്റ് ആർക്കിടെക്ചർ, മൈക്രോസോഫ്റ്റ് സെക്യൂരിറ്റി ഈകോസിസ്റ്റം സംയോജനം
- **വനരസ്രോതസംഘടനം**: ട്രിപ്പിൾ വിഭാഗങ്ങളായി (ഔദ്യോഗിക ഡോക്സ്, സ്റ്റാൻഡേർഡ്സ്, ഗവേഷണം, മൈക്രോസോഫ്റ്റ് സൊല്യൂഷൻസ്, ഇംപ്ലിമെന്റേഷൻ ഗൈഡ്‌സ്) സമഗ്രമായി ശ്രേണീകരിച്ചത്

### ഡോക്യുമെന്റേഷൻ ഗുണമേന്മയിലെ പുരോഗതി
- **ഘടനാപരമായ പഠനലക്ഷ്യങ്ങൾ**: നിർദ്ദേശപ്രദമായ, ചെയ്യുന്ന കാര്യങ്ങൾ ഉൾപ്പെടുത്തിയ പഠനലക്ഷ്യങ്ങൾ മെച്ചപ്പെടുത്തി
- **ക്രോസ് റഫറൻസുകൾ**: ബന്ധപ്പെട്ട സുരക്ഷാ, കോർ ആശയ വിഷയങ്ങൾ തമ്മിൽ ലിങ്കുകൾ ചേർത്തു
- **നിലവിലെ വിവരങ്ങൾ**: എല്ലാ തീയതി റഫറൻസുകളും സ്പെസിഫിക്കേഷൻ ലിങ്കുകളും നിലവിലെ സ്റ്റാൻഡേർഡുകൾക്ക് അനുയോജ്യമായി പുതുക്കി
- **ഇംപ്ലിമെന്റേഷൻ മാർഗ്ഗനിർദ്ദേശം**: രണ്ടു വിഭാഗങ്ങളിലും എളുപ്പത്തിൽ അല്ലെങ്കിൽ പൂർണ്ണമായി ഉപയോഗിക്കാൻ കഴിയുന്ന നിർദ്ദേശങ്ങൾ ചേർത്തു

- `<details>` ടാഗുകൾ പകരം കൂടുതൽ എളുപ്പത്തിലുള്ള ടേബിൾ അടിസ്ഥാനമായ ഫോർമാറ്റ് വച്ച്
- പുതിയ "alternative_layouts" ഫോൾഡറിൽ പകർപ്പുള്ള ലേയൗട്ട് ഓപ്ഷനുകൾ സൃഷ്ടിച്ചു
- കാർഡ് അടിസ്ഥാനമാക്കിയുള്ള, ടാബ് സ്റ്റൈൽ, ആക്കോർഡിയൻ സ്റ്റൈൽ നാവിഗേഷൻ ഉദാഹരണങ്ങൾ ചേർത്തു
- റിപ്പോസിറ്ററി ഘടന വിഭാഗം ഏറ്റവും പുതിയ ഫയലുകൾ ഉൾപ്പെടുത്താൻ പുതുക്കി
- "How to Use This Curriculum" വിഭാഗത്തിൽ സുതാര്യമായ ശുപാർശകൾ ചേർത്തു
- MCP സ്പെസിഫിക്കേഷൻ ലിങ്കുകൾ ശരിയായ URLs ലേക്ക് അപ്ഡേറ്റ് ചെയ്തു
- കോൺടെക്സ്റ്റ് എഞ്ചിനീയറിംഗ് സെക്ഷൻ (5.14) പാഠ്യപദ്ധതിയിൽ ചേർത്തു

### പഠന മാർഗ്ഗനിർദ്ദേശ അപ്ഡേറ്റുകൾ
- നിലവിലുള്ള റിപ്പോസിറ്ററി ഘടനയുമായി പൊരുത്തപ്പെടാൻ പഠന ഗൈഡ് പൂർണ്ണമായും നവീകരിച്ചു
- MCP ക്ലയന്റുകളും ടൂളുകളും, ജനപ്രിയ MCP സെർവറുകൾക്കുള്ള പുതിയ വിഭാഗങ്ങൾ ചേർത്തു
- വിജ്വൽ പാഠ്യപദ്ധതി ഭൂപടം മാറ്റങ്ങളെ പ്രതിഫലിപ്പിക്കുന്നതായി പുതുക്കി
- എല്ലാ സ്പെഷ്യലൈസ്ഡ് മേഖലയെയും ഉൾക്കൊള്ളുന്ന അഡ്വാൻസ്ഡ് വിഷയങ്ങളുടെ വിശദീകരണം മെച്ചപ്പെടുത്തി
- വാസ്തവ ഉദാഹരണങ്ങൾക്കായി കേസ്സ്റസ്റ്റഡി വിഭാഗം പുതുക്കി
- ഈ സമഗ്ര മാറ്റ സർവരേഖ ചേർത്തു

### കമ്മ്യൂണിറ്റി സംഭാവനകൾ (06-CommunityContributions/)
- ചിത്ര ഉൽപാദനത്തിനുള്ള MCP സെർവറുകളുടെ വിശദമായ വിവരങ്ങൾ ചേർത്തു
- VSCode-ൽ ക്ലോഡ് ഉപയോഗിക്കുന്നതിന്റെ സമഗ്ര സെക്ഷൻ ചേർത്തു
- ക്ലൈൻ ടെർമിനൽ ക്ലയന്റ് സെറ്റപ്പ്, ഉപയോഗ നിർദ്ദേശങ്ങൾ ചേർത്തു
- MCP ക്ലയന്റ് വിഭാഗം ജനപ്രിയ ക്ലയന്റ് ഓപ്ഷനുകളിൽ പുതുക്കി
- സംഭാവനയുടെ ഉദാഹരണങ്ങളിൽ കൂടുതൽ കൃത്യമായ കോഡ് സാമ്പിളുകൾ ചേർത്തു

### അഡ്വാൻസ്ഡ് വിഷയങ്ങൾ (05-AdvancedTopics/)
- എല്ലാ സ്പെഷ്യലൈസ്ഡ് വിഷയം ഫോൾഡറുകളും സുസ്ഥിര നാമകരണത്തോടെ ക്രമീകരിച്ചു
- കോൺടെക്സ്റ്റ് എഞ്ചിനീയറിംഗ് സാമഗ്രികളും ഉദാഹരണങ്ങളും ചേർത്തു
- ഫൗണ്ട്രി എജന്റ് ഇന്റഗ്രേഷൻ ഡോക്യുമെന്റേഷൻ ചേർത്തു
- എൻട്രഏ ഐഡി സുരക്ഷാ ഇന്റഗ്രേഷൻ ഡോക്യുമെന്റേഷൻ മെച്ചപ്പെടുത്തി
