# പിന്തുണ

## പ്രശ്‌നങ്ങൾ ഫയൽ ചെയ്ത് സഹായം നേടുന്നത് എങ്ങനെ  

ഈ പ്രോജക്ട് ബഗുകളും ഫീച്ചർ അഭ്യർത്ഥനകളും ട്രാക്കുചെയ്യാൻ GitHub Issues ഉപയോഗിക്കുന്നു. പുനരാവൃതികൾ ഒഴിവാക്കാൻ ദയവായി പുതിയ 
പ്രശ്‌നങ്ങൾ ഫയൽ ചെയ്യുന്നതിന് മുമ്പ് നിലവിലുള്ള പ്രശ്‌നങ്ങൾ തിരയുക. പുതിയ പ്രശ്‌നങ്ങൾക്കായി, നിങ്ങളുടെ ബഗ് അല്ലെങ്കിൽ 
ഫീച്ചർ അഭ്യർത്ഥന പുതിയ Issues ആയി ഫയൽ ചെയ്യുക.

ഈ പ്രോജക്ട് ഉപയോഗിക്കാനുളള സഹായത്തിനും ചോദ്യങ്ങൾക്കും ദയവായി:
- വിദഗ്ധരും മറ്റു ഡവലപ്പർമാരുമെ അഭിനിവേശിക്കുന്ന [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) പരിശോധിക്കുക
- പ്രോട്ടോക്കോൾ സംബന്ധിച്ച ചോദ്യങ്ങൾക്ക് [Model Context Protocol Community](https://modelcontextprotocol.io/community/) വിഭവങ്ങൾ ഉപയോഗിക്കുക

## Microsoft പിന്തുണ നയം  

Model Context Protocol (MCP) for Beginners പ്രോജക്റ്റിന് പിന്തുണ മുകളിൽ പട്ടികപ്പെടുത്തിയ വിഭവങ്ങളിലേക്കു മാത്രം പരിമിതമാണ്.
