# Chainlit & Microsoft Learn Docs MCP ഉപയോഗിച്ചുള്ള സ്റ്റഡി പ്ലാൻ ജനറേറ്റർ

## ആവശ്യമായ മുൻകൂട്ടി നിബന്ധനകൾ

- Python 3.8 അല്ലെങ്കിൽ അതിനുമുകളിൽ
- pip (Python പാക്കേജ് മാനേജർ)
- Microsoft Learn Docs MCP സർവറുമായി ബന്ധം കുറിക്കാൻ ഇന്റർനെറ്റ് ആക്‌സസ്

## ഇൻസ്റ്റലേഷൻ

1. ഈ റീപോസിറ്ററി ക്ലോൺ ചെയ്യുക അല്ലെങ്കിൽ പ്രോജക്ട് ഫയലുകൾ ഡൗൺലോഡ് ചെയ്യുക.
2. ആവശ്യമായ ഡിപ്പെൻഡൻസികൾ ഇൻസ്റ്റാൾ ചെയ്യുക:

   ```bash
   pip install -r requirements.txt
   ```

## ഉപയോഗം

### സീനാരിയോ 1: ഡോക്സ് MCP-ലേക്ക് സാധാരണ ക്വറി
Docs MCP സർവറുമായി കണക്റ്റ് ചെയ്ത്, ഒരു ക്വറി അയയ്ക്കുകയും ഫലം പ്രിന്റ് ചെയ്യുകയും ചെയ്യുന്ന കമാൻഡ്-ലൈൻ ക്ലയന്റ്.

1. സ്ക്രിപ്റ്റ് ഓടിക്കുക:
   ```bash
   python scenario1.py
   ```
2. പ്രോംപ്റ്റിൽ നിങ്ങളുടെ ഡോക്യൂമെന്റേഷൻ ചോദ്യം നൽകുക.

### സീനാരിയോ 2: സ്റ്റഡി പ്ലാൻ ജനറേറ്റർ (Chainlit വെബ് ആപ്പ്)
ഉപയോക്താക്കളെ коя അനുയോജ്യമായ സാങ്കേതിക വിഷയത്തിനായി ആഴ്ചയിൽ ഒരു സ്റ്റഡി പ്ലാൻ സൃഷ്ടിക്കാൻ സഹായിക്കുന്ന ഒരു വെബ് അടിസ്ഥാനമാക്കിയ അന്തർഫേസ് (Chainlit ഉപയോഗിച്ച്).

1. Chainlit ആപ്പ് ആരംഭിക്കുക:
   ```bash
   chainlit run scenario2.py
   ```
2. ടെർമിനലിൽ നൽകിയിരിക്കുന്ന ലോക്കൽ URL (ഉദാ: http://localhost:8000) നിങ്ങളുടെ ബ്രൗസറിൽ തുറക്കുക.
3. ചാറ്റ് വിൻഡോയിൽ നിങ്ങളുടെ പഠന വിഷയം, പഠിക്കാൻ ആഗ്രഹിക്കുന്ന ആഴ്ചകൾ (ഉദാ: "AI-900 സർട്ടിഫിക്കേഷൻ, 8 ആഴ്ചകൾ") നൽകുക.
4. ആപ്പ് ആഴ്ചയിലുള്ള പഠന പദ്ധതി, Microsoft Learn ഡോക്യുമെന്റേഷന്റെ ബന്ധപ്പെട്ട ലിങ്കുകളും ഉൾപ്പെടെ, പ്രതികരിക്കും.

**ആവശ്യമായ എൻവയോൺമെന്റ് വേരിയബിളുകൾ:**

സീനാരിയോ 2 (Azure OpenAI ഉപയോഗിച്ച് Chainlit വെബ് ആപ്പ്) ഉപയോഗിക്കാൻ, python ഡയറക്ടറിയിലെ `.env` ഫയലിൽ താഴെ പറയുന്ന എൻവയോൺമെന്റ് വേരിയബിളുകൾ സെറ്റ് ചെയ്യേണ്ടതാണ്:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

ആപ്പ് ഓടിക്കുന്നതിനുമുൻപ് നിങ്ങളുടെ Azure OpenAI റിസോഴ്സ് വിശദാംശങ്ങൾ ഈ മൂല്യങ്ങളിൽ പ്രവേശിപ്പിക്കുക.

> [!TIP]
> നിങ്ങൾക്കു സ്വന്തമായ മോഡലുകൾ എളുപ്പത്തിൽ [Microsoft Foundry](https://ai.azure.com/) ഉപയോഗിച്ച് ഡിപ്പ്ലോയ് ചെയ്യാം.

### സീനാരിയോ 3: VS Code-ൽ MCP സർവറുള്ള ഇൻ-എഡിറ്റർ ഡോക്സ്

ഡോക്യുമെന്റേഷൻ തിരയാൻ ബ്രൗസർ ടാബുകൾ മാറ്റുന്നതിന്റെ പകരം, Microsoft Learn Docs നേരിട്ട് VS Code-ലേക്ക് MCP സർവറിന്റെ സഹായത്തോടെ കൊണ്ടുവന്നാൽ നിങ്ങൾക്ക് സാധിക്കും:
- കോഡിംഗ് वातावरणം വിട്ട് പോകാതെ VS Code-ൽ ഉള്ളിൽ ഡോക്‌സ് തിരയുകയും വായിക്കുകയും ചെയ്യുക.
- README അല്ലെങ്കിൽ കോഴ്സ് ഫയലുകളിൽ നേരിട്ട് റഫറൻസ് ഡോക്യുമെന്റേഷൻ പിന്നീട് ലിങ്കുകൾ ഉൾപ്പെടുത്തുക.
- GitHub Copilot, MCP എന്നിവ സംയോജിപ്പിച്ച് എളുപ്പമുള്ള AI-സഹUPPORTDOC വർക്ക്‌ഫ്ലോ അനുഭവിക്കുക.

**ഉദാഹരണ ഉപയോഗങ്ങൾ:**
- കോഴ്സ് അല്ലെങ്കിൽ പ്രോജക്ട് ഡോക്യുമെന്റേഷൻ എഴുതുമ്പോൾ README-വിൽ റഫറൻസ് ലിങ്കുകൾ शीഗ്രം ചേർക്കുക.
- കോഡ് ജനറേറ്റ് ചെയ്യാൻ Copilot ഉപയോഗിക്കുകയും ബന്ധപ്പെട്ട ഡോക്യുമെന്റുകൾ കണ്ടെത്താൻ MCP-ഉം ഉപയോഗിക്കുക.
- എഡിറ്ററിലേക്ക് ശ്രദ്ധ കേന്ദ്രീകരിച്ച് ഉൽപ്പാദകത്വം വർദ്ധിപ്പിക്കുക.

> [!IMPORTANT]
> നിങ്ങളുടെ വർക്‌സ്‌പേസിൽ നിയമാനുസൃതമായ [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) കോൺഫിഗറേഷൻ ഉണ്ട് എന്ന് ഉറപ്പാക്കുക (സ്ഥാനം `.vscode/mcp.json`).

## സീനാരിയോ 2-ക്കായി Chainlit തിരഞ്ഞെടുക്കാനുള്ള കാരണങ്ങൾ

Chainlit ഒരു ആധുനിക open-source ഫ്രെയിംവർക്ക് ആണ് സംവാദാത്മക വെബ് ആപ്പുകൾ സൃഷ്ടിക്കാൻ. ഇത് Microsoft Learn Docs MCP സർവറിന്റെ പോലുള്ള ബാക്ക്എൻഡ് സർവീസുകളുമായി കണക്റ്റ് ചെയ്യുന്ന ചാറ്റ്-അധിഷ്ഠിത യൂസർ ഇന്റർഫേസുകൾ സൃഷ്ടിക്കുക എളുപ്പമാക്കുന്നു. ഈ പ്രോജക്ട് Chainlit ഉപയോഗിച്ച് വ്യക്തിഗത, വിശകലന സമയത്ത് പിൻഗാമികളെ സൃഷ്ടിക്കാൻ ഒരു ലളിതമായ, ഇന്ററാക്ടീവ് മാർഗം നൽകുന്നു. Chainlit വിപുലീകരിച്ച്, നിങ്ങൾ ഫാസ്റ്റായി ചാറ്റ്-ബേസ്‌ഡ് ടൂളുകൾ നിർമ്മിച്ച് പ്രോഡക്ടിവിറ്റിയും പഠനവും മെച്ചപ്പെടുത്താം.

## ഇപ്രകാരം ഇത് ചെയ്യുന്നത്

ഈ ആപ്പ് ഉപയോക്താക്കൾക്ക് ഒരു വിഷയം മാത്രം നൽകുകയും ആ ദൈർഘ്യം നൽകുകയും ചെയ്താൽ വ്യക്തിഗത പഠന പദ്ധതി സൃഷ്ടിക്കാൻ അനുവദിക്കുന്നു. ആപ്പ് നിങ്ങളുടെ ഇൻപുട്ട് പാഴ്സുചെയ്യുകയും, ബന്ധപ്പെട്ട ഉള്ളടക്കം അന്വേഷിച്ച് Microsoft Learn Docs MCP സർവറിൽ ക്വറി അയയ്ക്കുകയും, ഫലങ്ങൾ ആഴ്ചയ്ക്കു ആഴ്ചയായി ക്രമീകരിക്കുന്നു. ഓരോ ആഴ്ചയുടെ ശുപാർശകളും ചാറ്റിൽ പ്രദർശിപ്പിക്കപ്പെടുന്നു, അവ സ്വൈര്യത്തോടെ പിന്തുടരാനും പുരോഗതി ട്രാക്ക് ചെയ്യാനും എളുപ്പമാണ്. ഇന്റഗ്രേഷൻ നിർത്താതെ ഏറ്റവും പുതിയ, ഏറ്റവും അനുയോജ്യമായ പഠന വിഭവങ്ങൾ ലഭ്യമാക്കുന്നത് ഉറപ്പാക്കും.

## സാമ്പിൾ ക്വറികൾ

ആപ്പ് എങ്ങനെ പ്രതികരിക്കുന്നു എന്ന് നോക്കാൻ താഴെ കാണിച്ചിരിക്കുന്ന ക്വറികൾ ചാറ്റ് വിൻഡോയിൽ പരീക്ഷിക്കുക:

- `AI-900 certification, 8 weeks`
- `Learn Azure Functions, 4 weeks`
- `Azure DevOps, 6 weeks`
- `Data engineering on Azure, 10 weeks`
- `Microsoft security fundamentals, 5 weeks`
- `Power Platform, 7 weeks`
- `Azure AI services, 12 weeks`
- `Cloud architecture, 9 weeks`

വ്യത്യസ്ത പഠന ലക്ഷ്യങ്ങൾക്കും സമയപരിധികൾക്കും ഈ ഉദാഹരണങ്ങൾ ആപ്പ് എങ്ങനെ ഉപയോഗിക്കാമെന്ന് കാണിക്കുന്നു.
