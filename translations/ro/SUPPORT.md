# Suport

## Cum să raportați probleme și să obțineți ajutor  

Acest proiect utilizează GitHub Issues pentru a urmări erorile și cererile de funcționalități. Vă rugăm să căutați în problemele existente înainte de a raporta probleme noi pentru a evita duplicarea. Pentru probleme noi, creați un nou Issue pentru bug-ul sau cererea de funcționalitate.

Pentru ajutor și întrebări despre utilizarea acestui proiect, vă rugăm să:
- Verificați [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) unde experți și dezvoltatori se adună
- Folosiți resursele [Model Context Protocol Community](https://modelcontextprotocol.io/community/) pentru întrebări legate de protocol

## Politica de Suport Microsoft  

Suportul pentru proiectul Model Context Protocol (MCP) pentru Începători este limitat la resursele enumerate mai sus.
