# Suporte

## Como registrar problemas e obter ajuda  

Este projeto usa o GitHub Issues para rastrear bugs e solicitações de recursos. Por favor, pesquise os problemas existentes antes de registrar novos para evitar duplicatas. Para novos problemas, registre seu bug ou solicitação de recurso como um novo Issue.

Para ajuda e dúvidas sobre o uso deste projeto, por favor:
- Verifique o [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) onde especialistas e outros desenvolvedores se reúnem
- Utilize os recursos da [Model Context Protocol Community](https://modelcontextprotocol.io/community/) para perguntas relacionadas ao protocolo

## Política de Suporte da Microsoft  

O suporte para o projeto Model Context Protocol (MCP) para iniciantes é limitado aos recursos listados acima.
