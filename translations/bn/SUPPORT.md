# সমর্থন

## কীভাবে সমস্যা নিবন্ধন করবেন এবং সাহায্য পাবেন  

এই প্রকল্পে বাগ এবং ফিচার অনুরোধ ট্র্যাক করার জন্য GitHub Issues ব্যবহার করা হয়। অনুগ্রহ করে নতুন সমস্যা নিবন্ধনের আগে বিদ্যমান সমস্যাগুলো সার্চ করুন যাতে ডুপ্লিকেট এড়ানো যায়। নতুন সমস্যার জন্য, আপনার বাগ বা ফিচার অনুরোধ একটি নতুন Issue হিসেবে নিবন্ধন করুন।

এই প্রকল্প ব্যবহারের জন্য সাহায্য এবং প্রশ্নের জন্য, দয়া করে:
- [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) পরীক্ষা করুন যেখানে বিশেষজ্ঞ এবং অন্যান্য বিকাশকারীরা একত্রিত হন
- প্রোটোকল-সম্পর্কিত প্রশ্নের জন্য [Model Context Protocol Community](https://modelcontextprotocol.io/community/) রিসোর্স ব্যবহার করুন

## Microsoft সমর্থন নীতি  

Model Context Protocol (MCP) for Beginners প্রকল্পের জন্য সমর্থন শুধুমাত্র উপরের তালিকাভুক্ত রিসোর্সের মধ্যে সীমাবদ্ধ।
