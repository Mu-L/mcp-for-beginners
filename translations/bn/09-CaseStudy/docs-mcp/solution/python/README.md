# চেইনলিট এবং মাইক্রোসফট লার্ন ডকস MCP সহ স্টাডি প্ল্যান জেনারেটর

## প্রয়োজনীয়তা

- পাইথন ৩.৮ অথবা তার উপরে
- pip (পাইথন প্যাকেজ ম্যানেজার)
- মাইক্রোসফট লার্ন ডকস MCP সার্ভারে সংযোগের জন্য ইন্টারনেট অ্যাক্সেস

## ইনস্টলেশন

১. এই রিপোজিটরি ক্লোন করুন অথবা প্রজেক্ট ফাইলগুলি ডাউনলোড করুন।  
২. প্রয়োজনীয় ডিপেন্ডেন্সিগুলি ইনস্টল করুন:

   ```bash
   pip install -r requirements.txt
   ```

## ব্যবহার

### পরিস্থিতি ১: ডকস MCP-তে সোজা প্রশ্ন  
একটি কমান্ড-লাইন ক্লায়েন্ট যা ডকস MCP সার্ভারের সাথে সংযুক্ত হয়, একটি প্রশ্ন পাঠায় এবং ফলাফল প্রিন্ট করে।

১. স্ক্রিপ্ট চালান:  
   ```bash
   python scenario1.py
   ```
২. প্রম্পটে আপনার ডকুমেন্টেশন প্রশ্ন লিখুন।  

### পরিস্থিতি ২: স্টাডি প্ল্যান জেনারেটর (চেইনলিট ওয়েব অ্যাপ)  
একটি ওয়েব-ভিত্তিক ইন্টারফেস (চেইনলিট ব্যবহার করে) যা ব্যবহারকারীদের যেকোন প্রযুক্তিগত বিষয়ে ব্যক্তিগতকৃত, সপ্তাহ ভিত্তিক স্টাডি প্ল্যান তৈরি করতে দেয়।

১. চেইনলিট অ্যাপ শুরু করুন:  
   ```bash
   chainlit run scenario2.py
   ```
২. আপনার টার্মিনালে প্রদত্ত লোকাল URL (যেমন, http://localhost:8000) ব্রাউজারে খুলুন।  
৩. চ্যাট উইন্ডোতে আপনার স্টাডি বিষয় এবং আপনি কত সপ্তাহ পড়তে চান (যেমন, "AI-900 certification, 8 weeks") লিখুন।  
৪. অ্যাপ আপনার জন্য সপ্তাহ ভিত্তিক স্টাডি প্ল্যান এবং প্রাসঙ্গিক মাইক্রোসফট লার্ন ডকুমেন্টেশনের লিঙ্কগুলি দিয়ে উত্তর দেবে।  

**প্রয়োজনীয় পরিবেশ ভেরিয়েবল:**  

পরিস্থিতি ২ (Azure OpenAI সহ চেইনলিট ওয়েব অ্যাপ) ব্যবহার করতে হলে, আপনাকে `python` ডিরেক্টরির মধ্যে `.env` ফাইল এ নিম্নলিখিত পরিবেশ ভেরিয়েবলগুলি সেট করতে হবে:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```
  
অ্যাপ চালানোর আগে আপনার Azure OpenAI সম্পদের বিবরণ দিয়ে এই মানগুলি পূরণ করুন।  

> [!TIP]  
> আপনি সহজেই [Microsoft Foundry](https://ai.azure.com/) ব্যবহার করে আপনার নিজস্ব মডেলগুলি ডিপ্লয় করতে পারেন।  

### পরিস্থিতি ৩: VS কোডে MCP সার্ভারসহ ইন-এডিটর ডকস  

ডকুমেন্টেশন খুঁজতে ব্রাউজার ট্যাব পরিবর্তনের পরিবর্তে, আপনি সরাসরি VS কোডে মাইক্রোসফট লার্ন ডকস MCP সার্ভারের মাধ্যমে আনতে পারেন। এর ফলে আপনি পারবেন:  
- VS কোডের ভিতরে থেকে ডকস অনুসন্ধান এবং পড়া, আপনার কোডিং পরিবেশ ছাড়াই।  
- ডকুমেন্টেশন রেফারেন্স এবং লিঙ্কগুলি সরাসরি README বা কোর্স ফাইলে সন্নিবেশ করা।  
- GitHub Copilot এবং MCP একসাথে ব্যবহার করে একটি নিরবিচ্ছিন্ন, AI-চালিত ডকুমেন্টেশন ওয়ার্কফ্লো।  

**উদাহরণ ব্যবহার ক্ষেত্র:**  
- কোর্স বা প্রজেক্ট ডকুমেন্টেশন লিখার সময় README-তে দ্রুত রেফারেন্স লিঙ্ক যোগ করা।  
- Copilot দিয়ে কোড জেনারেট করা এবং MCP দিয়ে প্রাসঙ্গিক ডকস দ্রুত খুঁজে সাইট করা।  
- আপনার এডিটরে মনোযোগ বজায় রেখে উৎপাদনশীলতা বাড়ানো।  

> [!IMPORTANT]  
> নিশ্চিত করুন আপনার ওয়ার্কস্পেসে একটি বৈধ [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) কনফিগারেশন আছে (অবস্থান হলো `.vscode/mcp.json`)।  

## কেন পরিস্থিতি ২ এর জন্য চেইনলিট?  

চেইনলিট একটি আধুনিক ওপেন-সোর্স ফ্রেমওয়ার্ক যা কথোপকথনভিত্তিক ওয়েব অ্যাপ্লিকেশন তৈরির জন্য ব্যবহৃত হয়। এটি সহজে চ্যাট-ভিত্তিক ইউজার ইন্টারফেস তৈরি করতে সাহায্য করে যা মাইক্রোসফট লার্ন ডকস MCP সার্ভারের মত ব্যাকএন্ড সার্ভিসের সাথে সংযুক্ত হয়। এই প্রজেক্ট চেইনলিট ব্যবহার করে একটি সহজ, ইন্টারেক্টিভ উপায়ে ব্যক্তিগতকৃত স্টাডি প্ল্যান রিয়েল টাইমে জেনারেট করে। চেইনলিট ব্যবহার করে আপনি দ্রুত চ্যাট ভিত্তিক টুল তৈরি ও ডিপ্লয় করতে পারেন যা উৎপাদনশীলতা এবং শেখাকে উন্নত করে।  

## এটি কী করে  

এই অ্যাপ ব্যবহারকারীকে একটি বিষয় এবং সময়কাল লিখে ব্যক্তিগতকৃত স্টাডি প্ল্যান তৈরি করতে দেয়। অ্যাপ আপনার ইনপুট পার্স করে, প্রাসঙ্গিক বিষয়বস্তু খুঁজতে মাইক্রোসফট লার্ন ডকস MCP সার্ভারে প্রশ্ন পাঠায় এবং ফলাফল গুলোকে একটি কাঠামোবদ্ধ, সপ্তাহ ভিত্তিক প্ল্যানে সংগঠিত করে। প্রতিটি সপ্তাহের সুপারিশ চ্যাটে প্রদর্শিত হয়, যাতে সহজেই অনুসরণ ও অগ্রগতি ট্র্যাক করা যায়। সমন্বয়ের ফলে আপনি সর্বদা সর্বশেষ ও সবচেয়ে প্রাসঙ্গিক শেখার সম্পদ পান।
