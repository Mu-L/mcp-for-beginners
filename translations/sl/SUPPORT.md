# Podpora

## Kako oddati napake in pridobiti pomoč  

Ta projekt uporablja GitHub Issues za sledenje napakam in zahtevam po funkcijah. Pred oddajo novih napak prosimo, da preverite obstoječe težave, da se izognete podvojitvam. Za nove težave oddajte svojo napako ali zahtevo po funkciji kot novo težavo.

Za pomoč in vprašanja o uporabi tega projekta prosimo:
- Preverite [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4), kjer se zbirajo strokovnjaki in drugi razvijalci
- Uporabite vire [Model Context Protocol Community](https://modelcontextprotocol.io/community/) za vprašanja v zvezi s protokolom

## Politika podpore Microsoft  

Podpora za projekt Model Context Protocol (MCP) for Beginners je omejena na zgoraj navedene vire.
