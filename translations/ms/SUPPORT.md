# Sokongan

## Cara untuk mengemukakan isu dan mendapatkan bantuan  

Projek ini menggunakan Isu GitHub untuk menjejak pepijat dan permintaan ciri. Sila cari isu yang sedia ada sebelum mengemukakan isu baru untuk mengelakkan penduaan. Untuk isu baru, kemukakan pepijat atau permintaan ciri anda sebagai Isu baru.

Untuk bantuan dan soalan mengenai penggunaan projek ini, sila:
- Semak [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) di mana pakar dan pembangun rakan sekerja berkumpul
- Gunakan sumber [Model Context Protocol Community](https://modelcontextprotocol.io/community/) untuk soalan berkaitan protokol

## Polisi Sokongan Microsoft  

Sokongan untuk projek Model Context Protocol (MCP) untuk Pemula adalah terhad kepada sumber yang disenaraikan di atas.
