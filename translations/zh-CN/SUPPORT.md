# 支持

## 如何提交问题和获取帮助

该项目使用 GitHub Issues 来跟踪错误和功能请求。提交新问题前，请先搜索现有问题以避免重复。对于新的问题，请将您的错误报告或功能请求作为新 Issue 提交。

如需使用该项目的帮助和咨询，请：
- 查看 [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4)，这里有专家和开发者聚集
- 使用 [Model Context Protocol Community](https://modelcontextprotocol.io/community/) 的资源，针对协议相关问题
