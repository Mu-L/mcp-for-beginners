# ಚೇಂಜ್‌ಲಾಗ್: ಪ್ರಾರಂಭಿಕರಿಗಾಗಿ MCP ಪಠ್ಯಕ್ರಮ

ಈ ದಸ್ತಾವೇಜು ಮಾರುಕಟ್ಟೆಯಲ್ಲಿನ ಪ್ರಾಥಮಿಕರಿಗಾಗಿ ಮಾದರಿ ಪ್ರತ್ಯಯ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಪಠ್ಯಕ್ರಮಕ್ಕೆ ಮಾಡಲಾದ ಎಲ್ಲಾ ಪ್ರಮುಖ ಬದಲಾವಣೆಗಳ ದಾಖಲೆ ರೂಪದಲ್ಲಿದೆ. ಬದಲಾವಣೆಗಳನ್ನು ತರ್ಕಾತ್ಮಕ ಕಾಲಕ್ರಮದಲ್ಲಿ ಹಿಂದುಮುಂದಾಗಿ ದಾಖಲಿಸಲಾಗಿದೆ (ಹೊಸ ಬದಲಾವಣೆಗಳು ಮೊದಲು).

## ಜೂನ್ 16, 2026

### MCP ನಿರ್ಧಿಷ್ಟತೆ ಹೊಂದಾಣಿಕೆ ಮತ್ತು ಮಾದರಿ ಮಾನ್ಯತೆ

ಪ್ರಸ್ತುತ **MCP ನಿರ್ಧಿಷ್ಟತೆ 2025-11-25** ಮತ್ತು ಇತ್ತೀಚಿನ ಅಧಿಕೃತ SDKಗಳ ವಿರುದ್ಧ ಪಠ್ಯಕ್ರಮವನ್ನು ಪರಿಶೀಲಿಸಿ, ಉಳಿದ ಹಳೆಯ ನಿರ್ಧಿಷ್ಟತೆ ಉಲ್ಲೇಖಗಳನ್ನು ಸರಿಪಡಿಸಿ ಮತ್ತು ಅಂಕಿತ ಮಾದರಿಗಳು ಇನ್ನುತ್ತೀರುವಂತೆ ನಿರ್ಮಾಣ ಮತ್ತು ಚಾಲನೆ ಮಾಡುತ್ತವೆಯೆಂದು ಖಚಿತಪಡಿಸಿದರು.

#### ನಿರ್ಧಿಷ್ಟತೆ ಸಂಸ್ಕರಣಾ ತಿದ್ದೆಗಳು (2025-06-18 / 2025-03-26 → 2025-11-25)

ಹಿಂದಿನ ನಿರ್ಧಿಷ್ಟತೆ ಪರಿಷ್ಕರಣೆ ಪ್ರಸ್ತುತ/ಇತ್ತೀಚಿನ ಮಾನದಂಡ ಎಂಬ ಬಯಾನ ಮಾಡುತ್ತಿದ್ದ ಇಂಗ್ಲಿಷ್ ವಿಷಯವನ್ನು ನವೀಕರಿಸಿ ಮತ್ತು canonical `modelcontextprotocol.io` ನಿರ್ಧಿಷ್ಟತೆ ಮಾರ್ಗಗಳಿಗೆ ಲಿಂಕ್‌ಗಳನ್ನು ಪುನಃ ನಿರ್ದೇಶಿಸಿ:  
- **05-AdvancedTopics/mcp-security/README.md**: "ಪ್ರಸ್ತುತ ಮಾನದಂಡ" ಬ್ಯಾನರ್, ಪರಿಚಯ, ಕೋರ ಸೆಕ್ಯುರಿಟಿ ಸಿದ್ಧಾಂತಗಳ ಹೆಸರಿನ ಅಂಕಿತ, ಪರಿಗಣನೆ ಬೇಕಾದ ಅಗತ್ಯತೆಗಳ ಹೆಸರಿನ ಅಂಕಿತ, Microsoft Entra ID ವಿಭಾಗ, ಉಲ್ಲೇಖಗಳು ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳ ಲಿಂಕ್‌ಗಳು ಮತ್ತು ಮುಚ್ಚುವ ಸೆಕ್ಯುರಿಟಿ ಸೂಚನೆ (8 ಉಲ್ಲೇಖ) 2025-11-25ಕ್ಕೆ ನವೀಕರಿಸಿದವು  
- **05-AdvancedTopics/mcp-transport/README.md**: ಹೆಚ್ಚುವರಿ ಸಂಪನ್ಮೂಲಗಳ ನಿರ್ಧಿಷ್ಟತೆ ಲಿಂಕ್ ಮತ್ತು "ಪ್ರಸ್ತುತ ಮಾನದಂಡ" ಬ್ಯಾನರ್ 2025-11-25ಕ್ಕೆ ನವೀಕರಿಸಿದವು  
- **05-AdvancedTopics/mcp-realtimesearch/README.md**: ಹಳೆಯ `2025-03-26` ಸೆಕ್ಯುರಿಟಿ ಮತ್ತು ಟ್ರಸ್ಟ್ ಲಿಂಕ್ ಅನ್ನು ಪ್ರಸ್ತುತ 2025-11-25 ಕಡಿಮೆ ಸೆಕ್ಯುರಿಟಿ ಸ್ವತಂತ್ರದ ಪುಟಕ್ಕೆ ಬದಲಾಯಿಸಿದವು  
- **03-GettingStarted/14-sampling/README.md**: ಅಧಿಕೃತ ಸ್ಯಾಂಪ್ಲಿಂಗ್ ಡಾಕ್ಸ್ ಲಿಂಕ್ ಅನ್ನು 2025-11-25ಕ್ಕೆ ನವೀಕರಿಸಿದವು  
- **03-GettingStarted/05-stdio-server/README.md**: ಪ್ರಸ್ತುತ ಕಾಲದ "ಪ್ರಸ್ತುತ MCP ನಿರ್ಧಿಷ್ಟತೆ" ಉಲ್ಲೇಖ ಮತ್ತು ಹೆಚ್ಚುವರಿ ಸಂಪನ್ಮೂಲಗಳ ನಿರ್ಧಿಷ್ಟತೆ ಲಿಂಕ್ 2025-11-25ಕ್ಕೆ ನವೀಕರಿಸಿದವು (ಇತಿಹಾಸಾತ್ಮಕ SSE ನಿಷೇಧ ಸೂಚನೆಗಳು ನಿಖರತೆಗಾಗಿ ಉಳಿದಿವೆ)

#### ಪ್ರಸ್ತುತ SDKಗಳಿಗೆ ವಿರುದ್ಧ ಮಾದರಿ ಮಾನ್ಯತೆ

- **TypeScript (03-GettingStarted/01-first-server/solution/typescript)**: `npm install` ಮೂಲಕ `@modelcontextprotocol/sdk@1.29.0` ಅನುಸ್ಥಾಪಿಸಲಾಯಿತು; `tsc --noEmit` ಯಾವುದೇ ಪ್ರಕಾರದ ದೋಷವಿಲ್ಲದೆ ಉತ್ತೀರ್ಣ; ಸುಸ್ಥಿತಿಯಲ್ಲಿರುವ `McpServer`/`StdioServerTransport` APIs ಕಾಯ್ದುಕೊಂಡಿವೆ  
- **Python (03-GettingStarted/01-first-server/solution/python)**: ಸ್ವತಂತ್ರ `.venv` ನಲ್ಲಿ `mcp[cli]` (1.27.2) ಬಳಸಿ ಪರಿಶೀಲಿಸಲಾಯಿತು; `py_compile` ಉತ್ತೀರ್ಣ, `FastMCP.list_tools()` ಸರಿಯಾಗಿ `add` ಮತ್ತು `subtract` ಸಾಧನಗಳನ್ನು ಹಿಂತಿರುಗಿಸಿದೆ  
- ಎಲ್ಲಾ ಮಾದರಿ `@modelcontextprotocol/sdk` ಆವೃತ್ತಿ ಶ್ರೇಣಿಗಳು (`>=1.26.0` / `^1.26.0` / `^1.27.0`) ಪ್ರಸ್ತುತ `1.29.0` ಗೆ ಯಾವುದೇ API ಬ್ರೇಕಿಂಗ್ ಇಲ್ಲದೆ ಸಮರ್ಪಕವಾಗಿ ಪರಿಹರಿಸಲ್ಪಟ್ಟಿವೆ ಎಂದು ಖಚಿತಪಡಿಸಿದೆ

### ಸೆಕ್ಯುರಿಟಿ ಗಟ್ಟಿತನ ಮತ್ತು ಅವಲಂಬನೆ ಭದರತೆ ಪರಿಹಾರ

ಪ್ರತಿ ಅವಲಂಬನೆ ಮಾನಿಫೆಸ್ಟ್ ಮತ್ತು ಮಾದರಿ ಮೂಲ ಕೋಡ್ ನಲ್ಲಿ ಸಂಪೂರ್ಣ ಸೆಕ್ಯುರಿಟಿ ಪರಿಶೀಲನೆ ನಡೆಸಿ, ನಂತರ ಎಲ್ಲಾ npm ಎಚ್ಚರಿಕೆಗಳನ್ನು ಮತ್ತು ಒಂದೊಂದು ಕೋಡ್ ಅಶುದ್ದಿ ಪರಿಹರಿಸಲಾಯಿತು. ಪರಿಹಾರದ ನಂತರ, `npm audit` ಎಲ್ಲಾ ಪರಿಶೀಲಿತ ಡೈರೆಕ್ಟರಿಗಳಲ್ಲಿ **0 ಭದರತೆಗಳು** ವರದಿಮಾಡುತ್ತದೆ.

#### npm ಅವಲಂಬನಾ ಭದರತೆಗಳು (ಪರಿಣಾಮಕಾರಿ) — ಸರಿಪಡಿಸಲಾಗಿದೆ

15 ಎಚ್ಚರಿಕೆಗೊಳ್ಳಲ್ಪಟ್ಟ `package-lock.json` ಕಡತಗಳನ್ನು ಪರಿಶೀಲಿಸಲಾಯಿತು. ಭದರತೆಗಳು MCP ಇನ್ಸ್‌ಪೆಕ್ಟರ್ ಡೆವ್ ಟೂಲ್, OpenAI ಕ್ಲೈಂಟ್ ಮತ್ತು MCP SDK ಕರೆಯಲ್ಪಡುವ ಪರಿಣಾಮಕಾರಿ ಅವಲಂಬನೆಗಳಿಗೆ ಮಿತಿತಗೊಂಡಿದ್ದವು; ಈಗ ಮಾದರಿಗಳನ್ನು ಮುರಿಯದೆ ಈವರೆಗೆ ಪರಿಹಾರಗೊಂಡಿವೆ:  
- **10-StreamliningAIWorkflows.../lab4/code/github_mcp_server/inspector** ಮತ್ತು **lab3/code/weather_mcp/inspector**: `@modelcontextprotocol/inspector` (`0.16.6` / `0.14.1` → `0.22.0`) ಅನ್ನು ಹೆಚ್ಚಿಸಿದೆ, ಇದು ಪ್ಯಾಕೇಜಿನಿಂದ ಬೊಂಡಲಾದ `ajv`, `brace-expansion`, `diff`, `path-to-regexp` ಮತ್ತು `ws` ಎಚ್ಚರಿಕೆಗಳನ್ನು ಮುಗಿಸುವುದು; npm `overrides` ಕೊಡುಗೆ ಇದ್ದು `shell-quote@1.8.4` ಪ್ಯಾಚ್ ಮಾಡಲಾದ ರೂಪವನ್ನು ಬಲಜೋರಾಗಿ ಹಾಕಿದೆ, ಹೀಗಾಗಿ `concurrently` ಮೂಲಕ ಬಾಕಿ ಉಳಿದ ಗಂಭೀರ ಎಚ್ಚರಿಕೆಯನ್ನು ತೆಗೆಯುತ್ತದೆ; ಎರಡೂ ಲಾಕ್‌ಫೈಲ್‌ಗಳನ್ನು ಪುನಉತ್ಪಾದನೆ ಮಾಡಿದರು (ಪ್ರಸ್ತುತ 0 ಭದರತೆಗಳು)  
- **03-GettingStarted/samples/typescript**: `npm audit fix` ಪರಿಣಾಮಕಾರಿ `qs` (ಮಧ್ಯಮ) ಅನ್ನು ಪ್ಯಾಚ್ ಆಗಿರುವ ಬಿಡುಗಡೆಯಾಗಿ ನವೀಕರಿಸಿದೆ

#### ಕೋರ್ ಪರಿಕಲ್ಪನೆಗಳ ಸುಧಾರಣೆ (01-CoreConcepts/)
- **ಹೊಸ ಕ್ಲಾಯಂಟ್ ಪ್ರಿಮಿಟಿವ್ - ರೂಟ್ಸ್**: ಫೈಲ್‌ಸಿಸ್ಟಮ್ ಗಡಿಗಳನ್ನು ಮತ್ತು ಪ್ರವೇಶ ಅನುಮತಿಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸರ್ವರ್‌ಗಳಿಗೆ ರೂಟ್ಸ್ ಕ್ಲಾಯಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳ ಬಗ್ಗೆ ಸಮಗ್ರ ದಾಖಲಾತಿ ಸೇರಿಸಲಾಗಿದೆ
- **ಸಾಧನ ಅನೋಟೇಶನ್ಗಳು**: ಸಾಧನ ನಿರ್ವಹಣಾ ನಿರ್ಧಾರಗಳಿಗೆ ಉತ್ತಮ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ ಸಾಧನ ವರ್ತನೆಯ ಅನೋಟೇಶನ್ಗಳ (`readOnlyHint`, `destructiveHint`) ಬಗ್ಗೆ ದಾಖಲಾತಿ ಸೇರಿಸಲಾಗಿದೆ
- **ಸැಂಪ್ಲಿಂಗ್‌ನಲ್ಲಿ ಸಾಧನ ಕರೆ**: ಮಾದರಿಯಿಂದ ಚಾಲಿತ ಸಾಧನ ಕರೆ ಮಾಡಲು `tools` ಮತ್ತು `toolChoice` ಪ್ಯಾರಾಮೀಟರ್‌ಗಳೊಂದಿಗೆ ಸೇರ್ಪಡೆಗೊಂಡ ಸೆಂಪ್ಲಿಂಗ್ ದಾಖಲಾತಿ ನವೀಕರಿಸಲಾಗಿದೆ
- **URL ಮೋಡ್ ಪಡೆದಿದ್ದು**: ಸರ್ವರ್ ಪ್ರೇರಿತ ಬಾಹ್ಯ ವೆಬ್ ಸಂವಹನಕ್ಕಾಗಿ URL ಆಧಾರಿತ ಬೇಡಿಕೆಗಳ ವಿವರಗಳ ದಾಖಲಾತಿ ಸೇರಿಸಲಾಗಿದೆ
- **ಕಾರ್ಯಗಳು (ಪ್ರಯೋಗಾತ್ಮಕ)**: ದೀರ್ಘಕಾಲೀನ ಅನುಷ್ಠಾನ ರಾಪ್ಪರ್ಸ್ ಮತ್ತು ಸ್ಥಗಿತ ಫಲಿತಾಂಶ ಅನುಧ್ಯಾನಕ್ಕಾಗಿ ಪ್ರಯೋಗಾತ್ಮಕ ಕಾರ್ಯಗಳ ಹೊಸ ವಿಭಾಗ ಸೇರಿಸಲಾಗಿದೆ
- **ಐಕಾನ್ಗಳ ಬೆಂಬಲ**: ಈಗ ಸಾಧನಗಳು, ಸಂಪನ್ಮೂಲಗಳು, ಸಂಪನ್ಮೂಲ ಟೆಂಪ್ಲೇಟುಗಳು, ಪ್ರಾಂಪ್ಟ್‌ಗಳು ಹೆಚ್ಚುವರಿ ಮೆಟಾಡೇಟಾಗಾಗಿ ಐಕಾನ್ಗಳನ್ನು ಒಳಗೊಂಡಿರಬಹುದು

- **ಹೊಸ ಕೈಯಿಂದ ತರಬೇತಿ ಸಂಪನ್ಮೂಲ**: ಎಲ್ಲಾ ಭದ್ರತಾ ದಾಖಲೆಗಳಲ್ಲಿ [MCP ಭದ್ರತಾ ಶೃಂಗ ಸಮ್ಮೇಳನ ಕಾರ್ಯಾಗಾರ (Sherpa)](https://azure-samples.github.io/sherpa/) ಸಮಗ್ರ ಏಕರೂಪ ಸೇರಿಸಲಾಗಿದೆ
- **ಅಭಿಯಾನ ಮಾರ್ಗವಿವರಣೆ ನೋಟ**: ಬೇಸ್ ಕ್ಯಾಂಪಿನಿಂದ ಶೃಂಗಪರ್ವತದವರೆಗೆ ಸಂಪೂರ್ಣ ಶಿಬಿರದಿಂದ ಶಿಬಿರ ವಲಯದ ಪ್ರಗತಿ ದಾಖಲಾಗಿದ್ದುದು
- **OWASP ಹೊಂದಾಣಿಕೆ**: ಎಲ್ಲಾ ಭದ್ರತಾ ಮಾರ್ಗनिर्दೇಶಗಳು ಈಗ OWASP MCP ಆಜೂರ್ ಭದ್ರತಾ ಮಾರ್ಗದರ್ಶಿ ಅಪಾಯಗಳಿಗೆ ನಕ್ಷೆಗಳನ್ನು ಒಳಗೊಂಡಿವೆ

#### OWASP MCP Top 10 ಏಕರೂಪ
- **ಹೊಸ ವಿಭಾಗ**: ಮುಖ್ಯ ಭದ್ರತಾ README ಗೆ OWASP MCP ಟಾಪ್ 10 ಭದ್ರತಾ ಅಪಾಯಗಳು ಮತ್ತು Azure ಪರಿಹಾರಗಳ ಪಟ್ಟಿಯನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- **ಅಪಾಯ ಆಧಾರಿತ ದಾಖಲೆ**: ಪ್ರತಿ ಭದ್ರತಾ ಕ್ಷೇತ್ರಕ್ಕಾಗಿ OWASP MCP ಅಪಾಯ ಉಲ್ಲೇಖಗಳೊಂದಿಗೆ mcp-security-controls-2025.md ನವೀಕರಿಸಲಾಗಿದೆ
- **ಉಲ್ಲೇಖ ವಾಸ್ತುಶಿಲ್ಪ**: OWASP MCP ಆಜೂರ್ ಭದ್ರತಾ ಮಾರ್ಗದರ್ಶಿ ಉಲ್ಲೇಖ ವಾಸ್ತುಶಿಲ್ಪ ಮತ್ತು ಅನುಷ್ಠಾನ ಮಾದರಿಗಳಿಗೆ ಲಿಂಕ್

#### ನವೀಕರಿಸಿದ ಭದ್ರತಾ ಕಡತಗಳು
- **README.md**: ಶರ್ಪಾ ಕಾರ್ಯಾಗಾರ ಅವಲೋಕನ, ಅಭಿಯಾನ ಮಾರ್ಗಫಲಕ, OWASP MCP ಟಾಪ್ 10 ಅಪಾಯಗಳ ಸಾರಾಂಶ, ಮತ್ತು ಕೈಯಿಂದ ತರಬೇತಿ ವಿಭಾಗ ಸೇರಿಸಲಾಗಿದೆ
- **mcp-security-controls-2025.md**: ಫೆಬ್ರವರಿ 2026 ಗೆ ಹೆಡರ್ ನವೀಕರಿಸಲಾಗಿದೆ, OWASP ಅಪಾಯ ಉಲ್ಲೇಖಗಳು (MCP01-MCP08) ಸೇರಿಸಲಾಗಿದೆ, ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಆವೃತ್ತಿ ಅಸಂವೇದನಾಶೀಲತೆ ಸರಿಪಡಿಸಲಾಗಿದೆ
- **mcp-security-best-practices-2025.md**: ಶರ್ಪಾ ಮತ್ತು OWASP ಸಂಪನ್ಮೂಲಗಳು ವಿಭಾಗ ಸೇರಿಸಲಾಗಿದೆ, ದಿನಾಂಕ ನವೀಕರಣ
- **mcp-best-practices.md**: ಕೈಯಿಂದ ತರಬೇತಿ ವಿಭಾಗಕ್ಕೆ ಶರ್ಪಾ ಮತ್ತು OWASP ಲಿಂಕ್ಸ್ ಸೇರಿಸಲಾಗಿದೆ
- **azure-content-safety-implementation.md**: OWASP MCP06 ಉಲ್ಲೇಖ, ಶರ್ಪಾ ಶಿಬಿರ 3 ಹೊಂದಾಣಿಕೆ, ಮತ್ತು ಹೆಚ್ಚುವರಿ ಸಂಪನ್ಮೂಲಗಳು ವಿಭಾಗ ಸೇರಿಸಲಾಗಿದೆ

### ಪಠ್ಯಕ್ರಮ ವ್ಯಾಜ್ಯ MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25 ಹೊಂದಾಣಿಕೆ

#### ಮೊಡ್ಯೂಲ್ 03 - ಆರಂಭಿಸುವುದು
- **SDK ದಾಖಲೆಗಳು**: ಆಡಳಿತಿಕ SDK ಪಟ್ಟಿಗೆ Go SDK ಸೇರಿಸಲಾಗಿದೆ; ಎಲ್ಲಾ SDK ಉಲ್ಲೇಖಗಳನ್ನು MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25ಗೆ ಹೊಂದಿಸಲಾಗಿದೆ
- **ಸಾರಿಗೆ ಸ್ಪಷ್ಟೀಕರಣ**: STDIO ಮತ್ತು HTTP ಸ್ಟ್ರೀಮಿಂಗ್ ಸಾರಿಗೆ ವಿವರಣೆಗಳನ್ನೊಳಗೊಂಡ ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಉಲ್ಲೇಖಗಳು ನವೀಕರಿಸಲಾಗಿದೆ

#### ಮೊಡ್ಯೂಲ್ 04 - ಪ್ರಾಯೋಗಿಕ ಅನುಷ್ಠಾನ
- **SDK ನವೀಕರಣಗಳು**: Go SDK ಸೇರಿಸಲಾಗಿದೆ; ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಆವೃತ್ತಿಯೊಂದಿಗೆ SDK ಪಟ್ಟಿ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಅಧಿಕಾರದ ಸ್ಪೆಸಿಫಿಕೇಶನ್**: MCP ಅನುಮತಿ ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಲಿಂಕ್ ಅನ್ನು 2025-11-25 ಆವೃತ್ತಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ

#### ಮೊಡ್ಯೂಲ್ 05 - ಪ್ರಗತಿಶೀಲ ವಿಷಯಗಳು
- **ಹೊಸ ವೈಶಿಷ್ಟ್ಯಗಳು**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25 ಹೊಸ ವೈಶಿಷ್ಟ್ಯಗಳ (ಕಾರ್ಯಗಳು, ಸಾಧನ ಅನೋಟೇಶನ್ಗಳು, URL ಮೋಡ್, ರೂಟ್ಸ್) ಬಗ್ಗೆ ಟಿಪ್ಪಣಿ ಸೇರಿಸಲಾಗಿದೆ
- **ಭದ್ರತಾ ಸಂಪನ್ಮೂಲಗಳು**: OWASP MCP ಟಾಪ್ 10 ಮತ್ತು ಶರ್ಪಾ ಕಾರ್ಯಾಗಾರ ಲಿಂಕ್ಸ್ ಹೆಚ್ಚಿಸಲಾಗಿದೆ

#### ಮೊಡ್ಯೂಲ್ 06 - ಸಮುದಾಯ ಕೊಡುಗೆಗಳು
- **SDK ಪಟ್ಟಿ**: Swift ಮತ್ತು Rust SDK ಗಳು ಸೇರಿಸಿದ್ದು, ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಲಿಂಕ್ 2025-11-25 ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಉಲ್ಲೇಖ**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ ನೇರ URL ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ

#### ಮೊಡ್ಯೂಲ್ 07 - ಆರಂಭಿಕ ಸ್ವೀಕೃತಿ ಪಾಠಗಳು
- **ಸಂಪನ್ಮೂಲ ನವೀಕರಣಗಳು**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25 ಲಿಂಕ್ ಮತ್ತು OWASP MCP ಟಾಪ್ 10 ಹೆಚ್ಚಿಸಲಾಗಿದೆ

#### ಮೊಡ್ಯೂಲ್ 08 - ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು
- **ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಆವೃತ್ತಿ**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಉಲ್ಲೇಖ 2025-11-25 ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಭದ್ರತಾ ಸಂಪನ್ಮೂಲಗಳು**: OWASP MCP ಟಾಪ್ 10 ಮತ್ತು ಶರ್ಪಾ ಕಾರ್ಯಾಗಾರ ಲಿಂಕ್ಸ್ ಸೇರಿಸಲಾಗಿದೆ

#### ಮೊಡ್ಯೂಲ್ 10 - AI ಕಾರ್ಯವಾಹಕ ಸುಧಾರಣೆ
- **ಬ್ಯಾಡ್ಜ್ ನವೀಕರಣ**: MCP ಆವೃತ್ತಿ ಬ್ಯಾಡ್ಜ್ SDK ಆವೃತ್ತಿ (1.9.3) ಇಂದ ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಆವೃತ್ತಿ (2025-11-25) ಗೆ ಬದಲಾಗಿದೆ
- **ಸಂಪನ್ಮೂಲ ಲಿಂಕ್ಸ್**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಲಿಂಕ್ ನವೀಕರಿಸಲಾಗಿದೆ; OWASP MCP ಟಾಪ್ 10 ಸೇರಿಸಲಾಗಿದೆ

#### ಮೊಡ್ಯೂಲ್ 11 - MCP ಸರ್ವರ್ ಹ್ಯಾಂಡ್ಸ್-ಆನ್ ಲ್ಯಾಬ್ಸ್
- **ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಉಲ್ಲೇಖ**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25 ಆವೃತ್ತಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಭದ್ರತಾ ಸಂಪನ್ಮೂಲಗಳು**: ಅಧಿಕೃತ ಸಂಪನ್ಮೂಲಗಳಿಗೆ OWASP MCP ಟಾಪ್ 10 ಸೇರಿಸಲಾಗಿದೆ

## ಡಿಸೆಂಬರ್ 18, 2025

### ಭದ್ರತಾ ದಾಖಲೆ ನವೀಕರಣ - MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25

#### MCP ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು (02-Security/mcp-best-practices.md) - ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಆವೃತ್ತಿ ನವೀಕರಣ
- **ಪ್ರೋಟೋಕಾಲ್ ಆವೃತ್ತಿ ನವೀಕರಣ**: ಇತ್ತೀಚಿನ MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25 (ನವೆಂಬರ್ 25, 2025 ಬಿಡುಗಡೆ) ಉಲ್ಲೇಖಕ್ಕೆ ನವೀಕರಿಸಲಾಗಿದೆ
  - ಎಲ್ಲಾ ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಆವೃತ್ತಿ ಉಲ್ಲೇಖಗಳನ್ನು 2025-06-18 ರಿಂದ 2025-11-25 ಗೆ ಬದಲಿಸಲಾಗಿದೆ
  - ದಸ್ತಾವೇಜಿನ ದಿನಾಂಕ ಉಲ್ಲೇಖಗಳನ್ನು ಆಗಸ್ಟ್ 18, 2025 ರಿಂದ ಡಿಸೆಂಬರ್ 18, 2025 ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
  - ಎಲ್ಲಾ ಸ್ಪೆಸಿಫಿಕೇಶನ್ URL ಗಳು ಪ್ರಸ್ತುತ ದಾಖಲೆಗಳ ಕಡೆಗೆ ಕಳಿಸಲಾಗಿವೆ
- **ವಿಷಯ ದೃಢೀಕರಣ**: ಇತ್ತೀಚಿನ ಸ್ಟ್ಯಾಂಡರ್ಡ್ಗಳ ವಿರುದ್ಧ ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳ ಸಮಗ್ರ ಪರಿಶೀಲನೆ
  - **Microsoft ಭದ್ರತಾ ಪರಿಹಾರಗಳು**: ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್‌ಗಳು (ಹಿಂದಿನ "ಜೈಲ್ಬ್ರೇಕ್ ಅಪಾಯ ಪತ್ತೆ"), ಆಜೂರ್ ವಿಷಯ ಭದ್ರತೆ, Microsoft Entra ID, ಮತ್ತು ಆಜೂರ್ ಕೀ ವಾಲ್ಟ್ ನ ಪ್ರಸ್ತುತ ಪದ್ಯಾಪದಗಳು ಮತ್ತು ಲಿಂಕ್‌ಗಳು ದೃಢೀಕರಿಸಲಾಗಿದೆ
  - **OAuth 2.1 ಭದ್ರತೆ**: ಇತ್ತೀಚಿನ OAuth ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳೊಂದಿಗೆ ಹೊಂದಾಣಿಕೆ ದೃಢೀಕರಿಸಲಾಗಿದೆ
  - **OWASP ಮಾನಕಗಳು**: LLM ಗಾಗಿ OWASP ಟಾಪ್ 10 ಉಲ್ಲೇಖಗಳು ಪ್ರಸ್ತುತವಾಗಿಯೇ ಇರುವುದನ್ನು ದೃಢೀಕರಿಸಲಾಗಿದೆ
  - **ಆಜೂರ್ ಸೇವೆಗಳು**: ಎಲ್ಲಾ Microsoft Azure ದಾಖಲೆ ಲಿಂಕ್‌ಗಳು ಮತ್ತು ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು ಪರಿಶೀಲಿಸಲ್ಪಟ್ಟಿವೆ
- **ಮಾನಕ ಹೊಂದಾಣಿಕೆ**: ಎಲ್ಲಾ ಉಲ್ಲೇಖಿತ ಭದ್ರತಾ ಮಾನಕಗಳು ಪ್ರಸ್ತುತ ಮತ್ತು ಮಾನ್ಯವೆಂಬುದು ದೃಢೀಕರಿಸಲಾಗಿದೆ
  - NIST AI ಅಪಾಯ ನಿರ್ವಹಣಾ ಫ್ರೇಮ್ವರ್ಕ್
  - ISO 27001:2022
  - OAuth 2.1 ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು
  - ಆಜೂರ್ ಭದ್ರತೆ ಮತ್ತು ಅನುಕೂಲಾತ್ಮಕ ಫ್ರೇಮ್ವರ್ಕ್ ಗಳು
- **ಅನುಷ್ಠಾನ ಸಂಪನ್ಮೂಲಗಳು**: ಎಲ್ಲ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿ ಲಿಂಕ್ಸ್ ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳು ಪರಿಶೀಲಿಸಲ್ಪಟ್ಟಿವೆ
  - ಆಜೂರ್ API ನಿರ್ವಹಣೆ ಪ್ರಾಮಾಣಿಕತೆ ಮಾದರಿಗಳು
  - Microsoft Entra ID ಏಕರೂಪ ಮಾರ್ಗದರ್ಶಿಗಳು
  - ಆಜೂರ್ ಕೀ ವಾಲ್ಟ್ ರಹಸ್ಯ ನಿರ್ವಹಣೆ
  - DevSecOps ಪೈಪ್ಲೈನ್ಗಳು ಮತ್ತು ಮಾನಿಟರಿಂಗ್ ಪರಿಹಾರಗಳು

### ದಾಖಲೆ ಗುಣಮಟ್ಟದ ಶುದ್ಧೀಕರಣ
- **ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಅನುಕೂಲಾತ್ಮಕತೆ**: ಎಲ್ಲಾ MCP ಭದ್ರತಾ ಅಗತ್ಯತೆಗಳ (MUST/MUST NOT) ಇತ್ತೀಚಿನ ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಅನುಗುಣವಾಗಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಲಾಗಿದೆ
- **ಸಂಪನ್ಮೂಲ ಪ್ರಸ್ತುತತೆ**: ಎಲ್ಲಾ ಹೊರಗಿನ Microsoft ದಾಖಲೆ, ಭದ್ರತಾ ಮಾನಕ, ಮತ್ತು ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳ ಲಿಂಕ್‌ಗಳು ಪರಿಶೀಲಿಸಲ್ಪಟ್ಟಿವೆ
- **ಉತ್ತಮ ಅಭ್ಯಾಸ ವ್ಯಾಪ್ತಿ**: ಪ್ರಾಮಾಣಿಕತೆ, ಅನುಮತಿ, AI ನಿರ್ದಿಷ್ಟ ಅಪಾಯಗಳು, ಪೂರೈಕೆ ಸರಪಳಿ ಭದ್ರತೆ, ಮತ್ತು ಎಂಟರ್ಪ್ರೈಸ್ ಮಾದರಿಗಳ ಸಮಗ್ರ ವ್ಯಾಪ್ತಿ ದೃಢೀಕರಿಸಲಾಗಿದೆ

#### ಪ್ರಗತಿಶೀಲ ಸರ್ವರ್ ಬಳಕೆ (03-GettingStarted/10-advanced)
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರಿಸಲಾಗಿದೆ**: ನಿಯಮಿತ ಮತ್ತು ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್ ವಾಸ್ತುಶಿಲ್ಪಗಳನ್ನು ಒಳಗೊಂಡ ಸಂಪೂರ್ಣ MCP ಸರ್ವರ್ ಬಳಕೆಯ ಮಾರ್ಗದರ್ಶಿ ಪರಿಚಯಿಸಲಾಗಿದೆ
  - **ನಿಯಮಿತ ವಿರುದ್ಧ ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್**: ಎರಡು ವಿಧಾನಗಳ ಪೈಥಾನ್ ಮತ್ತು ಟೈಪ್ಸ್ಕ್ರಿಪ್ಟ್ ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ ವಿಶ್ಲೇಷಣೆ
  - **ಹ್ಯಾಂಡ್ಲರ್ ಆಧಾರಿತ ವಿನ್ಯಾಸ**: ಸಾರಿಗೆ, ಸಂಪನ್ಮೂಲ, ಪ್ರಾಂಪ್ಟ್ ನಿರ್ವಹಣೆಯ ಹ್ಯಾಂಡ್ಲರ್ ಆಧಾರಿತ ಮಾರ್ಗದಲ್ಲಿ ವಿವರ
  - **ಪ್ರಾಯೋಗಿಕ ಮಾದರಿಗಳು**: ಸಂಕೀರ್ಣ ವೈಶಿಷ್ಟ್ಯಗಳು ಮತ್ತು ವಾಸ್ತುಶಿಲ್ಪಕ್ಕಾಗಿ ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್ ಮಾದರಿಗಳ ಉಪಯೋಗದ ನೈಜ ಸಂದರ್ಭಗಳು
#### ಸರಳ ಪ್ರಮಾಣೀಕರಣ (03-GettingStarted/11-simple-auth)
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರಿಸಲಾಗಿದೆ**: MCP ಸರ್ವರ್‌ಗಳಲ್ಲಿ ಸರಳ ಪ್ರಮಾಣೀಕರಣವನ್ನು ಹಂತ ಹಂತವಾಗಿ ಜಾರಿಗೆ ತರಲು ಮಾರ್ಗದರ್ಶನ.
  - **ಪ್ರಮಾಣೀಕರಣ ತತ್ವಗಳು**: ಪ್ರಮಾಣೀಕರಣ ಮುಂತಾದ ಪರವಾನಿಗೆಗಳ ವ್ಯಾಖ್ಯಾನ ಮತ್ತು ಗುರುತು ನಿರ್ವಹಣೆ ಸ್ಪಷ್ಟ ವಿವರಣೆ.
  - **ಮೂಲಭೂತ ಪ್ರಮಾಣೀಕರಣ ಜಾರಿಗೆ ತರಿಕೆ**: ಪೈಥಾನ್ (Starlette) ಮತ್ತು ಟೈಪ್‌스크್ರಿಪ್ಟ್ (Express) ನಲ್ಲಿ ಮಧ್ಯವರ್ತಿ ಆಧಾರಿತ ಪ್ರಮಾಣೀಕರಣ ಮಾದರಿಗಳು, ಉದಾಹರಣೆಗಳೊಂದಿಗೆ.
  - **ಅತ್ಯಾಧುನಿಕ ಭದ್ರತೆಗೆ ಪ್ರಗತಿ**: ಸರಳ ಪ್ರಮಾಣೀಕರಣದಿಂದ ಪ್ರಾರಂಭಿಸಿ OAuth 2.1 ಮತ್ತು RBAC ಗೆ ಮುಂದುವರಿಯಲು ಮಾರ್ಗದರ್ಶನ, ಹಾಗೂ ಅತ್ಯಾಧುನಿಕ ಭದ್ರತಾ ಘಟಕಗಳಿಗೆ ಸೂಚನೆಗಳು.

ಈ ಸೇರ್ಪಡೆಗಳು MCP ಸರ್ವರ್ ಜಾರಿಗೆ ಹೆಚ್ಚು ಬಲಿಷ್ಠ, ಭದ್ರ ಮತ್ತು ರಾಜ್ಯಸಿ ಹೊಂದುವಂತಹ ಅನುಷ್ಠಾನಗಳಿಗೆ ಕಾರ್ಯಕ್ಷಮ, ಕೈಗೆಟುಕುವ ಮಾರ್ಗದರ್ಶನ ಒದಗಿಸುತ್ತವೆ, ಆಧಾರಶಿಲೆಯ ತತ್ವಗಳನ್ನು ಅತ್ಯಾಧುನಿಕ ಉತ್ಪಾದನಾ ಮಾದರಿಗಳೊಂದಿಗೆ ಸೇರುವಂತೆ ಮಾಡುತ್ತವೆ.

    - **ವ್ಯವಹಾರ ಪರಿಣಾಮ**: ಉತ್ತಮಗೊಂಡ ಡೆವಲಪರ್ ಆನ್ಬೋರ್ಡಿಂಗ್ ಮತ್ತು ಉತ್ಪಾದಕತೆ
    - **ತಂತ್ರಜ್ಞಾನ ಮೌಲ್ಯ**: ಮಾಡ್ಯೂಲರ್ ಏಜೆಂಟ್ ನಿಯೋಜನೆ ಮತ್ತು ಕ್ರಾಸ್-ಟೂಲ್ ಉಪಯೋಗ ಸಾಮರ್ಥ್ಯ
    - **ಪರಿಸರ ಅಭಿವೃದ್ಧಿ**: ಏಜೆಂಟಿಕ್ ಏಕೀಕರಣದ ಪುಸ್ತಕೀಯ ವೇದಿಕೆ ಆಗಿ ಸ್ಥಾಪನೆ
  - **ಬೆಳೆಸಿದ ಪ್ರಕರಣ ಅಧ್ಯಯನ ರೂಪರೇಖೆ**: ಏಳು ಪ್ರಕರಣ ಅಧ್ಯಯನಗಳಲ್ಲೂ ಸಮನ್ವಿತ ವಿನ್ಯಾಸ ಮತ್ತು ವ್ಯಾಪಕ ವಿವರಣೆ
    - ಅಜೂರ್ AI ಪ್ರವಾಸ ಏಜೆಂಟ್‌ಗಳು: ಬಹು-ಏಜೆಂಟ್ ವ್ಯವಸ್ಥಾಪನೆ ಒತ್ತು
    - ಅಜೂರ್ ಡೆವ್ಓಪ್ಸ್ ಏಕೀಕರಣ: ಕಾರ್ಯಪ್ರವಾಹ ಸ್ವಯಂಚಾಲನೆ
    - ರಿಯಲ್-ಟೈಮ್ ಡಾಕ್ಯುಮೆಂಟ್ ಪುನಃಪಡೆಯುವಿಕೆ: ಪೈಥಾನ್ ಕಾನ್‌ಸೋಲ್ ಕ್ಲೈಂಟ್ ಜಾರಿಗೆ ತರಿಕೆ
    - ಸಂವಹನಾತ್ಮಕ ಅಧ್ಯಯನ ಯೋಜನೆ ನಿರ್ಮಾಪಕ: ಚೈನ್‌ಲಿಟ್ ಸಂವಾದಾತ್ಮಕ ವೆಬ್ ಅಪ್ಲಿಕೇಶನ್
    - ಸಂಪಾದಕದಲ್ಲಿನ ಡಾಕ್ಯುಮೆಂಟೇಶನ್: VS ಕೋಡ್ ಮತ್ತು GitHub Copilot ಏಕೀಕರಣ
    - ಅಜೂರ್ API ವ್ಯವಸ್ಥಾಪನೆ: ಎಂಟರ್‌ಪ್ರೈಸ್ API ಏಕೀಕರಣ ಮಾದರಿಗಳು
    - GitHub MCP ರಿಜಿಸ್ಟ್ರಿ: ಪರಿಸರ ಅಭಿವೃದ್ಧಿ ಮತ್ತು ಸಮುದಾಯ ವೇದಿಕೆ
  - **ಸಮಗ್ರ ಸಾರಾಂಶ**: ಏಳು ಪ್ರಕರಣ ಅಧ್ಯಯನಗಳ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ವಿವರಿಸುವ ಮರುಬರೆದ ಅಂತಿಮ ಭಾಗ
    - ಎಂಟರ್‌ಪ್ರೈಸ್ ಏಕೀಕರಣ, ಬಹು-ಏಜೆಂಟ್ ವ್ಯವಸ್ಥಾಪನೆ, ಡೆವಲಪರ್ ಉತ್ಪಾದಕತೆ
    - ಪರಿಸರ ಅಭಿವೃದ್ಧಿ, ಶೈಕ್ಷಣಿಕ ಅನ್ವಯಿಕೆಗಳ ವರ್ಗೀಕರಣ
    - ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳು, ಅನುಷ್ಠಾನ ತಂತ್ರಗಳು, ಉತ್ತಮ ಅಭ್ಯಾಸಗಳ ಕುರಿತ ವಿವಿಧ ಪಾಠಗಳು
    - MCP ಸಮೃದ್ಧ, ಪ್ರಮಾಣಿತ ಪ್ರೋಟೋಕಾಲ್ ಆಗಿರುವುದರಲ್ಲಿ ಒತ್ತು

#### ಅಧ್ಯಯನ ಗೈಡ್ ನವೀಕರಣಗಳು (study_guide.md)
- **ದೃಶ್ಯ ಪಠ್ಯಕ್ರಮ ನಕ್ಷೆ**: GitHub MCP ರಿಜಿಸ್ಟ್ರಿಯನ್ನು ಪ್ರಕರಣ ಅಧ್ಯಯನ ವಿಭಾಗದಲ್ಲಿ ಸೇರಿಸಿ ಮನೋಚಿತ್ರ ನವೀಕರಣ
- **ಪ್ರಕರಣ ಅಧ್ಯಯನ ವರ್ಣನೆ**: ಸಾಮಾನ್ಯ ವಿವರಣೆಗಳಿಂದ ಏಳು ವ್ಯಾಪಕ ಪ್ರಕರಣ ಅಧ್ಯಯನಗಳ ವಿಶೇಷ ವಿಶ್ಲೇಷಣೆಗೆ ವಿಸ್ತರಣೆ
- **ರೆಪೋಸಿಟರಿ ರಚನೆ**: 10ನೇ ವಿಭಾಗವನ್ನು ಸಮಗ್ರ ಪ್ರಕರಣ ಅಧ್ಯಯನ ಆವರಣ ಮತ್ತು ಜಾರಿಗೆ ಸಂಬಂಧಿಸಿದ ವಿವರಗಳೊಂದಿಗೆ ನವೀಕರಣ
- **ಚೇಂಜ್‌ಲಾಗ್ ಏಕೀಕರಣ**: ಸೆಪ್ಟೆಂಬರ್ 26, 2025 ಎಂಟ್ರಿ ಸೇರಿಸುವ ಮೂಲಕ GitHub MCP ರಿಜಿಸ್ಟ್ರಿ ಸೌಲಭ್ಯ ಹಾಗೂ ಪ್ರಕರಣ ಅಧ್ಯಯನ ಸುಧಾರಣೆಗಳು ದಾಖಲಾಗಿದೆ
- **ದಿನಾಂಕ ನವೀಕರಣಗಳು**: ಪಾದಲೇಖ ಸಮಯದಲ್ಲಿನ ಕೊನೆಯ ಪರಿಷ್ಕರಣೆ (ಸೆಪ್ಟೆಂಬರ್ 26, 2025) ತೋರಿಸಲಾಗಿದೆ

### ದಾಖಲೆ ಗುಣಮಟ್ಟ ಸುಧಾರಣೆಗಳು
- **ಸಮನ್ವಯ ಸುಧಾರಣೆ**: ಎಲ್ಲಾ ಏಳು ಉದಾಹರಣೆಗಳಲ್ಲಿ ಪ್ರಕರಣ ಅಧ್ಯಯನ ವಿನ್ಯಾಸವನ್ನು ನಿಯಮಿತಗೊಳಿಸಲಾಗಿದೆ
- **ವಿಸ್ತೃತ ವ್ಯಾಪ್ತಿ**: ರೂಪಾಂತರ, ಡೆವಲಪರ್ ಉತ್ಪಾದಕತೆ, ಪರಿಸರ ಅಭಿವೃದ್ಧಿ ಸನ್ನಿವೇಶಗಳನ್ನು ಒಳಗೊಂಡಿತು
- **ತಂತ್ರಾಜ್ಞಾನದ ಸ್ಥಾನಮಾನ**: ಏಜೆಂಟಿಕ್ ವ್ಯವಸ್ಥೆ ನಿಯೋಜನೆಗೆ MCP ಮೇಲೆ ಹೆಚ್ಚು ಗಮನ
- **ಸಂಪನ್ಮೂಲ ಏಕೀಕರಣ**: GitHub MCP ರಿಜಿಸ್ಟ್ರಿ ಲಿಂಕ್ ಸೇರಿಸಲಾಗಿದೆ

### ಅತ್ಯಾಧುನಿಕ ವಿಷಯಗಳ ವಿಸ್ತರಣೆ - ಕಸ್ಟಮ್ ಸಂಚಾರಗಳು ಮತ್ತು ಸಂದರ್ಭ ಇಂಜಿನಿಯರಿಂಗ್

#### MCP ಕಸ್ಟಮ್ ಸಂಚಾರಗಳು (05-AdvancedTopics/mcp-transport/) - ಹೊಸ ಅತ್ಯಾಧುನಿಕ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನ
- **README.md**: ಕಸ್ಟಮ್ MCP ಸಂಚಾರ ಯಂತ್ರಣೆಯ ಸಂಪೂರ್ಣ ಜಾರಿಗೆ ಮಾರ್ಗದರ್ಶನ
  - **ಅಜೂರ್ ಇವೆಂಟ್ ಗ್ರಿಡ್ ಸಂಚಾರ**: ಸಮಗ್ರ ಸರ್ವರ್‌ಹೀನ ಘಟನೆ ಆಧಾರಿತ ಸಂಚಾರ ಜಾರಿಗೆ
    - C#, ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ ಮತ್ತು ಪೈಥಾನ್ ಉದಾಹರಣೆಗಳು ಅಜೂರ್ ಫಂಕ್ಷನ್‌ಗಳಿಗೆ ಏಕೀಕರಣದೊಂದಿಗೆ
    - MCP ಪರಿಹಾರಗಳಿಗೆ ವ್ಯಾಪಕವಾಗಿ ಸರಿಯುವ ಘಟನೆ ಆಧಾರಿತ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳು
    - ವೆಬ್‌ಹುಕ್ ಸ್ವೀಕಾರ ಮತ್ತು ಪುಶ್-ಆಧಾರಿತ ಸಂದೇಶ ನಿರ್ವಹಣೆ
  - **ಅಜೂರ್ ಇವೆಂಟ್ ಹಬ್ ಸಂಚಾರ**: ಅತ್ಯಂತ ವೇಗದ ಸ್ಟ್ರೀಮಿಂಗ್ ಸಂಚಾರ ಜಾರಿಗೆ
    - ಕಡಿಮೆಯಾದ ತಡೆ ಹೊಡಿತ ಪರಿಸ್ಥಿತಿಗಳಿಗೆ ವಾಸ್ತುಶಿಲ್ಪ ಸ್ಟ್ರೀಮಿಂಗ್ ಸಾಮರ್ಥ್ಯಗಳು
    - ವಿಭಾಗಿತ ಕಾರ್ಯತಂತ್ರಗಳು ಮತ್ತು ತಪಾಸಣಿ ನಿರ್ವಹಣೆ
    - ಸಂದೇಶ ಗುಂಪುಮಾಡಿಕೆ ಮತ್ತು ಕಾರ್ಯಕ್ಷಮತೆ ಉತ್ತಮಗೊಳಿಸುವಿಕೆ
  - **ಎಂಟರ್‌ಪ್ರೈಸ್ ಏಕೀಕರಣ ಮಾದರಿಗಳು**: ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ವಾಸ್ತುಶಿಲ್ಪ ಉದಾಹರಣೆಗಳು
    - ಅನೇಕ ಅಜೂರ್ ಫಂಕ್ಷನ್‌ಗಳ ಮೂಲಕ MCP ವಿತರಿತ ಪ್ರಕ್ರಿಯೆ
    - ಹಲವು ಸಂಚಾರ ಪ್ರಕಾರಗಳನ್ನು ಸಂಯೋಜಿಸುವ ಸಂಯುಕ್ತ ವಾಸ್ತುಶಿಲ್ಪ
    - ಸಂದೇಶ ಸಹನಶೀಲತೆ, ನಂಬಿಕೆ ಹಾಗೂ ದೋಷ ನಿರ್ವಹಣೆ ತಂತ್ರಗಳು
  - **ಭದ್ರತೆ ಮತ್ತು ನಿಗಾ**: ಅಜೂರ್ ಕೀ ವಾಲ್ಟ್ ಏಕೀಕರಣ ಹಾಗೂ ಗಮನವಿಡುವಿಕೆ ಮಾದರಿಗಳು
    - ನಿರ್ವಾಹಿತ ಗುರುತಿನ ಪ್ರಮಾಣೀಕರಣ ಮತ್ತು ಕನಿಷ್ಟ ಅನುವು ಅನ್ವಯಿಕೆ
    - ಅಪ್ಲಿಕೇಶನ್ ಇನ್ಸೈಟ್ಸ್ ದೂರದರ್ಶನ ಮತ್ತು ಕಾರ್ಯಕ್ಷಮತೆ ಮೇಲೆ ನಿಗಾ
    - ಸರ್ಕ್ಯೂಟ್ ಬ್ರೇಕರ್‌ಗಳು ಮತ್ತು ದೋಷ ಸಹಿಷ್ಣುತೆ ಮಾದರಿಗಳು
  - **ಪರೀಕ್ಷಣಾ ಚಟುವಟಿಕೆಗಳು**: ಕಸ್ಟಮ್ ಸಂಚಾರಗಳಿಗಾಗಿ ಸಮಗ್ರ ಪರೀಕ್ಷಾ ತಂತ್ರಗಳು
    - ಸೌಲಭ್ಯ ಟೆಸ್ಟಿಂಗ್ ಡಬಲ್‌ಗಳು ಮತ್ತು ಮೋಕಿಂಗ್ ಪ್ರಕ್ರಿಯೆಗಳನ್ನು ಒಳಗೊಂಡ ಯುನಿಟ್ ಟೆಸ್ಟಿಂಗ್
    - ಅಜೂರ್ ಟೆಸ್ಟ್ ಕಂಟೈನರ್‌ಗಳೊಂದಿಗೆ ಏಕೀಕರಣ ಪರೀಕ್ಷೆ
    - ಕಾರ್ಯಕ್ಷಮತೆ ಮತ್ತು ಲೋಡ್ ಪರೀಕ್ಷೆ ಪರಿಗಣನೆಗಳು

#### ಸಂದರ್ಭ ಇಂಜಿನಿಯರಿಂಗ್ (05-AdvancedTopics/mcp-contextengineering/) - ಉದಯೋನ್ಮುಖ AI ವಿಷಯ
- **README.md**: ಉದಯೋನ್ಮುಖ ಕ್ಷೇತ್ರವಿರುವ ಸಂದರ್ಭ ಇಂಜಿನಿಯರಿಂಗ್ ಸಮಗ್ರ ಅನ್ವೇಷಣೆ
  - **ಮೂಲ ಸಿದ್ಧಾಂತಗಳು**: ಸಂಪೂರ್ಣ ಸಂದರ್ಭ ಹಂಚಿಕೆ, ಕ್ರಮಕ್ರಮ ನಿರ್ಣಯ ಜಾಗೃತಿ ಮತ್ತು ಸಂದರ್ಭ ಕಿಟಕಿ ನಿರ್ವಹಣೆ
  - **MCP ಪ್ರೋಟೋಕಾಲ್ ಹೊಂದಾಣಿಕೆ**: MCP ವಿನ್ಯಾಸವು ಸಂದರ್ಭ ಇಂಜಿನಿಯರಿಂಗ್ ಸವಾಲುಗಳನ್ನು ಹೇಗೆ ಪ್ರತ್ಯುತ್ತರಿಸುತ್ತದೆ
    - ಸಂದರ್ಭ ಕಿಟಕಿ ನಿಯಮಗಳು ಮತ್ತು ಪ್ರಗತಿಶೀಲ ಲೋಡಿಂಗ್ ತಂತ್ರಗಳು
    - ಸಂಬಂಧಿತತೆ ನಿರ್ಣಯ ಮತ್ತು ಚಲನೆಯ ಸಂದರ್ಭ ಮರಳಿ ಪಡೆದಿಕೆ
    - ಬಹು-ಮೂಡಲ್ ಸಂದರ್ಭ ನಿರ್ವಹಣೆ ಮತ್ತು ಭದ್ರತಾ ಪರಿಗಣನೆಗಳು
  - **ಜಾರಿಗೆ ತರಕೆ ಉಪಾಧಿಗಳು**: ಏಕ-ಥ್ರೆಡ್ ಮತ್ತು ಬಹು-ಏಜೆಂಟ್ ವಾಸ್ತುಶಿಲ್ಪಗಳು
    - ಸಂದರ್ಭ ತುಂಡುಗಳಾಗಿ ವಿಭಾಗಿಸುವಿಕೆ ಮತ್ತು ಆದ್ಯತೆ ನಿಗದಿ ತಂತ್ರಗಳು
    - ಪ್ರಗತಿಶೀಲ ಸಂದರ್ಭ ಲೋಡಿಂಗ್ ಮತ್ತು ಸಂಗ್ರಹಣ ತಂತ್ರಗಳು
    - ಪದರಬದ್ಧ ಸಂದರ್ಭ ವಿಧಾನಗಳು ಮತ್ತು ಮರಳಿ ಪಡೆಯುವಿಕೆ ಸಧಾರಣೆ
  - **ಮಾಪನ ಚಟುವಟಿಕೆಗಳು**: ಸಂದರ್ಭ ಪರಿಣಾಮಕಾರಿ ಅಳೆಯುವಿಗಾಗಿ ಉದಯೋನ್ಮುಖ ಮೆಟ್ರಿಕ್‌ಗಳು
    - ಪ್ರವೇಶ ಕಾರ್ಯಕ್ಷಮತೆ, ಕಾರ್ಯಗತತೆ, ಗುಣಮಟ್ಟ ಮತ್ತು ಬಳಕೆದಾರ ಅನುಭವ ಪರಿಗಣನೆ
    - ವಿಷಯ ಸುಧಾರಣೆಗೆ ಪ್ರಯೋಗಾತ್ಮಕ ವಿಧಾನಗಳು
    - ವಿಫಲತೆ ವಿಶ್ಲೇಷಣೆ ಮತ್ತು ಸುಧಾರಣಾ ಕ್ರಮಗಳು

  - ಎಲ್ಲಾ ಮಾಡ್ಯೂಲ್‌ಗಳಲ್ಲಿ ಏಕರೂಪ ವಿನ್ಯಾಸ ಮತ್ತು ನವಿಗೇಷನ್ ಲಿಂಕುಗಳು
  - ವೃತ್ತಮಾನ ವಿಷಯ ವ್ಯಾಪ್ತಿಗೆ ಅನುಗುಣ ವಿವರಣೆ ನವೀಕರಣಗಳು

### ಡೈರೆಕ್ಟರಿ ರಚನೆ ಸುಧಾರಣೆಗಳು
- **ಹೆಸರುಕರಣ ನಿಯಮ**: "mcp transport" ಅನ್ನು "mcp-transport" ಎಂದು ಮರುಹೆಸರಿಸಿ ಇತರ ಅತ್ಯಾಧುನಿಕ ವಿಷಯ ಫೋಲ್ಡರ್‌ಗಳ ನಿಯಮಕ್ಕೆ ನೇರವಾಗಿ ಹೊಂದಿಸಲಾಗಿದೆ
- **ವಿಷಯ ಸಂಘಟನೆ**: ಎಲ್ಲ 05-AdvancedTopics ಫೋಲ್ಡರ್‌ಗಳು ಈಗ ನಿಯಮಿತ ಹೆಸರು ಮಾದರಿಯನ್ನು (mcp-[ವಿಷಯ]) ಅನುಸರಿಸುತ್ತವೆ

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಸುಧಾರಣೆಗಳು
- **MCP ನಿರ್ದಿಷ್ಟತೆ ಹೊಂದಾಣಿಕೆ**: ಎಲ್ಲಾ ಹೊಸ ವಿಷಯವು MCP ನಿರ್ದಿಷ್ಟತೆ 2025-06-18 ಅನ್ನು ಸೂಚಿಸುತ್ತದೆ
- **ಬಹು-ಭಾಷಾ ಉದಾಹರಣೆಗಳು**: C#, ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ ಮತ್ತು ಪೈಥಾನ್ ನಲ್ಲಿ ಸಮಗ್ರ ಕೋಡ್ ಉದಾಹರಣೆಗಳು
- **ಎಂಟರ್‌ಪ್ರೈಸ್ ಒತ್ತು**: ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ಮಾದರಿಗಳು ಮತ್ತು ಅಜೂರ್ ಕ್ಲೌಡ್ ಏಕೀಕರಣ
- **ದೃಶ್ಯ ಡಾಕ್ಯುಮೆಂಟೇಶನ್**: ವಾಸ್ತುಶಿಲ್ಪ ಮತ್ತು ಪ್ರವಾಹ ದೃಶ್ಯೀಕರಣಕ್ಕಾಗಿ ಮೆರ್ಮೇಡ್ ಚಿತ್ರರೇಖೆಗಳು

#### MCP ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು (02-Security/) - ಸಂಪೂರ್ಣ ನವೀಕರಣ
- **MCP-SECURITY-BEST-PRACTICES-2025.md**: MCP ನಿರ್ದಿಷ್ಟತೆ 2025-06-18 ಅನುಸಾರ ಸಂಪೂರ್ಣ ಮರುಬರೆಹ
  - **ಕಡ್ಡಾಯ ಅವಶ್ಯಕತೆಗಳು**: ಅಧಿಕೃತ ನಿರ್ದಿಷ್ಟತೆಯಿಂದ ನಿಖರವಾಗಿರುವ ಮಸ್ಟ್/ಮಸ್ಟ್ ನಾಟ್ ಅವಶ್ಯಕತೆಗಳು ದೃಶ್ಯ ಸೂಚಕಗಳೊಂದಿಗೆ ಸೇರಿಸಲಾಗಿದೆ
  - **12 ತತ್ವಭೂತ ಭದ್ರತಾ ಅಭ್ಯಾಸಗಳು**: 15 ಅಂಶಗಳ ಪಟ್ಟಿಯಿಂದ ಸಮಗ್ರ ಭದ್ರತಾ ಕ್ಷೇತ್ರಗಳಾಗಿ ಪುನರ್ವ್ಯವಸ್ಥೆ
    - ಟೋಕನ್ ಭದ್ರತೆ ಮತ್ತು ಗುರುತಿನ ಪ್ರಮಾಣೀಕರಣವನ್ನು ಹೊರಗಿನ ಗುರುತು ಪೂರೈಕೆದಾರದೊಂದಿಗೆ ಏಕೀಕರಣ
    - ಸೆಷನ್ ನಿರ್ವಹಣೆ ಮತ್ತು ಸಂಚಾರ ಭದ್ರತೆ ಕ್ರಿಪ್ಟೋಗ್ರಾಫಿಕ್ ಅಗತ್ಯಗಳೊಂದಿಗೆ
    - AI ವಿಶೇಷ ಭದ್ರತೆ ಮೈಕ್ರೋಸಾಫ್ಟ್ ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್ ಗಳೊಂದಿಗೆ ಏಕೀಕರಣ
    - ಪ್ರವೇಶ ನಿಯಂತ್ರಣ ಮತ್ತು ಅನುಮತಿಗಳು ಕನಿಷ್ಠ ಅನುವು ತತ್ವದೊಂದಿಗೆ
    - ವಿಷಯ ಭದ್ರತೆ ಮತ್ತು ಗಮನವಿಡುವಿಕೆ ಅಜೂರ್ ವಿಷಯ ಭದ್ರತೆ ಏಕೀಕರಣದೊಂದಿಗೆ
    - ಸರಬರಾಜು ಸರಣಿಯ ಭದ್ರತೆ ಸಮಗ್ರ ಘಟಕ ಪರಿಶೀಲನೆ
    - OAuth ಭದ್ರತೆ ಮತ್ತು ಗೊಂದಲದ ಡಿಪ್ಯುಟಿ ತಡೆ PKCE ಜಾರಿಗೆ
    - ಘಟನಾವಳಿ ಪ್ರತಿಕ್ರಿಯೆ ಮತ್ತು ಪುನಃಸ್ಥಾಪನೆ ಸ್ವಯಂಚಾಲಿತ ಸಮರ್ಥತೆಗಳೊಂದಿಗೆ
    - ಅನುಕೂಲಕತೆ ಮತ್ತು ಆಡಳಿತ ನಿಯಮಗಳು ನಿಯಂತ್ರಣಗಳಿಗೆ ಅನುಗುಣ
    - ಸುಧಾರಿತ ಭದ್ರತಾ ನಿಯಂತ್ರಣಗಳು ಶೂನ್ಯ ನಂಬಿಕೆ ವಾಸ್ತುಶಿಲ್ಪ
    - ಮೈಕ್ರೋಸಾಫ್ಟ್ ಭದ್ರತಾ ಪರಿಕಲ್ಪನೆಗಳು ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್, ಅಜೂರ್ ವಿಷಯ ಭದ್ರತೆ, ಎಂಟ್ರಾ ID ಮತ್ತು GitHub ಅತ್ಯಾಧುನಿಕ ಭದ್ರತೆ ಏಕೀಕರಣ
    - ನಿರಂತರ ಭದ್ರತಾ ವಿಕಾಸ ಕದಲಿಸುವ ಅಭ್ಯಾಸಗಳು
  - **ಮೈಕ್ರೋಸಾಫ್ಟ್ ಭದ್ರತಾ ಪರಿಹಾರಗಳು**: ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್, ಅಜೂರ್ ವಿಷಯ ಭದ್ರತೆ, ಎಂಟ್ರಾ ID, GitHub ಅವಿರತ ಭದ್ರತೆಯ ಏಕೀಕರಣ ಮಾರ್ಗದರ್ಶನ ವೃದ್ಧಿಸಿದೆ
  - **ಅನುಷ್ಠಾನ ಸಂಪನ್ಮೂಲಗಳು**: ಅಧಿಕೃತ MCP ಡಾಕ್ಯುಮೆಂಟೇಶನ್, ಮೈಕ್ರೋಸಾಫ್ಟ್ ಭದ್ರತಾ ಪರಿಹಾರಗಳು, ಭದ್ರತಾ ಮಾನದಂಡಗಳು ಮತ್ತು ಜಾರಿಗೆ ಮಾರ್ಗದರ್ಶಿಗಳನ್ನು ವರ್ಗಾವಣೆ ಮಾಡಿಕೊಂಡಿವೆ

#### ಸುಧಾರಿತ ಭದ್ರತಾ ನಿಯಂತ್ರಣಗಳು (02-Security/) - ಎಂಟರ್‌ಪ್ರೈಸ್ ಅನುಷ್ಠಾನ
- **MCP-SECURITY-CONTROLS-2025.md**: ಎಂಟರ್‌ಪ್ರೈಸ್-ಮಟ್ಟದ ಭದ್ರತಾ ಚಟುವಟಿಕೆಗಳೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಪುನರ್‌ರಚನೆ
  - **9 ಸಮಗ್ರ ಭದ್ರತಾ ಕ್ಷೇತ್ರಗಳು**: ಮೂಲ ನಿಯಂತ್ರಣಗಳಿಂದ ವಿವರಣಾತ್ಮಕ ಎಂಟರ್‌ಪ್ರೈಸ್ ಫ್ರೇಮ್ವರ್ಕ್ ಗೆ ವಿಸ್ತರಣೆ
    - ಮೈಕ್ರೋಸಾಫ್ಟ್ ಎಂಟ್ರಾ ID ಏಕೀಕರಣದೊಂದಿಗೆ ಸುಧಾರಿತ ಪ್ರಮಾಣೀಕರಣ ಮತ್ತು ಪರವಾನಿಗೆ
    - ಟೋಕನ್ ಭದ್ರತೆ ಮತ್ತು ಆಂಟಿ-ಪಾಸ್‌ಥ್ರೂ ನಿಯಂತ್ರಣ ವಿವರವಾದ ಮಾನ್ಯತೆ ಫ್ರೇಮ್‌ವರ್ಕ್
    - ಸೆಷನ್ ಭದ್ರತೆ ನಿಯಂತ್ರಣೆಗಳ ಅಂತರಗೊಳ್ಳುವಿಕೆ ತಡೆಗೆ ಅನುಸರಣೆ
    - AI ವಿಶೇಷ ಭದ್ರತೆ ನಿಯಂತ್ರಣಗಳು ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್ ಮತ್ತು ಉಪಕರಣ ವಿಷಪೂರಣ ತಡೆಗೆ
    - ಗೊಂದಲದ ಡಿಪ್ಯುಟಿ ದಾಳಿಗಳನ್ನು OAuth ಪ್ರಾಕ್ಸಿ ಭದ್ರತೆಯಿಂದ ತಡೆ
    - ಉಪಕರಣ ಕಾರ್ಯಾಚರಣೆ ಭದ್ರತೆ ಸಾವಧಾನದೊಂದಿಗೆ ಮತ್ತು ವಿಭಾಜನೆ
    - ಸರಬರಾಜು ಸರಣಿ ಭದ್ರತಾ ನಿಯಂತ್ರಣ ಅದೃಷ್ಟಾರ್ಹತೆ ಪರಿಶೀಲನೆ
    - ಗಮನವಿಡುವಿಕೆ ಮತ್ತು ಪತ್ತೆ ನಿಯಂತ್ರಣಗಳು SIEM ಏಕೀಕರಣದೊಂದಿಗೆ
    - ಘಟನಾವಳಿ ಪ್ರತಿಕ್ರಿಯೆ ಮತ್ತು ಪುನಃಸ್ಥಾಪನೆ ಸ್ವಯಂಚಾಲಿತ ಸಾಮರ್ಥ್ಯಗಳು
  - **ಜಾರಿಗೆಲ್ಲಾ ಉದಾಹರಣೆಗಳು**: YAML ಕಾನ್ಫಿಗರೇಶನ್ ಫಲಾಕೆಗಳು ಮತ್ತು ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ ವಿವರ
  - **ಮೈಕ್ರೋಸಾಫ್ಟ್ ಪರಿಹಾರಗಳ ಏಕೀಕರಣ**: ಅಜೂರ್ ಭದ್ರತಾ ಸೇವೆಗಳು, GitHub ಅತ್ಯಾಧುನಿಕ ಭದ್ರತೆ ಮತ್ತು ಎಂಟರ್‌ಪ್ರೈಸ್ ಗುರುತು ನಿರ್ವಹಣೆಯ ಸಮಗ್ರ ಕವರ್

#### ಅತ್ಯಾಧುನಿಕ ವಿಷಯಗಳ ಭದ್ರತೆ (05-AdvancedTopics/mcp-security/) - ಉತ್ಪಾದನಾ-ತಯಾರ ಅನುಷ್ಠಾನ
- **README.md**: ಎಂಟರ್‌ಪ್ರೈಸ್ ಭದ್ರತಾ ಅನುಷ್ಠಾನಕ್ಕೆ ಸಂಪೂರ್ಣ ಮರುಬರೆಹ
  - **ಪ್ರಸ್ತುತ ನಿರ್ದಿಷ್ಟತೆಗೆ ಹೊಂದಾಣಿಕೆ**: MCP ನಿರ್ದಿಷ್ಟತೆ 2025-06-18 ನೊಂದಿಗೆ ಕಡ್ಡಾಯ ಭದ್ರತಾ ಅವಶ್ಯಕತೆಗಳನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ
  - **ಶಕ್ತಿ ಪೂರ್ಣ ಪ್ರಮಾಣೀಕರಣ**: ಮೈಕ್ರೋಸಾಫ್ಟ್ ಎಂಟ್ರಾ ID ಏಕೀಕರಣ ಮತ್ತು .NET ಮತ್ತು ಜಾವಾ ಸ್ಪ್ರಿಂಗ್ ಭದ್ರತೆ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ
  - **AI ಭದ್ರತೆ ಏಕೀಕರಣ**: ಮೈಕ್ರೋಸಾಫ್ಟ್ ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್ ಮತ್ತು ಅಜೂರ್ ವಿಷಯ ಭದ್ರತೆ ಜಾರಿಗೆ ಪರಿಪೂರ್ಣ Python ಉದಾಹರಣೆಗಳೊಂದಿಗೆ
  - **ಅತ್ಯಾಧುನಿಕ ಬೆದರಿಕೆ ನಿವಾರಣೆ**: ವಿವರವಾದ ಜಾರಿಗೆ
    - PKCE ಮತ್ತು ಬಳಕೆದಾರ ಅನುಮತಿಗೊಂಥ ಸ್ಪಷ್ಟತೆ ನಿಯಂತ್ರಣೆಗಳೊಂದಿಗೆ ಗೊಂದಲದ ಡಿಪ್ಯುಟಿ ದಾಳಿಗಳ ತಡೆ
    - ಪ್ರೇಕ್ಷಕ ಮಾನ್ಯತೆ ಮತ್ತು ಸುರಕ್ಷಿತ ಟೋಕನ್ ನಿರ್ವಹಣೆ ಮೂಲಕ ಟೋಕನ್ ಪಾಸ್‌ಥ್ರೂ ತಡೆ
    - ಕ್ರಿಪ್ಟೋಗ್ರಾಫಿಕ್ ಬಾಂಧವ್ಯ ಮತ್ತು ವರ್ತನೆ ವಿಶ್ಲೇಷಣೆಯ ಮೂಲಕ ಸೆಷನ್ ಹೈಜ್ಯಾಕಿಂಗ್ ತಡೆ
  - **ಎಂಟರ್‌ಪ್ರೈಸ್ ಭದ್ರತೆ ಏಕೀಕರಣ**: ಅಜೂರ್ ಅಪ್ಲಿಕೇಶನ್ ಇನ್ಸೈಟ್ಸ್ ನಿಗಾ, ಬೆದರಿಕೆ ಪತ್ತೆ ಪೈಪ್ಲೈನ್‌ಗಳು ಮತ್ತು ಸರಬರಾಜು ಸರಣಿ ಭದ್ರತೆ
  - **ಜಾರಿಗೆ ತಾಳಿಕೆ ಪಟ್ಟಿಕೆ**: Microsoft's ಭದ್ರತಾ ಪರಿಸರ ಉಪಯೋಗಗಳೊಂದಿಗೆ ಕಡ್ಡಾಯ ಮತ್ತು ಶಿಫಾರಸು ಮಾಡಿರುವ ಭದ್ರತಾ ನಿಯಂತ್ರಣಗಳ ಸ್ಪಷ್ಟ ವಿಭಾಗ

### ದಾಖಲೆ ಗುಣಮಟ್ಟ ಮತ್ತು ಮಾನದಂಡ ಹೊಂದಾಣಿಕೆ
- **ವಿವರಣೆ ಉಲ್ಲೇಖಗಳು**: ಎಲ್ಲಾ ಉಲ್ಲೇಖಗಳನ್ನು ಪ್ರಸ್ತುತ MCP ವಿವರಣೆ 2025-06-18 ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಮೈಕ್ರೋಸಾಫ್ಟ್ ಭದ್ರತಾ ಪರಿಸರ**: ಎಲ್ಲಾ ಭದ್ರತಾ ಡಾಕ್ಯುಮೆಂಟೇಶನ್‌ನಲ್ಲಿ ಸಂಯೋಜನೆ ಮಾರ್ಗದರ್ಶನವನ್ನು ವೃದ್ಧಿಸಲಾಗಿದೆ
- **ಪ್ರಾಯೋಗಿಕ ಅನುಷ್ಠಾನ**: ಎಂಟರ್‌ಪ್ರೈಸ್ ಪ್ಯಾಟರ್ನ್‌ಗಳೊಂದಿಗೆ .NET, ಜಾವಾ, ಮತ್ತು ಪೈಥಾನ್‌ನಲ್ಲಿ ವಿವರವಾದ ಕೋಡ್ ಉದಾಹರಣೆಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- **ಸಾಧನಗಳ ಆಯೋಜನೆ**: ಅಧಿಕೃತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್, ಭದ್ರತಾ ಮಾನದಂಡಗಳು, ಮತ್ತು ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳ ಸಮಗ್ರ ವರ್ಗೀಕರಣ

#### ಕೋರ್ ಕಲ್ಪನೆಗಳು (01-CoreConcepts/) - ಸಂಪೂರ್ಣ ನವೀಕರಣ  
- **ಪ್ರೊಟೋಕಾಲ್ ಆವೃತ್ತಿ ನವೀಕರಣ**: ಪ್ರಸ್ತುತ MCP ವಿವರಣೆ 2025-06-18 ಗೆ ನವೀಕರಿಸಿ, ದಿನಾಂಕಾಧಾರಿತ ಆವೃತ್ತಿ ಸಂಖ್ಯೆಯೊಂದಿಗೆ (YYYY-MM-DD ಫಾರ್ಮ್ಯಾಟ್)
- **ವಿನ್ಯಾಸ ತಿದ್ದುಪಡಿ**: ಹೋಸ್ಟ್‌ಗಳು, ಕ್ಲೈಂಟ್‌ಗಳು ಮತ್ತು ಸರ್ವರ್‌ಗಳ ವಿವರಣೆಗಳನ್ನು ಸಧಾರಣ ಮಾಡಲಾಗಿದೆ ώστε ಪ್ರಸ್ತುತ MCP ವಿನ್ಯಾಸ ಪ್ಯಾಟರ್ನ್‌ಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸಲು
  - ಹೋಸ್ಟ್‌ಗಳು ಈಗ ಬಹು MCP ಕ್ಲೈಂಟ್ ಸಂಪರ್ಕಗಳನ್ನು ಸಂಯೋಜಿಸುವ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳಾಗಿ ಸ್ಪಷ್ಟವಾಗಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ
  - ಕ್ಲೈಂಟ್‌ಗಳು ಒಬ್ಬ ಸರ್ವರ್‌ಗೂ ಒಬ್ಬಕಾಲಿಕ ಸಂಪರ್ಕವನ್ನು ಕಾಯ್ದುಕೊಳ್ಳುವ ಪ್ರೊಟೋಕಾಲ್ ಸಂಪರ್ಕಕರಾಗಿ ವಿವರಣೆಗೊಂಡಿವೆ
  - ಸರ್ವರ್‌ಗಳ ಸ್ಥಳೀಯ ಮತ್ತು ದೂರಸ್ಥ ನಿಯೋಜನಾ ಪರिदೃಶ್ಯಗಳು ವೃದ್ಧಿಸಲಾಗಿದೆ
- **ಪ್ರಿಮಿಟಿವ್ ಪುನರ್‌ರಚನೆ**: ಸರ್ವರ್ ಮತ್ತು ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳ ಸಂಪೂರ್ಣ ಪುನರ್‌ರಚನೆ
  - ಸರ್ವರ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು: ಸಂಪನ್ಮೂಲಗಳು (ಡೇಟಾ ಮೂಲಗಳು), ಪ್ರಾಂಪ್ಟ್‌ಗಳು (ಟೆಂಪ್ಲೇಟುಗಳು), ಉಪಕರಣಗಳು (ಕಾರ್ಯನಿರ್ವಹಣಾ ಫಂಕ್ಷನ್‌ಗಳು) ವಿವರಗಳೊಂದಿಗೆ ಮತ್ತು ಉದಾಹರಣೆಗಳೊಂದಿಗೆ
  - ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು: ಸ್ಯಾಂಪ್ಲಿಂಗ್ (LLM ಪೂರ್ಣಗೊಳಿಸುವಿಕೆ), ವಿನಂತಿ (ಬಳಕೆದಾರ ಇನ್ಪುಟ್), ಲಾಗಿಂಗ್ (ಡೆಬಗ್/ಮೌಲ್ಯಮಾಪನ)
  - ಪ್ರಸ್ತುತ ಅನ್ವೇಷಣೆ (`*/list`), ಪಡೆಯುವಿಕೆ (`*/get`), ಮತ್ತು ಕಾರ್ಯಾಚರಣೆ (`*/call`) ವಿಧಾನ ಪ್ಯಾಟರ್ನ್‌ಗಳೊಂದಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಪ್ರೊಟೋಕಾಲ್ ವಿನ್ಯಾಸ**: ಇಬ್ಬರ ಹಂತದ ವಿನ್ಯಾಸ ಮಾದರಿಯನ್ನು ಪರಿಚಯಿಸಲಾಗಿದೆ
  - ಡೇಟಾ ಲೇಯರ್: JSON-RPC 2.0 ಮೂಲಭೂತ, ಜೀವನಚಕ್ರ ನಿರ್ವಹಣೆ ಮತ್ತು ಪ್ರಿಮಿಟಿವ್‌ಗಳು
  - ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಲೇಯರ್: STDIO (ಸ್ಥಳೀಯ) ಮತ್ತು SSE ಹೊಂದಿರುವ ಸ್ಟ್ರೀಮಬಲ್ HTTP (ದೂರಸ್ಥ) ಸಂಚಾರ ರೀತಿಗಳು
- **ಭದ್ರತಾ ಫ್ರೇಮ್ವರ್ಕ್**: ಸ್ಪಷ್ಟ ಬಳಕೆದಾರ ಅನುಮತಿ, ಡೇಟಾ ಗೌಪ್ಯತೆ ಸಂರಕ್ಷಣೆ, ಉಪಕರಣ ಕಾರ್ಯನಿರ್ವಹಣಾ ಸುರಕ್ಷತೆ ಮತ್ತು ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಲೇಯರ್ ಸುರಕ್ಷತೆ ಸೇರಿದಂತೆ ಸಮಗ್ರ ಭದ್ರತಾ ತತ್ವಗಳು
- **ಸಂವಹನ ಮಾದರಿಗಳು**: ಪ್ರಾರಂಭ, ಅನ್ವೇಷಣೆ, ಕಾರ್ಯಾಚರಣೆ, ಮತ್ತು ಸೂಚನೆ ಹರಿವು ಪ್ರದರ್ಶಿಸಲು ಪ್ರೊಟೋಕಾಲ್ ಸಂದೇಶಗಳನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ
- **ಕೋಡ್ ಉದಾಹರಣೆಗಳು**: ಪ್ರಸ್ತುತ MCP SDK ಪ್ಯಾಟರ್ನ್‌ಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸುವ ಬಹುಭಾಷಾ ಉದಾಹರಣೆಗಳನ್ನು ( .NET, ಜಾವಾ, ಪೈಥಾನ್, ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್ ) ನವೀಕರಿಸಲಾಗಿದೆ

#### ಭದ್ರತೆ (02-Security/) - ಸಮಗ್ರ ಭದ್ರತಾ ಪುನರ್‌ರಚನೆ  
- **ಮಾನದಂಡಗಳ ಸಂಯೋಜನೆ**: MCP ವಿವರಣೆ 2025-06-18 ಭದ್ರತಾ ಅಗತ್ಯಗಳೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಅಳಗಟ್ಟಿದವು
- **ಪ್ರಾಮಾಣೀಕರಣ ಅಭಿವೃದ್ಧಿ**: ಕಸ್ಟಮ್ OAuth ಸರ್ವರ್‌ಗಳಿಂದ ಬಾಹ್ಯ ಗುರುತಿನ ದಾತಾ ನಿಯೋಜನೆಯ (Microsoft Entra ID) ಅಭಿವೃದ್ಧಿ ದಾಖಲಾಗಿದೆ
- **AI-ನಿರ್ದಿಷ್ಟ ಹಿಂಸೆ ವಿಶ್ಲೇಷಣೆ**: ಆಧುನಿಕ AI ಹಲ್ಲೆ ವಹಿವಾಟಿನ ಮೇಲೆ ವಿಸ್ತಾರವಾದ ಹನಿಗೊಳಿಸುವಿಕೆ
  - ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್ ಹಲ್ಲೆಗಳ ವಿವರವಾದ ಪರಿದೃಶ್ಯಗಳು ನಿಜ ಜಾಗತಿಕ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ
  - ಉಪಕರಣ ವಿಷಶಿಲತೆ ವಿಧಾನಗಳು ಮತ್ತು "ರುಗ್ ಪುಲ್" ಹಲ್ಲೆ ಮಾದರಿಗಳು
  - ಸಂದರ್ಭ ವಿಂಡೋ ವಿಷಶಿಲತೆ ಮತ್ತು ಮಾದರಿ ಗೊಂದಲದ ಹಲ್ಲೆಗಳು
- **ಮೈಕ್ರೋಸಾಫ್ಟ್ AI ಭದ್ರತಾ ಪರಿಹಾರಗಳು**: ಮೈಕ್ರೋಸಾಫ್ಟ್ ಭದ್ರತಾ ಪರಿಸರದ ಸಮಗ್ರ ವೀಕ್ಷಣೆ
  - AI ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್‌ಗಳು ಹಾಗೂ ಸುಧಾರಿತ ಪಟ್ಟಿ, ಸ್ಪಾಟ್‌ಲೈಟ್ ಮತ್ತು ಡಿಲಿಮಿಟರ್ ತಂತ್ರಗಳು
  - ಅಜೂರ್ ಕಂಟೆಂಟ್ ಸೇಫ್ಟಿ ಸಂಯೋಜನಾ ಮಾದರಿಗಳು
  - ಗಿಥಬ್ ಆದ್ಯತೆಯುತ ಭದ್ರತೆ ಸರಬರಾಜು ಸರಪಳಿಯ ರಕ್ಷಣೆಗೆ
- **ಮುನ್ನಡೆಯ ಹಿಂಸೆ ನಿಬಾರಣೆ**:  
  - ಸೆಷನ್ ಹೈಜ್ಯಾಕಿಂಗ್ ಸाठी MCP-ನಿರ್ದಿಷ್ಟ ಹಲ್ಲೆ ಮಾರುಕಟ್ಟೆಗಳೊಂದಿಗೆ ಮತ್ತು ಸಂಕೇತಿತ ಸೆಷನ್ ID ಅವಶ್ಯಕತೆಗಳೊಂದಿಗೆ ಭದ್ರತಾ ನಿಯಂತ್ರಣಗಳು  
  - MCP ಪ್ರಾಕ್ಸಿ ಪರಿಸರದಲ್ಲಿ ಗೊಂದಲಗೊಂಡ ಕರ್ತವ್ಯ ಸಮಸ್ಯೆಗಳಿಗೆ ಸ್ಪಷ್ಟ ಅನುಮತಿ ನಿಯಮಗಳು  
  - ಟೋಕೆನ್ ಪಾಸ್ಸ್ತ್ರೂ ಘಾತಕಗಳಿಗೆ ಕಡ್ಡಾಯ ಪರಿಶೀಲನಾ ನಿಯಂತ್ರಣಗಳು  
- **ಸರಬರಾಜು ಸರಪಳಿ ಭದ್ರತೆ**: ಮೂಲ ಮಾದರಿಗಳು, ಎम्बೆಡ್ಡಿಂಗ್ ಸೇವೆಗಳು, ಸಂದರ್ಭ ಪೂರೈಕೆದಾರರು ಮತ್ತು ಮೂರನೇ ಹಕ್ಕು API-ಗಳನ್ನು ಒಳಗೊಂಡಂತೆ AI ಸರಬರಾಜು ಸರಪಳಿ ವಿಭಾಗ ವಿಸ್ತಾರಿಸಲಾಗಿದೆ  
- **ಮೂಲಭೂತ ಭದ್ರತೆ**: ಎಂಟರ್‌ಪ್ರೈಸ್ ಭದ್ರತಾ ಪ್ಯಾಟರ್ನ್‌ಗಳು ಸಹಿತ ಶೂನ್ಯ ನಂಬಿಕೆ ವಿನ್ಯಾಸ ಮತ್ತು ಮೈಕ್ರೋಸಾಫ್ಟ್ ಭದ್ರತಾ ಪರಿಸರದ ಸಂಯೋಜನೆ ವೃದ್ಧಿಸಲಾಗಿದೆ  
- **ಸಾಧನಗಳ ಆಯೋಜನೆ**: ವಿಧದ ಪ್ರಕಾರ ಸಮಗ್ರ ಸಂಪನ್ಮೂಲ ಲಿಂಕುಗಳನ್ನು ವರ್ಗೀಕರಿಸಲಾಗಿದೆ (ಅಧಿಕೃತ ಡಾಕ್ಸ್, ಮಾನದಂಡ, ಸಂಶೋಧನೆ, ಮೈಕ್ರೋಸಾಫ್ಟ್ ಪರಿಹಾರಗಳು, ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳು)

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಸುಧಾರಣೆಗಳು  
- **ರಚನೆಗೊಳಿಸಿದ ಕಲಿಕಾ ಉದ್ದೇಶಗಳು**: ವಿಶೇಷ, ಕಾರ್ಯತತ್ವಕ ಫಲಿತಾಂಶಗಳೊಂದಿಗೆ ಕಲಿಕಾ ಉದ್ದೇಶಗಳನ್ನು ವೃದ್ಧಿಸಲಾಗಿದೆ  
- **ಕ್ರಾಸ್-ಉಲ್ಲೇಖಗಳು**: ಸಂಬಂಧಿಸಿದ ಭದ್ರತಾ ಮತ್ತು ಕೋರ್ ಕಲ್ಪನೆ ವಿಷಯಗಳ ನಡುವೆ ಸಂಪರ್ಕ ಲಿಂಕುಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ  
- **ಪ್ರಸ್ತುತ ಮಾಹಿತಿ**: ಎಲ್ಲಾ ದಿನಾಂಕ ಉಲ್ಲೇಖಗಳು ಮತ್ತು ವಿವರಣೆ ಲಿಂಕುಗಳು ಪ್ರಸ್ತುತ ಮಾನದಂಡಗಳಿಗೆ ನವೀಕರಿಸಲ್ಪಟ್ಟವು  
- **ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನ**: ಇಬ್ಬೂ ವಿಭಾಗಗಳಲ್ಲಿ ನಿರ್ದಿಷ್ಟ, ಕಾರ್ಯತತ್ವಕ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ

### ಅಧ್ಯಯನ ಮಾರ್ಗದರ್ಶಿ ನವೀಕರಣಗಳು  
- ಪ್ರಸ್ತುತ ರೆಪೊಸಿಟರಿ ರಚನೆಗೆ ಹೊಂದಿಸಿ ಅಧ್ಯಯನ ಮಾರ್ಗದರ್ಶಿಯ ಸಂಪೂರ್ಣ ಮರುರಚನೆ  
- MCP ಕ್ಲೈಂಟ್‌ಗಳು ಮತ್ತು ಉಪಕರಣಗಳ, ಜನಪ್ರಿಯ MCP ಸರ್ವರ್‌ಗಳ ಹೊಸ ವಿಭಾಗಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ  
- ದೃಶ್ಯ ಪಠ್ಯಕ್ರಮ ನಕ್ಷೆಯನ್ನು ಎಲ್ಲಾ ವಿಷಯಗಳನ್ನು ನಿಖರವಾಗಿ ಪ್ರತಿಬಿಂಬಿಸಲು ನವೀಕರಿಸಲಾಗಿದೆ  
- ವಿಶೇಷ ವಿಷಯಗಳ ವಿವರಣೆಗಳನ್ನು ವೃದ್ಧಿಸಿ ಎಲ್ಲ ವಿಶಿಷ್ಟ ಕ್ಷೇತ್ರಗಳನ್ನು ಒಳಗೊಂಡಿದೆ  
- ಪ್ರಕರಣ ಅಧ್ಯಯನ ವಿಭಾಗವನ್ನು ನಿಜವಾದ ಉದಾಹರಣೆಗಳುಗಳಿಗೆ ಸರಿ ಮಾಡಿ ನವೀಕರಿಸಲಾಗಿದೆ  
- ಈ ಸಮಗ್ರ ಬದಲಾವಣೆ ನುಡಿಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ

- VSCode ನಲ್ಲಿ ಕ್ಲೌಡ್ ಬಳಕೆಯ ಸಮಗ್ರ ವಿಭಾಗವನ್ನು ಸೇರಿಸಲಾಗಿದೆ  
- ಕ್ಲೈನ್ ಟರ್ಮಿನಲ್ ಕ್ಲೈಂಟ್ ಸೆಟಪ್ ಮತ್ತು ಬಳಕೆ ಸೂಚನೆಗಳು ಸೇರಿಸಲಾಗಿದೆ  
- MCP ಕ್ಲೈಂಟ್ ವಿಭಾಗವನ್ನು ಎಲ್ಲಾ ಜನಪ್ರಿಯ ಕ್ಲೈಂಟ್ ಆಯ್ಕೆಗಳೊಂದಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ  
- ನಿಖರವಾದ ಕೋಡ್ ಮಾದರಿಗಳೊಂದಿಗೆ ಕೊಡುಗೆ ಉದಾಹರಣೆಗಳನ್ನು ವೃದ್ಧಿಸಲಾಗಿದೆ

### ಆಧುನಿಕ ವಿಷಯಗಳು (05-AdvancedTopics/)  
- ಎಲ್ಲಾ ವಿಶೇಷ ವಿಷಯ ಫೋಲ್ಡರ್‌ಗಳನ್ನು ಸुसಂಗತವಾಗಿ ಹೆಸರು ಕೊಟ್ಟು ಆಯೋಜಿಸಲಾಗಿದೆ  
- ಸಂದರ್ಭ ಇಂಜಿನಿಯರಿಂಗ್ ಪದಾರ್ಥಗಳು ಮತ್ತು ಉದಾಹರಣೆಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ  
- ಫೌಂಡ್ರಿ ಏಜೆಂಟ್ ಸಂಯೋಜನೆ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಸೇರಿಸಲಾಗಿದೆ  
- Entra ID ಭದ್ರತಾ ಸಂಯೋಜನೆ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ವೃದ್ಧಿಸಲಾಗಿದೆ
