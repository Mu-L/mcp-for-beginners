# ಬೆಂಬಲ

## ಸಮಸ್ಯೆಗಳನ್ನು ದಾಖಲಿಸುವುದು ಮತ್ತು ಸಹಾಯ ಪಡೆಯುವುದು ಹೇಗೆ  

ಈ ಯೋಜನೆ ದೋಷಗಳು ಮತ್ತು ವೈಶಿಷ್ಟ್ಯ ವಿನಂತಿಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲು GitHub Issues ಅನ್ನು ಬಳಸುತ್ತದೆ. ನಕಲಿ ಸಮಸ್ಯೆಗಳನ್ನು ತಪ್ಪಿಸಲು ದಯವಿಟ್ಟು ಹೊಸ ಸಮಸ್ಯೆಯನ್ನು ದಾಖಲಿಸುವ ಮೊದಲು ಇವುಗಳಿದ್ದ ಸಮಸ್ಯೆಗಳನ್ನು ಹುಡುಕಿ. ಹೊಸ ಸಮಸ್ಯೆಗಾಗಿ, ನಿಮ್ಮ ದೋಷ ಅಥವಾ ವೈಶಿಷ್ಟ್ಯ ವಿನಂತಿಯನ್ನು ಹೊಸ ಸಮಸ್ಯೆಯಾಗಿ ದಾಖಲಿಸಿ.

ಈ ಯೋಜನೆಯನ್ನು ಬಳಸುವುದರ ಬಗ್ಗೆ ಸಹಾಯ ಮತ್ತು ಪ್ರಶ್ನೆಗಳಿಗಾಗಿ, ದಯವಿಟ್ಟು:
- ತಜ್ಞರು ಮತ್ತು ಸಹ ಡೆವಲಪರ್‌ಗಳು ಸೇರಿಕೊಂಡಿರುವ [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ಪರಿಶೀಲಿಸಿ
- ಪ್ರೋಟೋಕಾಲ್ ಸಂಬಂಧಿತ ಪ್ರಶ್ನೆಗಳಿಗಾಗಿ [Model Context Protocol Community](https://modelcontextprotocol.io/community/) ಸಂಪನ್ಮೂಲಗಳನ್ನು ಬಳಸಿ

## Microsoft ಬೆಂಬಲ ನೀತಿ  

Beginnersಗಾಗಿ Model Context Protocol (MCP) ಗೆ ಬೆಂಬಲ ಮೇಲ್ಕನಿಷ್ಠ ಸಂಪನ್ಮೂಲಗಳಿಗಷ್ಟೇ ಸೀಮಿತವಾಗಿದೆ.
