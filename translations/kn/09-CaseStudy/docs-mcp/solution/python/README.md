# ಚೈನ್‌ಲಿಟ್ ಮತ್ತು ಮೈಕ್ರೋಸಾಫ್ಟ್ ಲರ್ನ್ ಡಾಕ್ಸ್ MCP ಮೂಲಕ ಅಧ್ಯಯನ ಯೋಜನೆ ಜನರೇಟರ್

## ಪೂರ್ವಾಪೇಕ್ಷಿತಗಳು

- ಪೈಥಾನ್ 3.8 ಅಥವಾ ಹಿಗರ್ನು
- ಪಿಪ್ (ಪೈಥಾನ್ ಪ್ಯಾಕೇಜ್ ನಿರ್ವಹಕ)
- ಮೈಕ್ರೋಸಾಫ್ಟ್ ಲರ್ನ್ ಡಾಕ್ಸ್ MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕ ಸಾಧಿಸಲು ಇಂಟರ್‌ನೆಟ್ ಪ್ರವೇಶ

## ಸ್ಥಾಪನೆ

1. ಈ ರೆಪೊಸಿಟರಿಯನ್ನು ಕ್ಲೋನ್ ಮಾಡಿ ಅಥವಾ ಪ್ರಾಜೆಕ್ಟ್ ಫೈಲ್‌ಗಳನ್ನು ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ.
2. ಅಗತ್ಯವಿರುವ ಡಿಪೆಂಡೆನ್ಸಿಗಳನ್ನು ಸ್ಥಾಪಿಸಿ:

   ```bash
   pip install -r requirements.txt
   ```

## ಬಳಕೆ

### ದೃಶ್ಯ 1: ಡಾಕ್ಸ್ MCP ಗೆ ಸರಳ ಪ್ರಶ್ನೆ
ಡಾಕ್ಸ್ MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕಿಸಿ, ಪ್ರಶ್ನೆ ಕಳುಹಿಸಿ ಮತ್ತು ಫಲಿತಾಂಶವನ್ನು ಮುದ್ರಿಸುವ ಕಮಾಂಡ್‌ಲೈನ್ ಕ್ಲೈಂಟ್.

1. ಸ್ಕ್ರಿಪ್ಟ್ ಅನ್ನು ಚಲಾಯಿಸಿ:
   ```bash
   python scenario1.py
   ```
2. ಪ್ರಾಂಪ್ಟ್‌ನಲ್ಲಿ ನಿಮ್ಮ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಪ್ರಶ್ನೆಯನ್ನು ನಮೂದಿಸಿ.

### ದೃಶ್ಯ 2: ಅಧ್ಯಯನ ಯೋಜನೆ ಜನರೇಟರ್ (ಚೈನ್‌ಲಿಟ್ ವೆಬ್ ಅಪ್)
ಚೈನ್‌ಲಿಟ್ ಬಳಸಿ ವೆಬ್ ಆಧಾರಿತ ಇಂಟರ್ಫೇಸ್, ಇದು ಬಳಕೆದಾರರಿಗೆ ಯಾವುದೇ ತಾಂತ್ರಿಕ ವಿಷಯದ ವೈಯಕ್ತಿಕೃತ, ವಾರಕ್ಕೊanವಾರ ಅಧ್ಯಯನ ಯೋಜನೆಯನ್ನು ರಚಿಸಲು ಅನುಮತಿಸುತ್ತದೆ.

1. ಚೈನ್‌ಲಿಟ್ ಅಪ್ಲಿಕೇಶನ್ ಪ್ರಾರಂಭಿಸಿ:
   ```bash
   chainlit run scenario2.py
   ```
2. ನಿಮ್ಮ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ನೀಡಲಾದ ಸ್ಥಳೀಯ URL ಅನ್ನು (ಉದಾ: http://localhost:8000) ನಿಮ್ಮ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ತೆರೆಯಿರಿ.
3. ಚಾಟ್ ವಿಂಡೋದಲ್ಲಿ ನಿಮ್ಮ ಅಧ್ಯಯನ ವಿಷಯ ಮತ್ತು ಪರಿಶಿಷ್ಟ ವಾರಗಳ ಸಂಖ್ಯೆಯನ್ನು ನಮೂದಿಸಿ (ಉದಾ: "AI-900 ಪ್ರಮಾಣಪತ್ರ, 8 ವಾರಗಳು").
4. ಅಪ್ ವಾರಕ್ಕೊanವಾರ ಅಧ್ಯಯನ ಯೋಜನೆಯನ್ನು ಪ್ರತಿಕ್ರಿಯಿಸುತ್ತದೆ, ಸಂಬಂಧಿತ ಮೈಕ್ರೋಸಾಫ್ಟ್ ಲರ್ನ್ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಲಿಂಕ್‌ಗಳೊಂದಿಗೆ.

**ಆವರಣ ವ್ಯತ್ಯಯಗಳು ಅಗತ್ಯವಿವೆ:**

ದೃಶ್ಯ 2 (ಅಜೂರ್ ಓಪನ್ ಎಐಯೊಂದಿಗೆ ಚೈನ್‌ಲಿಟ್ ವೆಬ್ ಅಪ್) ಬಳಸಲು, ನೀವು `python` ಡೈರೆಕ್ಟರಿಯಲ್ಲಿರುವ `.env` ಫೈಲ್‌ನಲ್ಲಿ ಕೆಳಕಂಡ ಆವರಣ ವ್ಯತ್ಯಯಗಳನ್ನು ಹೊಂದಿರಬೇಕು:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

ಈ ಮೌಲ್ಯಗಳನ್ನು ಅಜೂರ್ ಓಪನ್ ಎಐ ಸಂಸಾಧನ ವಿವರಗಳೊಂದಿಗೆ ಭರ್ತಿ ಮಾಡಿ ಮೊದಲು ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ರನ್ ಮಾಡುವ ಮೊದಲು.

> [!TIP]
> ನೀವು ಸುಲಭವಾಗಿ ನಿಮ್ಮ ಮಾದರಿಗಳನ್ನು [Microsoft Foundry](https://ai.azure.com/) ಬಳಸಿ ನಿಯೋಜಿಸಬಹುದು.

### ದೃಶ್ಯ 3: VS ಕೋಡ್‌ನಲ್ಲಿ MCP ಸರ್ವರ್ ಜೊತೆ ಎಡಿಟರ್ ಒಳಗಿನ ಡಾಕ್ಸ್

ಬ್ರೌಸರ್ ಟ್ಯಾಬ್‌ಗಳನ್ನು ಬದಲಾಯಿಸುವ ಬದಲು, ನೀವು MCP ಸರ್ವರ್ ಬಳಸಿ ಮೈಕ್ರೋಸಾಫ್ಟ್ ಲರ್ನ್ ಡಾಕ್ಸ್ ಅನ್ನು ನೇರವಾಗಿ VS ಕೋಡ್‌ನಲ್ಲಿ ತರಬಹುದು. ಇದು ನಿಮಗೆ ಅನುಮತಿಸುತ್ತದೆ:
- ನಿಮ್ಮ ಕೋಡಿಂಗ್ ಪರಿಸರವನ್ನು ಬಿಟ್ಟು ಬಿಡದೆ VS ಕೋಡ್ ಒಳಗೆ ಡಾಕ್ಸ್ ಹುಡುಕಿ ಓದಲು.
- ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ರೆಫರೆನ್ಸ್ ಮತ್ತು ಲಿಂಕ್‌ಗಳನ್ನು ನೇರವಾಗಿ README ಅಥವಾ ಕೋರ್ಸ್ ಫೈಲ್‌ಗಳಿಗೆ ಸೇರಿಸಲು.
- GitHub Copilot ಮತ್ತು MCP ನ್ನು ಒಟ್ಟಿಗೆ ಬಳಸಿ ಸ್ಲೀಕಾಗಿ, AI ಆಧಾರಿತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಕಾರ್ಯಪ್ರವಾಹ.

**ಉದಾಹರಣೆ ಬಳಕೆ ಪ್ರಕರಣಗಳು:**
- ಕೋರ್ಸ್ ಅಥವಾ ಪ್ರಾಜೆಕ್ಟ್ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಬರೆದಾಗ README ಗೆ ವೇಗವಾಗಿ ರೆಫರೆನ್ಸ್ ಲಿಂಕ್‌ಗಳನ್ನು ಸೇರ್ಪಡೆ ಮಾಡುವುದು.
- Copilot ಬಳಸಿ ಕೋಡ್ ರಚಿಸಿ ಮತ್ತು MCP ಬಳಸಿ ತಕ್ಷಣ ಸಂಬಂಧಿತ ಡಾಕ್ಸ್ ಕಂಡುಹಿಡಿಯುವುದು.
- ನಿಮ್ಮ ಎಡಿಟರ್‌ನಲ್ಲಿ ಗಮನ ಕೇಂದ್ರೀಕರಿಸಿ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು ಹೆಚ್ಚಿಸುವುದು.

> [!IMPORTANT]
> ನೀವು ಕಾರ್ಯಸ್ಥಳದಲ್ಲಿ ಮಾನ್ಯ [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) ಕಾನ್ಫಿಗರೇಶನ್ ಹೊಂದಿರುವದಾಗಿ ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ (ಸ್ಥಳ `.vscode/mcp.json`).

## ದೃಶ್ಯ 2ಕ್ಕೆ ಚೈನ್‌ಲಿಟ್ ಯಾಕೆ?

ಚೈನ್‌ಲಿಟ್ ಸಂವಾದಾತ್ಮಕ ವೆಬ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಲು ಆಧುನಿಕ ಓಪನ್ ಸಃಸರೋತ್ರ ಫ್ರೇಮ್ವರ್ಕ್. ಇದು ಮೈಕ್ರೋಸಾಫ್ಟ್ ಲರ್ನ್ ಡಾಕ್ಸ್ MCP ಸರ್ವರ್ ಪೂರ್ಣವಾಗಿ ಸಂಪರ್ಕವುಳ್ಳ ಬ್ಯಾಕ್‌ಎಂಡ್ ಸೆರ್ವಿಸ್‌ಗಳಿಗೆ ಸಂಪರ್ಕ ನೀಡುವ ಚಾಟ್ ಆಧಾರಿತ ಬಳಕೆದಾರ ಸಂವಾದೋಪಾದಿಗಳನ್ನು ಸೃಷ್ಟಿಸುವುದನ್ನು ಸುಲಭಗೊಳಿಸುತ್ತದೆ. ಈ ಪ್ರಾಜೆಕ್ಟ್ ಚೈನ್‌ಲಿಟ್ ಬಳಸಿ ಪರ್ಸನಲೈಸ್ ಮಾಡಿದ ಅಧ್ಯಯನ ಯೋಜನೆಗಳನ್ನ实时ವಾಗಿ ರಚಿಸಲು ಸರಳ ಹಾಗೂ ಸಂವಹನಾತ್ಮಕ ವಿಧಾನ ಒದಗಿಸುತ್ತದೆ. ಚೈನ್‌ಲಿಟ್ ಉಪಯೋಗಿಸುವ ಮೂಲಕ, ನೀವು ವೇಗವಾಗಿ ಚಾಟ್ ಆಧಾರಿತ ಉಪಕರಣಗಳನ್ನು ನಿರ್ಮಿಸಿ ನಿರ್ವಹಿಸಬಹುದು, ಇದು ಉತ್ಪಾದಕತೆ ಹಾಗೂ ಕಲಿಕೆಯನ್ನು ಉತ್ತೇಜಿಸುತ್ತದೆ.

## ಇದು ಏನು ಮಾಡುತ್ತದೆ

ಈ ಅಪ್ ಬಳಕೆದಾರರಿಗೆ ವಿಷಯ ಮತ್ತು ಅವಧಿಯನ್ನು ಸಿದೀಕರಿಸಿಯೇ ವೈಯಕ್ತಿಕೃತ ಅಧ್ಯಯನ ಯೋಜನೆ ಸೃಷ್ಟಿಸಲು ಅನುಮತಿಸುತ್ತದೆ. ಅಪ್ ನಿಮ್ಮ ಇನ್‌ಪುಟ್ ಪರಿಶೀಲಿಸಿ, ಸಂಬಂಧಿಸಿದ ವಿಷಯಕ್ಕಾಗಿ ಮೈಕ್ರೋಸಾಫ್ಟ್ ಲರ್ನ್ ಡಾಕ್ಸ್ MCP ಸರ್ವರ್‌ಗೆ ಪ್ರಶ್ನೆ ಕೇಳಿ, ಫಲಿತಾಂಶಗಳನ್ನು week-by-week ವಿನ್ಯಸ್ಥೀಕರಣಕ್ಕೆ ಆಯೋಜಿಸುತ್ತದೆ. ಪ್ರತಿ ವಾರದ ಶಿಫಾರಸುಗಳು ಚಾಟ್‌ನಲ್ಲಿ ಪ್ರದರ್ಶಿಸಲಾಗುತ್ತದೆ, ಇದು ಅನುಸರಿಸುವುದು ಮತ್ತು ಪ್ರಗತಿಯನ್ನ ತಪಾಸಿಸಲು ಸುಲಭಗೊಳಿಸುತ್ತದೆ. ಇಂಟಿಗ್ರೇಶನ್ ಮೂಲಕ ನೀವು ಸದಾ ಹೊಸ ಮತ್ತು ಅತ್ಯಂತ ಸಂಬಂಧಿಸಿದ ಕಲಿಕಾ ಸಂಪನ್ಮೂಲಗಳನ್ನು ಪಡೆಯುತ್ತೀರಿ.

## ಮಾದರಿ ಪ್ರಶ್ನೆಗಳು

ಎಲ್ಲಾ ಮೂಲಕ ಸಾಮಗ್ರಿಯನ್ನು ಪರೀಕ್ಷಿಸಲು ಈ ಪ್ರಶ್ನೆಗಳನ್ನು ಚಾಟ್ ವಿಂಡೋದಲ್ಲಿ ಪ್ರಯತ್ನಿಸಿ:

- `AI-900 ಪ್ರಮಾಣಪತ್ರ, 8 ವಾರಗಳು`
- `ಏಜ್ಯುರ್ ಫಂಕ್ಷನ್ಗಳನ್ನು ಕಲಿತುಕೊಳ್ಳಿ, 4 ವಾರಗಳು`
- `ಏಜ್ಯುರ್ ಡೆವ್ಓಪ್ಸ್, 6 ವಾರಗಳು`
- `ಏಜ್ಯುರ್‌ನಲ್ಲಿ ಡೇಟಾ ಎಂಜಿನಿಯರಿಂಗ್, 10 ವಾರಗಳು`
- `ಮೈಕ್ರೋಸಾಫ್ಟ್ ಸೆಕ್ಯುರಿಟಿ ಮೂಲಭೂತ, 5 ವಾರಗಳು`
- `ಪವರ್ ಪ್ಲಾಟ್‌ಫಾರಂ, 7 ವಾರಗಳು`
- `ಏಜ್ಯುರ್ AI ಸೇವೆಗಳು, 12 ವಾರಗಳು`
- `ಕ್ಲೌಡ್ ಆರ್ಕಿಟೆಕ್ಚರ್, 9 ವಾರಗಳು`

ಈ ಉದಾಹರಣೆಗಳು ಹಲವೆಲ್ಲಾ ಕಲಿಕಾ ಗುರಿಗಳು ಮತ್ತು ಅವಧಿಗಳಿಗಾಗಿ ಅಪ್‌ನ ಅನುಕೂಲತೆಯನ್ನು ತೋರಿಸುತ್ತವೆ.
