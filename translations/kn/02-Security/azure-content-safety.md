# ಅಡ್ವಾನ್ಸ್ಡ್ MCP ಭದ್ರತೆ ಅಜ್ಯೂರ್ ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಜೊತೆ

> **OWASP MCP ಅಪಾಯ ವಿಳಂಬಿಸಲಾಗಿದೆ**: [MCP06 - ಉದ್ದೇಶ ಹರಿವು ಮರುಪರಿಗಣನೆ](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

ಅಜ್ಯೂರ್ ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ನಿಮ್ಮ MCP ಕಾರ್ಯಾತ್ಮಕತೆಗಳ ಭದ್ರತೆಯನ್ನು ಹೆಚ್ಚಿಸುವ ಹಲವು ಶಕ್ತಿಶಾಲಿ ಟೂಲ್ಗಳನ್ನು ನೀಡುತ್ತದೆ. ಪ್ರಾಯೋಗಿಕ ಅನುಭವಕ್ಕಾಗಿ, ನೋಡಿ [MCP Security Summit Workshop (Sherpa)](https://azure-samples.github.io/sherpa/) Camp 3: I/O Security.

## ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್ಸ್

ಮೈಕ್ರೋಸಾಫ್ಟ್‌ನ AI ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್ಸ್ ನೇರ ಮತ್ತು ಪರೋಕ್ಷ ಪ್ರಾಂಪ್ಟ್ ಅಳವಡಿಕೆ ದಾಳಿಗಳ ವಿರುದ್ಧದ ಬಲಾತ್ಕಾರ ರಕ್ಷಣೆ ನೀಡುತ್ತದೆ:

1. **ಅಡ್ವಾನ್ಸ್ ಡಿಟಕ್ಷನ್**: ವಿಷಯದಲ್ಲಿ ನೊಂದಿರುವ ದುಷ್ಟ ಸೂಚನೆಗಳನ್ನು ಗುರುತಿಸಲು ಯಂತ್ರ ಅಧ್ಯಯನವನ್ನು ಉಪಯೋಗಿಸುತ್ತದೆ.
2. **ಸ್ಪಾಟ್‌ಲೈಟಿಂಗ್**: ಮಾನ್ಯ ಸೂಚನೆಗಳು ಮತ್ತು ಹೊರಗಿನ ಇನ್ಪುಟ್ ಗಳನ್ನು AI ವ್ಯವಸ್ಥೆಗಳು ವ್ಯತ್ಯಾಸಪಡಿಸುವಂತೆ ಇನ್‌ಪುಟ್ ಪಠ್ಯವನ್ನು ಪರಿವರ್ತನಗೊಳಿಸುತ್ತದೆ.
3. **ಡೆಲಿಮಿಟರ್ಸ್ ಮತ್ತು ಡೇಟಾಮಾರ್ಕಿಂಗ್**: ನಂಬಿಗಸ್ತ ಮತ್ತು ಅನಂಬಿಗಸ್ತ ಡೇಟಾ ನಡುವಿನ ಗಡಿಗಳನ್ನು ಗುರುತಿಸುತ್ತದೆ.
4. **ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಏಕೀಕರಣ**: ಅಜ್ಯೂರ್ AI ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಜೊತೆಯಾಗಿ ಡಿಟೆಕ್ಟ್ ಜೈಲ್ಬ್ರೇಕ್ ಪ್ರಯತ್ನಗಳು ಮತ್ತು ಹಾನಿಕಾರಕ ವಿಷಯಗಳ ಪತ್ತೆ ಹಚ್ಚುತ್ತದೆ.
5. **ತತ್ಕಾಲೀನ ನವೀಕರಣಗಳು**: ಮೈಕ್ರೋಸಾಫ್ಟ್ ನಿರಂತರವಾಗಿ ಬೆಳೆಯುವ ಧಮ್ಕಿಗಳನ್ನು ಎದುರಿಸುವ ರಕ್ಷಣಾ ಆವೃತ್ತಿಗಳನ್ನು ನವೀಕರಿಸುತ್ತದೆ.

## MCP ಜೊತೆಗೆ ಅಜ್ಯೂರ್ ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಅನ್ವಯಿಕೆ

ಈ ವಿಧಾನ ಬಹು-ಮಟ್ಟದ ರಕ್ಷಣೆಯನ್ನು ಒದಗಿಸುತ್ತದೆ:
- ಪ್ರಕ್ರಿಯೆಗಿಂತ ಮುಂಚೆ ಇನ್‌ಪುಟ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸುವುದು
- ಫಲಿತಾಂಶಗಳನ್ನು ಮರಳಿ ನೀಡುವ ಮುಂಚೆ ಪರಿಶೀಲಿಸುವುದು
- ಪರಿಚಿತ ಹಾನಿಕಾರಕ ಮಾದರಿಗಳಿಗೆ ಬ್ಲಾಕ್‌ಲಿಸ್ಟ್‌ಗಳನ್ನು ಉಪಯೋಗಿಸುವುದು
- ಅಜ್ಯೂರ್‌ನ ನಿರಂತರ ನವೀಕರಿಸುವ ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಮಾದರಿಗಳನ್ನು ಬಳಸುವುದು

## ಅಜ್ಯೂರ್ ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಸಂಪನ್ಮೂಲಗಳು

ನಿಮ್ಮ MCP ಸರ್ವರ್‌ಗಳೊಂದಿಗೆ ಅಜ್ಯೂರ್ ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಅನುಷ್ಠಾನ ಮಾಡಲು ಈ ಅಧಿಕೃತ ಸಂಪನ್ಮೂಲಗಳನ್ನು ನೋಡಿ:

1. [ಅಜ್ಯೂರ್ AI ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಡಾಕ್ಯುಮೆಂಟೇಷನ್](https://learn.microsoft.com/azure/ai-services/content-safety/) - ಅಜ್ಯೂರ್ ಕಂಟೆಂಟ್ ಸಫ್ಟಿಗೆ ಅಧಿಕೃತ ಡಾಕ್ಯುಮೆಂಟೇಷನ್.
2. [ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್ ಡಾಕ್ಯುಮೆಂಟೇಷನ್](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/prompt-shield) - ಪ್ರಾಂಪ್ಟ್ ಅಳವಡಿಕೆ ದಾಳಿಗಳನ್ನು ತಡೆಗಟ್ಟುವ ವಿಧಾನ.
3. [ಕಂಟೆಂಟ್ ಸಫ್ಟಿ API ಉಲ್ಲೇಖ](https://learn.microsoft.com/rest/api/contentsafety/) - ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಅನ್ವಯಿಕೆಗೆ ವಿವರವಾದ API ಉಲ್ಲೇಖ.
4. [ಕ್ವಿಕ್‌ಸ್ಟಾರ್ಟ್: C# ಜೊತೆಗೆ ಅಜ್ಯೂರ್ ಕಂಟೆಂಟ್ ಸಫ್ಟಿ](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-csharp) - C# ಬಳಸಿ ತ್ವರಿತ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿ.
5. [ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಕ್ಲೈಂಟ್ ಲೈಬ್ರರಿಗಳು](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-client-libraries-rest-api) - ವಿವಿಧ ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಭಾಷೆಗಳಿಗಾಗಿ ಕ್ಲೈಂಟ್ ಲೈಬ್ರರಿಗಳು.
6. [ಜೈಲ್ಬ್ರೇಕ್ ಪ್ರಯತ್ನಗಳನ್ನು ಪತ್ತೆಹಚ್ಚುವುದು](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/jailbreak-detection) - ಜೈಲ್ಬ್ರೇಕ್ ಪ್ರಯತ್ನಗಳನ್ನು ಪತ್ತೆಹಚ್ಚುವ ಮತ್ತು ತಡೆಯುವ ವಿಶೇಷ ಮಾರ್ಗದರ್ಶನ.
7. [ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/best-practices) - ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಕಂಟೆಂಟ್ ಸಫ್ಟಿಯನ್ನು ಅನುಷ್ಠಾನ ಮಾಡುವ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು.

ಹೆಚ್ಚು ಆಳವಾದ ಅನುಷ್ಠಾನಕ್ಕಾಗಿ, ನಮ್ಮ [ಅಜ್ಯೂರ್ ಕಂಟೆಂಟ್ ಸಫ್ಟಿ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿ](./azure-content-safety-implementation.md) ನೋಡಿ.
