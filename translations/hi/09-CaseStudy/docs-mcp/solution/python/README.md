# Chainlit & Microsoft Learn Docs MCP के साथ अध्ययन योजना जनरेटर

## आवश्यकताएँ

- Python 3.8 या उससे ऊपर
- pip (Python पैकेज प्रबंधक)
- Microsoft Learn Docs MCP सर्वर से कनेक्ट करने के लिए इंटरनेट एक्सेस

## स्थापना

1. इस रिपॉजिटरी को क्लोन करें या प्रोजेक्ट फाइलें डाउनलोड करें।
2. आवश्यक निर्भरताएँ इंस्टॉल करें:

   ```bash
   pip install -r requirements.txt
   ```

## उपयोग

### परिदृश्य 1: Docs MCP के लिए सरल प्रश्न
एक कमांड-लाइन क्लाइंट जो Docs MCP सर्वर से कनेक्ट होता है, प्रश्न भेजता है, और परिणाम प्रिंट करता है।

1. स्क्रिप्ट चलाएँ:
   ```bash
   python scenario1.py
   ```
2. प्रॉम्प्ट पर अपना दस्तावेज़ीकरण प्रश्न दर्ज करें।

### परिदृश्य 2: अध्ययन योजना जनरेटर (Chainlit वेब ऐप)
एक वेब-आधारित इंटरफेस (Chainlit का उपयोग करते हुए) जो उपयोगकर्ताओं को किसी भी तकनीकी विषय के लिए व्यक्तिगत, सप्ताह-दर-सप्ताह अध्ययन योजना जनरेट करने की अनुमति देता है।

1. Chainlit ऐप शुरू करें:
   ```bash
   chainlit run scenario2.py
   ```
2. अपने टर्मिनल में प्रदत्त स्थानीय URL (जैसे, http://localhost:8000) अपने ब्राउज़र में खोलें।
3. चैट विंडो में अपना अध्ययन विषय और अध्ययन के सप्ताहों की संख्या दर्ज करें (जैसे, "AI-900 certification, 8 weeks")।
4. ऐप सप्ताह-दर-सप्ताह अध्ययन योजना के साथ जवाब देगा, जिसमें संबंधित Microsoft Learn दस्तावेज़ीकरण के लिंक शामिल होंगे।

**आवश्यक पर्यावरण चर:**

परिदृश्य 2 (Azure OpenAI के साथ Chainlit वेब ऐप) का उपयोग करने के लिए, आपको `python` निर्देशिका में `.env` फ़ाइल में निम्नलिखित पर्यावरण चर सेट करना होगा:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

ऐप चलाने से पहले इन मानों को अपने Azure OpenAI संसाधन विवरण से भरें।

> [!TIP]
> आप आसानी से अपने स्वयं के मॉडल को [Microsoft Foundry](https://ai.azure.com/) का उपयोग करके तैनात कर सकते हैं।

### परिदृश्य 3: VS Code में MCP सर्वर के साथ इन-एडिटर डॉक

दस्तावेज़ीकरण खोजने के लिए ब्राउज़र टैब स्विच करने के बजाय, आप Microsoft Learn Docs को सीधे MCP सर्वर के माध्यम से अपने VS Code में ला सकते हैं। इससे आप कर सकते हैं:
- VS Code के अंदर ही कोडिंग वातावरण छोड़ें बिना दस्तावेज़ पढ़ें और खोजें।
- दस्तावेज़ीकरण का संदर्भ लें और लिंक सीधे अपने README या पाठ्यक्रम फाइलों में डालें।
- GitHub Copilot और MCP को साथ में उपयोग करें ताकि एक सहज, AI-संचालित दस्तावेज़ीकरण कार्यप्रवाह मिल सके।

**उदाहरण उपयोग के मामले:**
- पाठ्यक्रम या प्रोजेक्ट दस्तावेज़ लिखते समय README में त्वरित संदर्भ लिंक जोड़ें।
- Copilot का उपयोग करके कोड जनरेट करें और MCP द्वारा प्रासंगिक दस्तावेज तुरंत खोजें और संदर्भित करें।
- अपने संपादक में ध्यान केंद्रित रखें और उत्पादकता बढ़ाएँ।

> [!IMPORTANT]
> सुनिश्चित करें कि आपके वर्कस्पेस में एक मान्य [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) कॉन्फ़िगरेशन है (स्थान `.vscode/mcp.json` है)।

## परिदृश्य 2 के लिए Chainlit क्यों?

Chainlit चैट-आधारित वेब एप्लिकेशन बनाने के लिए एक आधुनिक खुला स्रोत फ्रेमवर्क है। यह Microsoft Learn Docs MCP सर्वर जैसे बैकएंड सेवाओं से जुड़ने वाले चैट आधारित यूजर इंटरफेस बनाना आसान बनाता है। यह प्रोजेक्ट Chainlit का उपयोग करके रियल टाइम में व्यक्तिगत अध्ययन योजनाएँ जनरेट करने का एक साधारण, इंटरैक्टिव तरीका प्रदान करता है। Chainlit का उपयोग करके, आप जल्दी से चैट-आधारित टूल बना और तैनात कर सकते हैं जो उत्पादकता और सीखने में सुधार करते हैं।

## यह क्या करता है

यह ऐप उपयोगकर्ताओं को केवल एक विषय और अवधि दर्ज करके एक व्यक्तिगत अध्ययन योजना बनाने की अनुमति देता है। ऐप आपके इनपुट को पार्स करता है, Microsoft Learn Docs MCP सर्वर से प्रासंगिक सामग्री के लिए प्रश्न करता है, और परिणामों को एक संरचित, सप्ताह-दर-सप्ताह योजना में व्यवस्थित करता है। प्रत्येक सप्ताह की सिफारिशें चैट में प्रदर्शित की जाती हैं, जिससे आपकी प्रगति का पालन और ट्रैक करना आसान हो जाता है। यह एकीकरण सुनिश्चित करता है कि आपको हमेशा नवीनतम, सबसे प्रासंगिक सीखने के संसाधन मिलें।

## नमूना प्रश्न

इन प्रश्नों को चैट विंडो में आज़माएँ और देखें कि ऐप कैसे प्रतिक्रिया देता है:

- `AI-900 certification, 8 weeks`
- `Learn Azure Functions, 4 weeks`
- `Azure DevOps, 6 weeks`
- `Data engineering on Azure, 10 weeks`
- `Microsoft security fundamentals, 5 weeks`
- `Power Platform, 7 weeks`
- `Azure AI services, 12 weeks`
- `Cloud architecture, 9 weeks`

ये उदाहरण विभिन्न सीखने के लक्ष्यों और समयसीमाओं के लिए ऐप की लचीलापन दिखाते हैं।
