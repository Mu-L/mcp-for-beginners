# सहायता

## मुद्दे कैसे दर्ज करें और सहायता प्राप्त करें  

यह परियोजना बग और फीचर अनुरोधों को ट्रैक करने के लिए GitHub Issues का उपयोग करती है। कृपया नए मुद्दे दर्ज करने से पहले मौजूदा 
मुद्दों को खोजें ताकि डुप्लिकेट टाल सकें। नए मुद्दों के लिए, अपनी बग या 
फीचर अनुरोध को एक नए Issue के रूप में दर्ज करें।

इस परियोजना का उपयोग करने के बारे में सहायता और प्रश्नों के लिए, कृपया:
- [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) देखें जहाँ विशेषज्ञ और साथी डेवलपर्स इकट्ठा होते हैं
- प्रोटोकॉल-संबंधित प्रश्नों के लिए [Model Context Protocol Community](https://modelcontextprotocol.io/community/) संसाधनों का उपयोग करें

## Microsoft सहायता नीति  

Model Context Protocol (MCP) for Beginners परियोजना के लिए समर्थन केवल ऊपर सूचीबद्ध संसाधनों तक सीमित है।
