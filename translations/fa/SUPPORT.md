# پشتیبانی

## نحوه ثبت مشکلات و دریافت کمک

این پروژه از GitHub Issues برای پیگیری باگ‌ها و درخواست‌های ویژگی استفاده می‌کند. لطفاً قبل از ثبت مشکلات جدید، مشکلات موجود را جستجو کنید تا از تکراری شدن جلوگیری شود. برای مسائل جدید، باگ یا درخواست ویژگی خود را به‌عنوان یک Issue جدید ثبت کنید.

برای کمک و سوالات مربوط به استفاده از این پروژه، لطفاً:
- بررسی کنید [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) جایی که کارشناسان و توسعه‌دهندگان هم‌فکر گرد هم می‌آیند
- از منابع [Model Context Protocol Community](https://modelcontextprotocol.io/community/) برای سوالات مربوط به پروتکل استفاده کنید

## سیاست پشتیبانی مایکروسافت

پشتیبانی از پروژه Model Context Protocol (MCP) برای مبتدیان محدود به منابع ذکر شده در بالا است.
