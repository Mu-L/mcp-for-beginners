# 支援

## 如何提交問題和獲取幫助  

本專案使用 GitHub Issues 來追蹤錯誤與功能請求。請在提交新問題前先搜尋現有的問題，以避免重複。對於新的問題，請將您的錯誤或功能請求作為新的 Issue 提交。

如需有關使用本專案的幫助和問題，請：
- 查看 [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4)，這裡是專家和開發者的聚集地
- 使用 [Model Context Protocol Community](https://modelcontextprotocol.io/community/) 的資源以詢問協議相關問題
