# бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ

## рҹҺҜ бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ

бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҠбҹғбһӣбһҫбһ“бҹҒбҹҮбһҺбҹӮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһІбҹ’бһҷбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„ PostgreSQL бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹҶбһЎбһҫбһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһ”бһүбҹ’бһҮбһјбһ“бһ’бһ“бһ’бһ¶бһ“ Azure бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҳбһ»бһ“бһ“бһ№бһ„бһ”бһ“бҹ’бһҸбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһӮбһәбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷбҹ” бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҺбҹӮбһ“бһ¶бҹҶбһҮбһ¶бһӮбһ“бҹ’бһӣбһ„бһҮбҹҶбһ бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„ Docker бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһҡбһ”бһҹбҹӢбһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбҹ”

бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ Zava Retail MCP бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- **бһҸбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһЎбһҫбһ„** бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ
- **бһ…бҹҶбһҺбһ¶бһҷбһ’бһ“бһ’бһ¶бһ“ Azure** бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
- **бһҸбҹҶбһЎбһҫбһ„бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡ Docker** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ PostgreSQL бһ“бһ·бһ„ бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
- **бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ** бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
- **бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶** бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҡбһ»бҹҶбһҡбһүбҹ’бһҮбһҪбһҷ бһ“бһ·бһ„бһ”бһүбҹ’бһ бһ¶бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ
- **бһҷбһӣбҹӢбһҠбһ№бһ„** бһўбҹҶбһ–бһёбһ…бһҡбһ“бҹ’бһҸбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҜбһҖбһҹбһ¶бһҡ

## рҹ“Ӣ бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ

бһҳбһ»бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бҹ–

### бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ
- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһҡбһ·бһ”бһ‘бһ”бһүбҹ’бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (Windows Command Prompt/PowerShell)
- бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ
- бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһўбҹҶбһ–бһё Git version control
- бһҷбһӣбҹӢбһҠбһ№бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһўбҹҶбһ–бһё Docker (containers, images, volumes)

### бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’
- **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ**: Windows 10/11, macOS бһ¬ Linux
- **RAM**: бһҷбҹүбһ¶бһ„бһҸбһ·бһ… 8GB (бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбҹ’бһҹбһ¶бһңбһҡбһёбһҷбҹҚ 16GB)
- **бһҗбһҸбһ•бҹ’бһ‘бһ»бһҖ**: бһҷбҹүбһ¶бһ„бһҸбһ·бһ… 10GB бһ‘бһ‘бҹҒ
- **бһ”бһҺбҹ’бһҸбһ¶бһү**: бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһ…бһ¶бһҖбҹӢбһ…бҹҒбһү Azure

### бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбһҺбһ“бһё
- **бһҖбһ¶бһҡбһҮбһ¶бһң Azure**: бһҖбҹҶбһ–бһјбһӣбһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢ
- **бһӮбһҺбһ“бһё GitHub**: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
- **бһӮбһҺбһ“бһё Docker Hub**: (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҡбһјбһ”бһ—бһ¶бһ–бһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһҸбҹҶбһҡбһјбһңбһҖбһ¶бһҡ

## рҹӣ пёҸ бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ§бһ”бһҖбһҡбһҺбҹҚ

### бҹЎ. бһҸбҹҶбһЎбһҫбһ„ Docker Desktop

Docker бһ•бҹ’бһҸбһӣбҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”

#### бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„ Windows

1. **бһ‘бһ¶бһүбһҷбһҖ Docker Desktop**бҹ–
   ```cmd
   # Visit https://desktop.docker.com/win/stable/Docker%20Desktop%20Installer.exe
   # Or use Windows Package Manager
   winget install Docker.DockerDesktop
   ```

2. **бһҸбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһЎбһҫбһ„**бҹ–
   - бһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһЎбһҫбһ„бһҮбһ¶бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„
   - бһ”бһҫбһҖбһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢ WSL 2 бһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһҹбҹҶбһҺбһјбһҳбһ–бһҡ
   - бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһЎбһҫбһ„бһңбһ·бһүбһ–бҹҒбһӣбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ

3. **бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„**бҹ–
   ```cmd
   docker --version
   docker-compose --version
   ```

#### бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„ macOS

1. **бһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһҸбҹҶбһЎбһҫбһ„**бҹ–
   ```bash
   # бһ‘бһ¶бһүбһҷбһҖбһ–бһё https://desktop.docker.com/mac/stable/Docker.dmg
   # бһ¬бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Homebrew
   brew install --cask docker
   ```

2. **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ Docker Desktop**бҹ–
   - бһ”бһҫбһҖ Docker Desktop бһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё
   - бһ”бһүбҹ’бһ…бһ”бҹӢбһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҠбҹҶбһ”бһјбһ„

3. **бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„**бҹ–
   ```bash
   docker --version
   docker-compose --version
   ```

#### бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„ Linux

1. **бһҸбҹҶбһЎбһҫбһ„ Docker Engine**бҹ–
   ```bash
   # Ubuntu/Debian
   curl -fsSL https://get.docker.com -o get-docker.sh
   ```

```bash
#!/usr/bin/env python3
"""
Environment validation script for MCP Server setup.
"""
import asyncio
import os
import sys
import subprocess
import requests
import asyncpg
from azure.identity import DefaultAzureCredential
from azure.ai.projects import AIProjectClient

async def validate_environment():
    """Comprehensive environment validation."""
    results = {}
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбҹӮ Python
    python_version = sys.version_info
    results['python'] = {
        'status': 'pass' if python_version >= (3, 8) else 'fail',
        'version': f"{python_version.major}.{python_version.minor}.{python_version.micro}",
        'required': '3.8+'
    }
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһҸбһҫбһҖбһүбҹ’бһ…бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҳбһ¶бһ“бһҡбһҪбһ…бһ бһҫбһҷбһ¬бһўбһҸбҹӢ
    required_packages = ['fastmcp', 'asyncpg', 'azure-ai-projects']
    for package in required_packages:
        try:
            __import__(package)
            results[f'package_{package}'] = {'status': 'pass'}
        except ImportError:
            results[f'package_{package}'] = {'status': 'fail', 'error': 'Not installed'}
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ Docker
    try:
        result = subprocess.run(['docker', '--version'], capture_output=True, text=True)
        results['docker'] = {
            'status': 'pass' if result.returncode == 0 else 'fail',
            'version': result.stdout.strip() if result.returncode == 0 else 'Not available'
        }
    except FileNotFoundError:
        results['docker'] = {'status': 'fail', 'error': 'Docker not found'}
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ Azure CLI
    try:
        result = subprocess.run(['az', '--version'], capture_output=True, text=True)
        results['azure_cli'] = {
            'status': 'pass' if result.returncode == 0 else 'fail',
            'version': result.stdout.split('\n')[0] if result.returncode == 0 else 'Not available'
        }
    except FileNotFoundError:
        results['azure_cli'] = {'status': 'fail', 'error': 'Azure CLI not found'}
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһ”бһ‘
    required_env_vars = [
        'PROJECT_ENDPOINT',
        'AZURE_OPENAI_ENDPOINT',
        'EMBEDDING_MODEL_DEPLOYMENT_NAME',
        'AZURE_CLIENT_ID',
        'AZURE_CLIENT_SECRET',
        'AZURE_TENANT_ID'
    ]
    
    for var in required_env_vars:
        value = os.getenv(var)
        results[f'env_{var}'] = {
            'status': 'pass' if value else 'fail',
            'value': '***' if value and 'SECRET' in var else value
        }
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
    try:
        conn = await asyncpg.connect(
            host=os.getenv('POSTGRES_HOST', 'localhost'),
            port=int(os.getenv('POSTGRES_PORT', 5432)),
            database=os.getenv('POSTGRES_DB', 'zava'),
            user=os.getenv('POSTGRES_USER', 'postgres'),
            password=os.getenv('POSTGRES_PASSWORD', 'secure_password')
        )
        
        # бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбҹҶбһҺбһҪбһҡ
        result = await conn.fetchval('SELECT COUNT(*) FROM retail.stores')
        await conn.close()
        
        results['database'] = {
            'status': 'pass',
            'store_count': result
        }
    except Exception as e:
        results['database'] = {
            'status': 'fail',
            'error': str(e)
        }
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
    try:
        response = requests.get('http://localhost:8000/health', timeout=5)
        results['mcp_server'] = {
            'status': 'pass' if response.status_code == 200 else 'fail',
            'response': response.json() if response.status_code == 200 else response.text
        }
    except Exception as e:
        results['mcp_server'] = {
            'status': 'fail',
            'error': str(e)
        }
    
    # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI бһҡбһ”бһҹбҹӢ Azure
    try:
        credential = DefaultAzureCredential()
        project_client = AIProjectClient(
            endpoint=os.getenv('PROJECT_ENDPOINT'),
            credential=credential
        )
        
        # бһ“бҹҒбҹҮбһ“бһ№бһ„бһ”бһҡбһ¶бһҮбҹҗбһҷбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһҖбһ·бһ…бҹ’бһ…бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
        results['azure_ai'] = {'status': 'pass'}
        
    except Exception as e:
        results['azure_ai'] = {
            'status': 'fail',
            'error': str(e)
        }
    
    return results

def print_results(results):
    """Print formatted validation results."""
    print("рҹ”Қ Environment Validation Results\n")
    print("=" * 50)
    
    passed = 0
    failed = 0
    
    for component, result in results.items():
        status = result.get('status', 'unknown')
        if status == 'pass':
            print(f"вң… {component}: PASS")
            passed += 1
        else:
            print(f"вқҢ {component}: FAIL")
            if 'error' in result:
                print(f"   Error: {result['error']}")
            failed += 1
    
    print("\n" + "=" * 50)
    print(f"Summary: {passed} passed, {failed} failed")
    
    if failed > 0:
        print("\nвқ— Please fix the failed components before proceeding.")

    else:
        print("\nрҹҺү All validations passed! Your environment is ready.")
        return True

if __name__ == "__main__":
```
