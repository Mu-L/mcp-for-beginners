# ឧទាហរណ៍ទី 3៖ ឯកសារជាមួយ MCP Server ក្នុង VS Code

## ទិដ្ឋភាពទូទៅ

នៅក្នុងឧទាហរណ៍នេះ អ្នកនឹងរៀនពីរបៀបនាំឯកសារ Microsoft Learn Docs មកទាញយកត្រង់ក្នុងបរិយាកាស Visual Studio Code របស់អ្នក ដោយប្រើ MCP server។ ជំនួសការប្ដូរទំព័រពហុកម្មវិធីរកឯកសារប៉ុន្មានដង អ្នកអាចចូលដំណើរការ ស្វែងរក និងយោងឯកសារផ្លូវការយ៉ាងត្រឹមត្រូវពីក្នុងកម្មវិធីកែសម្រួលរបស់អ្នក។ វិធីសាស្រ្តនេះធ្វើឱ្យដំណើរការងាររបស់អ្នកមានប្រសិទ្ធភាព ការផ្តោតលើការងារត្រូវបានរក្សាទុក ហើយអនុញ្ញាតឱ្យបញ្ចូលគ្រប់គ្រងជាមួយឧបករណ៍ដូចជា GitHub Copilot បានយ៉ាងរលូន។

- ស្វែងរក និងអានឯកសារជាប់ក្នុង VS Code ដោយគ្មានការចាកចេញពីបរិយាកាសកូដេីង។
- យោងឯកសារនិងបញ្ចូលតំណភ្ជាប់ដោយផ្ទាល់ក្នុង README ឬឯកសារតំណាក់កាលរបស់អ្នក។
- ប្រើ GitHub Copilot និង MCP រួមគ្នាវិញសម្រាប់ដំណើរការអត្ថបទជាមួយ AI បានដោយដំណើរការរលូន។

## គោលបំណងការរៀន

នៅចុងបញ្ចប់ជំពូកនេះ អ្នកនឹងយល់ពីរបៀបកំណត់ និងប្រើ MCP server ក្នុង VS Code ដើម្បីឲ្យបង្កើនគុណភាពឯកសារនិងដំណើរការអភិវឌ្ឍរបស់អ្នក។ អ្នកអាច៖

- កំណត់បរិយាកាសការងាររបស់អ្នកសម្រាប់ប្រើ MCP server ដើម្បីស្វែងរកឯកសារ។
- ស្វែងរក និងបញ្ចូលឯកសារដោយផ្ទាល់ពី VS Code។
- ផ្លាស់ប្តូរប្រាស់តំណាង GitHub Copilot និង MCP រួមគ្នាសម្រាប់ដំណើរការអត្ថបទដែលមាន AI ជំនួយបានប្រសើរឡើង។

ជំនាញទាំងនេះ នឹងជួយអ្នករក្សាភាពផ្ដោតលើការងារ ពង្រឹងគុណភាពឯកសារ និងបង្កើនផលិតភាពដូចជាកម្មវិធីករឬអ្នកសរសេរឯកសារបច្ចេកទេស។

## ដំណោះស្រាយ

ដើម្បីទទួលបានការចូលដំណើរការឯកសារជាមួយកម្មវិធីកែសម្រួល អ្នកនឹងធ្វើតាមជំហានជាច្រើនដែលបានរួមបញ្ចូល MCP server ជាមួយ VS Code និង GitHub Copilot។ ដំណោះស្រាយនេះសមរម្យសម្រាប់អ្នកសរសេរតំណាក់កាលវគ្គបណ្ដុះបណ្ដាល អ្នកសរសេរឯកសារ និងក្រុមអ្នកអភិវឌ្ឍដែលចង់រក្សាភាពផ្ដោតលើកម្មវិធីកែសម្រួលខណៈបំនុលកិច្ចការជាមួយឯកសារនិង Copilot។

- បន្ថែមតំណយោងទៅ README យ៉ាងលឿន ខណៈបង្កើតឯកសារតំណាក់កាលឬឯកសារគម្រោង។
- ប្រើ Copilot ដើម្បីបង្កើតកូដ និង MCP ដើម្បីស្វែងរក និងយោងឯកសារដែលទាក់ទងភ្លាមៗ។
- រក្សាភាពផ្ដោតលើកម្មវិធីកែសម្រួល និងបង្កើនផលិតភាព។

### មគ្គុទេសក៍​ជាន់ថ្នាក់

ដើម្បីចាប់ផ្តើម សូមអនុវត្តតាមជំហានខាងក្រោម។ សម្រាប់ជំហានទាំងអស់ អ្នកអាចបញ្ចូលរូបថតពីថតទ្រព្យសម្បត្តិ ដើម្បីបង្ហាញដំណើរការជាមួយរូបភាព។

1. **បន្ថែមការកំណត់ MCP:**
   នៅក្នុងប្រភពគម្រោងរបស់អ្នក បង្កើតឯកសារ `.vscode/mcp.json` ហើយបន្ថែមការកំណត់ដូចខាងក្រោម៖

   ```json
   {
     "servers": {

       }
     }
   }
   ```
  ការកំណត់នេះប្រាប់ VS Code ឲ្យដឹងពីរបៀបភ្ជាប់ទៅ [«Microsoft Learn Docs MCP server»](https://github.com/MicrosoftDocs/mcp)។
