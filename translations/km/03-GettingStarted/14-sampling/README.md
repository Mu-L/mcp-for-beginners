# бһҖбһ¶бһҡбһӮбҹҶбһҡбһј - бһұбҹ’бһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһ‘бҹ… Client

бһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһұбҹ’бһҷ MCP Client бһ“бһ·бһ„ MCP Server бһ”бҹ’бһҡбһҫбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһ”бһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбһҪбһҳбһҳбһҪбһҷбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ¶бһ“бһҖбһҡбһҺбһёбһҠбҹӮбһӣ Server бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷбһ–бһё LLM бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһӣбһҫ Clientбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһӮбһәбһҮбһ¶бһҡбһҝбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһ”бҹ’бһҡбһҫбҹ”

бһҺбҹӮбһ“бһ¶бҹҶбһҳбһҖбһҳбһҫбһӣбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҒбҹ’бһӣбҹҮбҹ— бһ бһҫбһҷбһҡбһ”бҹҖбһ”бһҹбһ¶бһ„бһҹбһ„бҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһӮбҹҶбһҡбһјбҹ”

## бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҹбһ„бҹ’бһҒбҹҒбһ”

бһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бҹҒбһӣбһҺбһ¶ бһ“бһ·бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶бһҠбҹӮбһӣбһӮбһҪбһҡбһ”бҹ’бһҡбһҫ Sampling бһ“бһ·бһ„бһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһңбһ¶бҹ”

## бһ•бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ“бҹ…бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„:

- бһ–бһ“бҹ’бһҷбһӣбҹӢбһҗбһ¶ Sampling бһҳбһ¶бһ“бһўбҹ’бһңбһё бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶бһҠбҹӮбһӣбһӮбһҪбһҡбһ”бҹ’бһҡбһҫбһңбһ¶бҹ”
- бһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢ Sampling бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCPбҹ”
- бһ•бҹ’бһҸбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ Sampling бһҖбҹ’бһ“бһ»бһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бҹ”

## Sampling бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё бһ бһҫбһҷбһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһңбһ¶?

Sampling бһӮбһәбһҮбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢбһҳбһҪбһҷ бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҠбһјбһ…бһҸбһ‘бҹ…бҹ–

```mermaid
sequenceDiagram
    participant User

    MCP Client->>MCP Server: бһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ (бһҹбҹ’бһӣбһ¶бһ”бһ–бҹ’бһҡбҹҗбһҸбҹ’бһҡвҖӢбһўбһҸбҹ’бһҗбһ”бһ‘вҖӢбһ бҹ’бһӮбһӣбҹӢ)
    MCP Server->>MCP Client: бһҖбһ¶бһҡбһҹбһ»бҹҶбһ…бһҳбҹ’бһӣбһ„ (бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ„бҹ’бһҒбҹҒбһ”)
    MCP Client->>LLM: бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ„бҹ’бһҒбҹҒбһ”бһўбһҸбҹ’бһҗбһ”бһ‘вҖӢбһ бҹ’бһӮбһ¶бһӣбҹӢ
    LLM->>MCP Client: бһӣбһ‘бҹ’бһ’бһ•бһӣвҖӢбһҹбһ„бҹ’бһҒбҹҒбһ”
    MCP Client->>MCP Server: бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ…бһҳбҹ’бһӣбһ„ (бһҹбһ„бҹ’бһҒбҹҒбһ”)
    MCP Server->>MCP Client: бһ”бһүбҹ’бһ…бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘вҖӢбһ бҹ’бһӮбһӣбҹӢ (бһҹбҹ’бһӣбһ¶бһ”бһ–бҹ’бһҡбҹҗбһҸбҹ’бһҡ + бһҹбһ„бҹ’бһҒбҹҒбһ”)
    MCP Client->>User: бһўбһҸбҹ’бһҗбһ”бһ‘вҖӢбһ бҹ’бһӮбһӣбҹӢбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢ
```

### бһ–бһ¶бһҖбҹ’бһҷбһҹбһ»бҹҶ Sampling

бһӣбҹ’бһўбһ бһҫбһҷ бһ–бҹҒбһӣбһ“бҹҒбҹҮбһҷбһҫбһ„бһҳбһ¶бһ“бһ‘бһҹбҹ’бһҹбһ“бҹҲбһҳбһҪбһҷбһҠбҹӮбһӣбһ’бҹҶбһ”бҹҶбһ•бһ»бһҸбһ“бҹғбһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһҠбҹӮбһӣбһ“бһ·бһҷбһҳбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ бһҳбһҖбһ“бһ·бһҷбһ¶бһҷбһ–бһёбһ–бһ¶бһҖбҹ’бһҷбһҹбһ»бҹҶ sampling бһҠбҹӮбһӣ Server бһ•бҹ’бһүбһҫбһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ… Clientбҹ” бһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҳбһ¶бһ“бһҡбһјбһ”бһҡбһ¶бһ„бһҠбһјбһ…бһҮбһ¶ JSON-RPC:

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "sampling/createMessage",
  "params": {
    "messages": [
      {
        "role": "user",
        "content": {
          "type": "text",
          "text": "Create a blog post summary of the following blog post: <BLOG POST>"
        }
      }
    ],
    "modelPreferences": {
      "hints": [
        {
          "name": "claude-3-sonnet"
        }
      ],
      "intelligencePriority": 0.8,
      "speedPriority": 0.5
    },
    "systemPrompt": "You are a helpful assistant.",
    "maxTokens": 100
  }
}
```

бһҳбһ¶бһ“бһҡбһҝбһ„бһҒбҹ’бһӣбҹҮбҹ—бһҠбҹӮбһӣбһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбҹ–

- Prompt бһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳ content -> text бһӮбһәбһҮбһ¶бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһ…бһјбһӣ бһҠбҹӮбһӣбһҮбһ¶бһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ LLM бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ„бҹ’бһҒбҹҒбһ”бһҳбһ¶бһҸбһ·бһҖбһ¶бһ”бҹ’бһӣбһҖбҹӢбҹ”

- **modelPreferences**. бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһҠбҹӮбһӣбһӮбһҪбһҡбһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷ LLMбҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбһ¶бһҳбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһ“бҹҒбҹҮ бһ¬бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһ…бһ·бһҸбҹ’бһҸбҹ” бһ“бҹ…бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹҒбҹҮ бһҳбһ¶бһ“бһҷбҹ„бһ”бһӣбҹӢбһӣбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһӮбһҪбһҡбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӣбһҫбһӣбҹ’бһ”бһҝбһ“ бһ“бһ·бһ„бһҳбһ“бһ»бһҹбҹ’бһҹбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”
- **systemPrompt**, бһ“бҹҒбҹҮбһҮбһ¶бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ’бһҳбҹ’бһҳбһҸбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһӣбһҖбҹ’бһҒбһҺбҹҲбһ‘бҹ… LLM бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбҹ”
- **maxTokens**, бһӮбһәбһҮбһ¶бһӣбһҖбҹ’бһҒбһҺбҹҲбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бһёбһ…бҹҶбһ“бһҪбһ“бһҸбһ»бһҖбһ·бһ“бһҠбҹӮбһӣбһ•бҹ’бһҠбһӣбҹӢбһҮбһјбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бҹҒбҹҮбҹ”

### бһҳбһҸбһ·бһҹбҹ’бһ“бһҫ Sampling

бһҳбһҸбһ·бһҸбһ”бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣ MCP Client бһ•бҹ’бһүбһҫбһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ… MCP Server бһ–бһёбһҖбһ¶бһҡбһ бҹ… LLM бһҡбһ”бһҹбҹӢ Client бһҡбһ„бҹӢбһ…бһ¶бҹҶбһҳбһҸбһ·бһҸбһ”бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ¶бһҡбһ“бҹҒбҹҮбһЎбһҫбһ„бһңбһ·бһүбҹ” бһҡбһјбһ”бһҡбһ¶бһ„бһңбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһ‘бҹҶбһҡбһ„бҹӢ JSON-RPC бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "role": "assistant",
    "content": {
      "type": "text",
      "text": "Here's your abstract <ABSTRACT>"
    },
    "model": "gpt-5",
    "stopReason": "endTurn"
  }
}
```

```python
from starlette.applications import Starlette
from starlette.routing import Mount, Host

from mcp.server.fastmcp import Context, FastMCP

from mcp.server.session import ServerSession
from mcp.types import SamplingMessage, TextContent

import json


from uuid import uuid4
from typing import List
from pydantic import BaseModel


mcp = FastMCP("Blog post generator")

# app = FastAPI()

posts = []

class BlogPost(BaseModel):
    id: int
    title: str
    content: str
    abstract: str

posts: List[BlogPost] = []

@mcp.tool()
async def create_blog(title: str, content: str, ctx: Context[ServerSession, None]) -> str:
    """Create a blog post and generate a summary"""

    post = BlogPost(
        id=len(posts) + 1,
        title=title,
        content=content,
        abstract=""
    )

    prompt = f"Create an abstract of the following blog post: title: {title} and draft: {content} "

    result = await ctx.session.create_message(
        messages=[
            SamplingMessage(
                role="user",
                content=TextContent(type="text", text=prompt),
            )
        ],
        max_tokens=100,
    )

    post.abstract = result.content.text

    posts.append(post)

    # бһ”бһ„бҹ’бһҡбһҪбһ”бһ”бһ„бҹ’бһҡбһҪбһҳбһўбһҸбҹ’бһҗбһ”бһ‘бһ”бҹ’бһӣбһ»бһҖбһҹбһҡбһ»бһ”
    return json.dumps({
        "id": post.title,
        "abstract": post.abstract
    })

if __name__ == "__main__":
    print("Starting server...")
    # mcp.run()
    mcp.run(transport="streamable-http")

# бһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ¶бһҳбһҪбһҷ бҹ– python server.py
```

1. бһҮбҹ’бһҡбһҫбһҹ "Configure Model Access", бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣ GitHub Copilot бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ”бҹ’бһҡбһҫбһ“бҹ…бһ–бҹҒбһӣбһ’бҹ’бһңбһҫ samplingбҹ” бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһҳбһҫбһӣбһ–бһ¶бһҖбҹ’бһҷбһҹбһ»бҹҶ sampling бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһЎбһҫбһ„бһҗбҹ’бһҳбһёбҹ—бһҠбҹ„бһҷбһҮбҹ’бһҡбһҫбһҹ "Show Sampling requests"бҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡ

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ Sampling бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҒбһ»бһҹбһ”бһ“бҹ’бһҸбһ·бһ…бһӮбҹ’бһ“бһ¶ бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ•бһӣбһ·бһҸбһ•бһӣбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

**бһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚ**: бһҖбһҳбҹ’бһҳбһҖбһҡбһҹбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҷбһҖбһ¶бһҡбһ·бһҷбһ¶бһӣбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһўбҹҠбһё-бһҖбһҳбҹүбҹӮбһҸбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷ бһңбһ¶бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ•бһӣбһ·бһҸбһ•бһӣбҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбһҪбһҷ бһҠбҹӮбһӣбһўбһ¶бһ…бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ "create_product" бһҮбһ¶бһҳбһҪбһҷ "title" бһ“бһ·бһ„ "keywords" бһҮбһ¶бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢ бһ бһҫбһҷбһңбһ¶бһӮбһҪбһҡбһўбҹ„бһҷбһ”бһүбҹ’бһ…бҹҒбһүбһ•бһӣбһ·бһҸбһ•бһӣбһ–бҹҒбһүбһӣбҹҒбһү бһҠбҹӮбһӣбһҳбһ¶бһ“бһңбһ¶бһӣ "description" бһҠбҹӮбһӣбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһүбһҠбҹ„бһҷ LLM бһҡбһ”бһҹбҹӢ Client бҹ”

TIP: бһ”бҹ’бһҡбһҫбһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһ–бһёбһҳбһ»бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ…бһ“бһ¶бһўбҹ’бһ“бһҖ Server бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһ–бһ¶бһҖбҹ’бһҷбһҹбһ»бҹҶ samplingбҹ”
