## рҹҺҜ бһҸбһҫбһҳбғқбғ“бҹ’бһҷбһјбһӣбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ

бһҳбһјбһҢбһ»бһӣбһ“бҹҒбҹҮбһ”бһ¶бһ“бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Model Context Protocol (MCP) бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһ“бһ·бһ„бһҮбҹҶбһҡбһ»бһүбһҖбһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбҹ” бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң, бһӮбҹҶбһҡбһјбһҠбҹғ бһ“бһ·бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҗбһ¶ MCP бһўбһ¶бһ…бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһ¶бһ“ бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҹбһ¶бһҳбҹүбһјбһҢбҹӮбһӣ бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

### рҹ“ҡ бһҳбһҫбһӣ MCP бһҖбҹ’бһ“бһ»бһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–

бһ…бһ„бҹӢбһғбһҫбһүбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҲбһ¶бһ“бһ‘бҹ…бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҡбһҪбһ…бһ бһҫбһҷбһ‘бҹҒ? бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ [**бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҡбһ”бһҹбҹӢ Microsoft бҹЎбҹ  бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ**](microsoft-mcp-servers.md) бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҡбһ”бһҹбҹӢ Microsoft бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҗбҹ’бһ„бҹғбһ“бҹҒбҹҮбҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһҷбһҖбһҠбҹҶбһ”бһјбһ„бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Model Context Protocol (MCP) бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһ“бһ·бһ„бһҮбҹҶбһҡбһ»бһүбһҖбһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһ‘бһјбһ‘бһ¶бҹҶбһ„бһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһӮбҹҶбһҡбһјбһҠбҹғ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҗбһ¶ MCP бһўбһ¶бһ…бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹҶбһҡбһјбһңбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ’бҹ’бһңбһҫбһҸбһ¶бһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ AI вҖ” бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶбҹ— бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ”бҹӮбһ”бһ”бһ‘бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶ бһ“бһ·бһ„бһҹбһ¶бһ„бһҹбһ„бҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһўбһ—бһ·бһ”бһ¶бһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP, бһҡбҹҖбһ“бһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ‘бҹ…бһ бһҫбһҷ, бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһўбһ“бһ»бһӮбҹ’бһҡбҹ„бҹҮбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ“бһ¶бҹҶбһ…бһјбһӣ MCP бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҖбҹҸбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ‘бһ·бһҹбһҠбҹ…бһҖбҹҶбһ–бһ»бһ„бһӣбҹҒбһ…бһ’бҹ’бһӣбҹ„ бһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһ”бһҫбһҖбһ…бҹҶбһ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ“бҹ…бһҶбҹ’бһңбҹҒбһ„бһ•бҹ’бһ“бҹӮбһҖбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ MCP бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбҹҒбһҖбһјбһҹбҹҠбһёбһҹбҹ’бһҠбһ·бһҳбһҖбҹҶбһ–бһ»бһ„бһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

- бһңбһ·бһ—бһ¶бһӮбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһ‘бһјбһ‘бһ¶бҹҶбһ„бһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—
- бһҡбһ…бһ“бһ¶ бһ“бһ·бһ„бһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP бһ–бҹҒбһүбһӣбҹҒбһү
- бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһ‘бһ·бһҹбһҠбҹ…бһҖбҹҶбһ–бһ»бһ„бһӣбҹҒбһ…бһ’бҹ’бһӣбҹ„ бһ“бһ·бһ„бһ‘бһ·бһҹбһҠбҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбһҖбҹ’бһ“бһ»бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ MCP
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһўбһ“бһ»бһӮбҹ’бһҡбҹ„бҹҮбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ

## бһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһ–бһёбһ–бһ·бһ—бһ–бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ

### бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶ бҹЎбҹ– бһўбһјбһҸбһјбһҳбҹүбһ¶бһ‘бһ·бһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһ–бһ бһ»бһҮбһ¶бһҸбһ·бһҖбҹҶбһ–бһ»бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹҶбһҡбһјбһңбһҳбҹүбһјбһҢбҹӮбһӣ AI бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҳбһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ бһ“бһ¶бҹҶбһ‘бҹ…бһҠбһӣбҹӢ:

- бһҡбһ…бһ“бһ¶бһ”бһ‘бһ•бҹ’бһ‘бҹғбһҳбһ»бһҒбһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ LLM бһ–бһ бһ»
- бһҡбһҖбҹ’бһҹбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ prompt бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбһ¶бһҡбһ—бһ¶бһ–бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ•бҹ’бһ“бҹӮбһҖ

using Microsoft.Extensions.DependencyInjection;
using ModelContextProtocol.SDK.Client;
using ModelContextProtocol.SDK.Security;
using ModelContextProtocol.SDK.Resources;

public class DiagnosticAssistant
{
    private readonly MCPHostClient _mcpClient;
    private readonly PatientContext _patientContext;
    
    public DiagnosticAssistant(PatientContext patientContext)
    {
        _patientContext = patientContext;
        
        // Configure MCP client with healthcare-specific settings
        var clientOptions = new ClientOptions
        {
            Name = "Healthcare Diagnostic Assistant",
            Version = "1.0.0",
            Security = new SecurityOptions
            {
                Encryption = EncryptionLevel.Medical,
                AuditEnabled = true
            }
        };
        
        _mcpClient = new MCPHostClientBuilder()
            .WithOptions(clientOptions)
            .WithTransport(new HttpTransport("https://healthcare-mcp.example.org"))
            .WithAuthentication(new HIPAACompliantAuthProvider())
            .Build();
    }
    
    public async Task<DiagnosticSuggestion> GetDiagnosticAssistance(
        string symptoms, string patientHistory)
    {
        // Create request with appropriate resources and tool access
        var resourceRequest = new ResourceRequest
        {
            Name = "patient_records",
            Parameters = new Dictionary<string, object>
            {
                ["patientId"] = _patientContext.PatientId,
                ["requestingProvider"] = _patientContext.ProviderId
            }
        };
        
        // Request diagnostic assistance using appropriate prompt
        var response = await _mcpClient.SendPromptRequestAsync(
            promptName: "diagnostic_assistance",
            parameters: new Dictionary<string, object>
            {
                ["symptoms"] = symptoms,
                patientHistory = patientHistory,
                relevantGuidelines = _patientContext.GetRelevantGuidelines()
            });
            
        return DiagnosticSuggestion.FromMCPResponse(response);
    }
}

        // бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҮбһ¶бһҳбһҪбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһўбһ“бһ»бһӣбҹ„бһҳбһ•бҹ’бһ“бҹӮбһҖбһ бһ·бһҡбһүбҹ’бһүбһңбһҸбҹ’бһҗбһ»
        MCPServer server = new MCPServerBuilder()
            .withModelProviders(
                new ModelProvider("risk-assessment-primary", new AzureOpenAIProvider()),
                new ModelProvider("risk-assessment-audit", new LocalLlamaProvider())
            )
            .withPromptTemplateDirectory("./compliance/templates")
            .withAccessControls(new SOCCompliantAccessControl())
            .withDataEncryption(EncryptionStandard.FINANCIAL_GRADE)
            .withVersionControl(true)
            .withAuditLogging(new DatabaseAuditLogger())
            .build();
            
        server.addRequestValidator(new FinancialDataValidator());
        server.addResponseFilter(new PII_RedactionFilter());
        
        server.start(9000);
        
        System.out.println("Financial Risk MCP Server running on port 9000");
    }
}

**бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ–**

```typescript
// TypeScript: бһҖбһ¶бһҡбһ…бҹҒбһүбһҲбҹ’бһҳбҹ„бҹҮбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ Playwright бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
import { createServer, ToolDefinition } from 'modelcontextprotocol';
import { launch } from 'playwright';

const server = createServer({
  name: 'Playwright MCP Server',
  version: '1.0.0',
  description: 'MCP server for browser automation using Playwright'
});

// бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ»бһҖбһҡбһҖбһ‘бҹ… URL бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ
server.tools.register(
  new ToolDefinition({
    name: 'navigate_and_screenshot',
    description: 'Navigate to a URL and capture a screenshot',
    parameters: {
      url: { type: 'string', description: 'The URL to visit' }
    }
  }),
  async ({ url }) => {
    const browser = await launch();
    const page = await browser.newPage();
    await page.goto(url);
    const screenshot = await page.screenshot();
    await browser.close();
    return { screenshot };
  }
);

// бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
server.listen(8080);
```

## бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶ бҹҰбҹ– NLWeb  
MCP (Model Context Protocol) бһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бһ·бһ’бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Chatbots бһ“бһ·бһ„бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҖбҹӢ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбҹ” бһҡбһ¶бһӣбҹӢбһ”бһҺбҹ’бһҠбһ»бҹҶ NLWeb бһӮбһәбһҮбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһ•бһ„ бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбһ”бҹҖбһ”бһҳбһҪбһҷбһ•бҹ’бһӣбһјбһңбһ…бһҳбҹ’бһ”бһ„, ask, бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҪбһҡбһ–бһ·бһҮбҹҗбһҷбһҸбҹҶбһ”бһ“бҹӢбһ”бһҺбҹ’бһҠбһ¶бһүбһҠбҹ„бһҷбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бҹ” бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ¶бҹҶбһҳбһҖбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ schema.org, бһңбһ·бһ—бһ¶бһӮбһңбһ…бһ“бһ¶бһ“бһ»бһҖбҹ’бһҡбһҳбһҠбҹӮбһӣбһӮбҹҒбһ”бҹ’бһҡбһҫбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһңбҹҒбһ”бһҹбһ¶бһҷбҹ” бһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP бһӮбһә NLWeb бһҠбһјбһ…бһҮбһ¶ Http бһ‘бҹ… HTMLбҹ” NLWeb бһ•бҹ’бһӮбһ»бҹҶбһҡбһҪбһҳбһ–бһёбһ–бһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ, бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ Schema.org бһ“бһ·бһ„бһҖбһјбһҠбһӮбҹҶбһҡбһј бһ‘бҹ…бһҸбһ¶бһҳбһ‘бһёбһҸбһ¶бҹҶбһ„бһ“бһ¶бһ“бһ¶бһұбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹ’бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹ, бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһҸбһ¶бһҳбһҡбһҷбҹҲбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһҹбһ“бҹ’бһ‘бһ“бһ¶ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бҹ”

**бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ–**

- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһҷбҹүбһ¶бһ„бһҸбһ·бһ…бһ”бһёбһҡбһјбһ” (бһ§. OpenAI, Anthropic, бһҳбҹүбһјбһҢбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ)  
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹҒбһҖбһ¶бһ“бһ·бһҖбһ•бҹ’бһүбһҫбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹҶбһҺбһҫ  
- бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ  
- бһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһ‘бһ»бһҖбһ”бһҺбҹ’бһҠбһ»бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҗбҹ’бһӣбҹғбһҠбһҫбһҳ  
- бһҹбһ¶бһ„бһҹбһ„бҹӢбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ¶бһҳбһүбҹ’бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ  

**бһҮбҹҶбһ бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–**

1. бһҸбҹҶбһЎбһҫбһ„бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“  
2. бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһңбһ¶бһҳбҹүбһјбһҢбҹӮбһӣ AI бһҸбҹӮбһҳбһҪбһҷбҹ—  
3. бһ”бһ„бҹ’бһҖбһҫбһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ•бҹ’бһүбһҫбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҺбһҫ  
4. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбҹҒбһҖбһ¶бһ“бһ·бһҖбһ•бҹ’бһ‘бһ»бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҫбһүбһ№бһҖбһүбһ¶бһ”бҹӢ  
5. бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ  
6. бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһҡбһјбһҹбҹҶбһҺбһҫбһ•бҹ’бһҹбҹҒбһ„бҹ—  

**бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бҹ–** бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ–бһёPython (.NET/Java/Python бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ), Redis бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖ, бһ“бһ·бһ„бһ”бҹҠбҹӮбһҸбһңбҹҒбһ”бһҹбһ¶бһҳбһүбҹ’бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бҹ”  

### бһӮбһҳбҹ’бһҡбҹ„бһ„ бҹўбҹ– бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Prompt бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

**бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бҹ–** бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһҖбҹҶбһҺбҹӮ бһ“бһ·бһ„бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһӮбҹҶбһҡбһј prompt бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҳбһҪбһҷбҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһғбҹ’бһӣбһ¶бҹҶбһ„бһҳбһҪбһҷбһҖбһҺбҹ’бһҸбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹҶбһҡбһјбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳ
- бһҹбһ¶бһ„бһҹбһ„бҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӮбҹҶбһҡбһјбһҮбһ¶бһҳбһҪбһҷбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹҶбһҡбһј
- бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бһјбһӣбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҸбһҪбһ“бһ¶бһ‘бһё
- бһ”бһ„бҹ’бһҖбһҫбһҸ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһӮбҹҶбһҡбһј

**бһҮбҹҶбһ бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбҹ–**

1. бһҡбҹҖбһ”бһ…бҹҶбһҹбҹ’бһҖбһёбһҳбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһӮбҹҶбһҡбһј
2. бһ”бһ„бҹ’бһҖбһҫбһҸ API бһҳбҹҒбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҖбһҳбҹ’бһҡбһ·бһҸ CRUD бһҡбһ”бһҹбҹӢбһӮбҹҶбһҡбһј
3. бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹҶбһҺбҹӮ
4. бһҹбһ¶бһ„бһҹбһ„бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳ
5. бһўбһ—бһ·бһңбһҢбҹ’бһҚбһңбҹҒбһ‘бһ·бһҖбһ¶бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
6. бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„
7. бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP

**бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бҹ–** бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ“бҹӮбһҖбһҖбҹ’бһҡбҹ„бһҷ, бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ SQL бһ¬ NoSQL, бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһҒбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бҹ”

### бһӮбһҳбҹ’бһҡбҹ„бһ„ бҹЈ: бһңбҹҒбһ‘бһ·бһҖбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹ„бһҷбһҳбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһӣбһҫ MCP

**бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бҹ–** бһҹбһ¶бһ„бһҹбһ„бҹӢбһңбҹҒбһ‘бһ·бһҖбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶ бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ MCP бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҠбһӣбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ¶бһ“бһҹбһ—бһ¶бһ–бһҡбһҪбһҳбһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҳбһ¶бһҸбһ·бһҖбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

**бһ‘бһ¶бһҳбһ‘бһ¶бһҡбҹ–**

- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҳбһ¶бһҸбһ·бһҖбһ¶бһ…бһҳбҹ’бһ”бһ„бҹ— (бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бҹ’бһӣбһ»бһҖ, бһ”бһҺбҹ’бһҠбһ¶бһүбһҹбһ„бҹ’бһӮбһҳ, бһ…бһҳбҹ’бһӣбһ„бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡ)
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӮбҹҶбһҡбһјбһҮбһ¶бһҳбһҪбһҷбһҮбһҳбҹ’бһҡбһҫбһҹбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
- бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһ¶бһҸбһ·бһҖбһ¶ бһ“бһ·бһ„бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ

### бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһҖбҹҶбһ–бһ»бһ„бһӣбҹҒбһ…бһЎбһҫбһ„

1. **MCP бһҠбҹҸбһ–бһ бһ»бһҡбһ”бҹҖбһ”**
   - бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ MCP бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһјбһ”бһ—бһ¶бһ– бһҹбһҳбҹ’бһӣбҹҒбһ„ бһ“бһ·бһ„бһңбһёбһҠбҹҒбһўбһј
   - бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ бҹҒбһҸбһ»бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҡбһ”бҹҖбһ”бһ•бҹ’бһҹбҹҒбһ„бҹ—
   - бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһҮбһ¶бһҡбһҷбҹҲбһ–бҹҒбһӣбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ”бҹҖбһ”бһ•бҹ’бһҹбҹҒбһ„бҹ—

2. **бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ MCP бһҹбһ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ**
   - бһ”бһҺбҹ’бһҸбһ¶бһү MCP бһ…бҹӮбһҖбһ…бһ¶бһҷбһўбһ¶бһ…бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ’бһ“бһ’бһ¶бһ“бһҡбһңбһ¶бһ„бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—
   - бһ–бһ·бһ’бһёбһҖбһ¶бһҡбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӮбһҺбһ“бһ¶бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“

   - бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ’бһ¶бһ“бһ¶бһӮбһ»бһҺбһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ
   - бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһ•бҹ’бһҹбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ

####бһҗбһҸбһ’бһ“бһ’бһ¶бһ“ MCP

бһҗбһҸ [MCP Resources directory](https://github.com/microsoft/mcp/tree/main/Resources) бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„ MCP бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢ Microsoft бһ•бҹ’бһҠбһӣбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбһҪбһҷбһҠбҹӮбһӣбһҹбҹҶбһҡбһ¶бһ”бҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ’бһ“бһ’бһ¶бһ“ бһӮбҹҶбһҡбһјбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ Model Context Protocolбҹ” бһҗбһҸбһ“бҹҒбҹҮбһҡбһ…бһ“бһ¶бһҡбһ”бһҹбҹӢбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҶбһ¶бһ”бҹӢбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ MCP бһҠбҹ„бһҷбһ•бҹ’бһҸбһӣбҹӢбһҲбһ»бһҸбһ”бҹ’бһӣбһ»бһҖбһҡбһ…бһ“бһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһ„бһ¶бһҷбҹ— бһҳбһ¶бһ“бһӮбҹҶбһҡбһјбһҒбҹ’бһ–бһҹбҹӢбһ“бҹғбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹӮбһ”бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбҹ–

- **бһӮбҹҶбһҡбһјбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бҹ–** бһӮбҹҶбһҡбһјбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…AI бһ“бһ·бһ„бһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ‘бһјбһ‘бҹ… бһҠбҹӮбһӣбһўбһ¶бһ…бһҖбҹӮбһ”бҹ’бһҡбҹӮбһІбҹ’бһҷбһҹбһҳбһҡбһҳбҹ’бһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”
- **бһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбҹ–** бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбҹ’бһҖбһёбһҳбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҳбҹҒбһҸбһ¶бһҠбһ¶бһҸбһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбҹҶбһ–бһ¶бһңбһ“бһ¶бһңбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
- **бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ’бһ“бһ’бһ¶бһ“бҹ–** бһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ API бһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҠбһ»бһҳ MCPбҹ”
- **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбҹ„бһ„бҹ–** бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһ…бҹҶ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһ’бһ“бһ’бһ¶бһ“ бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„ MCP бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ”

бһ’бһ“бһ’бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһӣбһҫбһҖбһҖбһҳбҹ’бһ–бһҹбҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ бһ“бһ·бһ„бһҮбһҪбһҷбһ’бһ¶бһ“бһ¶бһ“бһјбһңбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ MCPбҹ”

Model Context Protocol (MCP) бһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбһ“бһ¶бһӮбһҸбһ“бҹғбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶ бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһӣбһҝбһ“бһ“бҹ…бһ‘бһјбһ‘бһ¶бҹҶбһ„бһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбҹ” бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһҡбһҺбһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ бһ“бһ·бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһғбһҫбһүбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹҶбһ”бһјбһ„ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ Microsoft бһ“бһ·бһ„ Azure бһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ MCP бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ–бһ·бһҸ бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ AI бһҷбҹүбһ¶бһ„бһӣбһҝбһ“ бһ“бһ·бһ„бһ’бһ¶бһ“бһ¶бһ”бһ¶бһ“бһ“бһјбһңбһҖбһ¶бһҡбһўбһ“бһ»бһӣбҹ„бһҳ бһ‘бҹҶбһ„бһ“бҹӢ, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһҡбһ„бҹ’бһңбһ¶бһҹбҹӢбһҖбҹҶбһҺбһҸбҹӢбҹ” бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһјбһҢбһ»бһӣбһҡбһ”бһҹбҹӢ MCP бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶбҹ— бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ”бҹҖбһ”бһ бҹ…бһҳбһҪбһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһ¶бһ“бҹ” бһҒбһҺбҹҲбһҠбҹӮбһӣ MCP бһҖбҹҶбһ–бһ»бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ”бһ“бҹ’бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҹбһ бһӮбһҳбһ“бҹҚ бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһ’бһ“бһ’бһ¶бһ“бһ”бһҫбһҖбһ…бҹҶбһ  бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһӮбһәбһӮбһ“бҹ’бһӣбһ№бҹҮбһҠбһӣбҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһҡбһ№бһ„бһҳбһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ¶бһӮбһҸбҹ”
