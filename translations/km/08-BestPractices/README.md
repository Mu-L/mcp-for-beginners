_(бһ…бһ»бһ…бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһңбһёбһҠбҹҒбһўбһјбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“бһ“бҹҒбҹҮ)_

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ, бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ, бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢ MCP бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹ…бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһ•бһӣбһ·бһҸбҹ” бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ MCP бһҖбһҫбһ“бһЎбһҫбһ„бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ“бһ·бһ„бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ, бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһӣбҹҶбһ“бһ¶бҹҶбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҹбҹ’бһҗбһ¶бһ–бһҡбһҮбһ¶бһ—бһ¶бһӮбһ…бһҳбҹ’бһ”бһ„бһ’бһ¶бһ“бһ¶бһ”бһ¶бһ“бһ“бһјбһңбһ—бһ¶бһ–бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ, бһҖбһ¶бһҡбһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһҡбһҪбһҳбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһүбҹ’бһүбһ¶бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бһ‘бһҪбһӣбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһҠбһҫбһҳбҹ’бһ”бһёбһҺбҹӮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҡбһ№бһ„бһҳбһ¶бҹҶ, бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһҳбһҪбһҷбһ’бһ“бһ’бһ¶бһ“бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–, бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһҹбҹ’бһӮбҹ’бһҡбһёбһ” бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһ•бҹ’бһ‘бҹҒбһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP
- бһҡбһ…бһ“бһ¶бһӮбҹҶбһҡбһјбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһ¶бһ…мһ¬бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP бһҠбҹӮбһӣбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹбһҠбҹӮбһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң, бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһ–бһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP
- бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶

## бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP

бһҳбһ»бһ“бһ“бһ№бһ„бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһҮбһ¶бһҡбһҝбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹӮбһӣбһҺбҹӮбһ“бһ¶бҹҶбһўбһ—бһ·бһңбһҢбҹ’бһҚ MCP бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ–

1. **бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ**: MCP бһ”бҹ’бһҡбһҫ JSON-RPC 2.0 бһҮбһ¶бһӮбҹ’бһҡбҹ„бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢбһңбһ¶ бһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹғбһҹбҹҶбһҺбһҫ, бһ–бһҸбҹҢбһҳбһ¶бһ“бһҸбһ”бһҸбһ”, бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹбһ“бҹ…бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”

2. **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҳбҹ’бһҸбҹ„бһ„бһ‘бҹ…бһӣбһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ**: бһҸбҹӮбһ„бһҸбҹӮбһ•бҹ’бһҸбһӣбҹӢбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһҠбһӣбҹӢбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳ бһўбҹҶбһҺбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ—бһ¶бһ–бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

3. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҮбһ¶бһҠбҹҶбһ”бһјбһ“бҹ’бһҳбһ¶бһ“**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ’бһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ бһҡбһҪбһҳбһҳбһ¶бһ“ бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ, бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бҹ”

4. **бһўбһ¶бһӮбһ¶бһҡбһҳбҹүбһјбһҢбһ»бһӣ**: бһҡбһ…бһ“бһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһҪбһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҳбҹүбһјбһҢбһ»бһӣбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ“бһ·бһ„бһ•бҹ’бһҸбҹ„бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ”

5. **бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–**: бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ MCP бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҸбһ¶бһҳбһҡбһҷбҹҲбһҹбҹҶбһҺбһҫбһ…бҹ’бһҡбһҫбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбһ”бһҹбҹ’бһ“бһ„бһҗбҹ’бһӣбҹғбһҗбҹ’бһӣбһ¶ бһ“бһ·бһ„бһҳбһ¶бһ“бһ”бһҡбһ·бһ”бһ‘бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”

## бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ MCP бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶

бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҒбһүбһ–бһёбһҜбһҖбһҹбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢ Model Context Protocolбҹ–

### бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

1. **бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ**: бһҸбҹӮбһ„бһҸбҹӮбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһ»бһ“бһ–бҹҒбһӣбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ¬бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбҹ” бһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ бһ“бһ·бһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ”

2. **бһ—бһ¶бһ–бһҜбһҖбһҮбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ**: бһ”бһҫбһҖбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһҠбҹ„бһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ бһҫбһҷбһҖбһ¶бһҡбһ–бһ¶бһҡбһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһҳбһ·бһ“бһҳбһ¶бһ“бһўбһ¶бһҮбҹ’бһүбһ¶бһ”бһҺбҹ’бһҺбҹ”

3. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ§бһ”бһҖбһҡбһҺбҹҚ**: бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһ»бһ“бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбҹ” бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһёбһҳбһҪбһҷбҹ— бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҷбҹүбһ¶бһ„бһҡбһ№бһ„бһҳбһ¶бҹҶбҹ”

4. **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһ§бһ”бһҖбһҡбһҺбҹҚ**: бһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһӮбҹҶбһҡбһјбһ”бҹ’бһҡбһҫбһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ“бҹ…бһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹҗбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бҹ”

5. **бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ**: бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҳбһ»бһ“бһ–бҹҒбһӣбһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ, бһ’бһ“бһ’бһ¶бһ“ бһ¬бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбҹ’бһ„бһҫбһ…бһҹбҹ’бһ„бһ¶бһҸбҹӢ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһјбһ“бһҹбҹ„ API, бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ OAuth бһ¬бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

6. **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҒбһјбһ…бһ¬бһ§бһ”бһ¶бһ“бһҮбһ¶бһ…бҹ„бһҡбһҖбһҳбҹ’бһҳбһІбҹ’бһҷбһҳбһ·бһ“бһ‘бҹ…бһҠбһӣбҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

7. **бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбҹӮбһ„бһҸбҹӮ бһ“бһ·бһ„бһ’бһ¶бһ“бһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҷбҹүбһ¶бһ„бһҹбһҳбһҡбһҳбҹ’бһҷбҹ”
  
### бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ

1. **бһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–**: бһ“бҹ…бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ бһ”бһҳбҹ’бһӣбҹӮбһ„бһ–бһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ, бһҖбҹҶбһҺбҹӮбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶, бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бҹ”

2. **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶ бһ§бһ”бһҖбһҡбһҺбҹҚ**: бһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҡбһҝбһ„бһҳбһҪбһҷбһІбҹ’бһҷбһӣбҹ’бһў бһұбҹ’бһҷбһ бһҪбһҹбһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһӮбҹ’бһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһ“бҹ…бһӣбһҫбһ”бһүбҹ’бһ бһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”

3. **бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһ¶бһҡ бһ“бһ·бһ„бһҖбһјбһҠбһҖбҹҶбһ бһ»бһҹбһҠбҹӮбһӣбһӮбҹ„бһҡбһ–бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ”бһүбҹ’бһ бһ¶, бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷбһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“, бһ“бһ·бһ„бһ•бҹ’бһҠбһӣбҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҠбҹӮбһӣбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бҹ”

4. **бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶**: бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҸбһ¶бһҳбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһҹбҹ„бһ’бһ“бһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бҹ”

5. **бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–**: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҷбһјбһҡбһўбһ„бҹ’бһңбҹӮбһ„ бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҶбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҠбһӣбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”

6. **бһҖбһ¶бһҡбһӣбһ”бҹӢбһ”бҹ„бҹҮбһ”бһ„бҹӢбһҹбҹҶбһҺбһҫ**: бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһ·бһІбҹ’бһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ”бҹ„бҹҮбһ”бһ„бҹӢбһҹбҹҶбһҺбһҫбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһ–бҹҒбһӣбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһҠбҹӮбһӣбһ–бһ»бҹҶбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ¬бһҷбһәбһҸбһ–бҹҒбһҖбҹ”

#### 1. бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҸбҹӮбһҳбһҪбһҷ

бһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһҳбһҪбһҷбҹ—бһӮбһҪбһҡбһҸбҹӮбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ“бһ·бһ„бһ•бҹ’бһҸбҹ„бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ” бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹӮбһ”бһҳбһјбһ“бһјбһӣбһёбһҸбһ·бһ…бһҠбҹӮбһӣбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ…бҹ’бһҡбһҫбһ“бһ‘бҹҒ бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҜбһҖбһ‘бҹҒбһҹбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҡбһҝбһ„бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ”бһ¶бһ“бһӣбҹ’бһўбҹ”

```csharp
// A focused tool that does one thing well
public class WeatherForecastTool : ITool
{
    private readonly IWeatherService _weatherService;
    
    public WeatherForecastTool(IWeatherService weatherService)
    {
        _weatherService = weatherService;
    }
    
    public string Name => "weatherForecast";
    public string Description => "Gets weather forecast for a specific location";
    
    public ToolDefinition GetDefinition()
    {
        return new ToolDefinition
        {
            Name = Name,
            Description = Description,
            Parameters = new Dictionary<string, ParameterDefinition>
            {
                ["location"] = new ParameterDefinition
                {
                    Type = ParameterType.String,
                    Description = "City or location name"
                },
                ["days"] = new ParameterDefinition
                {
                    Type = ParameterType.Integer,
                    Description = "Number of forecast days",
                    Default = 3
                }
            },
            Required = new[] { "location" }
        };
    }
    
    public async Task<ToolResponse> ExecuteAsync(IDictionary<string, object> parameters)
    {
        var location = parameters["location"].ToString();
        var days = parameters.ContainsKey("days") 
            ? Convert.ToInt32(parameters["days"]) 
            : 3;
            
        var forecast = await _weatherService.GetForecastAsync(location, days);
        
        return new ToolResponse
        {
            Content = new List<ContentItem>
            {
                new TextContent(JsonSerializer.Serialize(forecast))
            }
        };
    }
}
```

#### 3. бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ

бһҸбҹӮбһ„бһҸбҹӮбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҷбҹүбһ¶бһ„бһҳбҹүбһҸбҹӢбһ…бһҸбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҒбһ»бһҹбһ¬бһ§бһ”бһ¶бһ“бһ…бҹ„бһҡбһҖбһҳбҹ’бһҳбҹ”

```javascript
class FileOperationTool {
  getName() {
    return "fileOperation";
  }
  
  getDescription() {
    return "Performs file operations like read, write, and delete";
  }
  
  getDefinition() {
    return {
      name: this.getName(),
      description: this.getDescription(),
      parameters: {
        operation: {
          type: "string",
          description: "Operation to perform",
          enum: ["read", "write", "delete"]
        },
        path: {
          type: "string",
          description: "File path (must be within allowed directories)"
        },
        content: {
          type: "string",
          description: "Content to write (only for write operation)",
          optional: true
        }
      },
      required: ["operation", "path"]
    };
  }
  
  async execute(parameters) {
    // 1. бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ
    if (!parameters.operation) {
      throw new ToolError("Missing required parameter: operation");
    }
    
    if (!parameters.path) {
      throw new ToolError("Missing required parameter: path");
    }
    
    // 2. бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ
    if (typeof parameters.operation !== "string") {
      throw new ToolError("Parameter 'operation' must be a string");
    }
    
    if (typeof parameters.path !== "string") {
```

```csharp
public class RateLimitingMiddleware
{
    private readonly RequestDelegate _next;
    private readonly IMemoryCache _cache;
    private readonly ILogger<RateLimitingMiddleware> _logger;
    
    // Configuration options
    private readonly int _maxRequestsPerMinute;
    
    public RateLimitingMiddleware(
        RequestDelegate next,
        IMemoryCache cache,
        ILogger<RateLimitingMiddleware> logger,
        IConfiguration config)
    {
        _next = next;
        _cache = cache;
        _logger = logger;
        _maxRequestsPerMinute = config.GetValue<int>("RateLimit:MaxRequestsPerMinute", 60);
    }
    
    public async Task InvokeAsync(HttpContext context)
    {
        // 1. Get client identifier (API key or user ID)
        string clientId = GetClientIdentifier(context);
        
        // 2. Get rate limiting key for this minute
        string cacheKey = $"rate_limit:{clientId}:{DateTime.UtcNow:yyyyMMddHHmm}";
        
        // 3. Check current request count
        if (!_cache.TryGetValue(cacheKey, out int requestCount))
        {
            requestCount = 0;
        }
        
        // 4. Enforce rate limit
        if (requestCount >= _maxRequestsPerMinute)
        {
            _logger.LogWarning("Rate limit exceeded for client {ClientId}", clientId);
            
            context.Response.StatusCode = StatusCodes.Status429TooManyRequests;
            context.Response.Headers.Add("Retry-After", "60");
            
            await context.Response.WriteAsJsonAsync(new
            {
                error = "Rate limit exceeded",
                message = "Too many requests. Please try again later.",
                retryAfterSeconds = 60
            });
            
            return;
        }
        
        // 5. Increment request count
        _cache.Set(cacheKey, requestCount + 1, TimeSpan.FromMinutes(2));
        
        // 6. Add rate limit headers
        context.Response.Headers.Add("X-RateLimit-Limit", _maxRequestsPerMinute.ToString());
        context.Response.Headers.Add("X-RateLimit-Remaining", (_maxRequestsPerMinute - requestCount - 1).ToString());
        
        // 7. Continue with the request
        await _next(context);
    }
```

```csharp
// C# example with caching
public class CachedWeatherTool : ITool
{
    private readonly IWeatherService _weatherService;
    private readonly IDistributedCache _cache;
    private readonly ILogger<CachedWeatherTool> _logger;
    
    public CachedWeatherTool(
        IWeatherService weatherService,
        IDistributedCache cache,
        ILogger<CachedWeatherTool> logger)
    {
        _weatherService = weatherService;
        _cache = cache;
        _logger = logger;
    }
    
    public string Name => "weatherForecast";
    
    public async Task<ToolResponse> ExecuteAsync(IDictionary<string, object> parameters)
    {
        var location = parameters["location"].ToString();
        var days = Convert.ToInt32(parameters.GetValueOrDefault("days", 3));
        
        // Create cache key
        string cacheKey = $"weather:{location}:{days}";
        
        // Try to get from cache
        string cachedForecast = await _cache.GetStringAsync(cacheKey);
        if (!string.IsNullOrEmpty(cachedForecast))
        {
            _logger.LogInformation("Cache hit for weather forecast: {Location}", location);
            return new ToolResponse
            {
                Content = new List<ContentItem>
                {
                    new TextContent(cachedForecast)
                }
            };
        }
        
        // Cache miss - get from service
        _logger.LogInformation("Cache miss for weather forecast: {Location}", location);
        var forecast = await _weatherService.GetForecastAsync(location, days);
        string forecastJson = JsonSerializer.Serialize(forecast);
        
        // Store in cache (weather forecasts valid for 1 hour)
        await _cache.SetStringAsync(
            cacheKey,
            forecastJson,
            new DistributedCacheEntryOptions
            {
                AbsoluteExpirationRelativeToNow = TimeSpan.FromHours(1)
            });
        
        return new ToolResponse
        {
            Content = new List<ContentItem>
            {
                new TextContent(forecastJson)
            }
        };
    }
}
```

```python
class TokenBucketRateLimiter:
    def __init__(self, tokens_per_second, bucket_size):
        self.tokens_per_second = tokens_per_second
        self.bucket_size = bucket_size
        self.tokens = bucket_size
        self.last_refill = time.time()
        self.lock = asyncio.Lock()
    
    async def get_delay_time(self):
        async with self.lock:
            self._refill()
            if self.tokens >= 1:
                return 0
            
            # бһӮбһҺбһ“бһ¶бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбһӣбҹӢбһ–бҹҒбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹ’бһӣбһ¶бһҖбһҖбһјбһҠбһҳбһҪбһҷбһ…бҹҒбһүбһҳбһҖ
            return (1 - self.tokens) / self.tokens_per_second
    
    async def consume(self):
        async with self.lock:
            self._refill()
            self.tokens -= 1
    
    def _refill(self):
        now = time.time()
        elapsed = now - self.last_refill
        
        # бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбҹ’бһӣбһ¶бһҖбһҖбһјбһҠбһҗбҹ’бһҳбһёбҹ—бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢ
        new_tokens = elapsed * self.tokens_per_second
        self.tokens = min(self.bucket_size, self.tokens + new_tokens)
        self.last_refill = now
```

бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹҶбһ—бһ¶бһҡбҹҲбһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бҹ–

```python
class SecureDataTool(Tool):
    def get_schema(self):
        return {
            "type": "object",
            "properties": {
                "userId": {"type": "string"},
                "includeSensitiveData": {"type": "boolean", "default": False}
            },
            "required": ["userId"]
        }
    
    async def execute_async(self, request):
        user_id = request.parameters["userId"]
        include_sensitive = request.parameters.get("includeSensitiveData", False)
        
        # бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
        user_data = await self.user_service.get_user_data(user_id)
        
        # бһҸбҹ’бһҡбһ„бҹӢбһңбһ¶бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹҶбһ„бһ¶бһҸбҹӢбһҖбҹ’бһҡбҹ…бһҸбҹӮбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ–бҹ’бһҡбһҳбһ‘бһ‘бһҪбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ
        if not include_sensitive or not self._is_authorized_for_sensitive_data(request):
            user_data = self._redact_sensitive_fields(user_data)
        
        return ToolResponse(result=user_data)
    
    def _is_authorized_for_sensitive_data(self, request):
        # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһҳбҹ’бһҡбһ·бһҸбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһ“бҹ…бһ”бһҡбһ·бһ”бһ‘бһҹбҹҶбһҺбһҫбһҡ
        auth_level = request.context.get("authorizationLevel")
        return auth_level == "admin"
    
    def _redact_sensitive_fields(self, user_data):
        # бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһ„бһҳбһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҠбһҫбһҳ
        redacted = user_data.copy()
        
        # бһӣбһ»бһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹҶбһ„бһ¶бһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
        sensitive_fields = ["ssn", "creditCardNumber", "password"]
        for field in sensitive_fields:
            if field in redacted:
                redacted[field] = "REDACTED"
        
        # бһӣбһ»бһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹҶбһ„бһ¶бһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһ„
        if "financialInfo" in redacted:
            redacted["financialInfo"] = {"available": True, "accessRestricted": True}
        
        return redacted
```

бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢ бһ‘бһёбһ“бҹҒбҹҮбһ•бһӣбһ·бһҸбһ•бһӣбһ…бҹҒбһүбһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһҪбһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ–

```python
# бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҒбҹ’бһҹбҹӮбһ§бһ”бһҖбһҡбһҺбҹҚ Python
class ChainWorkflow:
    def __init__(self, tools_chain):
        self.tools_chain = tools_chain  # бһ”бһүбҹ’бһҮбһёбһҲбҹ’бһҳбҹ„бҹҮбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бҹ…бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢ
    
    async def execute(self, mcp_client, initial_input):
        current_result = initial_input
        all_results = {"input": initial_input}
        
        for tool_name in self.tools_chain:
            # бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһёбһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһҹбҹӮ бһҠбҹ„бһҷбһ”бһүбҹ’бһҮбһјбһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ»бһ“
            response = await mcp_client.execute_tool(tool_name, current_result)
            
            # бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһҮбһ¶inputбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ
            all_results[tool_name] = response.result
            current_result = response.result
        
        return {
            "final_result": current_result,
            "all_results": all_results
        }

# бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
data_processing_chain = ChainWorkflow([
    "dataFetch",
    "dataCleaner",
    "dataAnalyzer",
    "dataVisualizer"
])

result = await data_processing_chain.execute(
    mcp_client,
    {"source": "sales_database", "table": "transactions"}
)
```

```csharp
public class ContentDispatcherTool : IMcpTool
{

    public ContentDispatcherTool(IMcpClient mcpClient)
    {
        _mcpClient = mcpClient;
    }
    
    public string Name => "contentProcessor";
    public string Description => "Processes content of various types";
    
    public object GetSchema()
    {
        return new {
            type = "object",
            properties = new {
                content = new { type = "string" },
                contentType = new { 
                    type = "string",
                    enum = new[] { "text", "html", "markdown", "csv", "code" }
                },
                operation = new { 
                    type = "string",
                    enum = new[] { "summarize", "analyze", "extract", "convert" }
                }
            },
            required = new[] { "content", "contentType", "operation" }
        };
    }
    
    public async Task<ToolResponse> ExecuteAsync(ToolRequest request)
    {
        var content = request.Parameters.GetProperty("content").GetString();
        var contentType = request.Parameters.GetProperty("contentType").GetString();
        var operation = request.Parameters.GetProperty("operation").GetString();
        
        // Determine which specialized tool to use
        string targetTool = DetermineTargetTool(contentType, operation);
        
        // Forward to the specialized tool
        var specializedResponse = await _mcpClient.ExecuteToolAsync(
            targetTool,
            new { content, options = GetOptionsForTool(targetTool, operation) }
        );
        
        return new ToolResponse { Result = specializedResponse.Result };
    }
    
    private string DetermineTargetTool(string contentType, string operation)
    {
        return (contentType, operation) switch
        {
            ("text", "summarize") => "textSummarizer",
            ("text", "analyze") => "textAnalyzer",
            ("html", _) => "htmlProcessor",
            ("markdown", _) => "markdownProcessor",
            ("csv", _) => "csvProcessor",
            ("code", _) => "codeAnalyzer",
            _ => throw new ToolExecutionException($"No tool available for {contentType}/{operation}")
        };
    }
    
    private object GetOptionsForTool(string toolName, string operation)
    {
        // Return appropriate options for each specialized tool
        return toolName switch
        {
            "textSummarizer" => new { length = "medium" },
            "htmlProcessor" => new { cleanUp = true, operation },
            // Options for other tools...
            _ => new { }
        };
    }
```

```java
                "method", "pearson"
            ))
        );

                "sensitivity", "medium"

        );

        Map<String, Object> combinedResults = new HashMap<>();

        combinedResults.put("statistics", statisticalAnalysis.join().getResult());
        combinedResults.put("correlations", correlationAnalysis.join().getResult());
        combinedResults.put("outliers", outlierDetection.join().getResult());
        
        // бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 4: бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҹбһ„бҹ’бһҒбҹҒбһ”
        ToolResponse summaryResponse = mcpClient.executeTool("reportGenerator",

        // бһ”бһүбҹ’бһҮбһјбһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһҡбһ»бһ”бһ…бҹҒбһү
        WorkflowResult result = new WorkflowResult();

        result.setAnalysisResults(combinedResults);

    }
}
```

```python
class ResilientWorkflow:
    def __init__(self, mcp_client):
        self.client = mcp_client
    
    async def execute_with_fallback(self, primary_tool, fallback_tool, parameters):
        try:
            # бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ§бһ”бһҖбһҡбһҺбҹҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһҮбһ¶бһҹбһ·бһ“
            response = await self.client.execute_tool(primary_tool, parameters)
            return {
                "result": response.result,
                "source": "primary",
                "tool": primary_tool
            }
        except ToolExecutionException as e:
            # бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷ
            logging.warning(f"Primary tool '{primary_tool}' failed: {str(e)}")
            
            # бһңбһ¶бһӣбҹӮбһ„бһҮбһҪбһҷбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҸбһ”бҹ’бһҡбһҷбҹ„бһӮбһ‘бһёбһ–бһёбһҡ
            try:
                # бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹҶбһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҸбһҮбҹҶбһ“бһҪбһҹ
                fallback_params = self._adapt_parameters(parameters, primary_tool, fallback_tool)
                
                response = await self.client.execute_tool(fallback_tool, fallback_params)
                return {
                    "result": response.result,
                    "source": "fallback",
                    "tool": fallback_tool,
                    "primaryError": str(e)
                }
            except ToolExecutionException as fallback_error:
                # бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ§бһ”бһҖбһҡбһҺбҹҸбһ”бһҡбһ¶бһҮбҹҗбһҷбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„
                logging.error(f"Both primary and fallback tools failed. Fallback error: {str(fallback_error)}")
                raise WorkflowExecutionException(
                    f"Workflow failed: primary error: {str(e)}; fallback error: {str(fallback_error)}"
                )
    
    def _adapt_parameters(self, params, from_tool, to_tool):
        """Adapt parameters between different tools if needed"""
        # бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҒбҹҮбһ“бһ№бһ„бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҸбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
        # бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ, бһҷбһҫбһ„бһ“бһ№бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһ–бһёбһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҠбһҫбһҳбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ
        return params
```

### 5. бһӮбҹҶбһҡбһјбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ

бһҹбһ„бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбҹ„бһҷбһ•бҹ’бһҹбҹҶбһ“бһјбһңбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҷбҹ—бҹ–

```csharp
public class CompositeWorkflow : IWorkflow
{
    private readonly List<IWorkflow> _workflows;
    
    public CompositeWorkflow(IEnumerable<IWorkflow> workflows)
    {
        _workflows = new List<IWorkflow>(workflows);
    }
    
    public async Task<WorkflowResult> ExecuteAsync(WorkflowContext context)
    {
        var results = new Dictionary<string, object>();
        
        foreach (var workflow in _workflows)
        {
            var workflowResult = await workflow.ExecuteAsync(context);
            
            // Store each workflow's result
            results[workflow.Name] = workflowResult;
            
            // Update context with the result for the next workflow
            context = context.WithResult(workflow.Name, workflowResult);
        }
        
        return new WorkflowResult(results);
    }
    
    public string Name => "CompositeWorkflow";
    public string Description => "Executes multiple workflows in sequence";
}

// Example usage
var documentWorkflow = new CompositeWorkflow(new IWorkflow[] {
    new DocumentFetchWorkflow(),
    new DocumentProcessingWorkflow(),
    new InsightGenerationWorkflow(),
    new ReportGenerationWorkflow()
});

var result = await documentWorkflow.ExecuteAsync(new WorkflowContext {
    Parameters = new { documentId = "12345" }
});
```

бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһ”бҹҶбһҡбһҫбһҮбһ¶бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһ“бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮбһӮбҹҶбһҡбһј AI бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҸбһ·бһҗбһ·бһҮбһ“бҹ” бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҷбҹүбһ¶бһ„бһҒбҹ’бһҮбһ¶бһ”бҹӢбһҒбҹ’бһҮбһҪбһ“бһ’бһ¶бһ“бһ¶бҹ–

- бһ—бһ¶бһ–бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһ•бһӣбһ·бһҸ
- бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”
- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ MCP
- бһўбһ¶бһ…бһ’бһ“бҹӢбһ‘бҹ’бһҡбһ¶бҹҶбһ“бһ№бһ„бһҖбһҡбһҺбһёбһ”бһҡбһ¶бһҮбҹҗбһҷ бһ“бһ·бһ„бһҖбһҡбһҺбһёбһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ
- бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–бһҹбҹҠбһёбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳбһ”бһ“бҹ’бһ‘бһ»бһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—

## бһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP

### бһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбһ¶ (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“)

бһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбһ¶ бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ•бҹ’бһ“бҹӮбһҖбһҜбһҖбһҸбҹ’бһҸбһ·бһҳбһҪбһҷбҹ—бһ“бҹғбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбҹ”

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„

1. **бһўбҹ’бһ“бһҖбһ…бһ¶бһҸбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶бһўбҹ’бһ“бһҖбһ…бһ¶бһҸбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹ„бһҷбһЎбҹӮбһҖ
2. **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚ**: бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҳбһҪбһҷбһ”бһүбҹ’бһ…бһјбһӣбһҗбҹӮбһҳбһҸбһ¶бһҡбһ¶бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—
3. **бһ‘бҹҶбһ–бҹҗбһҡбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһ„бҹ’бһ бһ¶бһү**: бһ’бһ¶бһ“бһ¶бһҗбһ¶бһ‘бҹҶбһ–бҹҗбһҡбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
4. **бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһ¶бһҡбһ¶бһ„**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„

3. **бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбҹҶбһҺбһҫ**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҡбҹҶбһӣбһ„бһҹбҹҶбһҺбһҫ-бһ…бһҳбҹ’бһӣбһҫбһҷбһ–бҹҒбһүбһӣбҹҒбһү
4. **бһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһ•бҹ’бһ‘бҹҒбһҡбһҖбҹҶбһ бһ»бһҹ**: бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбҹҶбһ бһ»бһҹбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—
5. **бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹҒбһҖбһ¶бһ“бһ·бһ…бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

3. **бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһҡбҹ„бһҳбһ”бһ“бҹ’бһ‘бһ»бһҖ**: бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҮбһ¶бһҳбһҪбһҷбһҹбҹҶбһҺбһҫбһ…бҹ’бһҡбһҫбһ“бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶

9. **бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ”бһ„бҹ’бһңбһ·бһӣбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһҺбһ»бҹҶбһҸбҹҒбһҹбҹ’бһҸбһҠбҹӮбһӣбһҡбһҸбҹӢбһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһјбһҠбһҡбһ¶бһӣбҹӢбһҠбһ„  
10. **бһ…бһ„бһҖбҹ’бһҡбһ„бһҜбһҖбһҹбһ¶бһҡбһҖбһҡбһҺбһёбһҸбҹҒбһҹбҹ’бһҸ**бҹ– бһҡбһҖбҹ’бһҹбһ¶бһҜбһҖбһҹбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһёбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҸбҹҒбһҹбҹ’бһҸ  

## бһ”бһүбҹ’бһ бһ¶бһ”бҹ’бһҡбһҲбһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ’бһҳбҹ’бһҳбһҸбһ¶

- **бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ•бҹ’бһӣбһјбһңбһҹбһ”бҹ’бһ”бһ¶бһҷбһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…**бҹ– бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸбһҖбһҡбһҺбһёбһҖбҹҶбһ бһ»бһҹбһҷбҹүбһ¶бһ„бһ–бҹҒбһүбһӣбҹҒбһү  
- **бһҳбһ·бһ“бһ”бһ¶бһ“бһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–**бҹ– бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһ“бҹ’бһӣбҹӮбһ„бһҲбһәбһ…бһ¶бһ”бҹӢбһҳбһ»бһ“бһңбһ¶бһҒбһјбһ…бһҒбһ¶бһҸбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ  
- **бһҸбҹҒбһҹбҹ’бһҸбһҠбҹ„бһҷбһҜбһҖбһ¶бһҹбҹ’бһҗбһ¶бһ“бһҸбҹӮбһҳбҹ’бһҠбһ„**бҹ– бһ•бҹ’бһҹбҹҶбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбһ¶, бһҸбҹҒбһҹбҹ’бһҸбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ, бһ“бһ·бһ„бһҸбҹҒбһҹбҹ’бһҸбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ  
- **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ API бһҳбһ·бһ“бһ–бҹҒбһүбһӣбҹҒбһү**бҹ– бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢ бһ“бһ·бһ„бһӣбһҖбҹ’бһҒбһҺбҹҲбһӮбҹ’бһҡбһ”бҹӢбһҶбҹ’бһңбҹҒбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ  
- **бһ”бһҡбһ·бһ”бһ‘бһҸбҹҒбһҹбҹ’бһҸбһҳбһ·бһ“бһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶**бҹ– бһ”бҹ’бһҡбһҫcontainers бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһ”бһҡбһ·бһ”бһ‘бһҸбҹҒбһҹбҹ’бһҸбһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶  

## бһ”бһүбҹ’бһ…бһ”бҹӢ

бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһӮбҹ’бһҡбһ”бҹӢбһӣбҹҶбһҠбһ¶бһ”бҹӢбһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҳбһҪбһҷбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбҹ” бһҠбҹ„бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ“бһ·бһ„бһҷбҹ„бһ”бһӣбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһӣбһҫбһҖбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бһ¶бһ“бһ¶бһ”бһ¶бһ“бһҗбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бҹҶбһ–бҹҒбһүбһҸбһ¶бһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһӮбһ»бһҺбһ—бһ¶бһ–, бһ—бһ¶бһ–бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ, бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҝбһ„бһҠбҹҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбҹ”  

## бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢ

1. **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҸбҹӮбһҳбһҪбһҷ, бһ”бҹ’бһҡбһҫ dependency injection, бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһ¶бһ“бһ—бһ¶бһ–бһ•бҹ’бһҹбҹҶ  
2. **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҹбҹ’бһӮбһёбһҳбҹүбһ¶**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ’бһӮбһёбһҳбҹүбһ¶бһҠбҹӮбһӣбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһӣбһҳбҹ’бһўбһ·бһҸбһҮбһ¶бһҳбһҪбһҷбһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң  
3. **бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбҹҶбһ бһ»бһҹ**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбҹҶбһ бһ»бһҹбһҷбҹүбһ¶бһ„бһҡбһ¶бһ”бһҹбһ¶бһҡ, бһ…бһҳбҹ’бһӣбһҫбһҷбһҖбҹҶбһ бһ»бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’, бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ retry  
4. **бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–**бҹ– бһ”бҹ’бһҡбһҫ caching, бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ¶бһҹбҹҠбһёбһ“бһҒбһҺбҹҲ (asynchronous), бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“  
5. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҷбҹүбһ¶бһ„бһҳбҹүбһҸбҹӢбһ…бһҸбҹӢ, бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ, бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһ”бҹ’бһҡбһүбһ¶бһ”бҹӢбһ–бҹҒбһӣ  
6. **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбһ¶, бһҸбҹҒбһҹбҹ’бһҸбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһҸбҹҒбһҹбҹ’бһҸбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҠбҹ„бһҷбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ  
7. **бһӣбҹҶбһ“бһ¶бҹҶбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡ**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҹбҹ’бһҠбһ„бҹӢбһҠбһ¶бһҡбһҠбһјбһ…бһҮбһ¶ бһҒбҹ’бһҹбҹӮ, dispatchers, бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһҪбһҳбһҹбһҳбһҳбһ¶бһҢ  

## бһ”бҹ’бһҡбһўбһ”бҹӢбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ

бһҡбһ…бһ“бһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһ“бһ·бһ„бһӣбҹҶбһ“бһ¶бҹҶбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҜбһҖбһҹбһ¶бһҡ бһҠбҹӮбһӣбҹ–

1. бһ‘бһ‘бһҪбһӣбһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ…бҹ’бһҡбһҫбһ“ (PDF, DOCX, TXT)  
2. бһ•бҹ’бһ‘бһ»бһҖбһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ–бһёбһҜбһҖбһҹбһ¶бһҡ  
3. бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҜбһҖбһҹбһ¶бһҡбһҸбһ¶бһҳбһ”бҹ’бһҡбһ—бҹҒбһ‘ бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶
