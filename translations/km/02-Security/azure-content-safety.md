# бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– MCP бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷ Azure Content Safety

> **бһ бһ¶бһ“бһ·бһ—бҹҗбһҷ OWASP MCP бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ**бҹ– [MCP06 - бһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ”бҹҶбһҺбһ„](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

Azure Content Safety бһ•бҹ’бһҸбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹҸбһҳбһ¶бһ“бһ—бһ¶бһ–бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһҠбҹғбһҹбҹ’бһ‘бһ¶бһҸбҹӢ бһҹбһјбһҳбһҳбһҫбһӣ [MCP Security Summit Workshop (Sherpa)](https://azure-samples.github.io/sherpa/) бһ бһ¶бһҡбһёвҖӢбһ‘бһё 3бҹ– бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– I/Oбҹ”

## бһӮбһҳбҹ’бһӣбһ¶бһҸ Prompt

бһӮбһҳбҹ’бһӣбһ¶бһҸ AI Prompt бһҡбһ”бһҹбҹӢ Microsoft бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҠбҹҸбһҳбһ¶бҹҶбһҳбһҪбһ“бһ‘бһӣбҹӢбһ“бһ№бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһ“бһ·бһ„бһӮбһ“бҹ’бһӣбһ„бһ”бһүбҹ’бһ…бһјбһӣбһ”бҹҶбһҺбһ„бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҸбһ¶бһҳбһҡбһҷбҹҲбҹ–

1. **бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҳбһ¶бһ“бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҠбҹӮбһӣбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”
2. **бһҖбһ¶бһҡбһ”бҹҶбһ—бҹ’бһӣбһә**бҹ– бһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһүбҹ’бһ…бһјбһӣбһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһ…бһ¶бһҸбҹӢбһ‘бһ»бһҖбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһ–бһёбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ”
3. **бһҹбһүбҹ’бһүбһ¶бһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“ бһ“бһ·бһ„бһҹбҹ’бһ“бһ¶бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ**бҹ– бһҖбҹҶбһҺбһҸбҹӢбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһҡбһңбһ¶бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһҳбһ·бһ“бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбҹ”
4. **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣ Content Safety**бҹ– бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ Azure AI Content Safety бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһғбһҫбһүбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ—бһҖбҹӢбһ…бҹҒбһүбһ–бһёбһҖбһ„бҹ’бһңбһӣбҹӢ бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбҹ”
5. **бһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ**бҹ– Microsoft бһ”бһ“бҹ’бһҸбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҳбһ’бҹ’бһҷбҹ„бһ”бһ¶бһҷбһҖбһ¶бһҡбһ–бһ¶бһҡбһ‘бһӣбҹӢбһ“бһ№бһ„бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҖбһҫбһҸбһЎбһҫбһ„бһҗбҹ’бһҳбһёбҹ—бҹ”

## бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Azure Content Safety бһҮбһ¶бһҳбһҪбһҷ MCP

бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҮбһ¶бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ…бҹ’бһҡбһҫбһ“бҹ–
- бһҹбҹ’бһҖбҹҒбһ“бһ”бһүбҹ’бһ…бһјбһӣбһҳбһ»бһ“бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ…бҹҒбһүбһҳбһ»бһ“бһ–бҹҒбһӣбһҸбҹ’бһҡбһЎбһ”бҹӢ
- бһ”бҹ’бһҡбһҫбһ”бһүбҹ’бһҮбһёбһҡбһ¶бҹҶбһ„бһҒбҹ’бһҹбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ“бһ¶бҹҶбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһӮбһ·бһҸбһғбһҫбһү
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹүбһјбһҢбҹӮбһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһ”бһҹбҹӢ Azure бһҠбҹӮбһӣбһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ

## бһ’бһ“бһ’бһ¶бһ“ Azure Content Safety

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Azure Content Safety бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹүбҹ„бһӣ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ’бһ“бһ’бһ¶бһ“бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–

1. [бһҜбһҖбһҹбһ¶бһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ Azure AI Content Safety](https://learn.microsoft.com/azure/ai-services/content-safety/) - бһҜбһҖбһҹбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Azure Content Safetyбҹ”
2. [бһҜбһҖбһҹбһ¶бһҡбһӮбһҳбҹ’бһӣбһ¶бһҸ Prompt](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/prompt-shield) - бһҡбҹҖбһ“бһ–бһёбһҡбһ”бҹҖбһ”бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһ”бҹҶбһҺбһ„бҹ”
3. [бһҷбҹ„бһ„ API Content Safety](https://learn.microsoft.com/rest/api/contentsafety/) - бһҷбҹ„бһ„ API бһӣбһҳбҹ’бһўбһ·бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸ Content Safetyбҹ”
4. [Quickstart: Azure Content Safety бһҮбһ¶бһҳбһҪбһҷ C#](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-csharp) - бһҳбһ¶бһӮбҹ’бһңбһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҡбһ бҹҗбһҹбһҮбһ¶бһҳбһҪбһҷ C#бҹ”
5. [бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһўбҹ’бһ“бһҖбһўбһҸбһ·бһҗбһ·бһҮбһ“ Content Safety](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-client-libraries-rest-api) - бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
6. [бһҡбһҖбһғбһҫбһүбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ—бһҖбҹӢбһ…бҹҒбһүбһ–бһёбһҖбһ„бҹ’бһңбһӣбҹӢ](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/jailbreak-detection) - бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбһғбһҫбһүбһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ—бһҖбҹӢбһ…бҹҒбһүбҹ”
7. [бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Content Safety](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/best-practices) - бһ•бҹӮбһ“бһҖбһ¶бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбҹ’бһҡбҹ…бһҮбһ¶бһ„бһ“бҹҒбҹҮ бһҹбһјбһҳбһҳбһҫбһӣ [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ Azure Content Safety](./azure-content-safety-implementation.md)бҹ”
