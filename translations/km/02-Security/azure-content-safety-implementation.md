# бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бһ¶бһ–бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶ Azure бһҮбһ¶бһҳбһҪбһҷ MCP

> **бһ бһ¶бһ“бһ·бһ—бҹҗбһҷ OWASP MCP бһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ**: [MCP06 - Intent Flow Subversion](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– MCP бһҖбҹҶбһ–бһёбһҖбһ¶бһҡбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ„бҹ’бһ бһ¶бһү (prompt injection), бһҖбһ¶бһҡбһ”бҹҶбһ–бһ»бһӣбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһҹбҹ„бһҷбһ–бһ·бһҹбҹҒбһҹ AI бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure Content Safety бһӮбһәбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбҹ” бһҳбҹҒбһҡбҹҖбһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһ¶бһҳбһҪбһҷ [бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбһӮбҹ’бһӮбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– MCP Security Summit Workshop (Sherpa)](https://azure-samples.github.io/sherpa/) Tab 3: бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– I/Oбҹ”

## бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһ»бһ„ MCP

бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure Content Safety бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһ»бһ„ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҹбһјбһҳбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбҹүбһјбһҢбһ»бһӣбһ бҹ’бһңбҹҠбһёбһӣбһ‘бһҪбһҡбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбһҺбҹ’бһҸбһ¶бһӣбһҖбҹ’бһ“бһ»бһ„бһҹбһ„бҹӢбһ…бһ„бһҒбҹ’бһҹбҹӮбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбҹ–

1. бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹүбһјбһҢбһ»бһӣбһ бҹ’бһңбҹҠбһёбһӣбһ“бҹ…бһ–бҹҒбһӣбһ•бҹ’бһҠбһ¶бһ…бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһ»бһ„
2. бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбҹҶбһҺбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһјбһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһ»бһ“бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
3. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ…бһҳбҹ’бһӣбһҫбһҷбһҡбһ¶бһӣбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһүбҹ’бһҮбһјбһ“бһ”бһ„бҹ’бһңбһ·бһӣбһ‘бҹ…бһўбһҸбһ·бһҗбһ·бһҮбһ“
4. бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһ“бһ·бһ„бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҡбҹҶбһӣбҹ„бһ—бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
5. бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһҳбҹ’бһ–бһҹбҹӢбһҖбҹҶбһ бһ»бһҹбһҹбһҳбһҡбһҳбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһ”бһҡбһ¶бһҮбҹҗбһҷ

бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбһ¶бһҡбһ–бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ№бһ„бһҳбһ¶бҹҶбһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһ“бһ№бһ„бҹ–
- бһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ„бҹ’бһ бһ¶бһү
- бһҖбһ¶бһҡбһ”бҹҶбһ–бһ»бһӣбһ§бһ”бһҖбһҡбһҺбҹҚ
- бһҖбһ¶бһҡбһ…бҹҒбһүбһҗбҹ’бһ“бһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бһүбҹ’бһ…бһјбһӣбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢ
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһўбһ¶бһҖбҹ’бһҡбһҖбҹӢ

## бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure Content Safety

1. **бһ”бһүбҹ’бһҮбһёбһҲбһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“**: бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһүбҹ’бһҮбһёбһҲбһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹҶбһҡбһјбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ MCP
2. **бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡ**: бһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡбһҸбһ¶бһҳбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ—бһ¶бһ–бһӣбҹӮбһ„бһҹбҹ’бһҡбһ”
3. **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бҹ…бһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһјбһӣ бһ“бһ·бһ„бһ…бҹҒбһүбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
4. **бһ”бһ„бҹ’бһңбһ№бһҖбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–**: бһӮбһ·бһҸбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (cache) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҳбҹ’бһҠбһ„бһҳбҹ’бһҖбһ¶бһӣбһЎбһҫбһ„бһңбһ·бһү
5. **бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ’бҹ’бһӣбһ¶бһҖбҹӢбһҒбҹ’бһӣбһҪбһ“**: бһҖбҹҶбһҺбһҸбҹӢбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ”бһүбҹ’бһ…бһ¶бһҖбҹӢбһ‘бҹ’бһ”бһ¶бһ“бһ–бҹҒбһӣбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“
6. **бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ**: бһ•бҹ’бһҸбһӣбҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҠбһӣбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ–бҹҒбһӣбһҠбҹӮбһӣбһҳбһ¶бһҸбһ·бһҖбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ¶бһҡбһ¶бҹҶбһ„бһҠбҹ„бһҷбһҹбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҖбһ¶бһҡбһ–бһ¶бһҡ
7. **бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ”бһ“бҹ’бһҠбһҡбһ”бһҹбҹӢбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶ**: бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһ”бһүбҹ’бһҮбһёбһҲбһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһӮбҹҶбһҡбһјбһҸбһ¶бһҳбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһҗбҹ’бһҳбһёбҹ—
