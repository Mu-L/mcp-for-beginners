# Model Context Protocol (MCP) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ - бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бһҹбһ·бһҖбҹ’бһҹбһ¶

бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҮбһ¶бһҹбһ„бҹ’бһҒбҹҒбһ”бһўбҹҶбһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһ“бҹғбһғбҹ’бһӣбһ¶бҹҶбһ„бһҖбһјбһҠвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ·бһҖбҹ’бһҹбһ¶ "Model Context Protocol (MCP) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ"бҹ” бһ”бҹ’бһҡбһҫбһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ»бһҖбһҡбһҖбһғбҹ’бһӣбһ¶бҹҶбһ„бһҖбһјбһҠбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһёбһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ”

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”бһғбҹ’бһӣбһ¶бҹҶбһ„бһҖбһјбһҠ

Model Context Protocol (MCP) бһӮбһәбһҮбһ¶бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҡбһ·бһ”бһ‘бһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡвҖӢбһ…бһҳбҹ’бһ”бһ„бһҡбһңбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣ AI бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһ‘бһҪбһӣбҹ” бһҠбһҫбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ Anthropic, MCP бһҘбһЎбһјбһңбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһҹбһ бһӮбһҳбһ“бҹҚ MCP бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһ„бҹ’бһӮбһ—бһ¶бһ– GitHub бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҳбһҪбһҷбҹ” бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһ–бҹҒбһүбһӣбҹҒбһүбһҮбһ¶бһҳбһҪбһҷбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹ„бһҷбһҠбҹғбһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶ C#, Java, JavaScript, Python, бһ“бһ·бһ„ TypeScript бһҠбҹӮбһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ AI, бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ–бһ бһ»бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’, бһ“бһ·бһ„бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ•бҹ’бһ“бҹӮбһҖбһ‘бһ“бҹӢбҹ”

## бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһғбҹ’бһӣбһ¶бҹҶбһ„бһҖбһјбһҠ

бһғбҹ’бһӣбһ¶бҹҶбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһ”бҹҶбһ”бҹӮбһҖбһ…бҹҒбһүбһҮбһ¶бҹЎбҹўбһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— бһҠбҹӮбһӣбһҳбһҪбһҷбҹ—бһ•бҹ’бһҸбҹ„бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӣбһҫбһ•бҹ’бһ“бҹӮбһҖбһҒбһ»бһҹбҹ—бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCPбҹ–

1. **бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ (00-Introduction/)**
   - бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҮбһјбһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бһјбһ‘бҹ…бһўбҹҶбһ–бһё Model Context Protocol
   - бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҠбҹӮбһӣбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ„бһ¶бһҡ AI
   - бһӮбҹҶбһҡбһјбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ

2. **бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (01-CoreConcepts/)**
   - бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳ Client-server
   - бһҹбһҳбһ¶бһҹбһ—бһ¶бһӮбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҷбһ¶бһ“ MCP
   - бһӣбҹҶбһ“бһ¶бҹҶбһҹбһ¶бһҡвҖӢбһ•бҹ’бһ‘бҹҒбһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCP

3. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (02-Security/)**
   - бһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбһ¶бһҳбһӣбҹҶбһ“бһ¶бҹҶ MCP
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ
   -бһңбһёбһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһўбһ¶бһҮбҹ’бһүбһ¶бһ”бҹҗбһҺбҹ’бһҺ
   - **бһҜбһҖбһҹбһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ–бҹҒбһүбһӣбҹҒбһү**:
     - MCP Security Best Practices 2025
     - Azure Content Safety Implementation Guide
     - MCP Security Controls and Techniques
     - MCP Best Practices Quick Reference
   - **бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—**:
     - бһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ¶бһҡ (prompt injection) бһ“бһ·бһ„бһҮбһёбһңбһ·бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ»бһӣ
     - бһ”бһүбҹ’бһ бһ¶бһҹбһ¶бһҡбһҒбҹ’бһӣбһҪбһ“бһЎбһҫбһ„бһңбҹүбһ¶бһ…бһ¶бһ”бҹӢбһ‘бһ»бһҖ (session hijacking) бһ“бһ·бһ„бһ“бҹ…бһ…бҹ’бһҡбһ”бһ¶бһ…бҹӢ (confused deputy)
     - бһ”бһүбҹ’бһ бһ¶бһҮбһјбһ“бһҹбһүбҹ’бһүбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ (token passthrough)
     - бһўбҹҶбһҺбһ¶бһ…бһӣбһҫбһҹ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫ
     - бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһҹбҹӮбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһҝбһ„бһ•бҹ’бһҹбһ¶бһҡбһ—бһ¶бһӮ AI
     - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Microsoft Prompt Shields

4. **бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (03-GettingStarted/)**
   - бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
   - бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҗбһҸбһҖбҹ’бһҡбҹ…
   - бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҡбһ¶бһ”бҹӢ
   - бһҡбһҪбһҳбһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ:
     - бһҖбһ¶бһҡвҖӢбһўбһ“бһ»бһңбһҸбҹ’бһҸвҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһҳбҹҒбһҠбҹҶбһ”бһјбһ„
     - бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҗбһҸбһҖбҹ’бһҡбҹ…
     - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҗбһҸ LLM
     - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ VS Code
     - бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ•бҹ’бһүбһҫбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ
     - бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
     - бһҖбһ¶бһҡбһ•бҹ’бһҸбһҪбһ…бһ•бҹ’бһҸбһҫбһҳ HTTP streaming
     - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI Toolkit
     - бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
     - бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ

5. **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ (04-PracticalImplementation/)**
   - бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ SDK бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһ…бһҳбҹ’бһҡбһ»бҹҮ
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ
   - бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣ prompt бһ“бһ·бһ„ workflow бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸ
   - бһӮбҹҶбһҡбһјбһӮбһҳбҹ’бһҡбҹ„бһ„бһҮбһ¶бһҳбһҪбһҷбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ

6. **бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһңбһҢбҹ’бһҚбһ“бҹҲ (05-AdvancedTopics/)**
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһҡбһ·бһ”бһ‘ (context engineering)
   - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ agent Foundry
   - workflow AI бһ”бҹ’бһҡбһҫбһҡбһ”бҹҖбһ”бһҳбһјбһӣбһ»бһҷбһ‘бҹ’бһҡбһёбһҳбҹүбһјбһҠ
   - бһҠбҹҒбһҳбһјбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ OAuth2
   - бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҳбһҪбһҷ
   - бһ”бҹҠбһ·бһ…бһҳбҹүбһјбһҸбһ“бҹ…бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҳбһҪбһҷ
   - бһўбһ“бһ»бһңбһҸбҹ’бһҸ context бһҠбһҫбһҳ
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһүбҹ’бһҮбһјбһ“
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӮбҹҶбһҡбһјбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ–бһ„бҹ’бһҡбһёбһҖ
   - бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
   - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– Entra ID
   - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҸбһ¶бһҳбһңбҹҒбһ”
   - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ“бҹ’бһ‘бһҖбһҡбһ–бһ бһ»бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһӮбҹҶбһ“бһ·бһҸбһҮбһҳбҹ’бһӣбһҫбһҷбһ”бҹ’бһҡбһҲбһҳ (adversarial multi-agent reasoning)

7. **бһңбҹҒбһ‘бһ·бһҖбһ¶бһҹбһ бһӮбһҳбһ“бҹҚ (06-CommunityContributions/)**
   - бһҳбһ’бҹ’бһҷбҹ„бһ”бһ¶бһҷбһ…бһјбһӣбһҡбһҪбһҳбһҖбһјбһҠбһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡ
   - бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ GitHub
   - бһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһ“бһ·бһ„бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҠбҹ„бһҷбһҹбһ бһӮбһҳбһ“бҹҚ
   - бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ MCP clients бһ…бһҳбҹ’бһҡбһ»бҹҮ (Claude Desktop, Cline, VSCode)
   - бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ–бҹҒбһүбһ“бһ·бһҷбһҳ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–

8. **бһҳбҹҒбһҡбҹҖбһ“бһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹҶбһ”бһјбһ„ (07-LessonsfromEarlyAdoption/)**
   - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бһ·бһҸ бһ“бһ·бһ„бһҡбһҝбһ„бһҮбҹ„бһӮбһҮбҹҗбһҷ
   - бһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһ“бһ·бһ„бһҠбһ¶бһҖбҹӢ MCP бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢ
   - бһҹбҹ’бһ‘бһёбһӣбһ“бһ¶бһ“бһ¶бһ“бһ·бһ„бһ•бҹӮбһ“бһ‘бһёбһўбһ“бһ¶бһӮбһҸ
   - **бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бһҹбҹҖбһңбһ—бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ Microsoft MCP**: бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бһҹбҹҖбһңбһ—бҹ…бһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ Microsoft MCP 10 бһ”бҹ’бһҡбһ—бҹҒбһ‘вҖӢбһҠбһјбһ…бһҮбһ¶:
     - Microsoft Learn Docs MCP Server
     - Azure MCP Server (15+ connectors бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҒбҹ’бһ–бһҹбҹӢ)

9. **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ (08-BestPractices/)**
   - бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһ“бһ·бһ„бһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһ„бҹӢ
   - бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ MCP бһҳбһ¶бһ“бһ—бһ¶бһ–бһ’бһ“бҹӢбһ“бһ№бһ„бһҖбҹҶбһ бһ»бһҹ
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ“бһ·бһ„бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ

10. **бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶ (09-CaseStudy/)**
    - **бһ”бҹ’бһҡбһ¶бҹҶбһ–бһёбһҡбһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бҹҒбһүбһӣбҹҒбһү** бһ”бһ„бҹ’бһ бһ¶бһүбһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“бһҡбһ”бһҹбҹӢ MCP бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶:
    - **Azure AI Travel Agents**: бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһ¶бһҳбһҪбһҷ multi-agent бһҮбһ¶бһҳбһҪбһҷ Azure OpenAI бһ“бһ·бһ„ AI Search
    - **бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure DevOps**: бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһўбһјбһҸбһјбһҳбҹүбһ¶бһ‘бһ·бһҖбһ“бҹғWorkflow бһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ YouTube
    - **бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶**: бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҗбһҸ Python console бһ”бҹ’бһҡбһҫ HTTP streaming
    - **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶**: бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһҺбҹ’бһҠбһ¶бһү Chainlit бһҮбһ¶бһҳбһҪбһҷ AI бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶
    - **бһҜбһҖбһҹбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡ**: бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ VS Code бһҮбһ¶бһҳбһҪбһҷ GitHub Copilot workflows
    - **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Azure API**: бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ API бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
    - **GitHub MCP Registry**: бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһҡбһ·бһҷбһ¶бһ”бҹҗбһҺбһ“бһ·бһ„бһңбҹҒбһ‘бһ·бһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
    - бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ•бһӣбһ·бһҸбһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ

11. **бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ (10-StreamliningAIWorkflowsBuildingAnMCPServerWithAIToolkit/)**
    - бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ MCP бһ“бһ·бһ„ AI Toolkit
    - бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ
    - бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣвҖӢбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бһӣбһ·бһҸ
    - **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ**:
      - Lab 1: бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
      - Lab 2: бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢ
      - Lab 3: бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI Toolkit
      - Lab 4: бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһ“бһ·бһ„бһ–бһ„бҹ’бһҡбһёбһҖ
    - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҮбһ¶бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бҹ—

12. **бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP (11-MCPServerHandsOnLabs/)**
    - **бһ•бҹ’бһӣбһјбһңбһҹбһ·бһҖбҹ’бһҹбһ¶ 13-бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ–бҹҒбһүбһӣбҹҒбһү** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бһӣбһ·бһҸбһҠбҹ„бһҷбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ PostgreSQL
    - **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһӣбһҖбҹӢбһ“бҹ…бһ–бҹҒбһӣбһ–бһ·бһҸ** бһ”бҹ’бһҡбһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ‘бҹҶбһ“бһ·бһү Zava Retail
    - **бһӣбҹҶбһ“бһ¶бҹҶбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ** бһҡбһҪбһҳбһҳбһ¶бһ“ Row Level Security (RLS), бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢ бһ“бһ·бһ„бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһ бһ»бһўбһҸбһ·бһҗбһ·бһҮбһ“
    - **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ–бҹҒбһүбһӣбҹҒбһү**:
      - **Labs 00-03: бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“** - бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ, бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳ, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–, бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ
      - **Labs 04-06: бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP** - бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ, бһҮбҹҶбһ“бһҪбһҷбһ”бҹ’бһӣбһ„бҹӢ
      - **Labs 07-09: бһӣбһҖбҹ’бһҒбһҺбҹҲбһҒбҹ’бһ–бһҹбҹӢ** - бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢ, бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ, бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ VS Code
      - **Labs 10-12: бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ** - бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮ, бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ, бһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһ„бҹӢ
    - **бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ**: FastMCP framework, PostgreSQL, Azure OpenAI, Azure Container Apps, Application Insights
    - **бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶**: бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ•бһӣбһ·бһҸбһ”бһ¶бһ“, бһӣбҹҶбһ“бһ¶бҹҶбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһңбһ·бһ—бһ¶бһӮбһҠбҹ„бһҷ AI, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

13. **бһ§бһ”бһҖбһҡбһҺбҹҚ (12-tooling/)**
    - бһҡбҹҖбһ“бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫ MCP бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Copilot бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—

## бһ’бһ“бһ’бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ

бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ’бһ“бһ’бһ¶бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ–

- **бһҗбһҸбһҡбһјбһ”бһ—бһ¶бһ–**бҹ– бһҳбһ¶бһ“бһӮбҹҶбһ“бһјбһҡбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһ–бһҺбҹҢбһ”бҹҶбһ•бһ»бһҸбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶
- **бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ**бҹ– бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
- **бһ’бһ“бһ’бһ¶бһ“ MCP бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ**:
  - [бһҜбһҖбһҹбһ¶бһҡ MCP](https://modelcontextprotocol.io/)
  - [бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ MCP](https://spec.modelcontextprotocol.io/)
  - [бһғбҹ’бһӣбһ¶бҹҶбһ„бһҖбһјбһҠ MCP GitHub](https://github.com/modelcontextprotocol)

## бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮ

1. **бһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһӣбҹҶбһҠбһ¶бһ”бҹӢ**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҮбҹҶбһ–бһјбһҖбһҠбҹ„бһҷбһӣбҹҶбһҠбһ¶бһ”бҹӢ (00 бһҠбһӣбҹӢ 11) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶бһҹбһҳбҹ’бһҡбһҪбһӣ
2. **бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҺбһ¶бһҳбһҪбһҷ бһ—бһ¶бһҹбһ¶бһ•бҹ’бһҹбҹҶбһЎбһҫбһ„бһӮбҹҶбһҡбһјбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹғбһ—бһ¶бһҹбһ¶бһ“бҹ„бҹҮ
3. **бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһҖбһ¶бһҡбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**бҹ– бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһ•бҹ’бһ“бҹӮбһҖ "Getting Started" бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҗбһҸ MCP бһҠбҹҶбһ”бһјбһ„
4. **бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһӣбһјбһ“ бһҹбһјбһҳбһҮбҹ’бһҡбҹҒбһёбһҹбҹ’бһҡбһ¶бһӣбһ‘бҹ…бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһңбһҢбҹ’бһҚбһ“бҹҲбһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ„бҹ’бһҡбһёбһҖбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„
5. **бһ…бһјбһӣбһҡбһҪбһҳбһҹбһ бһӮбһҳбһ“бҹҚ**бҹ– бһ…бһјбһӣбһҡбһҪбһҳбһҹбһ бһӮбһҳбһ“бҹҚ MCP бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһ“бҹ… GitHub бһ“бһ·бһ„ Discord бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ

## бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҗбһҸ MCP бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚ

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҗбһҸ MCP бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһҳбҹ’бһҡбһ»бҹҮбҹ–
