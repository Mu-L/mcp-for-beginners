# Model Context Protocol (MCP) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ - бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶

бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹғбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһ”бһҹбҹӢд»“еә“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶ "Model Context Protocol (MCP) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ"бҹ” бһ”бҹ’бһҡбһҫбһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ»бһҖбһҡбһҖд»“еә“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһҷбһҖбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһёбһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһҡбһ”бһҹбҹӢд»“еә“

Model Context Protocol (MCP) бһӮбһәбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҸбҹҶбһӣбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҡбһңбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣ AI бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“бҹ” бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһ”бһјбһ„бһҠбҹ„бһҷ Anthropic, MCP бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһҹбһ бһӮбһҳбһ“бҹҚ MCP бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһӣбһҫ GitHubбҹ”д»“еә“бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһҳбҹ’бһҡбҹ…бһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһҠбҹӮбһӣбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶ C#, Java, JavaScript, Python бһ“бһ·бһ„ TypeScript бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ AI, бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ•бҹ’бһ“бҹӮбһҖбһ‘бһ“бҹӢ бҹ”

## бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’д»“еә“

д»“еә“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһҪбһ“бһҠбһҫбһҳ бһҮбҹ’бһҡбһ¶бһ”бһ–бһ¶бһҺбһ·бһҮбҹ’бһҮбһҖбһҳбҹ’бһҳбһҠбһӣбҹӢбһңбһ¶бһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹғ MCPбҹ–

1. **бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ (00-Introduction/)**
   - бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹғ Model Context Protocol
   - бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҠбҹӮбһӣбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһҹбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбҹ’бһ“бһҖбһҶбҹ’бһӣбһҫбһҷ AI
   - бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ

2. **бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (01-CoreConcepts/)**
   - бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһўбһҸбһ·бһҗбһ·бһҮбһ“-бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶
   - бһ’бһ¶бһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ protocol
   - бһӮбҹҶбһҡбһјбһҹбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCP

3. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (02-Security/)**
   - бһҗбһ¶бһҳбһ–бһӣбһӮбҹҶбһҡбһ¶бһҳбһҖбҹҶбһ бҹӮбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ
   - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ бһ“бһ·бһ„бһўбһ¶бһ‘бһ·бһ—бһ¶бһ–
   - **бһҜбһҖбһҹбһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бһ·бһҷбһҳ**бҹ–
     - MCP бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ”жңҖдҪі2025
     - бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһұбҹ’бһҷбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶ Azure
     - бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– MCP
     - бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ“бҹғбһҸбһ·бһ…бһ“бһ·бһҖбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
   - **бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—**бҹ–
     - бһҖбһ¶бһҡбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ”бһүбҹ’бһҮбһ¶ бһ“бһ·бһ„бһӮбҹҶбһҡбһ¶бһҳбһҖбҹҶбһ бҹӮбһ„бһҹбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚ
     - бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһңбҹҒбһҷбҹҗбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ“бһ·бһ„бһ”бһүбҹ’бһ бһ¶бһӮбһҪбһҡбһұбҹ’бһҷбһ…бҹ’бһҡбһ”бһјбһҖбһ…бҹ’бһҡбһ”бһӣбҹӢ
     - бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһүбҹ’бһүбһ¶бһ”бҹҗбһҺбҹ’бһҺбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҸбҹӢ
     - бһҹбһ·бһ‘бҹ’бһ’бһ·бһ–бҹҒбһҖбһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫ
     - бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһҹбҹӮбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢ AI
     - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Microsoft Prompt Shields

4. **бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (03-GettingStarted/)**

   - бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶ MCP бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“
   - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
   - бһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–
     - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбҹҶбһ”бһјбһ„
     - бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһўбһҸбһ·бһҗбһ·бһҮбһ“
     - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбһ·бһҗбһ·бһҮбһ“ LLM
     - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ VS Code

     - бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
     - бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҡбһјбһ”бһ—бһ¶бһ– HTTP streaming
     - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI Toolkit
     - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
     - бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ

5. **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ (04-PracticalImplementation/)**
   - бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ SDKs бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—
   - бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһ бһ»бһҹ, бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
   - бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһј бһ“бһ·бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһ”бһ¶бһ“
   - бһӮбһҳбҹ’бһҡбҹ„бһ„бһӮбҹҶбһҡбһјбһҮбһ¶бһҳбһҪбһҷбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ

6. **бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ (05-AdvancedTopics/)**
   - бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸ context
   - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ Foundry
   - бһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһўбһ¶бһҮбһёбһ– AI бһ–бһ бһ»бһҡбһ”бҹҖбһ”
   - бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ OAuth2
   - бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
   - бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
   - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ context бһҠбһҫбһҳ
   - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ
   - бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҸбҹҶбһӣбҹғбһ•бҹ’бһҹбҹҒбһ„бҹ—
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҶбһ бҹҶ
   - бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
   - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– Entra ID
   - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһҺбҹ’бһҠбһ¶бһү
   - бһҖбһ¶бһҡбһӮбһ·бһҸбһҗбҹ’бһӣбҹӮбһ„бһҳбһҸбһ·бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ agent (бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҷбҹ„бһ”бһӣбҹӢ)

7. **бһҖбһ¶бһҡбһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһҹбһ бһӮбһҳбһ“бҹҚ (06-CommunityContributions/)**
   - бһҡбһ”бҹҖбһ”бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҖбһјбһҠ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡ
   - бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ GitHub бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ
   - бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһў бһ“бһ·бһ„бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһ–бһёбһҹбһ бһӮбһҳбһ“бҹҚ
   - бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“ MCP бһ•бҹ’бһҹбҹҒбһ„бҹ— (Claude Desktop, Cline, VSCode)
   - бһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶ MCP бһ–бҹҒбһүбһ“бһ·бһҷбһҳ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–

8. **бһҳбҹҒбһҡбҹҖбһ“бһ–бһёбһҖбһ¶бһҡбһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһҠбҹҶбһ”бһјбһ„ (07-LessonsfromEarlyAdoption/)**
   - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһ“бһ·бһ„бһҡбһҝбһ„бһҮбҹ„бһӮбһҮбҹҗбһҷ
   - бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP
   - бһ‘бһ·бһҹбһҠбҹ… бһ“бһ·бһ„бһ•бҹӮбһ“бһ‘бһёбһўбһ“бһ¶бһӮбһҸ
   - **бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶ Microsoft MCP**бҹ– бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҮбһҳбҹ’бһҡбҹ…бһҠбһӣбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶ Microsoft MCP бҹЎбҹ  бһҷбҹүбһ¶бһ„бһҡбһҪбһҳбһҳбһ¶бһ“бҹ–
     - Microsoft Learn Docs MCP Server
     - Azure MCP Server (бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҮбһ¶бһ„ бҹЎбҹҘ)
     - GitHub MCP Server
     - Azure DevOps MCP Server
     - MarkItDown MCP Server
     - SQL Server MCP Server
     - Playwright MCP Server
     - Dev Box MCP Server
     - Microsoft Foundry MCP Server

9. **бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ (08-BestPractices/)**
   - бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһ”бһҳбҹ’бһҡбҹӮбһ”бһҳбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
   - бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ MCP бһҠбҹӮбһӣбһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһ—бһ¶бһ–бһҒбһјбһ…бһҒбһ¶бһҸ
   - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҡбһ№бһ„бһҳбһ¶бҹҶ

10. **бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶ (09-CaseStudy/)**
    - **бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһ…бҹҶбһ“бһҪбһ“бһ”бҹ’бһҡбһ¶бҹҶбһ–бһёбһҡбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһҫбһ“бһҒбһ¶бһ„ MCP бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ•бҹ’бһҹбҹҒбһ„бҹ—**бҹ–
    - **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ–бһ бһ»бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ Azure AI**бҹ– бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһ¶бһҳбһҪбһҷ Azure OpenAI бһ“бһ·бһ„ AI Search
    - **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure DevOps**бҹ– бһҖбһ¶бһҡбһҮбһҪбһүбһҠбһјбһҡ бһ’бҹ’бһңбһҫбһҸбһҳбҹ’бһӣбһҫбһ„бһҸбһ¶бһҳ YouTube
    - **бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷ**бҹ– бһўбһҸбһ·бһҗбһ·бһҮбһ“ Python console бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҡбһјбһ”бһ—бһ¶бһ– HTTP
    - **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ•бҹ’бһӣбһјбһңбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„**бҹ– бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Chainlit бһҮбһ¶бһҳбһҪбһҷ AI бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶
    - **бһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠ VS Code**бҹ– бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ GitHub Copilot workflows
    - **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ API Azure**бҹ– бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ API бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ MCP server
    - **бһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһё GitHub MCP Registry**бҹ– бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһңбҹҒбһ‘бһ·бһҖбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
    - бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ бһ—бһ¶бһ–бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҹбһ бһӮбһҳбһ“бҹҚ

11. **бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһўбһ“бһ»бһңбһҸбҹ’бһҸ (10-StreamliningAIWorkflowsBuildingAnMCPServerWithAIToolkit/)**
    - бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ MCP бһ“бһ·бһ„ AI Toolkit
    - бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбһ–бһ·бһҸ
    - бһҳбһјбһҢбһ»бһӣбһ”бҹ’бһҡбһ¶бһҖбҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ–бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ‘бҹ…бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ
    - **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ**бҹ–
      - бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ бҹЎбҹ– бһўбһҳбһҸбҹҲ MCP Server
      - бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ бҹўбҹ– бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ MCP Server бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
      - бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ бҹЈбҹ– бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI Toolkit
      - бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ бҹӨбҹ– бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҶбһ бҹҶ
    - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҸбһ¶бһҳбһҡбҹ„бһ„бһ…бһҖбҹ’бһҡвҖӢбһҮбһ¶бһҳбһҪбһҷбһҮбҹҶбһ бһ¶бһ“бһ бҹ’бһңбһ¶бһ„бһҮбҹҶбһ бһ¶бһ“

12. **бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡвҖӢбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҠбһ¶бһү MCP Server (11-MCPServerHandsOnLabs/)**
    - **бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ”бһҹбҹӢбһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ бҹЎбҹЈ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶ MCP бһҸбҹ’бһҡбҹҖбһҳбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣ PostgreSQL**
    - **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ—бһ¶бһӮбһӣбһҖбҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ–бһ·бһ—бһ–бһӣбҹ„бһҖ** бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Zava Retail
    - **бһӣбҹҶбһ“бһ¶бҹҶбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ** бһҡбһҪбһҳбһҳбһ¶бһ“ Row Level Security (RLS), бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһўбһҸбҹ’бһҗбһ“бҹҗбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһўбһҸбһ·бһҗбһ·бһҮбһ“
    - **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡвҖӢбһ–бҹҒбһүбһӣбҹҒбһү**бҹ–
      - **бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ 00-03бҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“** - бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ, бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–, бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“
      - **бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ 04-06бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ MCP Server** - бһҡбһ…бһ“бһ¶бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP Server, бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ§бһ”бһҖбһҡбһҺбҹҚ
      - **бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ 07-09бҹ– бһӣбһҖбҹ’бһҒбһҺбҹҲбһҒбҹ’бһ–бһҹбҹӢ** - бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһўбһҸбҹ’бһҗбһ“бҹҗбһҷ, бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҖбҹӮбһҖбҹҶбһ бһ»бһҹ, бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ VS Code
      - **бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ 10-12бҹ– бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӣбҹ’бһў** - бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ, бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ, бһ”бһ“бҹ’бһ’бһјбһҡбһ”бһ“бҹ’бһҗбһҷ
    - **бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ**бҹ– бһҹбҹҠбһ»бһҳ FastMCP, PostgreSQL, Azure OpenAI, Azure Container Apps, Application Insights
    - **бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶**бҹ– бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶ MCP бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ, бһӣбҹҶбһ“бһ¶бҹҶбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһңбһ·бһ—бһ¶бһӮбһҠбҹ„бһҷбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„ AI, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

## бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫд»“еә“бһ“бҹҒбҹҮ

1. **бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһӣбҹҶбһҠбһ¶бһ”бҹӢ**бҹ– бһҸбһ¶бһҳбһҠбһ¶бһ“бһҮбҹҶбһ–бһјбһҖбһ–бһё бҹ бҹ  бһҠбһӣбҹӢ бҹЎбҹЎ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбҹ”
2. **бһҖбһ¶бһҡбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһӣбһҫбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҺбһ¶бһҳбһҪбһҷ бһҹбһјбһҳбһҡбһҖбһҳбһҫбһӣбһҗбһҸбһӮбҹҶбһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбҹ”
3. **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**бҹ– бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһ•бҹ’бһ“бҹӮбһҖ "Getting Started" бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹҒбһңбһ¶ MCP бһ“бһ·бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
4. **бһҖбһ¶бһҡбһҡбһ»бһҖбһҡбһҖбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһўбһ¶бһ…бһ…бһјбһӣбһҡбһҪбһҳбһңбһ·бһҳбһҮбҹ’бһҲбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ„бҹ’бһҡбһёбһҖбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бҹ”
