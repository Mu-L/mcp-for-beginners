# គាំទ្រ

## របៀបដាក់បញ្ហា និងទទួលបានជំនួយ  

គម្រោងនេះប្រើ GitHub Issues ដើម្បីតាមដានបញ្ហារួច និងសំណើរពិសេស។ សូមស្វែងរកបញ្ហា​ ដែលមានរួចមុនពេលដាក់បញ្ហាថ្មី ដើម្បីជៀសវាងការចម្លង។ សម្រាប់បញ្ហាថ្មី សូមដាក់bug របស់អ្នកឬសំណើពិសេស ជាបញ្ហាថ្មី។

សម្រាប់ជំនួយ និងសំណួរអំពីការប្រើប្រាស់គម្រោងនេះ សូម៖
- ពិនិត្យមើល [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ដែលជាកន្លែងអ្នកជំនាញ និងអ្នកអភិវឌ្ឍន៍ផ្សេងទៀតបែងចែក
- ប្រើធនធាន [Model Context Protocol Community](https://modelcontextprotocol.io/community/) សម្រាប់សំណួរដែលពាក់ព័ន្ធនឹងពិធីការណ៏

## និតិវិធីគាំទ្ររបស់ Microsoft  

ការគាំទ្រសម្រាប់គម្រោង Model Context Protocol (MCP) សម្រាប់អ្នកចាប់ផ្តើម គឺមានកំណត់ត្រឹមតែធនធានដែលបានរាយការណ៍ខាងលើ។
