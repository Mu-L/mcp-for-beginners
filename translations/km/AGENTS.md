## Project Overview

**MCP бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ** бһӮбһәбһҮбһ¶бһ–бһ„бҹ’бһҡбһёбһҖбһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶбһӮбҹҶбһҡбһјбһ”бһҫбһҖбһ…бҹҶбһ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ–бһё Model Context Protocol (MCP) - бһҹбҹҠбһ»бһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҡбһңбһ¶бһ„бһӮбҹҶбһҡбһј AI бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҸбһ·бһҗбһ·бһҮбһ“бҹ” бһ‘бһёбһҸбһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҜбһҖбһҹбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ…бһҳбҹ’бһҡбһ»бҹҮ бһ“бһ·бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһҠбһ¶бһҖбҹӢбһўбһ¶бһҮбһёбһ–бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”

### Architecture

- **11 бһҳбһјбһҢбһ»бһӣ бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (00-11)**: бһ•бҹ’бһӣбһјбһңбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһҸбһҳбҹ’бһҡбһ№бһҳбһ–бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бҹ…бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
- **бһҹбһ·бһҖбҹ’бһҸбҹҶбһЎбһҫбһ„бһ бһҸбҹ’бһҗбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ**: бһӣбҹҶбһ бһ¶бһҸбҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҳбһҪбһҷбһ“бһ·бһ„бһҖбһјбһҠбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ–бҹҒбһүбһӣбҹҒбһүбһҮбһ¶бһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“
- **бһӮбһҳбҹ’бһҡбҹ„бһ„бһӮбҹҶбһҡбһј**: бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ“бһ·бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһҖбһ”бҹ’бһҡбҹӮ**: бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ GitHub Actions бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“
- **бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһҡбһјбһ”бһ—бһ¶бһ–**: бһҗбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһңбҹӮбһҡбһёбһҹбҹ’бһҷбһ»бһ„бһ”бһҖбһ”бҹ’бһҡбҹӮ

## Development Workflow

### Documentation Structure

- **Module 00-11**: бһҳбһ¶бһҸбһ·бһҖбһ¶бһӮбҹ’бһҡбһ”бҹӢбһҜбһҖбһҹбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҡбҹҖбһ”бһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’
- **translations/**: бһңбҹӮбһҡбһёбһҹбҹ’бһҷбһ»бһ„бһ—бһ¶бһҹбһ¶бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ (бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„)
- **translated_images/**: бһңбҹӮбһҡбһёбһҹбҹ’бһҷбһ»бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҸбһ¶бһҳбһ—бһ¶бһҹбһ¶ (бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·)
- **images/**: бһҡбһјбһ”бһ—бһ¶бһ–бһҠбһҫбһҳбһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһҸбһ¶бһҡбһ¶бһ„

3. бһ”бҹҠбһ·бһ… GitHub Co-op translator бҹ” бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһҖбһ”бҹ’бһҡбҹӮбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
4. бһ”бһҖбһ”бҹ’бһҡбҹӮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһңбһ·бһүбһ–бҹҒбһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҹбһ¶бһҒбһ¶бһ…бһҳбҹ’бһ”бһ„

### Working with Translations

- **бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·**: бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ GitHub Actions бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
- **бһҖбһ»бҹҶбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹ„бһҷбһҠбҹғ** бһҜбһҖбһҹбһ¶бһҡбһ“бҹ…бһҗбһҸ `translations/`
- бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣ
- бһ—бһ¶бһҹбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡ: бҹӨбҹЁ+ бһ—бһ¶бһҹбһ¶ бһҡбһҪбһҳбһҳбһ¶бһ“ бһўбһ¶бһҡбҹүбһ¶бһ”бһ·бһ…, бһ…бһ·бһ“, бһ”бһ¶бһҡбһ¶бҹҶбһ„, бһўбһ¶бһӣбҹ’бһӣбһәбһҳбҹүбһ„бҹӢ, бһ бһ·бһҺбҹ’бһҢбһ», бһҮбһ”бҹүбһ»бһ“, бһҖбһјбһҡбҹүбҹҒ, бһ–бҹҗбһҡбһ‘бһ»бһҷбһ бҹ’бһӮбһ¶бһӣбҹӢ, бһҡбһ»бһҹбҹ’бһҹбҹҠбһё, бһҹбҹ’бһ”бҹүбһ¶бһ“, бһ“бһ·бһ„бһ…бҹ’бһҡбһҫбһ“бһ‘бҹҖбһҸ

## Contributing Guidelines

### Pull Request Process

1. **бһ…бһ¶бһҖбҹӢ Fork бһ“бһ·бһ„ Clone**: бһ…бһ¶бһҖбҹӢ Fork бһ‘бһёбһҸбһ¶бҹҶбһ„ бһ“бһ·бһ„ Clone Fork бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһҖбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ
2. **бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ¶бһҒбһ¶ Branch**: бһ”бҹ’бһҡбһҫбһҲбҹ’бһҳбҹ„бҹҮбһҹбһ¶бһҒбһ¶бһҳбһҪбһҷбһҹбҹҶбһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ (бһ§. `fix/typo-module-3`, `add/python-example`)
3. **бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ**: бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҜбһҖбһҹбһ¶бһҡбһӣбҹҒбһҒбһЎбһҫбһ„ markdown бһҮбһ¶бһ—бһ¶бһҹбһ¶бһўбһ„бҹӢбһӮбҹ’бһӣбҹҒбһҹбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ (бһҳбһ·бһ“бһҳбҹӮбһ“бһ”бһҖбһ”бҹ’бһҡбҹӮ)
4. **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“**: бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ markdown бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
5. **бһҠбһ¶бһҖбҹӢ PR**: бһ”бҹ’бһҡбһҫбһ…бҹҶбһҺбһ„бһҮбһҫбһ„ бһ“бһ·бһ„бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ
6. **CLA**: бһ…бһ»бҹҮбһ бһҸбҹ’бһҗбһӣбҹҒбһҒбһ¶бһӣбһҫ Microsoft Contributor License Agreement бһ“бҹ…бһ–бҹҒбһӣбһҸбҹ’бһҡбһјбһңбһҹбҹ’бһ“бһҫбһҡ

### Common Troubleshooting

**Q: PR бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҖбҹҶбһ–бһ»бһ„бһ”бһҡбһ¶бһҮбҹҗбһҷбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһҖбһ”бҹ’бһҡбҹӮ**
A: бһҹбһјбһҳбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ¶бһ“бһҸбҹӮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһўбһ„бҹӢбһӮбҹ’бһӣбҹҒбһҹбһ“бҹғбһҳбҹүбһјбһҢбһ»бһӣбһҠбһҫбһҳ бһҳбһ·бһ“бһҳбҹӮбһ“бһңбҹӮбһҡбһёбһҹбҹ’бһҷбһ»бһ„бһ”бһҖбһ”бҹ’бһҡбҹӮбһ‘бҹҒбҹ”

**Q: бһҸбһҫбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бһ¶бһҹбһ¶бһҗбҹ’бһҳбһё?**
A: бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҸбһ¶бһҳбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ co-op-translatorбҹ” бһҹбһјбһҳбһ”бһҫбһҖбһ”бһүбҹ’бһ бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бһ¶бһҹбһ¶бһҗбҹ’бһҳбһёбҹ”

**Q: бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ**
A: бһҹбһјбһҳбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҸбһҳбҹ’бһӣбһҫбһ„бһҖбҹ’бһ“бһ»бһ„ README бһ“бҹғбһӮбҹҶбһҡбһјбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бҹ” бһ”бҹ’бһҡбһ№бһ„бһ”бҹ’бһҡбҹӮбһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбҹӮбһңбһҸбҹ’бһҗбһ»бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”
