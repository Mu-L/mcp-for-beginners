## рҹҺҜ бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

[![Build AI Agents in VS Code: 4 Hands-On Labs with MCP and Microsoft Foundry Toolkit](../../../translated_images/km/11.0f6db6a0fb606885.webp)](https://youtu.be/r34Csn3rkeQ)

_(бһ…бһ»бһ…бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһңбһёбһҠбҹҒбһўбһјбһҹбһҳбҹ’бһҡбҹҖбһ”бһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ)_

бһҹбһјбһҳбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҳбһҖбһҖбһ¶бһ“бҹӢ **бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶ Model Context Protocol (MCP)**! бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһ“бһ»бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹҒбҹҮбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҠбҹӮбһӣбһ‘бҹҶбһ“бһҫбһ”бһ‘бҹҶбһ“бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҳбҹ’бһӣбҹӮбһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AIбҹ–

- **рҹ”— Model Context Protocol (MCP)**: бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ”бһҫбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ AI бһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“
- **рҹӣ пёҸ Microsoft Foundry Toolkit Extension бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ VS Code**: бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢ Microsoft

### рҹҺ“ бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ” бһ…бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһЎбһҫбһҷбһҠбһӣбҹӢбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ API бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһ‘бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҡбһ”бһҹбҹӢбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбҹ”

## рҹҸ—пёҸ бһ”бһҺбҹ’бһҠбһ¶бһүбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶

### рҹ”Ң Model Context Protocol (MCP)

MCP бһӮбһәбһҮбһ¶ **"USB-C бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI"** - бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҖбһӣбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ“бһ·бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

**вңЁ бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бҹ–**

- рҹ”„ **бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҸбһҳбҹ’бһҡбһ№бһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ**: бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһ‘бһјбһ‘бҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ AI
- рҹҸӣпёҸ **бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһўбһ¶бһ…бһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бһ¶бһ“**: бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһ“бһ·бһ„бһ…бһҳбҹ’бһ„бһ¶бһҷбһҸбһ¶бһҳбһҡбһҷбҹҲ stdio/SSE бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„
- рҹ§° **бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҹбһҳбҹ’бһ”бһјбһҡ**: бһ§бһ”бһҖбһҡбһҺбҹҚ бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҹбһүбҹ’бһүбһ¶ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ Protocol бһҳбһҪбһҷ
- рҹ”’ **бһҡбһҪбһҹбһҡбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ**: бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһ—бһ¶бһ–бһҮбһҝбһҮбһ¶бһҖбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣ

**рҹҺҜ бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ MCP бһҹбҹҶбһҒбһ¶бһ“бҹӢ:**
бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ…бҹҒбһүбһҒбҹ’бһҹбҹӮ USB-C бһ”бһ¶бһ“бһ”бҹҶбһ”бһ¶бһҸбҹӢбһ—бһ¶бһ–бһ…бһӣбһ¶бһ…бһӣбһ“бҹғбһҒбҹ’бһҹбҹӮ бһ бҹҒбһҸбһ»бһ“бҹҒбҹҮ MCP бһҖбҹҸбһ”бһ¶бһ“бһ”бҹҶбһ”бһ¶бһҸбҹӢбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ“бҹғбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ AI бһ•бһ„бһҠбҹӮбһҡбҹ” Protocol бһҸбҹӮбһҳбһҪбһҷ бһ”бһ·бһ‘бһ…бҹҶбһҺбһ»бһ…бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбҹ”

### рҹӨ– Microsoft Foundry Toolkit Extension бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ VS Code

бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ AI бһҠбҹҸбһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһҡбһ”бһҹбҹӢ Microsoft бһҠбҹӮбһӣбһ”бһҳбҹ’бһӣбҹӮбһ„ VS Code бһ‘бҹ…бһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’AI бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҳбһ»бһҒбҹ”

**рҹҡҖ бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһ“бһјбһӣбҹ–**

- рҹ“Ұ **бһ”бһүбҹ’бһҮбһёбһҳбҹүбһјбһҠбҹӮбһӣ**: бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҳбҹүбһјбһҠбҹӮбһӣбһ–бһё Azure AI, GitHub, Hugging Face, Ollama
- вҡЎ **бһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ**: бһ”бһ„бҹ’бһ бҹ„бҹҮбһ–бһё CPU/GPU/NPU бһҸбһ¶бһҳ ONNX бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- рҹҸ—пёҸ **бһҒбҹӢбһ”бһјбһҢбһөбһҡбһўбҹҒбһ бҹ’бһҹбһ·бһ“**: бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһҷбһўбһ¶бһҷбһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһ»бһ„бһҖбҹ’бһҡбһ»бһҳ MCP
- рҹҺӯ **бһңбһӮбҹ’бһӮбһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘**: бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбһҖбҹ’бһҹбһҡ бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ– бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

**рҹ’Ў бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбҹ–**

- бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҖбҹҶбһҺбһҸбҹӢ
- бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһүбҹ’бһүбһ¶
- бһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸ
- бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҡбһӣбһјбһ“

- [ ] **бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮ**: бһ—бҹ’бһҮбһ¶бһ”бҹӢ AI бһҮбһ¶бһҳбһҪбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
