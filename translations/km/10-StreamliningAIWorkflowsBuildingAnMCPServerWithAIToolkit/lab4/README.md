   - **бһҲбҹ’бһҳбҹ„бҹҮбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ:** `git_mcp_server`

- вң… **Microsoft Foundry Toolkit**бҹ– бһҳбҹүбһјбһҢбҹӮбһӣ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бһ·бһ„бһӮбҹҶбһҡбһјбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ
- вң… **бһҹбҹҶбһҺбһ»бҹҶ MCP**бҹ– бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһўбһҸбһ·бһҗбһ·бһҮбһ“-бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ, бһ–бһ·бһ’бһёбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҒбһҡ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
- вң… **бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ**бҹ– бһ–бһё Playground бһ‘бҹ… Inspector бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бҹ…бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ
- вң… **бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸ бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
- вң… **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**бҹ– бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҮбһ¶бһҳбһҪбһҷ AI

### рҹ”® бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

1. **бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбҹҶбһ“бһ¶бһүбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбҹ’бһ“бһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢ
2. **бһ…бһјбһӣбһҡбһҪбһҳбһҹбһ бһӮбһҳбһ“бҹҚ MCP**бҹ– бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһўбҹ’бһ“бһҖбһҠбһ‘бҹғ
3. **бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ‘бҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
4. **бһ…бһјбһӣбһҡбһҪбһҳбһўбҹ„бһҷбһ—бһ¶бһ–бһ”бҹ’бһҡбһҹбһҫбһҡ Open Source**бҹ– бһҮбһҪбһҷбһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡ

бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ бһҡбһ¶бһӣбҹӢбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһӮбһҪбһҡбһҹбһҳбһ’бһ“бһ’бһ¶бһ“ Model Context Protocol бһҖбҹҶбһ–бһ»бһ„бһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһҷбҹүбһ¶бһ„бһӣбһҝбһ“ бһ бһҫбһҷбһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҡбһҪбһ…бһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ“бҹ…бһҮбһ¶бһҳбһҪбһҷбһңбһ¶бҹ”

**бһўбһҡбһӮбһ»бһҺбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҸбҹҒбһҮбҹ’бһүбһ¶бһ…бһ·бһҸбҹ’бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбҹҖбһ“!**

бһҷбһҫбһ„бһҹбһ„бҹ’бһғбһ№бһҳбһҗбһ¶бһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһ“бҹҒбҹҮбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҸбһҠбҹӮбһӣбһ“бһ№бһ„бһ”бҹҶбһӣбҹӮбһ„бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚ AI бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
