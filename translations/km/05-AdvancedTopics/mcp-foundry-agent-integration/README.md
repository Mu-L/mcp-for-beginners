## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

Model Context Protocol (MCP) бһӮбһәбһҮбһ¶вҖӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ”бһҫбһҖ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҷбҹүбһ¶бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ” бһ“бҹ…бһ–бҹҒбһӣбһ”бһ¶бһ“бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ Microsoft Foundry MCP бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһўбһ¶бһҮбһёбһ–бһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ… API бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбҹ”

бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹҒбҹҮбһ”бһүбҹ’бһ…бһјбһӣбһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“бһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһҮбһ¶бһҳбһҪбһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһ бҹӮбһӣбһ“бҹ… Microsoft Foundry бһ•бҹ’бһҸбһӣбҹӢбһҠбһӣбҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ”бҹҗбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбһ¶бһҳбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбҹ”

**бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбҹ–** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫ MCP бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Microsoft Foundry Agent Service бһҹбһ–бҹ’бһңбһҗбҹ’бһ„бҹғбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ”бһҺбҹ’бһҠбһ¶бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ– westus, westus2, uaenorth, southindia бһ“бһ·бһ„ switzerlandnorth

бһ“бҹ…бһ…бһ”бҹӢбһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһҸбҹҶбһЎбһҫбһ„ MCP server бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҒбһ¶бһ„ Microsoft Foundry
- бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ MCP
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ MCP server бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ

## Model Context Protocol (MCP) бһҮбһ¶бһўбҹ’бһңбһё?

Model Context Protocol бһҮбһ¶бһңбһ·бһ’бһёбһҹбҹ’бһ“бһјбһӣбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ” бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹ„бһҷбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ**бҹ– бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһҹбһ–бҹ’бһңбһҸбҹӮбһҳбһҪбһҷ

```python
    # бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚ MCP
    agent = agents_client.create_agent(
        model="Your AOAI Model Deployment",
        name="my-mcp-agent",
        instructions="You are a helpful agent that can use MCP tools to assist users. Use the available MCP tools to answer questions and perform tasks.",
        tools=mcp_tool.definitions,
    )
    print(f"Created agent, ID: {agent.id}")
    print(f"MCP Server: {mcp_tool.server_label} at {mcp_tool.server_url}")

    # бһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһҹбҹӮбһҡбһ…бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„
    thread = agents_client.threads.create()
    print(f"Created thread, ID: {thread.id}")

    # бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ¶бһҡбһ‘бҹ…бһҒбҹ’бһҹбҹӮбһҡбһ…бһ„
    message = agents_client.messages.create(
        thread_id=thread.id,
        role="user",
        content="What's difference between Azure OpenAI and OpenAI?",
    )
    print(f"Created message, ID: {message.id}")

    # бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһўбһ“бһ»бһҳбҹҗбһҸбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„вҖӢбһҡбһҸбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
    mcp_tool.update_headers("SuperSecret", "123456")
    run = agents_client.runs.create(thread_id=thread.id, agent_id=agent.id, tool_resources=mcp_tool.resources)
    print(f"Created run, ID: {run.id}")

    while run.status in ["queued", "in_progress", "requires_action"]:
        time.sleep(1)
        run = agents_client.runs.get(thread_id=thread.id, run_id=run.id)

        if run.status == "requires_action" and isinstance(run.required_action, SubmitToolApprovalAction):
            tool_calls = run.required_action.submit_tool_approval.tool_calls
            if not tool_calls:
                print("No tool calls provided - cancelling run")
                agents_client.runs.cancel(thread_id=thread.id, run_id=run.id)
                break

            tool_approvals = []
            for tool_call in tool_calls:
                if isinstance(tool_call, RequiredMcpToolCall):
                    try:
                        print(f"Approving tool call: {tool_call}")
                        tool_approvals.append(
                            ToolApproval(
                                tool_call_id=tool_call.id,
                                approve=True,
                                headers=mcp_tool.headers,
                            )
                        )
                    except Exception as e:
                        print(f"Error approving tool_call {tool_call.id}: {e}")

            if tool_approvals:
                agents_client.runs.submit_tool_outputs(
                    thread_id=thread.id, run_id=run.id, tool_approvals=tool_approvals
                )

        print(f"Current run status: {run.status}")

    print(f"Run completed with status: {run.status}")

    # бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶
    messages = agents_client.messages.list(thread_id=thread.id)
    print("\nConversation:")
    print("-" * 50)
    for msg in messages:
        if msg.text_messages:
            last_text = msg.text_messages[-1]
            print(f"{msg.role.upper()}: {last_text.text.value}")
            print("-" * 50)
```

4. **бһ”бһ„бҹ’бһҖбһҫбһ“бһ‘бҹҶбһ бҹҶбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ**бҹ– бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҠбһ»бҹҶбһ”бһ“бҹ’бһ‘бһ»бһҖ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP server бһ…бҹӮбһҖбһ…бһ¶бһҷ
