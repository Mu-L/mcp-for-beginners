# समर्थन

## कसरी समस्या दर्ता गर्ने र सहयोग पाउने  

यस परियोजनाले GitHub Issues प्रयोग गर्दछ बगहरू र सुविधाहरू अनुरोधहरू ट्र्याक गर्न। नयाँ समस्याहरू दर्ता गर्नु अघि कृपया विद्यमान 
समस्याहरू खोज्नुस् ताकि पुनरावृत्ति नहोस्। नयाँ समस्याहरूको लागि, आफ्नो बग वा 
विशेषता अनुरोध नयाँ Issueको रूपमा दर्ता गर्नुहोस्।

यस परियोजना प्रयोग गर्ने सन्दर्भमा सहयोग र प्रश्नहरूको लागि, कृपया:
- [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) जाँच गर्नुहोस् जहाँ विज्ञहरू र अन्य विकासकर्ताहरू भेला हुन्छन्
- प्रोटोकल-सम्बन्धी प्रश्नहरूको लागि [Model Context Protocol Community](https://modelcontextprotocol.io/community/) स्रोतहरू प्रयोग गर्नुहोस्

## माइक्रोसफ्ट समर्थन नीति  

Model Context Protocol (MCP) for Beginners परियोजनाको समर्थन माथि सूचीबद्ध स्रोतहरूमा सीमित छ।
