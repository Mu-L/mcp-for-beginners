# चेनलिट र माइक्रोसफ्ट लर्न डकस् MCP सँग अध्ययन योजना जेनेरेटर

## पूर्वापेक्षितहरू

- पायथन ३.८ वा माथि
- पिप (पायथन प्याकेज मेनेजर)
- माइक्रोसफ्ट लर्न डकस् MCP सर्भरमा जडान गर्न इन्टरनेट पहुँच

## स्थापना

१. यो रिपोजिटरी क्लोन गर्नुहोस् वा प्रोजेक्ट फाइलहरू डाउनलोड गर्नुहोस्।
२. आवश्यक निर्भरता स्थापना गर्नुहोस्:

   ```bash
   pip install -r requirements.txt
   ```

## प्रयोग

### परिदृश्य १: डकस् MCP मा सरल प्रश्न
एक कमाण्ड-लाइन क्लाइन्ट जुन डकस् MCP सर्भरमा जडान हुन्छ, प्रश्न पठाउँछ, र परिणाम छाप्छ।

१. स्क्रिप्ट चलाउनुहोस्:
   ```bash
   python scenario1.py
   ```
२. संकेतमा तपाईंको दस्तावेजीकरण प्रश्न प्रविष्ट गर्नुहोस्।

### परिदृश्य २: अध्ययन योजना जनरेटर (चेनलिट वेब एप)
एक वेब-आधारित इन्टरफेस (चेनलिट प्रयोग गरेर) जसले प्रयोगकर्ताहरूलाई कुनै पनि प्राविधिक विषयका लागि व्यक्तिगत, हप्ता-दर-हप्ता अध्ययन योजना जेनेरेट गर्न अनुमति दिन्छ।

१. चेनलिट एप सुरु गर्नुहोस्:
   ```bash
   chainlit run scenario2.py
   ```
२. तपाइँको टर्मिनलमा दिइएको स्थानीय URL (जस्तै http://localhost:8000) आफ्नो ब्राउजरमा खोल्नुहोस्।
३. च्याट विन्डोमा, आफ्नो अध्ययन विषय र तपाईंले अध्ययन गर्न चाहनुभएको हप्ता संख्यालाई प्रविष्ट गर्नुहोस् (जस्तै, "AI-900 certification, 8 weeks")।
४. एपले तपाईंलाई हप्ता-दर-हप्ता अध्ययन योजना प्रतिक्रिया दिनेछ, जसमा सान्दर्भिक माइक्रोसफ्ट लर्न दस्तावेजहरूको लिंकहरू समावेश हुनेछन्।

**आवश्यक वातावरण चरहरू:**

परिदृश्य २ (चेनलिट वेब एप संग Azure OpenAI) प्रयोग गर्न, तपाईंले `python` निर्देशिकामा `.env` फाइलमा तलका वातावरण चरहरू सेट गर्नुपर्छ:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

एप चलाउनु अघि यी मानहरू आफ्नो Azure OpenAI स्रोत विवरणहरूका साथ भर्नुहोस्।

> [!TIP]
> तपाईं सजिलै आफ्नो मोडेलहरू [Microsoft Foundry](https://ai.azure.com/) प्रयोग गरेर तैनाथ गर्न सक्नुहुन्छ।

### परिदृश्य ३: VS कोडमा MCP सर्भरसँग इन-एडिटर डकस्

ब्राउजर ट्याबहरू परिवर्तन गरेर दस्तावेज खोज्नुभन्दा, माइक्रोसफ्ट लर्न डकस् MCP सर्भर प्रयोग गरी सिधै VS कोडमा ल्याउन सक्नुहुन्छ। यसले तपाईंलाई सक्षम बनाउँछ:
- कोडिङ वातावरण छोडेर बाहिर नजाई VS कोड भित्रै डकसमाथि खोजी र पढ्न।
- सन्दर्भ दस्तावेजहरू र लिंकहरू सिधै README वा कोर्स फाइलहरूमा राख्न।
- GitHub Copilot र MCP सँग मिलेर एक सहज, AI-सञ्चालित दस्तावेजीकरण कार्यप्रवाह प्रयोग गर्न।

**उदाहरण उपयोगहरू:**
- कोर्स वा प्रोजेक्ट दस्तावेज लेख्दा छिटो सन्दर्भ लिंकहरू README मा थप्न।
- कोड जेनेरेट गर्न Copilot र सम्बन्धित डकस तुरुन्त पाउन MCP सँग प्रयोग गर्न।
- सम्पादकमा केन्द्रित रहन र उत्पादकत्व बढाउन।

> [!IMPORTANT]
> सुनिश्चित गर्नुहोस् कि तपाईंको कार्यक्षेत्रमा मान्य [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) कन्फिगरेसन छ (स्थान `.vscode/mcp.json`)।

## किन परिदृश्य २ का लागि चेनलिट?

चेनलिट एक आधुनिक खुला स्रोत फ्रेमवर्क हो जसले संवादात्मक वेब अनुप्रयोगहरू निर्माण गर्न सजिलो बनाउँछ। यसले माइक्रोसफ्ट लर्न डकस् MCP सर्भर जस्ता ब्याकएन्ड सेवामा जडान हुने च्याट-आधारित प्रयोगकर्ता अन्तरफलकहरू सजिलै सिर्जना गर्न अनुमति दिन्छ। यस प्रोजेक्टले चेनलिट प्रयोग गरी व्यक्तिगत अध्ययन योजना वास्तविक समयमा सरल र अन्तरक्रियात्मक तरिकाले जेनेरेट गर्ने सुविधा प्रदान गर्दछ। चेनलिटको उपयोगबाट, तपाईं सिजलिङ, त्वरित उत्पादनशीलता र सिकाइ सुधार गर्ने च्याट-आधारित उपकरणहरू सजिलै बनाउनु र डिप्लॉय गर्नु सक्छ।

## के यो गर्छ

यो एपले प्रयोगकर्ताहरूलाई केवल विषय र अवधि प्रविष्ट गरेर व्यक्तिगत अध्ययन योजना बनाउन अनुमति दिन्छ। एपले तपाईंको इनपुट विश्लेषण गर्छ, माइक्रोसफ्ट लर्न डकस् MCP सर्भरमा सान्दर्भिक सामग्रीको लागि प्रश्न गर्छ, र नतिजाहरूलाई संरचित, हप्ता-दर-हप्ता योजनामा आयोजन गर्छ। हरेक हप्ताको सिफारिशहरू च्याटमा देखाइन्छ, जसले तपाईंलाई पालना गर्न र प्रगति ट्र्याक गर्न सजिलो बनाउँछ। एकीकरणले सुनिश्चित गर्छ कि तपाईंले सधैं नवीनतम र सबैभन्दा सान्दर्भिक सिकाइ स्रोतहरू प्राप्त गर्नुहुन्छ।

## नमुना प्रश्नहरू

च्याट विन्डोमा यी प्रश्नहरू प्रयास गर्नुहोस् र एपले कसरी प्रतिक्रिया जनाउँछ हेर्नुहोस्:

- `AI-900 certification, 8 weeks`
- `Learn Azure Functions, 4 weeks`
- `Azure DevOps, 6 weeks`
- `Data engineering on Azure, 10 weeks`
- `Microsoft security fundamentals, 5 weeks`
- `Power Platform, 7 weeks`
- `Azure AI services, 12 weeks`
- `Cloud architecture, 9 weeks`

यी उदाहरणहरूले फरक सिकाइ लक्ष्य र अवधिको लागि एपको लचकता प्रदर्शन गर्छन्।
