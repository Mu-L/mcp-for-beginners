# 🚀 MCP Tooling

## 🧠 Огляд навчального шляху інтеграції бази даних MCP

Тут ви навчитеся використовувати інструменти, які використовують MCP. Ваш досвід роботи з цими інструментами покращиться, коли ви підключите до них сервери MCP.

## 🔗 Офіційні ресурси MCP

- 📘 [Документація MCP](https://modelcontextprotocol.io/) – Детальні посібники та керівництва для користувачів
- 📜 [Специфікація MCP (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – Архітектура протоколу та технічні посилання
- 🧑‍💻 [Репозиторій MCP на GitHub](https://github.com/modelcontextprotocol) – Відкриті SDK, інструменти та зразки коду
- 🌐 [Спільнота MCP](https://github.com/orgs/modelcontextprotocol/discussions) – Приєднуйтесь до обговорень та робіть внесок у спільноту
- 🔒 [OWASP MCP Top 10](https://microsoft.github.io/mcp-azure-security-guide/mcp/) – Кращі практики безпеки та заходи зі зменшення ризиків
