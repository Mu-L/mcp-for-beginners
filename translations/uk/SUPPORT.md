# Підтримка

## Як подавати звернення та отримувати допомогу  

Цей проєкт використовує GitHub Issues для відстеження помилок та запитів на функції. Будь ласка, спочатку пошукайте наявні  
звернення, щоб уникнути дублювання. Для нових звернень подайте вашу помилку або  
запит на функцію як нове звернення.

Для допомоги та питань щодо використання цього проєкту, будь ласка:
- Перевірте [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4), де збираються експерти та інші розробники
- Використовуйте ресурси [Model Context Protocol Community](https://modelcontextprotocol.io/community/) для питань, пов’язаних із протоколом

## Політика підтримки Microsoft  

Підтримка проекту Model Context Protocol (MCP) for Beginners обмежується ресурсами, наведеними вище.
