# Podpora

## Ako podať hlásenia a získať pomoc

Tento projekt používa GitHub Issues na sledovanie chýb a požiadaviek na funkcie. Pred podaním nových hlásení prosím vyhľadajte existujúce problémy, aby ste sa vyhli duplicitám. Nové chyby alebo požiadavky na funkcie podajte ako nové hlásenie.

Pre pomoc a otázky týkajúce sa používania tohto projektu, prosím:
- Skontrolujte [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4), kde sa stretávajú odborníci a vývojári
- Využite zdroje [Model Context Protocol Community](https://modelcontextprotocol.io/community/) pre otázky súvisiace s protokolom

## Politika podpory Microsoftu

Podpora pre projekt Model Context Protocol (MCP) pre začiatočníkov je obmedzená na vyššie uvedené zdroje.
