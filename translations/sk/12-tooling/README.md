# 🚀 MCP Nástroje

## 🧠 Prehľad učebnej cesty integrácie MCP databázy

Tu sa naučíte, ako používať nástroje, ktoré používajú MCP. Vaša skúsenosť s používaním týchto nástrojov sa zlepší, keď k nim pripojíte MCP servery.

## 🔗 Oficiálne zdroje MCP

- 📘 [MCP Dokumentácia](https://modelcontextprotocol.io/) – Podrobné návody a používateľské príručky
- 📜 [Špecifikácia MCP (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – Architektúra protokolu a technické odkazy
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – Open-source SDK, nástroje a ukážky kódu
- 🌐 [MCP Komunita](https://github.com/orgs/modelcontextprotocol/discussions) – Pridajte sa do diskusií a prispievajte do komunity
- 🔒 [OWASP MCP Top 10](https://microsoft.github.io/mcp-azure-security-guide/mcp/) – Najlepšie bezpečnostné postupy a zmierňovanie rizík
