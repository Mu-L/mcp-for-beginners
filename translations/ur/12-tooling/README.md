# 🚀 ایم سی پی ٹولنگ

## 🧠 ایم سی پی ڈیٹا بیس انٹیگریشن لرننگ پاتھ کا جائزہ

یہاں آپ سیکھیں گے کہ ایسے اوزار کیسے استعمال کیے جائیں جو ایم سی پی استعمال کرتے ہیں۔ جب آپ ایم سی پی سرورز کو ان اوزار سے جوڑیں گے تو آپ کے استعمال کا تجربہ بہتر ہو جائے گا۔

- 📜 [ایم سی پی وضاحت (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – پروٹوکول کی ساخت اور تکنیکی حوالہ جات
