# منظرنامہ 3: VS کوڈ میں MCP سرور کے ساتھ ایڈیٹر میں دستاویزات

## عمومی جائزہ

اس منظرنامے میں، آپ سیکھیں گے کہ Microsoft Learn Docs کو براہ راست اپنے Visual Studio Code ماحول میں MCP سرور کے ذریعے کیسے لایا جائے۔ مستقل براؤزر ٹیبز کے درمیان سوئچ کرنے کی بجائے دستاویزات تلاش کرنے کے لیے، آپ رسمی دستاویزات کو اپنے ایڈیٹر کے اندر ہی تلاش، ریفرنس، اور پڑھ سکتے ہیں۔ یہ طریقہ کار آپ کے ورک فلو کو ہموار کرتا ہے، آپ کو مرکوز رکھتا ہے، اور GitHub Copilot جیسے اوزار کے ساتھ بغیر کسی رکاوٹ کے انضمام ممکن بناتا ہے۔

- VS کوڈ کے اندر اپنے کوڈنگ ماحول کو چھوڑے بغیر دستاویزات تلاش اور پڑھیں۔
- دستاویزات کی طرف حوالہ دیں اور لنکس کو براہ راست اپنے README یا کورس فائلوں میں داخل کریں۔
- GitHub Copilot اور MCP کو ساتھ استعمال کریں تاکہ AI کے ذریعے چلنے والا ہموار دستاویزی ورک فلو حاصل کیا جا سکے۔

## سیکھنے کے مقاصد

اس باب کے آخر میں، آپ سمجھ جائیں گے کہ VS کوڈ میں MCP سرور کو کیسے سیٹ اپ اور استعمال کیا جائے تاکہ آپ کی دستاویزات اور ترقیاتی ورک فلو بہتر ہو سکے۔ آپ یہ کر سکیں گے:

- اپنے ورک اسپیس کو MCP سرور کے لیے دستاویزات تلاش کرنے کے لیے ترتیب دیں۔
- VS کوڈ میں سے براہ راست دستاویزات تلاش کریں اور داخل کریں۔
- GitHub Copilot اور MCP کی طاقت کو ملا کر زیادہ پیداواری، AI سے بڑھا ہوا ورک فلو تخلیق کریں۔

یہ مہارتیں آپ کو مرکوز رہنے، دستاویزات کے معیار کو بہتر بنانے، اور بطور ڈویلپر یا تکنیکی مصنف اپنی پیداوری بڑھانے میں مدد دیں گی۔

## حل

ایڈیٹر میں دستاویزات تک رسائی حاصل کرنے کے لیے، آپ چند مراحل پر عمل کریں گے جو MCP سرور کو VS کوڈ اور GitHub Copilot کے ساتھ مربوط کرتے ہیں۔ یہ حل کورس مصنفین، دستاویزی مصنفین، اور ڈویلپرز کے لیے مثالی ہے جو چاہیں کہ وہ دستاویزات اور Copilot کے ساتھ کام کرتے ہوئے اپنے ایڈیٹر میں توجہ مرکوز رکھیں۔

- کورس یا پروجیکٹ کی دستاویزات لکھتے ہوئے README میں حوالہ جاتی لنکس جلدی شامل کریں۔
- کوڈ لکھنے کے لیے Copilot استعمال کریں اور MCP کے ذریعے فوری طور پر متعلقہ دستاویزات تلاش کریں اور حوالہ دیں۔
- اپنے ایڈیٹر میں مرکوز رہیں اور پیداوری کو بڑھائیں۔

### مرحلہ وار رہنمائی

شروع کرنے کے لیے، ان اقدامات پر عمل کریں۔ ہر قدم کے لیے، آپ بصری وضاحت کے لیے اسٹس فولڈر سے اسکرین شاٹ شامل کر سکتے ہیں۔

1. **MCP کنفیگریشن شامل کریں:**
   اپنے پروجیکٹ کے روٹ میں `.vscode/mcp.json` فائل بنائیں اور درج ذیل کنفیگریشن شامل کریں:
   ```json
   {
     "servers": {
       "LearnDocsMCP": {
         "url": "https://learn.microsoft.com/api/mcp"
       }
     }
   }
   ```
   
یہ کنفیگریشن VS کوڈ کو بتاتا ہے کہ [`Microsoft Learn Docs MCP سرور`](https://github.com/MicrosoftDocs/mcp) سے کیسے رابطہ قائم کرنا ہے۔
   
   ![مرحلہ 1: mcp.json کو .vscode فولڈر میں شامل کریں](../../../../../../translated_images/ur/step1-mcp-json.c06a007fccc3edfa.webp)
    
2. **GitHub Copilot چیٹ پینل کھولیں:**
   اگر آپ کے پاس GitHub Copilot ایکسٹینشن انسٹال نہیں ہے، تو VS کوڈ کے ایکسٹینشن ویو میں جا کر اسے انسٹال کریں۔ آپ اسے براہ راست [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot-chat) سے ڈاؤن لوڈ کر سکتے ہیں۔ پھر سائڈبار سے Copilot چیٹ پینل کھولیں۔

   ![مرحلہ 2: Copilot چیٹ پینل کھولیں](../../../../../../translated_images/ur/step2-copilot-panel.f1cc86e9b9b8cd1a.webp)

3. **ایجنٹ موڈ فعال کریں اور ٹولز کی تصدیق کریں:**
   Copilot چیٹ پینل میں، ایجنٹ موڈ کو فعال کریں۔

   ![مرحلہ 3: ایجنٹ موڈ فعال کریں اور ٹولز کی تصدیق کریں](../../../../../../translated_images/ur/step3-agent-mode.cdc32520fd7dd1d1.webp)

   ایجنٹ موڈ فعال کرنے کے بعد، تصدیق کریں کہ MCP سرور دستیاب ٹولز میں شامل ہے۔ اس سے یقین دہانی ہوتی ہے کہ Copilot ایجنٹ متعلقہ معلومات لانے کے لیے دستاویزات سرور تک رسائی حاصل کر سکتا ہے۔
   
   ![مرحلہ 3: MCP سرور ٹول کی تصدیق کریں](../../../../../../translated_images/ur/step3-verify-mcp-tool.76096a6329cbfecd.webp)
4. **نیا چیٹ شروع کریں اور ایجنٹ کو سوال کریں:**
   Copilot چیٹ پینل میں نیا چیٹ کھولیں۔ اب آپ اپنے دستاویزی سوالات ایجنٹ کو دے سکتے ہیں۔ ایجنٹ MCP سرور کا استعمال کرتے ہوئے متعلقہ Microsoft Learn دستاویزات آپ کے ایڈیٹر میں براہ راست دکھائے گا۔

   - *"میں موضوع X کے لیے مطالعہ کا پلان لکھنے کی کوشش کر رہا ہوں۔ میں اسے 8 ہفتوں کے لیے پڑھوں گا، ہر ہفتے کے لیے، مواد تجویز کریں جو مجھے لینا چاہیے۔"*

   ![مرحلہ 4: چیٹ میں ایجنٹ کو سوال کریں](../../../../../../translated_images/ur/step4-prompt-chat.12187bb001605efc.webp)

5. **لائیو سوالات:**

   > آئیے Microsoft Foundry Discord کے [#get-help](https://discord.gg/D6cRhjHWSC) سیکشن سے لائیو سوال لیتے ہیں ([اصل پیغام دیکھیں](https://discord.com/channels/1113626258182504448/1385498306720829572)):
   
   *"میں Azure AI Foundry پر تیار کردہ AI ایجنٹس کے ساتھ ملٹی ایجنٹ حل تعینات کرنے کے طریقوں کے بارے میں جوابات تلاش کر رہا ہوں۔ میں نے دیکھا ہے کہ کوئی براہ راست تعیناتی کا طریقہ نہیں ہے، جیسے Copilot Studio چینلز۔ تو، انٹرپرائز صارفین کے لیے کام کو مکمل کرنے اور بات چیت کرنے کے لیے مختلف تعیناتی کے طریقے کیا ہیں؟
کئی مضامین/بلاگز موجود ہیں جو کہتے ہیں کہ ہم Azure Bot سروس استعمال کر سکتے ہیں جو MS Teams اور Azure AI Foundry Agents کے درمیان پل کا کام کرے گی، کیا یہ کام کرے گا اگر میں Azure function کے ذریعے Orchestrator Agent کو Azure AI Foundry پر منظم کرنے کے لیے Azure bot سیٹ اپ کروں یا مجھے ہر AI ایجنٹ کے لیے Azure function بنانا ہوگا جو ملٹی ایجنٹ حل کا حصہ ہو تاکہ Bot فریم ورک پر آرکیسٹریشن کی جا سکے؟ کوئی اور مشورے خوش آمدید ہیں۔"*

   ![مرحلہ 5: زندہ سوالات](../../../../../../translated_images/ur/step5-live-queries.49db3e4a50bea273.webp)

   ایجنٹ متعلقہ دستاویزات کے لنکس اور خلاصے کے ساتھ جواب دے گا، جنہیں آپ پھر اپنے مارک ڈاؤن فائلوں میں براہ راست داخل کر سکتے ہیں یا اپنے کوڈ میں حوالہ کے طور پر استعمال کر سکتے ہیں۔
   
### نمونہ سوالات

یہاں کچھ مثالیں دی گئی ہیں جنہیں آپ آزما سکتے ہیں۔ یہ سوالات دکھائیں گے کہ MCP سرور اور Copilot کس طرح مل کر فوری، سیاق و سباق سے آگاہ کردہ دستاویزات اور حوالہ جات فراہم کر سکتے ہیں بغیر VS کوڈ چھوڑے:

- "مجھے دکھائیں کہ Azure Functions triggers کو کیسے استعمال کریں۔"
- "Azure Key Vault کی سرکاری دستاویزات کا لنک داخل کریں۔"
- "Azure وسائل کو محفوظ بنانے کے بہترین طریقے کیا ہیں؟"
- "Azure AI سروسز کے لیے کوئی quickstart تلاش کریں۔"

یہ سوالات ظاہر کریں گے کہ MCP سرور اور Copilot کیسے مل کر فوری، سیاق و سباق سے آگاہ دستاویزات اور حوالہ جات فراہم کرتے ہیں بغیر VS کوڈ چھوڑے۔
