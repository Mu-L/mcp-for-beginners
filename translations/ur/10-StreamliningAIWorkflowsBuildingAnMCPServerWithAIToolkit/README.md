خوش آمدید **Model Context Protocol (MCP) ورکشاپ** میں! یہ جامع عملی ورکشاپ دو جدید ٹیکنالوجیز کو یکجا کرتی ہے تاکہ AI ایپلیکیشن کی ترقی میں انقلاب لایا جا سکے:

- **🔗 Model Context Protocol (MCP)**: AI اور ٹولز کے درمیان بے جوڑ انضمام کے لیے ایک کھلا معیار
- **🛠️ Microsoft Foundry Toolkit کا VS Code ایکسٹینشن**: مائیکروسافٹ کا طاقتور AI ترقیاتی ایکسٹینشن

### 🎓 آپ کیا سیکھیں گے

اس ورکشاپ کے اختتام تک، آپ ذہین ایپلیکیشنز بنانے میں مہارت حاصل کر لیں گے جو AI ماڈلز کو حقیقی دنیا کے ٹولز اور سروسز سے مربوط کرتی ہیں۔ خودکار ٹیسٹنگ سے لے کر حسبِ ضرورت API انضمام تک، آپ پیچیدہ کاروباری چیلنجز حل کرنے کی عملی مہارتیں حاصل کریں گے۔

## 🏗️ ٹیکنالوجی کا مجموعہ

### 🔌 Model Context Protocol (MCP)

MCP ہے **"USB-C برائے AI"** — ایک یونیورسل معیار جو AI ماڈلز کو بیرونی ٹولز اور ڈیٹا ذرائع سے مربوط کرتا ہے۔

**✨ اہم خصوصیات:**

- 🔄 **معیاری انضمام**: AI-ٹول کنکشنز کے لیے یونیورسل انٹرفیس
- 🏛️ **لچکدار فن تعمیر**: لوکل اور ریموٹ سرورز stdio/SSE ٹرانسپورٹ کے ذریعے
- 🧰 **دیرپا ماحولیاتی نظام**: ٹولز، پرامپٹس، اور وسائل ایک پروٹوکول میں
- 🔒 **انٹرپرائز کے لیے تیار**: بلٹ ان سیکیورٹی اور اعتبار

**🎯 MCP کیوں اہم ہے:**
جیسے USB-C کیبل کے الجھن کو ختم کر دیتا ہے، MCP AI انٹیگریشنز کی پیچیدگی کو ختم کرتا ہے۔ ایک پروٹوکول، لا محدود امکانات۔

### 🤖 Microsoft Foundry Toolkit کا VS Code ایکسٹینشن

مائیکروسافٹ کا مرکزی AI ترقیاتی ایکسٹینشن جو VS Code کو AI طاقتور مرکز میں بدل دیتا ہے۔

**🚀 بنیادی صلاحیتیں:**

- 📦 **ماڈل کیٹلاگ**: Azure AI، GitHub، Hugging Face، Ollama کے ماڈلز تک رسائی
- ⚡ **لوکل انفرنس**: ONNX سے بہتر بنایا گیا CPU/GPU/NPU اجرا
- 🏗️ **ایجنٹ بلڈر**: MCP انضمام کے ساتھ بصری AI ایجنٹ کی ترقی
- 🎭 **کثیر النوعی**: متن، بصارت، اور منظم آؤٹ پٹ کی حمایت

**💡 ترقی کے فوائد:**

- بغیر کنفیگریشن کے ماڈل کی تعیناتی
- بصری پرامپٹ انجینیئرنگ
- حقیقی وقت کی ٹیسٹنگ کے لئے پلے گراؤنڈ
- MCP سرور کا بے جوڑ انضمام

## 💡 حقیقی دنیا کی درخواستیں اور اثر

### 🏢 انٹرپرائز استعمال کے کیسز

#### 🔄 ڈیوآپ خودکاری

اپنے ترقیاتی ورک فلو کو ذہین خودکاری کے ساتھ تبدیل کریں:

- **سمارٹ ریپازیٹری مینجمنٹ**: AI سے چلنے والا کوڈ کے جائزے اور ضم کرنے کے فیصلے
- **ذہین CI/CD**: کوڈ میں تبدیلیوں کی بنیاد پر خودکار پائپ لائن کی اصلاح
- **مسئلہ ٹریاج**: خودکار بگ کی درجہ بندی اور تفویض

#### 🧪 کوالٹی ایشورنس میں انقلاب

AI سے چلنے والے خودکار طریقوں کے ذریعے جانچ کو بہتر بنائیں:

- **ذہین ٹیسٹ جنریشن**: مکمل ٹیسٹ سوئٹس خودکار طریقے سے بنائیں
- **بصری ریگریشن ٹیسٹنگ**: AI سے چلنے والا UI تبدیلی کا پتہ لگانا
- **کارکردگی کی نگرانی**: مسائل کی پیشگی شناخت اور حل

#### 📊 ڈیٹا پائپ لائن کی ذہانت

ذہین ڈیٹا پروسیسنگ ورک فلو تعمیر کریں:

- **مطابقت پذیر ETL عمل**: خود کو بہتر بنانے والی ڈیٹا تبدیلیاں
- **انوملی ڈیٹیکشن**: حقیقی وقت میں ڈیٹا معیار کی نگرانی
- **ذہین راستہ سازی**: سمارٹ ڈیٹا فلو مینجمنٹ

#### 🎧 کسٹمر تجربے کی بہتری

بہترین کسٹمر انٹریکشنز بنائیں:

- **سیاق و سباق سے آگاہ مدد**: AI ایجنٹس جنہیں کسٹمر ہسٹری تک رسائی حاصل ہے
- **پیشگی مسئلہ حل**: پیشگوئی کنندہ کسٹمر سروس
- **کثیر چینل انضمام**: مختلف پلیٹ فارمز پر یکجا AI تجربہ

## 🛠️ ضروریات اور سیٹ اپ

### 💻 نظام کے تقاضے

| جزو | تقاضہ | نوٹس |
|-----------|-------------|-------|
| **آپریٹنگ سسٹم** | Windows 10+، macOS 10.15+، Linux | کوئی بھی جدید OS |
| **Visual Studio Code** | جدید مستحکم ورژن | Microsoft Foundry Toolkit کے لیے ضروری |
| **Node.js** | v18.0+ اور npm | MCP سرور کی ترقی کے لیے |
| **Python** | 3.10+ | Python MCP سرورز کے لیے اختیاری |
| **میموری** | کم از کم 8GB RAM | لوکل ماڈلز کے لیے 16GB تجویز کردہ |

### 🔧 ترقیاتی ماحول

#### تجویز کردہ VS Code ایکسٹینشنز

- **Microsoft Foundry Toolkit** (ms-windows-ai-studio.windows-ai-studio)
- **Python** (ms-python.python)
- **Python Debugger** (ms-python.debugpy)
- **GitHub Copilot** (GitHub.copilot) - اختیاری مگر مفید

#### اختیاری ٹولز

- **uv**: جدید Python پیکیج مینیجر
- **MCP Inspector**: MCP سرورز کے لیے بصری ڈی بگنگ ٹول
- **Playwright**: ویب آٹومیشن کی مثالوں کے لیے
