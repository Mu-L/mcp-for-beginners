# سپورٹ

## مسائل درج کروانے اور مدد حاصل کرنے کا طریقہ  

یہ پروجیکٹ بگز اور فیچر ریکویسٹز کو ٹریک کرنے کے لیے GitHub Issues استعمال کرتا ہے۔ براہ کرم نئے مسائل درج کرنے سے پہلے موجودہ مسائل تلاش کریں تاکہ نقل سے بچا جا سکے۔ نئے مسائل کے لیے، اپنا بگ یا فیچر ریکویسٹ ایک نئی Issue کے طور پر درج کریں۔

اس پروجیکٹ کو استعمال کرنے کے بارے میں مدد اور سوالات کے لیے، براہ کرم:
- [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) چیک کریں جہاں ماہرین اور دیگر ڈویلپرز جمع ہوتے ہیں
- پروٹوکول سے متعلق سوالات کے لیے [Model Context Protocol Community](https://modelcontextprotocol.io/community/) کے وسائل استعمال کریں

## مائیکروسافٹ سپورٹ پالیسی
