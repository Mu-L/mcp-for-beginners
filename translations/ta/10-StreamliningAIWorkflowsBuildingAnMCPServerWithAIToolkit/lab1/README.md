## 🧠 Microsoft Foundry Toolkit அறிமுகம்

**Microsoft Foundry Toolkit விரிவாக்கம் VS Code க்கான** Microsoft இன் முக்கிய விரிவாக்கமாகும், இது VS Code ஐ முழுமையான AI மேம்பாட்டு சூழலாக்கமாக மாற்றுகிறது. இது AI ஆராய்ச்சி மற்றும் நடைமுறை பயன்பாட்டு மேம்பாட்டினை இணைத்துக்கொண்டு, ஜெனரேட்டிவ் AI ஐ எல்லா திறன் நிலைகளில் உள்ள டெவலப்பராக்களுக்கு அணுகுமுறையாக்குகிறது.

### 🎯 ஏன் Microsoft Foundry Toolkit முக்கியம்

- **🚀 விரைவான மேம்பாடு**: யோசனை முதல் முன்மாதிரி வரை நிமிடங்களில்
- **🔄 ஒருங்கிணைந்த வேலைநடை**: பல AI வழங்குநர்களுக்கான ஒரே இடைமுகம்
- **🧪 எளிய பரிசீலனை**: சிக்கலான அமைப்பு இல்லாமல் மாதிரிகளை ஒப்பிடல்
- **📈 தயாரிப்பிற்கான தயாராக 있음**: முன்மாதிரியிலிருந்து பரிமாற்றம் எளிதாக

### 📊 படி 1: முறை பட்டியலை ஆய்வு செய்க

முறை பட்டியல் உங்கள் AI சூழலுக்கான வாயிலாகும். இது பல வழங்குநர்களின் மாதிரிகளை ஒருங்கிணைத்து, கண்டுபிடிக்க மற்றும் ஒப்பிட எளிதாக அமைக்கிறது.

**🔍 பயணம் வழிகாட்டி:**

Microsoft Foundry Toolkit பக்க பட்டியலில் **MODELS - Catalog** என்பதை கிளிக் செய்க

![Model Catalog](../../../../translated_images/ta/aimodel.263ed2be013d8fb0.webp)

**💡 சிறந்த பரிந்துரை**: உங்கள் பயன்பாட்டுக்கு ஏற்ற தனித்திறன்கள் கொண்ட மாதிரிகளைத் தேடு (எ.கா. குறியீடு உருவாக்கல், படைப்பாற்றல் எழுதுதல், பகுப்பாய்வு).

**⚠️ குறிப்பு**: GitHub ஹோஸ்ட் செய்யப்பட்ட மாதிரிகள் (GitHub Models)ใช้ฟรี, ஆனால் கோரிக்கைகள் மற்றும் டோக்கன்களுக்கு வரம்புகள் உள்ளன. நீங்களே Azure AI அல்லது பிற இடங்களிலிருந்து மற்ற மாதிரிகளை பயன்படுத்த விரும்பினால் சரியான API விசை அல்லது அங்கீகாரத்தை வழங்க வேண்டும்.

### 🎯 படி 3: விளையாட்டுத் தளத்தைக் கையாளும் திறன்

விளையாட்டு தளம் உங்கள் AI பரிசோதனை ஆய்வுகூடம். இதன் முழு திறனைக் கொண்டு பயன் படுத்துவது எப்படி என்பதைக் காணலாம்:

**🎨 ப்ராம்ட் பொறியியல் சிறந்த நடைமுறைகள்:**
1. **விபரமாக இரு**: தெளிவான, விரிவான வழிமுறைகள் சிறந்த முடிவுகளை தரும்
2. **பின்புலம் கொடு**: சம்பந்தப்பட்ட பின்னணி தகவலைச் சேர்க்கவும்
3. **உதாரணங்கள் கொடு**: நீங்கள் விரும்பும்தை மாதிரிக்கு காண்பி
4. **மீண்டும் முயற்சி செய்**: ஆரம்ப முடிவுகளை வைத்து ப்ராம்ட்களை செம்மைப்படுத்து

**🧪 சோதனை நிலைகள்:**
```markdown
# Example 1: Code Generation
"Write a Python function that calculates the factorial of a number using recursion. Include error handling and docstrings."

# Example 2: Creative Writing
"Write a professional email to a client explaining a project delay, maintaining a positive tone while being transparent about challenges."

# Example 3: Data Analysis
"Analyze this sales data and provide insights: [paste your data]. Focus on trends, anomalies, and actionable recommendations."
```

### 🏗️ படி 1: Agent Builder அறிதல்

Agent Builder யார் Microsoft Foundry Toolkit இல் மிக்க முக்கியமானது. இது பெரிய மொழி மாதிரிகள் மற்றும் தனிப்பயன் வழிமுறைகள், குறிப்பிட்ட அளவுகோல்கள் மற்றும் சிறப்பு அறிவுடன் மொத்த AI உதவியாளர்களை உருவாக்க அனுமதிக்கிறது.

**🧠 முகவர் கட்டமைப்பு கூறுகள்:**
- **முதல் மாதிரி**: அடிக்கலம் LLM (GPT-4, Groks, Phi, மற்றவை)
- **கணினி ப்ராம்ட்**: முகவர் தனிமை மற்றும் நடத்தை வரையறு
- **அளவுகோல்கள்**: சிறந்த செயல்திறனுக்கான மேம்படுத்தப்பட்ட அமைப்புகள்
- **கருவிகள் ஒருங்கிணைப்பு**: வெளியித்த API மற்றும் MCP சேவைகளை இணைக்கவும்
- **நினைவு**: உரையாடல் சூழல் மற்றும் அமர்வு நிலைத்தன்மை
