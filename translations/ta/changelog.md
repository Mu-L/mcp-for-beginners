# மாற்றம் பதிவு: MCP தொடக்கக்காரர்கள் பாடத்திட்டம்

இந்த ஆவணம் Model Context Protocol (MCP) தொடக்கக்காரர்கள் பாடத்திட்டத்தில் செய்யப்பட்ட அனைத்து முக்கிய மாற்றங்களின் பதிவாக செயல்படுகிறது. மாற்றங்கள் மரபுசார் கால வரிசைப்படி (புதிய மாற்றங்கள் முதலில்) பதிவாகக்கொள்ளப்பட்டுள்ளன.

### MCP விவரக்குறிப்பு 2025-11-25 ஒத்திசைவை மதிப்பாய்வு

#### கரு கருத்துக்கள் மேம்பாடு (01-CoreConcepts/)  
- **புதிய கிளையண்ட் முதன்மை - Roots**: சேவையகங்கள் கோப்பமைப்பு எல்லைகள் மற்றும் அணுகல் அனுமதிகள் புரிந்துகொள்ளும் வகையில் Roots கிளையண்ட் முதன்மை குறித்த விரிவான ஆவணங்கள் சேர்க்கப்பட்டன  
- **கருவி 注解(Annotations)**: கருவி நடத்தலும் (`readOnlyHint`, `destructiveHint`) குறித்த விளக்கங்கள் கருவி செயல்முறைகளுக்கு மேலும்நற்செயலாக்க உதவியாக சேர்க்கப்பட்டன  
- **சேம்பிளிங் எனக்கென கருவி அழைப்பு**: சேம்பிளிங் ஆவணத்தில் `tools` மற்றும் `toolChoice` அளவுருக்கள் சேர்க்கப்பட்டன, மாதிரியைக் கொண்டு கருவி அழைப்பை கட்டுப்படுத்தும் முறைக்கான விளக்கம்  
- **URL பயன்முறை அறிமுகம்**: சேவையகம் ஆரம்பிக்கும் வெளிப்புற இணையச் செயல்பாடுகளுக்கான URL அடிப்படையிலான அழைப்பு விவரங்கள்  
- **பணிகள் (அனுபவ வீச்சு)**: தொடர் நிறைவேற்றும் மூடியான பணிகளுக்கான மற்றும் பிறக்கச் சாத்திய முடிவுகளை முன்கூட்டியே வரைவது குறித்த புதிய பகுதி  
- **ஐகான்கள் ஆதரவு**: கருவிகள், வளங்கள், வள வார்ப்புருக்கள் மற்றும் பணிக் கட்டளைகள் இனி கூடுதலாக ஐகான்கள் தகவலை சேர்க்க முடியும் என குறிப்பு

#### ஆவண புதுப்பிப்புகள்  
- **README.md**: MCP விவரக்குறிப்பு 2025-11-25 பதிப்பு குறிப்பு மற்றும் தேதி அடிப்படையிலான பதிப்பு விளக்கம்  
- **study_guide.md**: கருத்துக் கோரிக்கைகள் பகுதியில் பணிகள் மற்றும் கருவி 注解கள் சேர்க்கப்பட்டு ஆவண காலத்தை புதுப்பித்தது

#### விவரக்குறிப்பு ஒத்திசைவு சரிபார்ப்பு  
- **புரொடோக்கால் பதிப்பு**: அனைத்து ஆவணங்களும் தற்போதைய MCP விவரக்குறிப்பு 2025-11-25 ஐ குறிப்பிடுவதாக உறுதி செய்யப்பட்டது  
- **கட்டமைப்பு ஒத்திசைவு**: இரண்டு அடுக்கு கட்டமைப்பு (தரவு அடுக்கு + பரிமாற்ற அடுக்கு) ஆவணங்களின் துல்லியத்தன்மை உறுதி செய்யப்பட்டது  
- **முதன்மை தன்மைகளின் ஆவணப்படுத்தல்**: சேவையக முதற்பணிகள் (வளங்கள், பணிக் கட்டளைகள், கருவிகள்) மற்றும் கிளையண்ட் முதற்பணிகள் (சேம்பிளிங், பயன்முறை, பதிவு, Roots) ஆவணப்படுத்தல் சரிபார்க்கப்பட்டது  
- **பரிமாற்ற இயந்திரங்கள்**: STDIO மற்றும் ஸ்ட்ரீமிங் HTTP பரிமாற்ற ஆவணங்களின் துல்லியம் உறுதி செய்யப்பட்டது  
- **பாதுகாப்பு வழிகாட்டுதல்கள்**: சமீபத்திய MCP பாதுகாப்பு சிறந்த நடைமுறைகளுடன் ஒத்திசைவு உறுதி செய்யப்பட்டது

#### முக்கிய MCP 2025-11-25 அம்சங்கள் ஆவணப்படுத்தப்பட்டன  
- **OpenID இணைவு கண்டறிதல்**: OIDC மூலம் அங்கீகார சேவை கண்டுபிடிப்பு

இந்த சேர்க்கைகள் MCP சர்வர் செயலாக்கங்களை மேலதிக வலுவான, பாதுகாப்பான மற்றும் நெகிழ்வானதாக கட்டியெழுப்புவதற்கான நடைமுறை வழிகாட்டல்களை வழங்குகின்றன, அடிப்படை கருத்துக்களை மேம்பட்ட உற்பத்தி மாதிரிகளுடன் இணைப்பதாக.

    - சூழல் துண்டாக்கம் மற்றும் முன்னுரிமை விதிகள்
    - முன்னேற்றமான சூழல் ஏற்றல் மற்றும் சுருக்கல் உத்திகள்
    - அடுக்கப்பட்ட சூழல் அணுகுமுறைகள் மற்றும் பெறுதல் மேம்பாடு
  - **அளவிடும் அமைப்பு**: வளர்ந்து வரும் சூழல் செயல்திறன் மதிப்பீடு மீட்டல்கள்
    - உள்ளீடு திறன், செயல்திறன், தரம் மற்றும் பயனர் அனுபவ பரிசீலனைகள்
    - பரிசோதனை முறைகள் சூழல் மேம்பாட்டுக்கு
    - தோல்வி பகுப்பு மற்றும் முன்னேற்ற நடைமுறைகள்

#### பாடத்திட்ட வழிசெலுத்தல் புதுப்பிப்புகள் (README.md)

  - சர்வர்கள் உள்ளூரிலும் தொலைதூர பணி நடத்திய சூழ்நிலைகளுடனும் மேம்படுத்தப்பட்டன
- **அசல் அம்சங்கள் மறுமொழியா அமைப்பது**: சர்வர் மற்றும் கிளையண்ட் அடிக்கடி அம்சங்களின் முழுமையான மறுமொழியா அமைப்பு
  - சர்வர் அடிக்கடி அம்சங்கள்: வளங்கள் (தரவு ஆதாரங்கள்), முன்முனைப்புகள் (வடிவங்கள்), கருவிகள் (கைப்பற்ற செயல்பாடுகள்) விரிவான விளக்கங்கள் மற்றும் உதாரணங்களோடு
  - கிளையண்ட் அடிக்கடி அம்சங்கள்: மாதிரியிடல் (LLM முடிவுகள்), கேட்குதல் (பயனர் உள்ளீடு), பதிவேடு (பிழைதிருத்தம்/பார்வை)
  - தற்போதைய கண்டுபிடிப்பு (`*/list`), மீட்பு (`*/get`), மற்றும் செயல்படுத்தல் (`*/call`) முறைபாடுகளுடன் புதுப்பிக்கப்பட்டது
- **சம்பிரதாய கட்டமைப்பு**: இரண்டு அடுக்கு கட்டமைப்பு மாடல் அறிமுகம்
  - தரவு அடுக்கு: ஜேஎஸ்ஓஎன்-RPC 2.0 அடித்தளம் உயிர்நிலை மேலாண்மை மற்றும் அடிக்கடி அம்சங்களுடன்
  - போக்குவரத்து அடுக்கு: STDIO (உள்ளூர்தல்) மற்றும் Streamable HTTP SSE (தொலைதூர) போக்குவரத்து முறைமைகள்
- **பாதுகாப்பு கட்டமைப்பு**: பயனர் தெளிவான ஒப்புதலுடன், தரவு தனியுரிமை பாதுகாப்பு, கருவி செயல்பாட்டு பாதுகாப்பு மற்றும் போக்குவரத்து அடுக்குப் பாதுகாப்பு போன்ற விரிவான பாதுகாப்பு கொள்கைகள்
- **உரையாடல் முறைபாடுகள்**: ஆரம்பித்தல், கண்டுபிடிப்பு, செயல்படுத்தல் மற்றும் அறிவிப்பு ஓட்டங்களை வெளிப்படுத்தும் சம்பிரதாய செய்தி முறைபாடுகள் புதுப்பிக்கப்பட்டன
- **குறியீடு உதாரணங்கள்**: தற்போதைய MCP SDK முறைபாடுகளை பிரதிபலிக்கும் பன்மொழி உதாரணங்கள் (.NET, ஜாவா, பைதான், ஜாவாஸ்கிரிப்ட்) புதுப்பிக்கப்பட்டன

#### பாதுகாப்பு (02-Security/) - முழுமையான பாதுகாப்பு மறுமலர்ச்சி  
- **தரநிலை ஒத்துழைப்பு**: MCP குறிப்பிடப்பட்ட விதிகள் 2025-06-18 பாதுகாப்பு தேவைகளுடன் முழுமையாக ஒத்துழைப்பு செய்தது
- **அங்கீகார முன்னேற்றம்**: தனிப்பட்ட OAuth சர்வர் இருந்து வெளிப்புற அடையாள வழங்குநருக்கு பொறுப்புரிமை (Microsoft Entra ID) மாற்ற விவரிக்கப்பட்டது
- **AI-பிரத்தியேக ஆபத்து பகுப்பாய்வு**: நவீன AI தாக்குதல் வழிகளின் விரிவான கவர் செய்வுடன
  - உண்மையான உலக உதாரணங்களுடன் விரிவான முன்முனைப் ஊன்று தாக்கல் சூழ்நிலைகள்
  - கருவி நச்சுக்கூற்று இயந்திரங்கள் மற்றும் "ரக் புல்" தாக்கல் முறைபாடுகள்
  - சூழல் சாளரம் நச்சுக்கூற்று மற்றும் மாடல் குழப்பி தாக்குதல்கள்
- **Microsoft AI பாதுகாப்பு தீர்வுகள்**: Microsoft பாதுகாப்பு சூழலின் விரிவான கவர் செய்தல்
  - AI முன்முனை கவசங்கள் மேம்பட்ட கண்டறிதல், காந்தம் மற்றும் வரம்பு வகைகள் கொண்டவை
  - Azure உள்ளடக்கம் பாதுகாப்பு ஒருங்கிணைப்பு முறைபாடுகள்
  - GitHub மேம்பட்ட பாதுகாப்பு வழங்கல் சங்கிலி பாதுகாப்புக்காக
- **மேம்பட்ட அச்சுறுத்தல் தடுப்பு**: சென்னை படை தாக்கல் சூழ்நிலைகளுடன் MCP சிறப்பு பாதுகாப்பு கட்டுப்பாடுகள்
  - உட்புகு கைப்பற்றல் மற்றும் குரிப்டோகிராபிக் அமர்வு ஐடிய சார்பு தேவைகள்
  - MCP தூதர் சூழ்நிலைகளில் குழப்பிய துணைக் கொள்ளையன் பிரச்சினைகள் தெளிவான ஒப்புதல் தேவைகளுடன்
  - குறியீட்டு கடத்தல் செயல்திறன் விரைவுகளுடன் கட்டாய மறுமதிப்பீடு கட்டுப்பாடுகள்
- **வழங்கல் சங்கிலி பாதுகாப்பு**: அடித்தளம் மாதிரிகள், ஊக்கிய சேவைகள், சூழல் வழங்குநர்கள் மற்றும் மூன்றாம் தரப்பு API கள் ஆகியவற்றுடன் விரிவான AI வழங்கல் சங்கிலி கவர் செய்தல்
- **அடித்தளம் பாதுகாப்பு**: சுழற்சி நம்பிக்கை கட்டமைப்பு மற்றும் Microsoft பாதுகாப்பு சூழலுடன் மேம்பட்ட ஒருங்கிணைப்பு

- LLM கிளையண்ட் ஒருங்கிணைப்பு வழிமுறைகள் இணைக்கப்பட்டன
- VS Code ஒருங்கிணைப்பு ஆவணம் சேர்க்கப்பட்டது
- Server-Sent Events (SSE) சர்வர் உதாரணங்கள் அமல்படுத்தப்பட்டன

### நெருக்கடி கொள்கைகள் (01-CoreConcepts/)

- முக்கிய சம்பிரதாய கூறுகள் பற்றிய ஆவணங்கள் உருவாக்கப்பட்டன
- MCP இல் செய்தி முறைபாடுகள் ஆவணப்படுத்தப்பட்டன
