### பாதுகாப்பு கடுமைப்படுத்தல் மற்றும் சார்பு பாதுகாப்பு குறைபாடுகள் சரிசெய்தல்

அனைத்து சார்பு நிரல்தாள் மற்றும் மாதிரி மூலக் குறியீடு முழுமையாய் பாதுகாப்பு ஆய்வு செய்யப்பட்டது, பிறகு npm அறிவிப்புகள் மற்றும் ஒரு குறியீட்டு நிலை கண்டுபிடிப்பை சரிசெய்தோம். சரிசெய்தலுக்குப்பின்பு, `npm audit` ஒவ்வொரு ஆய்வு செய்யப்பட்ட அடைவு அடைவுவதிலும் **0 பாதுகாப்பு குறைபாடுகள்** காட்டுகிறது.

#### முக்கியக் கருத்துக்கள் மேம்பாடு (01-CoreConcepts/)
- **புதிய கிளையண்ட் முதன்மை - ரூட்டுகள்**: சேவையகங்களுக்கு கோப்பு அமைப்பு எல்லைகள் மற்றும் அணுகல் அனுமதிகளை புரிந்து கொள்ள உதவும் ரூட்டுகள் கிளையண்ட் முதன்மை குறியீட்டைக் கொண்ட பரிசீலனை ஆவணங்கள் சேர்க்கப்பட்டது
- **கருவி குறிப்பு**: சிறந்த கருவி செயல்பாட்டு முடிவுகளுக்கான (`readOnlyHint`, `destructiveHint`) கருவி நடைமுறை குறிப்பு ஆவணங்கள் சேர்க்கப்பட்டது
- **சிலம்பிக் கருவி அழைப்பு**: மாதிரி இயக்கப்படும் கருவி அழைப்புக்கான சாம்பிளிங் ஆவணங்களில் `tools` மற்றும் `toolChoice` வகைகள் புதுப்பிக்கப்பட்டன
- **URL முறை வெளிப்படல்**: சேவையகம் ஏற்படுத்தும் புற இணையச் தொடர்புகளுக்கான URL வழி வெளிப்படல் ஆவணங்கள் சேர்க்கப்பட்டன
- **பணி(கள்) (சோதனை நிலை)**: நீடித்த அமல் தொகுப்புகளுக்கான மற்றும் பிண்ணரைவிளைவுகளை பெறுவதற்கான சோதனை பணி அம்சங்களுக்கான புதிய பகுதி சேர்க்கப்பட்டது
- **ஐகான்கள் ஆதரவு**: கருவிகள், வளங்கள், வள மாதிரிகள் மற்றும் கேள்வி முனைகளை மேலும் விவர metadata ஐகான்களுடன் வழங்க முடியும் என்பது குறிக்கப்பட்டுள்ளது

#### முன்னேறிய சேவை பயன்பாடு (03-GettingStarted/10-advanced)
- **புதிய அதிகாரப்பூர்வ அத்தியாயம்**: தனிப்பட்ட மற்றும் குறைந்த அடுக்கு சேவை கட்டமைப்புகளை உள்ளடக்கிய முழுமையான MCP சேவை பயன்பாடு கையேடு அறிமுகப்படுத்தப்பட்டது.
  - **சாதாரண vs. கீழ்நிலை சர்வர்**: இரு அணுகுமுறைகளுக்கும் பைதான் மற்றும் டைப் ஸ்கிரிப்ட் மொழிகளில் விரிவான ஒப்பீடு மற்றும் குறியீட்டு எடுத்துக்காட்டுகள்.
  - **ஹேண்ட்லர் அடிப்படையிலான வடிவமைப்பு**: விரிவான, நெகிழ்வான சர்வர் செயலாக்கங்களுக்கு ஹேண்ட்லர் அடிப்படையிலான கருவி/வள/துடிப்பு மேலாண்மை விளக்கம்.
  - **பயன்முறை வடிவங்கள்**: கீழ்நிலை சர்வர் வடிவமைப்புகள் மேம்பட்ட அம்சங்களுக்கும் கட்டமைப்புக்கும் பயனுள்ளதாக இருக்கும் உண்மையான உலக பரிசுரங்கள்.

#### எளிய அங்கீகாரம் (03-GettingStarted/11-simple-auth)
- **புதிய அத்தியாயம் சேர்க்கப்பட்டது**: MCP சர்வர்களில் எளிய அங்கீகாரத்தை நடைமுறைப்படுத்தும் படி படி வழிகாட்டி.
  - **அங்கீகாரக் கூறுகள்**: அங்கீகாரம் மற்றும் அங்கீகாரத்தின் தெளிவான விளக்கம் மற்றும் அங்கீகாரக் குறிப்புகளின் கையாளல்.
  - **அடிப்படை அங்கீகாரம் நடாத்துதல்**: பைதான் (Starlette) மற்றும் டைப் ஸ்கிரிப்ட் (Express) மொழிகளில் மிடில்வேர் அடிப்படையிலான அங்கீகாரம் வடிவமைப்புகள், குறியீட்டு எடுத்துக்காட்டுகளுடன்.
  - **முன்னேற்றமாக பாதுகாப்பு**: எளிய அங்கீகாரத்துடன் தொடங்குதல் மற்றும் OAuth 2.1 மற்றும் RBAC திரும்ப உத்தரவாதங்களை முன்னோக்கி முன்னேற்ற வழிகாட்டி, மேம்பட்ட பாதுகாப்பு தொகுதிகளுக்கு குறிப்பு.
  
இந்த சேர்ப்புகள் MCP சர்வர் செயலாக்கங்களை நம்பகமான, பாதுகாப்பான மற்றும் நெகிழ்வான முறையில் கட்டுவதற்கான பயன்முறை வழிகாட்டியினை வழங்குகின்றன, அடிப்படைக் கருத்துக்களையும் மேம்பட்ட உற்பத்தி வடிவங்களையும் இணைக்கும்.

#### மூலக் கருத்துக்கள் (01-CoreConcepts/) - முழுமையான நவீனமயமாக்கல்
- **பட்டறைக் பதிப்பு மேம்பாடு**: தற்போதைய MCP விவரக் குறிப்புக்கு 2025-06-18 தேதி அடிப்படையிலான பதிப்பு குறிப்பு (YYYY-MM-DD வடிவம்)
- **கட்டமைப்பு சீரமைப்பு**: தற்போதைய MCP கட்டமைப்பு மாதிரிகளை பிரதிபலிக்கும் தொகுத்துரைகள் மேம்படுத்தல்
  - ஹோஸ்டுகள் தற்கட்டுப்பாடான MCP கிளையன்ட் இணைப்புகளை ஒருங்கிணைக்கும் AI பயன்பாடுகள் என தெளிவாக வரையறுக்கப்படல்
  - கிளையன்டுகள் ஒரு-ஒரு சர்வர் தொடர்புகளை பராமரிக்கும் பட்டறைக் இணைப்பாளர்களாக விளக்கப்பட்டது
  - சர்வர்: உள்ளூரானவையா அல்லது தூரயானவையா என்பதில் மேம்படுத்தல்
- **அகிலகார மாற்றங்கள்**: சர்வர் மற்றும் கிளையன்ட் அகிலகாரங்களில் முழுமையான மறுசீரமைப்பு
  - சர்வர் அகிலகார்கள்: வளங்கள் (தரவு ஆதாரங்கள்), ஊக்கவாதங்கள் (வார்ப்புருக்கள்), கருவிகள் (ஏற்றுமதி செயல்பாடுகள்) விரிவான விளக்கங்களுடன்
  - கிளையன்ட் அகிலகார்கள்: மாதிரியாக்கல் (LLM முடிவுகள்), கேள்விப்பட்டல் (பயனர் உள்ளீடு), பதிவேட்டல் (தொலைபார்வை/கண்காணிப்பு)
  - தற்போதைய கண்டுபிடிப்பு (`*/list`), பெறுதல் (`*/get`), மற்றும் நிறைவேற்றுதல் (`*/call`) முறைகளுடன் புதுப்பிக்கப்பட்டது
- **பட்டறைக் கட்டமைப்பு**: இரு அடுக்கு கட்டமைப்பு முறை அறிமுகம்
  - தரவு அடுக்கு: JSON-RPC 2.0 அடித்தளம் மற்றும் அகிலகார்கள் வாழ்க்கைச்சுற்று மேலாண்மை
  - பரிமாற்ற அடுக்கு: STDIO (உள்ளூர்) மற்றும் Streamable HTTP SSE (தூரம்) பரிமாற்ற சாதனங்கள்
- **பாதுகாப்பு சுருக்கம்**: தெளிவான பயனர் ஒப்புதல், தரவு தனியுரிமை பாதுகாப்பு, கருவி செயல்பாடு பாதுகாப்பு மற்றும் பரிமாற்ற அடுக்கு பாதுகாப்பு உள்ளிட்ட முழுமையான பாதுகாப்பு கொள்கைகள்
- **தொடர்பு மாதிரிகள்**: ஆர்.டி., கண்டுபிடித்தல், நிறைவேற்றுதல் மற்றும் அறிவித்தல் தாங்கிய செய்திச் சான்றுகளை புதுப்பித்தல்
- **குறியீட்டு உதாரணங்கள்**: தற்போதைய MCP SDK மாதிரிகளை பிரதிபலிக்கும் பன்மொழி உதாரணங்கள் (.NET, ஜாவா, பைதான், ஜாவாஸ்கிரிப்ட்) புதுப்பிப்பு

#### பாதுகாப்பு (02-Security/) - முழுமையான பாதுகாப்பு மறுசீரமைப்பு  
- **தரநிலை ஒத்திசைவு**: MCP விவரக் குறிப்பின் 2025-06-18 பாதுகாப்பு தேவைகளுடன் முழுமையான ஒத்திசைவு
- **அங்கீகார முன்னேற்றம்**: தனிப்பயன் OAuth சர்வர்களிலிருந்து வெளிப்புற அடையாள வழங்குநர் பணிவீடு (Microsoft Entra ID) வரை முன்னேற்றக் குறிப்புகள்
- **AI-சார்ந்த அச்சுறுத்தல் பகுப்பாய்வு**: நவீன AI தாக்குதல் வழிகளில் மேம்பட்ட தொகுப்பு
  - விரிவான ஊக்கவாதச் சுளுகை தாக்கல் சூழ்நிலைகள் மற்றும் இயல்பான உதாரணங்கள்
  - கருவி நச்சு மற்றும் "ரக் புல்" தாக்கல் முறைகள்
  - சூழல் சாளர நச்சு மற்றும் மாதிரி குழப்புதல் தாக்கல்கள்
- **Microsoft AI பாதுகாப்பு தீர்வுகள்**: Microsoft பாதுகாப்பு சூழலை ஒருங்கிணைத்த முழுமையான தொகுப்பு
  - AI ஊக்கவாத காக்கிகள் முன்னேற்ற கண்டுபிடிப்பு, விரிவாக்கம் மற்றும் பிரிவினை முறைகளுடன்
  - Azure உள்ளடக்கப் பாதுகாப்பு ஒருங்கிணைப்பு மாதிரிகள்
  - GitHub மேம்பட்ட பாதுகாப்பு மூலம் சப்ளைச் செயின் பாதுகாப்பு
- **மேம்பட்ட அச்சுறுத்தல் தடுப்பு**: விவரமான பாதுகாப்பு கட்டுப்பாடுகள்
  - MCP-சார்ந்த தாக்கல் சூழ்நிலைகளுடன் செஷன் ஹைஜாக்கிங் மற்றும் கிரிப்டோகிராபிக் செஷன் ID தேவைகள்
  - MCP புராக்ஸி சூழ்நிலைகளில் குழப்பமான துணை பிரச்சினைகள் மற்றும் தெளிவான ஒப்புதல் தேவைகள்
  - டோக்கன் வழியாக்கம் ஆபத்துகளுக்கு கட்டாய சோதனை கட்டுப்பாடுகள்
- **சப்ளைச் செயின் பாதுகாப்பு**: அடிப்படை மாதிரிகள், உள்ளமைப்பு சேவைகள், சூழல் வழங்குநர்கள் மற்றும் மூன்றாம் தரப்பு APIகளை உள்ளடக்கிய AI சப்ளைச் செயின் தாக்கலுக்கு விரிவாக்கம்
- **அடித்தளம் பாதுகாப்பு**: சிகராத் நம்பிக்கைக்கான கட்டமைப்பு மற்றும் Microsoft பாதுகாப்பு சூழலாச்சியுடன் ஒருங்கிணைப்பு மேம்பாடு
- **வள அமைப்பு**: வகைப்படுத்தப்பட்ட முதன்மை வள இணைப்புகள் (அதிகாரப்பூர்வ ஆவணங்கள், தரநிலைகள், ஆராய்ச்சி, Microsoft தீர்வுகள், நடைமுறை வழிகாட்டிகள்)
