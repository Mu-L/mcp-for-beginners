# 🚀 MCP கருவிகள்

## 🧠 MCP தரவுத்தள ஒருங்கிணைவு கற்றல் பாதையின் சுருக்கம்

இங்கே நீங்கள் MCP பயன்படுத்தும் கருவிகளை எப்படி பயன்படுத்துவது என்பதை கற்றுக்கொள்ளுவீர்கள். MCP சேவைகள் அவற்றிற்கு இணைக்கும்போது உங்கள் அனுபவம் மேம்படும்.

## 🔗 அதிகாரப்பூர்வ MCP வளங்கள்

- 📘 [MCP ஆவணம்](https://modelcontextprotocol.io/) – விரிவான பயிற்சிகள் மற்றும் பயனர் கையேடுகள்  
- 📜 [MCP விவரணம் (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – நெறிமுறை கட்டமைப்பு மற்றும் தொழில்நுட்ப குறிப்புகள்  
- 🧑‍💻 [MCP GitHub களஞ்சியம்](https://github.com/modelcontextprotocol) – திறந்த மூல SDKகள், கருவிகள், மற்றும் குறியீட்டு உதாரணங்கள்  
- 🌐 [MCP சமூகம்](https://github.com/orgs/modelcontextprotocol/discussions) – விவாதங்களில் கலந்துகொண்டு சமூகத்தில் பங்களிக்கவும்  
- 🔒 [OWASP MCP Top 10](https://microsoft.github.io/mcp-azure-security-guide/mcp/) – பாதுகாப்பு சிறந்த நடைமுறைகள் மற்றும் ஆபத்து குறைப்புகள்
