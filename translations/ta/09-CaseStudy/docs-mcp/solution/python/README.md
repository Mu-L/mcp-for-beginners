# Chainlit & Microsoft Learn Docs MCP உடன் படிப்பு திட்டம் உருவாக்கி

## தேவைகள்

- Python 3.8 அல்லது அதற்கு மேல்
- pip (Python தொகுப்பு முகவர்)
- Microsoft Learn Docs MCP சர்வருடன் இணைக்க இணைய அணுகல்

## நிறுவல்

1. இந்த ரெப்போஸிடரியை கிளோன் செய்யவும் அல்லது திட்ட கோப்புகளை பதிவிறக்கம் செய்யவும்.
2. அவசியமான சார்புகளை நிறுவவும்:

   ```bash
   pip install -r requirements.txt
   ```

## பயன்பாடு

### நிகழ்ச்சி 1: எளிய கேள்வி Docs MCP க்கு
Docs MCP சர்வருடன் இணைந்து, ஒரு கேள்வியை அனுப்பி, முடிவை அச்சிடும் கட்டளை வரி கிளையண்ட்.

1. ஸ்கிரிப்டை இயக்கவும்:
   ```bash
   python scenario1.py
   ```
2. செயல்பாட்டில் உங்கள் ஆவண கேள்வியை உள்ளிடவும்.

### நிகழ்ச்சி 2: படிப்பு திட்டம் உருவாக்கி (Chainlit வலை பயன்பாடு)
பயனர்களுக்கு எந்தவொரு தொழில்நுட்ப தலைப்பிற்கும் தனிப்பயன் வாரத்திற்கு வார படிப்பு திட்டத்தை உருவாக்க அனுமதிக்கும் Chainlit பயன்படுத்தி வலை அடிப்படையிலான இடைமுகம்.

1. Chainlit செயலியை துவங்கவும்:
   ```bash
   chainlit run scenario2.py
   ```
2. உங்கள் டெர்மினலில் கொடுக்கப்பட்ட உள்ளூர் URL (எ.கா., http://localhost:8000) ஐ உலாவியில் திறக்கவும்.
3. உரையாடல் சாளரத்தில் உங்கள் படிப்பு தலைப்பையும் நீங்கள் படிக்க விரும்பும் வாரங்களின் எண்ணிக்கையையும் உள்ளிடவும் (எ.கா., "AI-900 செர்டிபிகேஷன், 8 வாரங்கள்").
4. பயன்பாடு வாரத்திற்கு வார படிப்பு திட்டத்துடன் பதில் அளிக்கும், அதில் தொடர்புடைய Microsoft Learn ஆவணங்களுக்கான இணைப்புகள் அடங்கும்.

**தேவைப்படும் சுற்றுச்சூழல் மாறிலிகள்:**

நிகழ்ச்சி 2 (Azure OpenAI உடன் Chainlit வலை பயன்பாடு) பயன்படுத்த, நீங்கள் `python` கோப்புறையில் `.env` கோப்பில் கீழ்காணும் சுற்றுச்சூழல் மாறிலிகளை அமைக்க வேண்டும்:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

பயன்பாட்டை இயக்கும் முன், உங்கள் Azure OpenAI வள விபரங்களுடன் இவற்றை நிரப்பவும்.

> [!TIP]
> நீங்கள் [Microsoft Foundry](https://ai.azure.com/) மூலம் உங்கள் சொந்த மாதிரிகளையும் எளிதாக செயல்படுத்தலாம்.

### நிகழ்ச்சி 3: VS Code இல் MCP சர்வருடன் ஆடியோக்கியல் ஆவணங்கள்

உலாவி டேப்களை மாற்றி ஆவணங்களைத் தேடுவதற்குப் பதிலாக, Microsoft Learn Docs ஐ நேரடியாக உங்கள் VS Code க்கு MCP சர்வர் பயன்படுத்தி கொண்டு வர முடியும். இதனால் நீங்கள்:
- உங்கள் குறியாக்க சூழலை விட்டு வெளியேறாமலே VS Code சொல்லி ஆவணங்களை தேடி படிக்க முடியும்.
- README அல்லது பாடத்திட்ட கோப்புகளில் நேரடியாக ஆவண குறிப்புக்கள் மற்றும் இணைப்புகளைச் சேர்க்கலாம்.
- GitHub Copilot மற்றும் MCP ஐ இணைத்து ஒரு எளிமையான, AI-ஆதாரப்பட்ட ஆவண வேலைநிரலை பெறலாம்.

**காட்சிப்படுத்தப்பட்ட பயன்பாடுகள்:**
- பாடத்திட்டம் அல்லது திட்ட ஆவணத்தை எழுதும் பொழுது README இல் குறிப்பு இணைப்புகளை விரைவில் சேர்க்க.
- Copilot கொண்டு குறியீட்டை உருவாக்கி MCP மூலம் உடனே பொருத்தமான ஆவணங்களை கண்டுபிடி.
- உங்கள் தொகுப்பியில் கவனம் செலுத்தி உற்பத்தித்திறனை மேம்படுத்து.

> [!IMPORTANT]
> உங்கள் பணியிடத்தில் செல்லுபடியாகும் [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) கட்டமைப்பை உறுதி செய்யவும் (இது `.vscode/mcp.json` இல் இருக்கிறது).

## ஏன் நிகழ்ச்சி 2 க்காக Chainlit?

Chainlit என்பது உரையாடல் அடிப்படையிலான வலை பயன்பாடுகளை உருவாக்குவதற்கான நவீன திறந்த முகவரி கட்டமைப்பு. இது Microsoft Learn Docs MCP சர்வர் போன்ற பின்புல சேவைகளுடன் இணைக்கக்கூடிய சாட் சார்ந்த பயனர் இடைமுகங்களை எளிதாக்குகிறது. இந்த திட்டம் தனிப்பயன் படிப்பு திட்டங்களை நேரத்தில் உருவாக்க எளிய, தொடர்புடைய வழியை வழங்க Chainlit ஐப் பயன்படுத்துகிறது. Chainlit கொண்டு, உற்பத்தித்திறன் மற்றும் கற்றலை மேம்படுத்தும் சாட் அடிப்படையிலான கருவிகளை விரைவாக உருவாக்கி வெளியிடலாம்.

## இதன் செயல்பாடு

இந்த பயன்பாடு பயனர்களுக்கு ஒரு தலைப்பு மற்றும் கால அவகாசத்தை நுழைத்து தனிப்பயன் படிப்பு திட்டத்தை உருவாக்க அனுமதிக்கிறது. பயன்பாடு உங்கள் உள்ளீட்டை பகுப்பாய்வு செய்து, Microsoft Learn Docs MCP சர்வராக்ஷியுடன் தொடர்பு கொண்டு பொருத்தமான உள்ளடக்கத்தை கேட்டு, முடிவுகளை.week-to-week திட்டமாக ஒழுங்குபடுத்துகிறது. ஒவ்வொரு வாரத்திற்குமான பரிந்துரைகள் உரையாடலில் காட்டப்பட்டு, உங்கள் முன்னேற்றத்தை எளிதாக பின்தொடரலாம். ஒரு ஒருங்கிணைப்பு மூலம் நீங்கள் எப்போதும் புதிய, எனவு பொருத்தமான கற்றல் வளங்களை பெற்றுக்கொள்வீர்கள்.

## மாதிரி கேள்விகள்

இருந்து கேள்விகளை உரையாடல் சாளரத்தில் முயற்சி செய்து, பயன்பாடு எப்படி பதிலளிப்பதை காணுங்கள்:

- `AI-900 செர்டிபிகேஷன், 8 வாரங்கள்`
- `Azure Functions கற்றுக்கொள்ளவும், 4 வாரங்கள்`
- `Azure DevOps, 6 வாரங்கள்`
- `Azure இல் தரவு பொறியியல், 10 வாரங்கள்`
- `Microsoft பாதுகாப்பு அடித்தளம், 5 வாரங்கள்`
- `Power Platform, 7 வாரங்கள்`
- `Azure AI சேவைகள், 12 வாரங்கள்`
- `மேக கட்டமைப்பு, 9 வாரங்கள்`

இந்த உதாரணங்கள் பயன்பாட்டின் வெவ்வேறு கற்றல் குறிக்கோள்களுக்கும் காலக்கெடுகளுக்கும் எதிர்நோக்கிய தகுதியை காட்டுகின்றன.
