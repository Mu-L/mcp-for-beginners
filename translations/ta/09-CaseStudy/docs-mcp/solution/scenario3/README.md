# காட்சி 3: VS Code இல் MCP சர்வருடன் உள்ளீட்டாளரின் ஆவணங்கள்

## அவலோகவிடம்

இந்த காட்சியில், Microsoft Learn Docs ஐ நேரடியாக உங்கள் Visual Studio Code சுற்றுச்சூழலில் MCP சர்வர் மூலம் கொண்டு வருவது எப்படி என்பதை நீங்கள் கற்றுக்கொள்ளப்போகிறீர்கள். அதிகாரப்பூர்வ ஆவணங்களைத் தேட இன்புடிர் பரிசுத்தங்களைத் தாங்கிய இணைய உலாவி தாவல்களை தொடர்ச்சியாக மாற்றாமல், நீங்கள் உங்கள் உள்ளீட்டாளருக்குள் நேரடியாக அட்டவணைகள் அணுகல், தேடல் மற்றும் குறிப்பிட்டு பயன்படுத்த முடியும். இது உங்கள் பணிவழியை எளிதாக்கி, கவனம் தக்கவைத்து, GitHub Copilot போன்ற கருவிகளுடன் சரளமான ஒருங்கிணைப்பை உண்டாக்கும்.

- VS Code இல் உங்கள் குறியிடும் சூழலை விட்டு வெளியேறாமல் ஆவணங்களை தேடவும் படிக்கவும்.
- ஆவணங்களை குறிப்பிட்டு உங்கள் README அல்லது பாடநெறி கோப்புகளில் நேரடியாக இணைப்புகளைச் சேர்க்கவும்.
- GitHub Copilot மற்றும் MCP ஐ இணைத்து, பரபரப்பில்லா, AI இயக்கப்படும் ஆவண பணிவழியைப் பயன்படுத்தவும்.

## கற்றல் குறிக்கோள்கள்

இந்த அத்தியாயம் முடிவடையும் போது, VS Code இல் MCP சர்வரை அமைத்து பயன்படுத்தி உங்கள் ஆவண மற்றும் வளர்ப்பு பணிவழியை மேம்படுத்துவது எப்படி என்பது புரிய தயாராக இருப்பீர்கள். நீங்கள்:

- ஆவண தேடலுக்காக MCP சர்வரை பயன்படுத்த உங்கள் பணிச்சூழலை அமைக்க முடியும்.
- VS Code உள்ளே இருந்து நேரடியாக ஆவணங்களை தேடி சேர்க்க முடியும்.
- அதிகும், AI-வளமான பணிவழிக்கு GitHub Copilot மற்றும் MCP திறனை சேர்ந்தடிக்கை செய்ய முடியும்.

இந்த திறன்கள் உங்களுக்கு கவனத்தை நிலைத்திருக்க, ஆவண தரத்தை மேம்படுத்த, மற்றும் வளர்ப்பாளராக அல்லது தொழில்நுட்ப எழுத்தாளர் ஆக உற்பத்தித்திறனை அதிகரிக்க உதவும்.

## தீர்வு

உள்ளீட்டாளரின் ஆவண அணுகலை எட்ட, MCP சர்வரை VS Code மற்றும் GitHub Copilot உடன் ஒருங்கிணைக்கும் ஒரு தொடர் நடவடிக்கைகளை நீங்கள் பின்பற்றுவீர்கள். இந்த தீர்வு பாடநெறி எழுத்தாளர்கள், ஆவணம் எழுத்தாளர்கள் மற்றும் ஆவணங்களுடன் மற்றும் Copilot யுடன் வேலை செய்யும் போது உங்கள் கவனத்தை உள்ளீட்டாளரிலேயே வைத்திருக்க விரும்புவோருக்கு உகந்தது.

- பாடநெறி அல்லது திட்ட ஆவணங்களை எழுதும் போது README இல் விரைவாக குறிப்பான இணைப்புகளைச் சேர்க்கவும்.
- Copilot மூலம் குறியீடு உருவாக்கி MCP மூலம் உடனுக்குடன் பொருத்தமான ஆவணங்களை கண்டறிந்து மேற்கோள் இடவும்.
- உங்கள் உள்ளீட்டாளரில் கவனம் செலுத்தி, உற்பத்தித்திறனை அதிகரிக்கவும்.

### படிப்படியாக வழிகாட்டி

தொடக்கம் எடுக்க, கீழ்க்காணும் படிகளை பின்பற்றவும். ஒவ்வொரு படியுக்கும், செயல்முறையை காட்சிப்படுத்த உருவகப்படத்தை assets பணியில் சேர்க்கலாம்.

1. **MCP கட்டமைப்பைச் சேர்க்க:**
   உங்கள் திட்ட வேரியில் `.vscode/mcp.json` கோப்பை உருவாக்கி கீழ்காணும் கட்டமைப்பைச் சேர்:
   ```json
   {
     "servers": {
       "LearnDocsMCP": {
         "url": "https://learn.microsoft.com/api/mcp"
       }
     }
   }
   ```
   இந்த கட்டமைப்பு VS Code க்கு [`Microsoft Learn Docs MCP சர்வர்`](https://github.com/MicrosoftDocs/mcp) உடன் எப்படி இணையவேண்டும் என்பதை கூறுகிறது.
   
   ![Step 1: Add mcp.json to .vscode folder](../../../../../../translated_images/ta/step1-mcp-json.c06a007fccc3edfa.webp)
    
2. **GitHub Copilot Chat குழுமத்தை திற:**
   நீங்கள் GitHub Copilot விரிவாக்கத்தை அன்செய நல்லதில்லை என்றால், VS Code இல் Extensions தோற்றம் செல்லவும் மற்றும் அதை நிறுவவும். நீங்கள் அதை நேரடியாக [Visual Studio Code சந்தையில்](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot-chat) பதிவிறக்கம் செய்யலாம். பின்னர், Copilot Chat குழுமத்தை பக்ககாட்டியில் திறக்கவும்.

   ![Step 2: Open Copilot Chat panel](../../../../../../translated_images/ta/step2-copilot-panel.f1cc86e9b9b8cd1a.webp)

3. **ஏஜெண்ட் பயன்முறையை இயக்கி கருவிகளை சரிபார்:**
   Copilot Chat குழுமத்தில் ஏஜெண்ட் பயன்முறையை இயக்கு.

   ![Step 3: Enable agent mode and verify tools](../../../../../../translated_images/ta/step3-agent-mode.cdc32520fd7dd1d1.webp)

   ஏஜெண்ட் பயன்முறையை இயக்கு பின்னர், MCP சர்வர் கிடைக்கும் கருவிகளில் ஒன்றாக பட்டியலிடப்பட்டுள்ளதா என்று உறுதி செய்யவும். இது Copilot ஏஜெண்ட் ஆவண சர்வரை அணுகி பொருத்தமான தகவலை பெறும் என்பதைக் குறிப்பதாகும்.
   
   ![Step 3: Verify MCP server tool](../../../../../../translated_images/ta/step3-verify-mcp-tool.76096a6329cbfecd.webp)
4. **புதிய உரையாடலை துவங்கி ஏஜெண்டை தூண்டுக:**
   Copilot Chat குழுமத்தில் புதிய உரையாடலைத் திறக்கவும். இப்போது உங்கள் ஆவண கேள்விகளை ஏஜெண்ட்டிற்கு வழங்கலாம். ஏஜெண்ட் MCP சர்வரை பயன்படுத்தி Microsoft Learn ஆவணத்தை நேரடியாக உங்கள் உள்ளீட்டாளரில் காட்சிப்படுத்தும்.

   - *"நான் X விஷயத்திற்கு 8 வாரங்கள் படிக்க ஒரு படிப்புத்திட்டம் எழுத முயற்கிறேன். ஒவ்வொரு வாரத்திற்கும் எனக்கு எடுக்க வேண்டிய உள்ளடக்கங்களை பரிந்துரைக்கவும்."*

   ![Step 4: Prompt the agent in chat](../../../../../../translated_images/ta/step4-prompt-chat.12187bb001605efc.webp)

5. **படிப்படை கேள்வி:**

   > Microsoft Foundry Discord இல் உள்ள [#get-help](https://discord.gg/D6cRhjHWSC) பிரிவிலிருந்து ஒரு நேரடி கேள்வியை எடுத்துக்கொள்வோம் ([அசல் செய்தியைப் பார்க்க](https://discord.com/channels/1113626258182504448/1385498306720829572)):
   
   *"Azure AI Foundry இல் உருவாக்கிய AI ஏஜெண்ட்கள் கொண்ட பல ஏஜெண்ட் தீர்வை வரைபடமாக்குவது எப்படி என்பதில் நான் பதில்களை தேடுகிறேன். Copilot Studio சேனல் போன்ற நேரடி வரைபடம் முறை இல்லாதது போல உள்ளது. எனவே, ஏன் மற்றும் எவ்வாறு தொழில் பயனாளர்களுக்கு இணைய முகாமையாளராக இருப்பதற்கு மற்றும் பணியினை இயற்படுத்துவதற்கு பல்வேறு முறைகள் உள்ளன?
Azure Bot சேவை MS குழுக்கள் மற்றும் Azure AI Foundry ஏஜெண்ட்களை இடையில் پل நகர்த்தும் பணியை செய்வதாகக் கூறும் பல கட்டுரைகள்/வலைப்பதிவுகள் உள்ளன, நான் Azure function மூலம் Orchestrator Agent உடன் Azure bot ஐ அமைத்தால் இது வேலை செய்யுமா அல்லது பல ஏஜெண்ட் தீர்வின் ஒவ்வொரு AI ஏஜெண்டுக்கும் Bot கட்டமைப்பில் orchestration செய்ய Azure function உருவாக்க வேண்டுமா? வேறு எந்த ஆலோசனைகளும் வரவேற்கப்படுகின்றன."*

   ![Step 5: Live queries](../../../../../../translated_images/ta/step5-live-queries.49db3e4a50bea273.webp)

   ஏஜெண்ட் பொருத்தமான ஆவண இணைப்புகள் மற்றும் சுருக்கங்களை வழங்கி பதில் அளிக்கும், அவற்றை நீங்கள் markdown கோப்புகளில் நேரடியாகச் சேர்க்கலாம் அல்லது உங்கள் குறியீட்டில் குறிப்பிடலாம்.
   
### உதாரண கேள்விகள்

இங்கே நீங்கள் முயற்சிக்கக்கூடிய சில உதாரண கேள்விகள் உள்ளன. இந்த கேள்விகள் MCP சர்வரும் Copilot என்றும் ஒருங்கிணைந்து விரைவாக, உள்ளடக்க உணர்வுடன் கூடிய ஆவணங்கள் மற்றும் குறிப்புகளை VS Code விட்டு வெளியேறாதே வழங்கும் முறையை காண்பிக்கும்:

- "Azure Functions டிரிகர்கள் பயன்படுத்தும் விதம் காட்டவும்."
- "Azure Key Vault அதிகாரப்பூர்வ ஆவணத்திற்கான இணைப்பைச் சேர்க்கவும்."
- "Azure வளங்களை பாதுகாப்பதற்கான சிறந்த நடைமுறைகள் என்ன?"
- "Azure AI சேவைகளுக்கு ஒரு விரைவுகூடிய துவக்கம் காண்பிக்கவும்."

இந்த கேள்விகள் MCP சர்வர் மற்றும் Copilot இணைந்து வேலை செய்யும் முறையை வெளிப்படுத்தும், அப்படியே VS Code விட்டு வெளியேறாமலேயே தானாகவே நிலைப்பட்டிருக்கும் உள்ளடக்கம் மற்றும் குறிப்புகளை வழங்கும்.
