# ஆதரவு

## பிரச்சனைகளை பதிவு செய்வது மற்றும் உதவி பெறுவது எப்படி  

இந்த திட்டம் பிழைகளையும் அம்ச கோரிக்கைகளையும் கண்காணிப்பதற்கு GitHub Issues ஐ பயன்படுத்துகிறது. அச்செய்யப்பட்ட பிரச்சனைகளைத் தவிர்க்க புதிய பிரச்சனைகளை பதிவு செய்வதற்கு முன் உள்ள பிரச்சனைகளை தேடவும். புதிய பிரச்சனைகள் இருந்தால் உங்களது பிழை அல்லது அம்ச கோரிக்கையை புதிய பிரச்சனையாக பதிவு செய்யவும்.

இந்தத் திட்டத்தைப் பயன்படுத்தி உதவி மற்றும் கேள்விகளுக்கு, தயவு செய்து:
- நிபுணர்கள் மற்றும் சக தயாரிப்பாளர்கள் கூடும் [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ஐ சரிபார்க்கவும்
- முனை தொடர்புடைய கேள்விகளுக்கு [Model Context Protocol Community](https://modelcontextprotocol.io/community/) வளங்களைப் பயன்படுத்தவும்

## Microsoft ஆதரவு கொள்கை  

Model Context Protocol (MCP) for Beginners திட்டத்துக்கான ஆதரவு மேலே பட்டியலிடப்பட்ட வளங்களுக்குதான் வரையறுக்கப்பட்டுள்ளது.
