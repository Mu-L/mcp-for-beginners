# 🚀 தொடக்கக்காரர்களுக்கான மாடல் சூழல் நடைமுறை (MCP) பாடத்திட்டம்

## **C#, Java, JavaScript, Rust, Python மற்றும் TypeScript ஆகிய மொழிகளில் கையாளும் குறியீடுகளுடன் MCP கற்றுக்கொள்ளவும்**

## 🧠 மாடல் சூழல் நடைமுறை பாடத்திட்டத்தின் மேலோட்டம்
மாடல் சூழல் நடைமுறைக்கான உங்கள் பயணத்திற்கு வரவேற்கிறோம்! ஏதேனும் AI பயன்பாடுகள் வெவ்வேறு கருவிகள் மற்றும் சேவைகளுடன் எப்படிப் பேசுகின்றன என்று நீங்கள் யோசித்திருந்தால், இங்கு அந்த அழகான தீர்வை கண்டுபிடிக்கப் போகிறீர்கள், இது எப்படிச் செவிலியர்கள் நுண்ணறிவு அமைப்புகளை உருவாக்குகின்றனர் என்பதை மாற்றிவிட்டது.

MCP ஐ AI பயன்பாடுகளுக்கான பொது மொழிபெயர்ப்பாளராக நினைக்கவும் — USB போர்டுகள் உங்கள் கணினியில் எந்த சாதனத்தையும் இணைக்க உதவுவதுபோல், MCP AI மாதிரிகளுக்கு எந்த கருவி அல்லது சேவையுடனும் ஒரே மாதிரியான முறையில் இணைக்க உதவுகிறது. நீங்கள் உங்கள் முதலாவது உரையாடல் இயந்திரத்தை உருவாக்கினாலும், அல்லது சிக்கலான AI வேலைவாய்ப்புகளில் ஈடுபடுகிறீர்களானால், MCP ஐப் புரிந்துகொள்வது அதிகமாகச் செயல்படும் மற்றும் நெகிழ்வூட்டுகின்ற பயன்பாடுகளை உருவாக்கும் சக்தியை உங்களுக்கு வழங்கும்.

இந்த பாடத்திட்டம் உங்கள் கற்றல் பயணத்திற்கு பொறுமையுடன் மற்றும் கவனத்துடன் வடிவமைக்கப்பட்டுள்ளது. நீங்கள் ஏற்கனவே புரிந்துகொண்ட எளிய கருத்துக்களிலிருந்து துவங்கி, உங்கள் விருப்பமான நிரலாக்க மொழியில் நன்கு கையாளும் பயிற்சிகளின் மூலம் உங்கள் திறமையை கட்டியெழுப்புவோம். ஒவ்வொரு படியிலும் தெளிவான விளக்கங்கள், நடைமுறை உதாரணங்கள் மற்றும் நிறைய ஊக்கமளிப்புகள் உள்ளன.

இந்த பயணத்தை முடித்துள்ள காலத்தில், நீங்கள் உங்கள் சொந்த MCP சேவையகங்களை உருவாக்கும் தெளிவையும், பிரபலமான AI தளங்களுடன் இணைக்கும் திறனையும், இந்த தொழில்நுட்பம் AI அபிவிருத்தி எதிர்காலத்தை எப்படிச் சுற்றியுள்ளதென்றும் புரிந்துகொள்வீர்கள். இந்த சுவாரஸ்யமான பயணத்தை தொடங்குவோம்!

## கற்றல் குறிக்கோள்கள்

இந்த பாடத்திட்டத்தை முடிக்கும் போது, நீங்கள் உங்கள் புதிய திறமைகள் பற்றி முழுமையான நம்பிக்கை மற்றும் உற்சாகத்தில் இருப்பீர்கள். எதனை அடையப்போகிறீர்கள் என்பதை இங்கே காண்கிறீர்கள்:

• **MCP அடிப்படைகளை புரிந்துகொள்வது**: மாடல் சூழல் நடைமுறை என்றால் என்ன, AI பயன்பாடுகள் எப்படி ஒன்றாகவே செயல்படுவதாக மாற்றிவிடுகிறது என்பதை, தெளிவான உதாரணங்கள் மற்றும் விளக்கங்களுடன் நாம் புரிந்துகொள்ளுவோம்.

• **முதலாவது MCP சேவையகத்தை உருவாக்குதல்**: உங்கள் விருப்ப மொழியில் செயல்படும் MCP சேவையகத்தை எளிய எடுத்துக்காட்டுகளுடன் ஆரம்பித்து, படி படியாக திறமையை மேம்படுத்துவீர்கள்.

• **AI மாதிரிகளை நிஜ கருவிகளுடன் இணைக்கும் திறன்**: AI மாதிரிகளுக்கும் உண்மை சேவைகளுக்கும் இடையேயான இடைவெளியை எப்படி நிரப்புவது என்பதை கற்பீர்கள், உங்கள் பயன்பாடுகளுக்கு பல வலுவான திறன்களை அளிக்கும்.

• **பாதுகாப்பு சிறந்த நடைமுறைகளை செயல்படுத்தல்**: உங்கள் MCP நடைமுறைகளை பாதுகாப்பாகவும், உங்கள் பயன்பாடுகளையும் பயனர்களையும் பாதுகாக்கவும் எப்படி செய்வது என்பதை விளங்குவீர்கள்.

• **நம்பிக்கையுடன் விரம்புதல்**: உங்கள் MCP திட்டங்களை உருவாக்கிய நிலையில் இருந்து தயாரிப்பு நிலைக்கு எவ்வாறு கொண்டு வருவது என்பதை, நடைமுறை விரம்பும் முறைமைகளை கற்றுக் கொள்வீர்கள்.

• **MCP சமூகத்தில் சேரல்**: AI பயன்பாட்டுத் துறையின் எதிர்காலத்தை வடிவமைக்கும் விரிவாகிற சமூகத்தின் ஒரு பகுதியாக நீங்கள் இருப்பீர்கள். 

## அவசியமான பின்னணி அறிவு

MCP குறிப்புகளுக்குள் செல்வதற்குமுன், சில அடிப்படை கருத்துக்களை நீங்கள் நன்றாக புரிந்திருப்பீர்கள் என்பதை உறுதிப்படுத்திக்கொள்ளலாம். நீங்கள் அவற்றில் நிபுணர் அல்லாவிட்டாலும் கவலைப்பட வேண்டாம் - நாம் எல்லாவற்றையும் தேவையானபடி விளக்கும்!

### நடைமுறை என்பதைப் புரிந்து கொள்ளுதல் (அடித்தளம்)

நடைமுறை என்பது உரையாடலில் விதிகள் போல தான். நண்பரை அழைக்கும் பொழுது, நீங்கள் "ஹலோ" என்று சொல்ல வேண்டும் என்று இருவரும் தெரிந்திருக்கும், பேசும் முறையை மாற்றிக் கொள்வீர்கள், முடிந்தபின் "பிரியா" சொல்லுவீர்கள். கணினி செயலிகள் சிறப்பாக தொடர்பு கொள்ள இதே விதமான விதிகளைக் கடைபிடிக்கவேண்டும்.

MCP என்பது படிமுறையாக ஆகிய விதிமுறைகளின் தொகுப்பு, இது AI மாதிரிகள் மற்றும் பயன்பாடுகள் கருவிகள் மற்றும் சேவைகளுடன் பயனுள்ள "உரையாடல்கள்" மேற்கொள்ள உதவுகிறது. மனித உரையாடலில் விதிகள் உள்ளதால் பேச்சு மென்மையாக இருப்பது போல், MCP வை உருவாக்குவதால் AI பயன்பாடு தொடர்புகள் நம்பகமாகவும் வலுவாகவும் இருக்கும்.

### க்ளையன்ட்-சேவரர் உறவுகள் (செயலிகள் ஒன்றாக இணையும் முறை)

நீங்கள் தினமும் க்ளையன்ட்-சேவரர் உறவைப் பயன்படுத்துகிறீர்கள்! இணைய உலாவர் (க்ளையன்ட்) மூலம் இணையதளத்தைப் பார்க்கும்போது, நீங்கள் அந்த இணையத்தள பக்கத்தை அனுப்பும் இணைய சேவருடன் இணைகிறீர்கள். உலாவி தகவலை கேட்கும் முறையும், சேவர் பதில் அளிக்கும் முறையும் தெரியும்.

MCP யில் இதே மாதிரியான உறவு உள்ளது: AI மாதிரிகள் தகவல் அல்லது செயல் கோரல் தெரிவித்துக் கொள்ளும் க்ளையன்டாக செயல்படுகிறது, MCP சேவர்கள் அந்த திறன்களை வழங்குகின்றன. உதவியாளராக செயல்படும் சேவரை வைத்து AI கூறிய பணிகளைச் செய்யும்படி கேட்டுக்கொள்ளும் வகை.

### நிலையான அமைப்பு ஏன் அவசியம் (ஒன்றிணைந்த செயல்பாடு)

ஒவ்வொரு கார் தயாரிப்பாளரும் வித்தியாசமான அமைப்பிற்கான எரிபொருள் பம்புகளை உபயோகித்தால் - ஒவ்வொரு காருக்கும் தனி அடாப்டர் தேவைப்படும்! நிலையான அமைப்பு பொருள்கள் ஒருங்கிணைத்து வேலை செய்ய சொந்த விசைப்பலகைகளை ஒத்துக் கொள்வதே ஆகும்.

MCP AI பயன்பாடுகளுக்கான இந்த நிலையான அமைப்பை வழங்குகிறது. ஒவ்வொரு AI மாதிரிக்கு ஒவ்வொரு கருவியுடனும் தனித்துவமான குறியீட்டு தேவையைத் தவிர்த்து, MCP மூலம் அனைவரும் ஒரே விதமாக தொடர்பு கொள்ள முடியும். இதனால், உருவாக்குநர்கள் ஒருமுறை கருவிகளை உருவாக்கி பல வித்தியாசமான AI அமைப்புகளில் இயங்கச் செய்யலாம்.

## 🧭 உங்கள் கற்றல் பாதை மேலோட்டம்

MCP பயணம் நெருக்கமாக அமைக்கப்பட்டுள்ளது, உங்கள் நம்பிக்கையையும் திறமையையும் படிப்படியாக வளர்க்கிறது. ஒவ்வொரு கட்டமும் புதிய கருத்துக்களை அறிமுகம் செய்வதுடன் அதற்கெல்லாம் இறுதியில் மீண்டும் உறுதிப்படுத்துகிறது.

### 🌱 அடித்தளம் கட்டம்: அடிப்படைகளை புரிந்துகொள்ளுதல் (மாடுல்கள் 0-2)

இங்கே உங்கள் பயணம் துவங்குகிறது! MCP கருத்துக்களை அறிமுகப்படுத்துவோம், பரிச்சயமான ஒப்புமைகள் மற்றும் எளிய எடுத்துக்காட்டுக்களுடன். MCP என்றால் என்ன, அது ஏன் முக்கியம் ஆனது மற்றும் AI அபிவிருத்தி உலகத்தில் எவ்வாறு பொருந்தும் என்பதைப் புரிந்துகொள்ளுவீர்கள்.

• **மாடுல் 0 - MCP அறிமுகம்**: MCP எப்படிச் செயல்படுகிறது என்பது மற்றும் இதன் முக்கியத்துவம் பற்றி அறியும். MCP இயங்கும் நிஜ உலக பயிற்சிகளைக் காண்பீர்கள் மற்றும் உருவாக்குநர்கள் எதிர்கொள்ளும் வழக்கமான பிரச்சனைகளை எப்படி தீர்க்கிறது என்பதையும் புரிதல் பெறுவீர்கள்.

• **மாடுல் 1 - முக்கிய கருத்துகள் விளக்கம்**: MCP அடிப்படை கூறுகளை அடையக் கற்றுக்கொள்ளவும். இந்த அதிகாரத்தில் நிறைய ஒப்புமைகள் மற்றும் காட்சிய உதாரணங்கள் தரப்படும், இதனால் MVC கருதுகோள்கள் இயல்பாகவும் புரிந்துகொள்ளக்கூடியதாகவும் இருக்கும்.

• **மாடுல் 2 - MCP இல் பாதுகாப்பு**: பாதுகாப்பு பயமுறுத்துவது போல இருக்கும், ஆனால் MCP எப்படி உள்ளமைந்த பாதுகாப்பு அம்சங்களைக் கொண்டுள்ளது என்றும் உங்கள் பயன்பாடுகளை ஆரம்பத்திலிருந்தே பாதுகாக்க கூடிய சிறந்த நடைமுறைகளை கற்றுக்கொடுக்கிறோம்.

### 🔨 கட்டுமான கட்டம்: உங்கள் முதல் செயலாக்கங்களை உருவாக்குதல் (மாடுல் 3)
இப்போது உண்மையான வேடிக்கை ஆரம்பமாகிறது! நீங்கள் நிஜ MCP சேவையகங்கள் மற்றும் கிளையண்ட்களை உருவாக்கும் நடைமுறை அனுபவத்தை பெறுவீர்கள். கவலைப்படாதீர்கள் - நாம் எளிமையாகத் தொடங்கி ஒவ்வொரு படியையும் வழிகாட்டுவோம்.

இந்த மாட்யூலில் நீங்கள் விருப்பமான நிரலாக்க மொழியில் பயிற்சி செய்ய பல நடைமுறை வழிகாட்டிகள் உள்ளன. நீங்கள் உங்கள் முதல் சேவையகத்தை உருவாக்கி, அதுடன் இணைப்பதற்கான கிளையண்டை கட்டி, மற்றும் VS Code போன்ற பிரபல மேம்பாட்டு கருவிகளுடனும் ஒருங்கிணைக்கப் போகிறீர்கள்.

ஒவ்வொரு வழிகாட்டியிலும் முழுமையான குறியீட்டு உதாரணங்கள், பிழைதிருத்தக் குறிப்புகள் மற்றும் நாம் குறிப்பிட்ட வடிவமைப்பு முடிவுகளை எதனால் எடுக்கிறோம் என்பதை விளக்குகிறது. இந்த கட்டத்தின் முடிவில், நீங்கள் பெருமையடையக்கூடிய செயல்படும் MCP அமல்படுத்தல்களை தோற்றுவிக்க இருப்பீர்கள்!

### 🚀 வளர்ச்சி கட்டம்: மேம்பட்ட கருத்துக்களும் நிஜ உலகில் பயன்பாடும் (மாட்யூல்கள் 4-5)

அடிப்படைகள் தெரிந்து கொண்ட பிறகு, நீங்கள் சொகுசான MCP அம்சங்களை ஆராயத் தயாராக இருக்கிறீர்கள். நடைமுறை அமல்படுத்தல் நுட்பங்கள், பிழைதிருத்த முறைகள் மற்றும் பல்துறை AI ஒருங்கிணைப்புகள் போன்ற மேம்பட்ட தலைப்புக்களை நாம் கையாள்வோம்.

மேலும், உற்பத்தி பயன்பாட்டிற்கு உங்கள் MCP அமல்படுத்தல்களை எப்படி அளவுரு செய்யுவது மற்றும் Azure போன்ற மேக தளங்களின் ஒருங்கிணைப்பையும் நீங்கள் கற்றுக்கொள்வீர்கள். இவை நிஜ உலகின் தேவைகளை கையாளக்கூடிய MCP சாதனைகளை கட்டும் முன் تیار்ப்புகளை செய்ய உதவும்.

### 🌟 திறனாய்வு கட்டம்: சமூகம் மற்றும் திறனமைப்பு (மாட்யூல்கள் 6-11)

இறுதி கட்டம் MCP சமூகத்தில் சேருவதையும் மற்றும் உங்களுக்கு மிகவும் பிடிக்கும் துறைகளில் சிறப்பு திறனை பெறுவதை சார்ந்தது. நீங்கள் திறந்த மூல MCP திட்டங்களுக்கு பங்களிப்பது, மேம்பட்ட அங்கீகார மாதிரிகள் மற்றும் முழுமையான தரவுத்தள ஒருங்கிணைந்த தீர்வுகளை உருவாக்குவது போன்றவை கற்றுக் கொள்வீர்கள்.

மாட்யூல் 11 சிறப்பு குறிப்பிடத்தக்கது - இது முழுமையான 13-லேப்கள் கொண்ட நடைமுறை பயிற்சி பாதை ஆகும், இதில் நீங்கள் PostgreSQL ஒருங்கிணைப்புடன் உற்பத்திக்கு தயாரான MCP சேவையகங்களை உருவாக்க கற்றுக்கொள்கிறீர்கள். இது நீங்கள் கற்ற அனைத்தையும் ஒன்றிணைக்கும் முக்கிய திட்டப் போன்றது!

## 📚 கற்கை வழிகாட்டி மற்றும் ஆதாரங்கள்

இந்த களஞ்சியம் பல ஆதாரங்களை உள்ளடக்கியுள்ளது, இது உங்களுக்கு வழிமுறையை அறிவதில் உதவும்:

### கற்கை வழிகாட்டி

ஒரு முழுமையான [கற்கை வழிகாட்டி](./study_guide.md) உள்ளடக்கத்துடன் இந்த களஞ்சியத்தை திறம்பட பயன்படுத்த உதவுகிறது. இந்த காட்சி பாடத்திட்ட வரைபடம் அனைத்து தலைப்புகளும் எவ்வாறு இணைக்கப்பட்டுள்ளன என்பதை göstermுகின்றது மற்றும் உதாரண திட்டங்களை திறம்பட எப்படி பயன்படுத்துவது என்பதை வழிகாட்டுகிறது. இது பெரும்பாலும் மட்டும் பார்வையாளர்களுக்கு மிகவும் உதவியாக இருக்கும்.

வழிகாட்டியில் உள்ளன:
- அனைத்து தலைப்புகளை காட்டும் காட்சி பாடத்திட்ட வரைபடம்
- ஒவ்வொரு களஞ்சியப் பகுதிக்கும் விரிவான பகுப்பு
- உதாரண திட்டங்களை எவ்வாறு பயன்படுத்துவது என்பதை வழிகாட்டுதல்
- வேறுபட்ட திறன்களின் அடிப்படையில் பரிந்துரைக்கப்பட்ட கற்றல் பாதைகள்
- உங்கள் கற்றலை ஒத்துழைக்கும் கூடுதல் ஆதாரங்கள்

### மாற்றம் பதிவேடு

பாடத்திட்டப் பொருட்களில் அனைத்து முக்கியமான புதுப்பிப்புகளை பதிவுசெய்ய நாங்கள் ஒரு விரிவான [மாற்றம் பதிவேடு](./changelog.md) பராமரிக்கிறோம், ஆகையால் நீங்கள் சமீபத்திய மேம்பாடுகள் மற்றும் சேர்க்கைகளைப் பற்றி பட்டியலிடலாம்.
- புதிய உள்ளடக்கம் சேர்க்கைகள்
- கட்டமைப்பு மாற்றங்கள்
- அம்ச மேம்பாடுகள்
- ஆவண புதுப்பிப்புகள்

## 🎓 உங்கள் MCP பயணம் தொடங்கியது

 வாழ்த்துகள்! நீங்கள் உங்கள் குறுந்தொகுப்பு திறன்களை விரிவுசெய்து AI வளர்ச்சியின் முன்வாசலில் இணையும் ஒரு சுவாரஸ்யமான பயணத்தின் முதல் படியை எடுத்து உள்ளீர்கள்.

### நீங்கள் ஏற்கனவே எதை சாதித்துள்ளீர்கள்

இந்த அறிமுகத்தை வாசிப்பதன் மூலம், நீங்கள் MCP அறிவு அடித்தளத்தை உருவாக்க ஆரம்பித்துள்ளீர்கள். MCP என்பது என்ன, ஏன் இது முக்கியம், மற்றும் இந்த பாடத்திட்டம் உங்கள் கற்றல் பயணத்தை எவ்வாறு ஆதரிக்கிறது என்பதைக் கருத்தில் கொண்டு இருக்கிறீர்கள். இது முக்கியமான சாதனை மற்றும் இந்த முக்கிய தொழில்நுட்பத்தில் உங்கள் திறமையின் துவக்கம் ஆகும்.

### எதிர்கால  சவால்கள்

நீங்கள் படிப்பை தொடர அனுக்கொள்ளுங்கள், ஒவ்வொரு நிபுணர் ஒருகாலையில் ஒரு தொடர் செய்தியாளர் என்பதைக் நினைவில் வையுங்கள். இப்போது சிக்கலாக தெரியும் கருத்துக்கள் பயிற்சிப் பெற்று பயன்படுத்தும் போது இயற்கையானதாக மாறும். ஒவ்வொரு சிறிய படியும் உங்கள் வளர்ச்சி மேம்பாட்டின் சக்திவாய்ந்த திறன்களை உருவாக்கும்.

### உங்கள் ஆதரவுக் குழு

MCP பற்றிய ஆர்வமுள்ள கற்றல்பவர்கள் மற்றும் நிபுணர்களுடன் கூடிய சமுதாயத்தில் நீங்கள் சேர்ந்துள்ளீர்கள். குறியீடு சிக்கலில் சிக்கினாலும் அல்லது முன்னேற்றத்தை பகிர்ந்துகொள்ள ஆர்வமுள்ளவர்களும், இந்த சமூக உங்களை ஆதரிக்க உள்ளது.

நீங்கள் சிக்கல் சிக்கினால் அல்லது AI பயன்பாடுகளை உருவாக்குவதற்கான கேள்விகள் இருந்தால், MCP பற்றிய விவாதங்களில் வெளிப்படையான கற்றலாளர்களோடு மற்றும் அனுபவம் வாய்ந்த டெவலப்பர்களோடு கலந்துகொள்ளுங்கள். இது ஆதரவுச் சமூகமாக, கேள்விகள் வரவேற்கப்படுகின்றன மற்றும் அறிவு அரசுசெலுத்தப்படுகிறது.
