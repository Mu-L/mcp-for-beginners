# 🚀 MCP ကိရိယာများ

## 🧠 MCP ဒေတာဘေ့စ် ပေါင်းစည်းခြင်း သင်ယူမှု လမ်းကြောင်းအကြောင်းအရာ

ဒီမှာ သင်သည် MCP ကို အသုံးပြုသည့် ကိရိယာများကို မည်သို့ အသုံးပြုရမည်ကို သင်ယူပါလိမ့်မည်။ MCP ဆာဗာများကို ထို့ ကိရိယာများနှင့် ချိတ်ဆက်သည်နှင့် သင်၏ အတွေ့အကြုံ ပိုမိုကောင်းမွန်လာပါလိမ့်မည်။

## 🔗 အတည်ပြု MCP အရင်းအမြစ်များ

- 📘 [MCP စာတမ်းများ](https://modelcontextprotocol.io/) – အသေးစိတ် သင်ခန်းစာများနှင့် အသုံးပြုသူလမ်းညွှန်များ
- 📜 [MCP ဖော်ပြချက် (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – Protocol ဖွဲ့စည်းပုံနှင့် နည်းပညာဆိုင်ရာ အညွှန်းများ
- 🧑‍💻 [MCP GitHub အဆောက်အအုံ](https://github.com/modelcontextprotocol) – Open-source SDK များ၊ ကိရိယာများနှင့် ကုဒ်နမူနာများ
- 🌐 [MCP အသိုင်းအဝိုင်း](https://github.com/orgs/modelcontextprotocol/discussions) – ဆွေးနွေးပွဲများမှာ ပါဝင်ပြီး အသိုင်းအဝိုင်းသို့ ကျယ်ပြန့်စွာ ဝင်ရောက် ဆောင်ရွက်ပါ
- 🔒 [OWASP MCP ထိပ်တန်း ၁၀ ခု](https://microsoft.github.io/mcp-azure-security-guide/mcp/) – လုံခြုံရေးအကောင်းဆုံးနည်းလမ်းများနှင့် အန္တရာယ်လျော့နည်းစေနည်းများ
