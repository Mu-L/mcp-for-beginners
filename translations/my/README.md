# 🚀 Model Context Protocol (MCP) Curriculum for Beginners

## **C#, Java, JavaScript, Rust, Python, နဲ့ TypeScript အတွက် လက်တွေ့ကုဒ်နမူနာတွေနဲ့ MCP ကို လေ့လာပါ**

## 🧠 Model Context Protocol သင်ရိုးညွှန်းတမ်းအကြောင်းအနှစ်ချုပ်
Model Context Protocol ဖန်တီးသည့် ခရီးစဉ်သို့ ကြိုဆိုပါတယ်။ AI အပလီကေးရှင်းများကို မည်သို့ ကိရိယာနှင့် ဝန်ဆောင်မှုများနှင့် ဆက်သွယ်သလဲဆိုတာ စဉ်းစားဖူးတယ်ဆိုရင် အခုတော့ ဒီနည်းလမ်းက စူးစမ်းရအောင်။ ဒါဟာ တီထွင်သူတွေကို စမတ်စနစ်တွေ ဆောက်လုပ်ရာမှာ ပြောင်းလဲခွင့်ပေးနေသော အောင်မြင်နည်းလမ်းဖြစ်ပါတယ်။

MCP ကို အော်ဟစ်နှင့် AI အပလီကေးရှင်းများအတွက် ယေဘုယျ ဘာသာပြန်ပြောဆိုသူလိုထင်ပါစေ – USB ပေါက်တွေက သင့်ကွန်ပျူတာနဲ့ ဘယ်လိုမဆို စက်ကိရိယာတွေကို ချိတ်ဆက်ပေးနိုင်တဲ့အတိုင်း MCP က AI ပုံစံတွေကို ကိရိယာ သို့ ဝန်ဆောင်မှု မဆို စံပြုနည်းလမ်းနဲ့ ချိတ်ဆက်စေပါတယ်။ သင့်ပထမဆုံး chatbot လုပ်ဖို့ဖြစ်စေ၊ ရှုပ်ထွေးတဲ့ AI workflow တွေဆောက်ဖို့ဖြစ်စေ MCP ကို နားလည်ရင် ပိုမိုစွမ်းရည်မြင့်တဲ့၊ ထိန်းချုပ်နိုင်ရခက်တဲ့ အပလီကေးရှင်းတွေ ဖန်တီးနိုင်မှာဖြစ်ပါသည်။

ဒီသင်ရိုးညွှန်းတမ်းကို သင်ယူသူတိုင်းအတွက် သည်းခံမှုနဲ့ သတိထားပြီး ပြင်ဆင်ထားပါတယ်။ သင်းလွယ်ကူစွာ နားလည်နိုင်တဲ့ အကြောင်းအရာအသစ်များနဲ့ စစတင်ကနေ သင်၏ အကြို သူငယ်ချင်းအနေနဲ့ သင့်နှစ်သက်ရာ Programming ဘာသာစကားအသုံးပြု၍ လက်တွေ့လေ့ကျင့်မှုတွေးငြိမ်း လေ့လာနိုင်မှာ ဖြစ်ပါတယ်။ တစ်ခုခြင်းအား ဖြင့် ရှင်းလင်းချက်များ၊ လက်တွေ့ နမူနာများနဲ့ အားပေးမှုများ တင်ပြပေးမှာဖြစ်ပါသည်။

ဤခရီးစဉ်တစ်ခု ပြီးရာမှာ မိမိရဲ့ ကိုယ်ပိုင် MCP Server များကို ဆောက်လုပ်နိုင်အောင်၊ လူကြိုက်များသော AI ပလပ်ဖောင်းများနှင့် ပေါင်းစပ်အသုံးပြုနိုင်ပြီး ဒီနည်းပညာသည် AI ဖန်တီးမှု၏ အနာဂတ်ကို ဘယ်လို ပြောင်းလဲနေသည်ကို နားလည်နိုင်မှာ ဖြစ်သည်။ လှုပ်ရှားဖွယ် ဒီခရီးစဉ်ကို စတင်လိုက်ရအောင်။

### တရားဝင်စာရွက်စာတမ်းများနှင့် သတ်မှတ်ချက်များ

ဒီသင်ရိုးညွှန်းတမ်းဟာ **MCP Specification 2025-11-25** (နောက်ဆုံးအတည်ပြုထုတ်ပြန်မှု) ကို လိုက်နာထားပါတယ်။ MCP သတ်မှတ်ချက်က ရက်စွဲပေါ် မူတည်ပြီး ဗားရှင်းများ ခြားနားစေသည့် (YYYY-MM-DD ပုံစံ) စနစ်ကို အသုံးပြုပါတယ်။

ဒီရင်းမြစ်တွေကို သင့်နားလည်မှု များလာသလို ပိုမိုတန်ဖိုးရှိလာမယ်၊ ဒါပေမဲ့ ချက်ချင်းအားလုံးကို ဖတ်ရှုရန် ဖိအားမခံပါနဲ့။ သင်စိတ်ဝင်စားတဲ့ပိုင်းတွေနဲ့ စတင်လိုက်ပါ!
- 📘 [MCP Documentation](https://modelcontextprotocol.io/) – စတင်လေ့လာသူတွေအတွက် စနစ်တကျ လေ့လာနိုင်တဲ့ အသေးစိတ် လမ်းညွှန်စာတမ်းဖြစ်ပါသည်။ ကောင်းမွန်သော နမူနာများနှင့် အသုံးပြုသူ လက်လွှတ်စေပါတယ်။
- 📜 [MCP Specification](https://modelcontextprotocol.io/specification/2025-11-25) – သင်ရိုးညွှန်းတမ်း များ လေ့လာရင်း ကိစ္စရပ်အရ များသောအသေးစိတ် နှင့်တကွ အဆင့်မြင့် လုပ္ဆောင်ချက်များကို ရှာဖွေနိုင်ရန် သုံးပါ။
- 📜 [MCP Specification Versioning](https://modelcontextprotocol.io/specification/versioning) – ဤတွင် MCP ဆက်သွယ်စနစ် ဗားရှင်းသမိုင်းနှင့် ရက်စွဲပေါ် ဗားရှင်းစနစ် (YYYY-MM-DD ပုံစံ) အကြောင်း ရှင်းလင်းချက်များပါရှိသည်။
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – SDK များ၊ ကိရိယာများနဲ့ ကုဒ်နမူနာများကို တစ်ခါတည်း လက်လှမ်းမှီစေသည့်အရာများ ရှိပါသည်။ လက်တွေ့အသုံးပြုနိုင်တဲ့ စိတ်ကြိုက်ပစ္စည်း မှတ်တမ်းကဲ့သို့ဖြစ်သည်။
- 🌐 [MCP Community](https://github.com/orgs/modelcontextprotocol/discussions) – MCP အကြောင်း သင်ယူသူအသင်းနှင့် အတွေ့ကြုံရှိသူများနဲ့ ဆွေးနွေးမှုများ ဆောင်ရွက် Neaux ပါ။ မေးခွန်းများကိုချီးမြှင့်ကူညီပြီး အသိပညာများ သယ်ဆောင်တိုက်ရိုက်ရွေးချယ်ခြင်းများ ဖြစ်ပါသည်။

## သင်ယူရမည့် ရည်မှန်းချက်များ

ဤသင်ရိုးညွှန်းတမ်း ပြီးဆုံးရာတွင် သင့် ကိုယ်ပိုင် အတတ်ပညာအသစ်များအပေါ် ယုံကြည်မှုနဲ့ စိတ်လှုပ်ရှားဖွယ် ရရှိမယ်။ သင်ရရှိမယ့် အချက်များမှာ -

• **MCP အခြေခံများ နားလည်ခြင်း**: Model Context Protocol ဆိုတာဘာလဲ၊ AI အပလီကေးရှင်းများ ပေါင်းစည်း ဆောင်ရွက်ရာမှာ ဘယ်လို ပြောင်းလဲမှု ပေးနေသလဲဆိုတာကို ရိုးရှင်းတဲ့ သွယ်ဝိုက်ချက်များ၊ နမူနာအပြည့်နဲ့ နားလည်ပါလိမ့်မယ်။

• **ပထမဆုံး MCP Server တည်ဆောက်ခြင်း**: သင်နှစ်သက်သော programming language ဖြင့် လွယ်ကူသည့် နမူနာများစွာဖြင့် MCP Server တစ်ခု ဖန်တီးနိုင်ပါလိမ့်မည်။

• **AI မော်ဒယ်များကို နိုင်ငံတကာ ကိရိယာများနဲ့ ချိတ်ဆက်ခြင်း**: AI မော်ဒယ်များကို တကယ့် ဝန်ဆောင်မှုများနဲ့ ချိတ်ဆက်ပေးနိုင်ရန် နည်းလမ်းများ သင်ယူပါမည်။

• **လုံခြုံရေး အကောင်းဆုံး လေ့ကျင့်မှုများ အကောင်အထည်ဖော်ခြင်း**: MCP ကို လုံခြုံစိတ်ချစွာ အကောင်အထည်ဖော်ရန် လုံခြုံရေး ဆောင်ရွက်ချက်များကို နားလည်ပါလိမ့်မယ်။

• **ယုံကြည်စိတ်ချပြီး တင်တော့ခြင်း**: MCP ပရောဂျက်များကို ဖန်တီးပြီးမှ စမ်းသပ်ထားသည့် တီထွင်မှုတင်ပို့ နည်းလမ်းများကို အသုံးပြုပြီး အသုံးချနိုင်အောင် လေ့လာပါမည်။

• **MCP အသိုင်းအဝိုင်းတွင် ပူးပေါင်းပါဝင်ခြင်း**: AI ဖန်တီးမှု နယ်ပယ်တွင် ဟန်ချက်ညီဖွံ့ဖြိုးမှု တိုးတက်အောင် စိတ်ဝင်စားမှုရှိသူ Developer အသင်းတစ်စုကို ပူးပေါင်းပါဝင်နိုင်ပါလိမ့်မယ်။

## မ Voraussetzung အချက်များ

MCP အထူးသဖြင့် ဆက်လက်ဝင်ရောက်မသွားခင် အခြေခံ အကြောင်းအရာများကို နားလည်ထားလိုက်ပါစို့။ ကျွမ်းကျင်သူ မဟုတ်လဲ အဆင်ပြေပါသည် - လိုအပ်သမျှကို ရှင်းပြပေးပါမယ်။

### Protocols နားလည်မှု (အခြေခံစီမံချက်)

Protocol ကို စကားပြောဆိုခြင်းနည်းစနစ်လို ထင်ပါစို့။ မိတ်ဆွေတစ်ဦးကို ခေါ်လိုက်တဲ့အခါ “ဟယ်လို” လို့ ပြောဆက်သည်၊ သင့်တို့နားလိုက်တော့ ဆက်လက်ကြားနာသလို ဖျော်ဖြေပျော်ပွဲ လုပ်ပါသည်။ ကွန်ပျူတာ အစီအစဉ်တွေက ဒီလို စည်းမျဉ်းစည်းကမ်းတွေ မရှိရင် ဆက်သွယ်နိုင်တာ ခက်ခဲပါတယ်။

MCP က တစ်ခုသော protocol ဖြစ်သည် - AI မော်ဒယ်နဲ့ အပလီကေးရှင်းများကို ကိရိယာနဲ့ ဝန်ဆောင်မှုများနှင့် ပူးပေါင်း ဆက်သွယ်နိုင်ဖို့ ညီညွတ်သည့် စည်းမျဉ်းစနစ်တစ်ခု ဖြစ်သည်။ လူတွေ ဆက်သွယ်မှု ပိုပြီး ပျော်ရွှင်စေတဲ့ စည်းမျဉ်းမျိုး လိုအပ်သလို MCP က AI ပလက်ဖောင်းတွေပါ ပိုတိုးတက် ၊ ယုံကြည်စိတ်ချစေတဲ့ ဆက်သွယ်မှု ပေးနိုင်ဖို့ အရေးကြီးပါတယ်။

### Client-Server ကွန်ယက်ဆက်သွယ်မှု (ပရိုဂရမ်းတွေ ဘယ်လို ပူးပေါင်းလုပ်ဆောင်ကြသလဲ)

သင့်ရဲ့နေ့စဉ်ဘဝမှာ Client-Server ဆက်သွယ်မှုတွေကို သုံးနေပါပြီ! Web browser (client) မှာ ဝဘ်ဆိုက်တစ်ခုကို သွားရောက်တဲ့အခါ ကျရောက်တဲ့ website server က web page ကို ပေးပို့ပါသည်။ Browser က မေးခွန်းမေးတယ်၊ server က ဖြေပါတယ်။

MCP မှာလည်း ဒီလို ပူးပေါင်းဆောင်ရွက်မှုရှိသည် - AI မော်ဒယ်များက client အဖြစ် တောင်းဆိုချက် များ ပေးပို့ပြီး MCP Server များက လိုအပ်တဲ့ တာဝန်များကို ဆောင်ရွက်ပေးသည်။ AI မော်ဒယ်ရဲ့ ကူညီပေးသူအဖြစ် Server ရှိနေသလို ဖြစ်ပါတယ်။

### စံနှုန်းသတ်မှတ်ခြင်းအရေးကြီးမှု (အလွယ်တကူ တွဲဖက် အသုံးပြုနိုင်စေရန်)

ကားထုတ်လုပ်သူ ကားတိုင်းအတွက် ဆီထိုးပုံးပုံစံ မတူသလို ဖြစ်ရင် ထုတ်လုပ်သူ တစ်ခုစီကို သင့်တော်တဲ့ အထူးပေါင်းစည်းကိရိယာလိုအပ်ပါမယ်။ စံသတ်မှတ်မှုကတော့ အားလုံးဝိုင်းကပ်၍ တစ်ပေါ်တည်း အဆင်ပြေစေခြင်းပါ။

MCP က AI ပလက်ဖောင်းများအတွက် ဒီလို စံနှုန်းတကျပေးသည့် နည်းလမ်းဖြစ်သည်။ AI မော်ဒယ်တစ်ခုစီဘာသာစကား ဘာသာပြန်သတ်မှတ်ချက် မလိုအပ်ဘဲ MCP က စံနည်းလမ်းကောင်းတစ်ခုဖြင့် ဆက်သွယ်ပေးသည်။ ဒါကြောင့် ပရိုဂရမ်းရေးသူတွေအနေနဲ့ မျိုးစုံ AI စနစ်တွေရဲ့ ကိရိယာတွေကို တစ်ကြိမ်တည်း ဖန်တီးပြီး သုံးနိုင်တာ ဖြစ်ပါတယ်။

## 🧭 သင်ယူမည့် လမ်းကြောင်းအနှစ်ချုပ်

သင့်ရဲ့ MCP ခရီးစဉ်ဟာ သင့်ယုံကြည်မှုနဲ့ ကျွမ်းကျင်မှု တိုးတက်အောင် အဆင့်လိုက် ဖွဲ့စည်းထားပါတယ်။ အပိုင်းတိုင်းမှာ သင်ယူထားသော အကြောင်းအရာများအား အားကောင်းစွာ မှန်ကန်မှုရှိစေသည်။

### 🌱 အခြေခံအဆင့် - အခြေခံများနားလည်ခြင်း (Module 0-2)

ဒီမှာ သင့် စူးစမ်းမှု စတင်ပါမည်။ MCP နှင့်ပတ်သက်တဲ့ ဥပမာရိုးရှင်းများနဲ့ ဟိုင်းတယ် Analogies များမှတဆင့် မိတ်ဆက်ပေးမှာပါ။ MCP ဟာ ဘာလဲ၊ ဘာကြောင့် တည်ရှိတာလဲ၊ AI ဖန်တီးမှု နယ်ပယ်မှာ ဘယ်လိုပါဝင်လဲ ဆိုတာ နားလည်ပါလိမ့်မယ်။

• **Module 0 - MCP ချိတ်ဆက်မိတ်ဆက်ခြင်း**: MCP အကြောင်းနှင့် ဒါတွေဟာ ခေတ်သစ် AI ပလက်ဖောင်းတွေမှာ ဘယ်လောက်အရေးပါတဲ့အကြောင်းကို စတင်လေ့လာပါမယ်။ MCP ဖြေရှင်းပေးနေတဲ့ အဆင်မပြေမှုများကို လက်တွေ့ဥပမာတွေနဲ့ ပြသပါမယ်။ 

• **Module 1 - အဓိက အတွေးအခေါ်များ ရှင်းပြခြင်း**: ဒီပိုင်းမှာ MCP ၏ အင်္ဂါရပ်အခြေခံများကို အများပြည်သူနားလည်အောင် အများအပြား Analogy နဲ့ ဂရပ်ဖစ်များဖြင့် သင်ကြားပါမယ်။

• **Module 2 - MCP တွင် လုံခြုံရေး**: လုံခြုံရေးဆိုတာ ကြောက်မက်စရာ ဖြစ်နိုင်ပါတယ်၊ ဒါပေမယ့် MCP မှာ built-in လုံခြုံရေး အင်္ဂါရပ်များ ရှိပုံ၊ သင့် application များကိုကာကွယ်ရန် လုပ်ဆောင်သင့်သည့် ဆောင်ရွက်ချက်များကို သင်ကြားပေးပါမယ်။

### 🔨 တည်ဆောက်ခြင်းအဆင့် - သင်၏ ပထမဆုံး ဆောင်ရွက်ချက်များ ဖန်တီးခြင်း (Module 3)

အခုမှ စတင်ပြီး စစ်မှန်စွာအကြမ်းဖက်မှုတွဲဖြစ်သည်! သင်က MCP Server နဲ့ Client တည်ဆောက် နဲ့ အတွေ့အကြုံ လက်တွေ့အသုံးပြုခွင့် ရမည်။ စိုးရိမ်စရာ မလိုပါ - ပိုမိုလွယ်ကူစွာ စတင်ပြီး လမ်းညွှန်ချက်တိုင်းနဲ့အတူ သင်ယူပါမယ်။
ဒီမော်ဂျူးမှာ သင်ကြိုက်နှစ်သက်တဲ့ပရိုဂရမ်မင်းဘာသာစကားဖြင့် လက်တွေ့လုပ်ဆောင်နိုင်တဲ့ လမ်းညွှန့် များစွာပါဝင်ပါတယ်။ သင့်ရဲ့ ပထမဆုံးဆာဗာကို ဖန်တီးပြီး၊ 그것에 연결하기 위한 클라이언트를 구축하며၊ VS Code 같은 인기 개발 도구와 통합하는 방법도 배울 것입니다.

각 지침에는 완전한 코드 예제, 문제 해결 팁, 그리고 특정 설계 선택 이유에 대한 설명이 포함되어 있습니다. 이 단계가 끝나면 자신이 자랑스러워할 수 있는 작동하는 MCP 구현물을 갖게 될 것입니다!

### 학습 가이드
ဒီ repository ကို ထိရောက်စွာ ဦးတည်လမ်းညွှန်ဖို့ အကူအညီပေးနိုင်တဲ့ စုံလင်သော [သင်ယူမှု လမ်းညွှန်](./study_guide.md) ရနိုင်ပါတယ်။ ဒီ视觉ပညာအစီအစဉ်မြေပုံက ခေါင်းစဉ်များအားလုံး ဘယ်လိုဆက်စပ်နေပြီး နမူနာပရောဂျက်များကို ထိရောက်စွာ ဘယ်လိုအသုံးပြုရမယ်ဆိုတာ ဦးတည်ချက်ပေးပါတယ်။ ဓာတ်ပုံတွေကို ကြည့်ရတာနှစ်သက်တဲ့ အသိပညာရှင်တွေအတွက် အထူးအသုံးဝင်ပါတယ်။  

လမ်းညွှန်မှာ ပါရှိတာတွေက –  
- အားလုံးပါတဲ့ ခေါင်းစဉ်များ ပြသထားတဲ့ 视觉ပညာအစီအစဉ်မြေပုံ  
- repository အပိုင်းဆီလက်ကျန်အသေးစိတ်ဖွင့်ပြချက်  
- နမူနာပရောဂျက်များကို ဘယ်လိုအသုံးပြုရမယ်ဆိုတဲ့ ဦးတည်ချက်များ  
- ကျွမ်းကျင်မှုပိုင်းအဆင့်အလိုက် သင်ယူရမယ့် လမ်းကြောင်းများ  
- သင်ယူမှု ခရီးမှာ အထောက်အကူဖြစ်စေမယ့် အပိုဆောင်းရင်းမြစ်များ

### စတင်ဖို့ အဆင်သင့်လား?  

သင့် MCP ခရီးနဲ့ပထမဆုံးလက်တွေ့ ဇယားကို Module 0 နဲ့စလိုက်ပါ၊ ဒါမှမဟုတ် နမူနာပရောဂျက်တွေကို ရှာဖွေပြီး ဘာလုပ်ထားဦးမှာလဲ ကြည့်ရှုကြပါ။ ဖြစ်တည်သူတိုင်း အစအဆုံးသင်ကြားခဲ့သူဖြစ်ပြီး စိတ်ရှည်လည်းရှိတယ်ဆိုရင် သင်တတ်နိုင်တာတွေကို မျှော်လင့်ရမှာပါ။  

Model Context Protocol အကြောင်း ကမ္ဘာမှာ ကြိုဆိုပါတယ်။ အံ့ဩဖွယ်ရာကို တစ်ပြိုင်နက်တည်း တည်ဆောက်ကြမယ်!  

## 🤝 သင်ယူမှု အသိုင်းအဝိုင်းကို ဝင်ရောက် ပါဝင်ရန်  

ဒီ သင်ခန်းစာဟာ သင်လို ရည်ရွယ်ထားသူတွေရဲ့ ပံ့ပိုးမှုနဲ့ ပို၍ ပြင်းထန်လာမှာပါ။ ကျောင်းသင်စာလုံးအမှားတစ်ခုမှန်ဂျံပေးခြင်း၊ ပိုရှင်းလင်းတဲ့ ရှင်းပြချက် တစ်ခု သို့မဟုတ် နမူနာအသစ်တစ်ခု ထည့်သွင်းပေးခြင်းမှတစ်ဆင့် သင့်ထောက်ပံ့မှုများက အခြား သမားရိုးကျတွေကို အောင်မြင်စေမှာဖြစ်ပါတယ်။  

Microsoft Valued Professional [Shivam Goyal](https://www.linkedin.com/in/shivam2003/) ကို ကုဒ်နမူနာတွေ ထည့်သွင်းပေးမှုအတွက် ကျေးဇူးတင်ပါတယ်။  

ပံ့ပိုးမှုလုပ်ငန်းစဉ်က လက်ခံရေးနဲ့ တာဝန်ယူမှုရှိစေဖို့ ဖြစ်ပါတယ်။ ပိုင်းခြားအားလုံးအတွက် Contributor License Agreement (CLA) လိုအပ်တတ်ပေမယ့် အလိုအလျောက်ပစ္စည်းတွေက အဆင်ပြေစွာ လမ်းညွှန်ပါလိမ့်မယ်။  

## 📜 အခမဲ့ မူရင်းကုဒ် သင်ယူမှု  

ဒီ သင်ခန်းစာအတန်းအကုန်လုံးကို MIT [LICENSE](../../LICENSE) လက်မှတ်ဖြင့် ရရှိနိုင်ပြီး မည်သည့်သူမဆို လွတ်လပ်စွာ အသုံးပြု၊ ပြင်ဆင်၊ မျှဝေပိုင်ခွင့်ရပါတယ်။ MCP အသိပညာကို အားလုံးသုံးနိုင်ဖို့ ကျွန်ုပ်တို့ရဲ့ ရည်မှန်းချက်ကို ပံ့ပိုးပေးပါတယ်။  

## 🤝 ပံ့ပိုးမှု ညွှန်ကြားချက်များ
