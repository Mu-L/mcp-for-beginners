# 🚀 Model Context Protocol (MCP) အခြေခံသင်တန်း

## **C#, Java, JavaScript, Rust, Python, နှင့် TypeScript ဘာသာစကားများဖြင့်လက်တွေ့ ကုဒ် ဥပမာများဖြင့် MCP ကို လေ့လာပါ**

## 🧠 Model Context Protocol သင်ရိုးညြှန်း

Model Context Protocol ၏ ခရီးစဉ်သို့ ကြိုဆိုပါသည်! AI အက်ပလီကေးရှင်းများသည် ကိရိယာများနှင့် ဝန်ဆောင်မှုများနှင့် မည်ကဲ့သို့ ဆက်သွယ်ကြသည်ကို မေးမြန်းဖူးပါက၊ ဒီကွန့်လုံးကိုလေ့လာရန်ပြင်ဆင်နေပါပြီ။

MCP ကို AI အက်ပလီကေးရှင်းများအတွက် စာနာခွင့်ပြု ဘာသာပြန်တစ်ခုအဖြစ် တွေးပါ - USB ပေါက်များက သင့်ကွန်ပျူတာနှင့် မည်သည့်ကိရိယာတစ်ခုမဆို ချိတ်ဆက်နိုင်သည့်အတိုင်း MCP သည် AI မော်ဒယ်များအတွက် မည်သည့်ကိရိယာ သို့ ဝန်ဆောင်မှုကိုမဆို စံတမ်းတစ်ရပ်ဖြင့် ချိတ်ဆက်ဖို့ ခွင့်ပြုသည်။ သင်၏ ပထမ chatbot တည်ဆောက်ခြင်းမှစ၍ အဆင်ပြေတဲ့ AI လုပ်ငန်းစဉ်ရှစ်ဆန်းဆန်းကို ရည်ရွယ်နေစဉ်အထိ MCP ကို နားလည်ခြင်းမှတဆင့် ပိုမိုပါးလွှာပြီး တိကျသော အက်ပလီကေးရှင်းများ တည်ဆောက်နိုင်မည်။

ဤသင်ရိုးညွှန်းမှာ သင်၏သင်ယူခရီးကို သည်းခံမှုနှင့် ဂရုစိုက်မှုဖြင့် ဖန်တီးထားသည်။ သင်သိပြီးသား အခြေခံအယူအဆများဖြင့် စတင်ကာ သင်နှစ်သက်ရာ programming ဘာသာစကားဖြင့် လက်တွေ့ ပြုလုပ်ခြင်းမှတဆင့် နည်းဗျူဟာအသစ်များ တိုးတက်စေမည်။ လုပ်ဆောင်မှုတိုင်းတွင် ရှင်းလင်းသဘောပေါက်သော အဓိပ္ပါယ်ရှင်းချက်များ၊ လေ့လာနိုင်သော ဥပမာများနှင့် အားပေးမှုများ ပါဝင်သည်။

ဤခရီးစဉ်ကို ပြီးဆုံးသည့်အချိန်၊ သင့်အနေနှင့် သင်၏ MCP ဆာဗာများကို တည်ဆောက်နိုင်ခဲ့မှာဖြစ်ပြီး လူကြိုက်များသော AI ပလက်ဖောင်းများနှင့်ပေါင်းစည်းနိုင်မှုရှိပြီး AI ဖွံ့ဖြိုးမှု၏ အနာဂတ်ကို ပြောင်းလဲနေသည့်နည်းပညာကို နားလည်စွာ အသုံးချနိုင်မည် ဖြစ်သည်။ အတူတကွ စိတ်ဓာတ်မြှင့်ပြီး စတင်လိုက်ကြပါစို့!

### တရားဝင် စာရွက်စာတမ်းများ နှင့် ဖော်ပြချက်များ

ဤသင်ရိုးညွှန်းသည် **MCP Specification 2025-11-25** (နောက်ဆုံး အတည်ပြုထွက်ရှိမှု) နှင့် ကိုက်ညီပါသည်။ MCP specification သည် ရက်စွဲပေါ် အခြေခံ Versioning (YYYY-MM-DD ပုံစံ) ကို အသုံးပြု၍ ပရိုတိုကော များအား ထိန်းသိမ်းသည်။

များသောအားဖြင့် သင်၏ နားလည်မှု ကြီးပွားလာသည်နှင့်အမျှ ဤရင်းမြစ်များမှာ တန်ဖိုးဖြစ်လာမည်ဖြစ်ပြီး အကုန်လုံးကို ချက်ချင်း ဖတ်ရန် မလိုအပ်ပါ။ သင် စိတ်ဝင်စားသော နေရာများဖြင့် စတင်တတ်နိုင်ပါသည်။
- 📘 [MCP စာရွက်စာတမ်း](https://modelcontextprotocol.io/) – လမ်းညွှန်ချက်အဆင့်ဆင့်နှင့် အသုံးပြုသူလမ်းညွှန်များအတွက် သင့်ရဲ့ အဓိကရင်းမြစ်ဖြစ်သည်။ သာမန်လေ့လာသူများအတွက် ရေးသားထားပြီး ပိုမို မလွဲမှားစွာ နားလည်နိုင်ရန် ဥပမာများ ပါဝင်သည်။
- 📜 [MCP ဖော်ပြချက်](https://modelcontextprotocol.io/specification/2025-11-25) – သင်ရိုးကို လေ့လာသည့်အခါ နောက်ပြန် ရှာဖွေကြည့်ရှုရန် နှင့် တိုးတက်သော အင်္ဂါရပ်များကို အကြောင်းကြားရန်လက်စွဲစာအုပ်အဖြစ် သုံးနိုင်သည်။
- 📜 [MCP ဖော်ပြချက် Versioning](https://modelcontextprotocol.io/specification/versioning) – ပရိုတိုကော Version များ၏ သမိုင်းကြောင်းနှင့် MCP က ရက်စွဲပေါ် အခြေခံ Versioning ကို ဘယ်လိုအသုံးပြုသည် များပါရှိသည်။
- 🧑‍💻 [MCP GitHub Repository](https://github.com/modelcontextprotocol) – SDK များ၊ ကိရိယာများနှင့် ဘာသာစကားစုံဖြင့် ကုဒ်နမူနာများပါရှိသည်။ လေ့လာသူများအတွက် အသုံးဝင်သော ဥပမာများနှင့် အသုံးပြုနိုင်သော ပစ္စည်းများ ပါဝင်သည်။
- 🌐 [MCP လူမှုအဖွဲ့](https://github.com/orgs/modelcontextprotocol/discussions) – MCP အကြောင်း တွေ့ဆုံဆွေးနွေးပြီး ကြွယ်ဝသော အသိပညာများကို မျှဝေရာ လူမှုအဖွဲ့ပါဝင်ခြင်း။

## သင်ယူရမည့် ရည်ရွယ်ချက်များ

ဤသင်ရိုးကို ပြီးမြောက်အပြီး သင်သည် သင်၏အသစ်ရရှိသော တတ်မြောက်မှုများနှင့် ယုံကြည်မှု ရရှိပြီး စိတ်လှုပ်ရှားမှုဖြင့် ပြည့်နက်သွားမည်ဖြစ်သည်။ အောက်ပါအရာများကို ရရှိမည်ဖြစ်သည်။

• **MCP အခြေခံအယူအဆများ နားလည်ခြင်း** - Model Context Protocol သည် မည်သည့်အတိုင်း AI အက်ပလီကေးရှင်းများ တွဲဖက်လုပ်ဆောင်ရေးကို ပြောင်းလဲနေသည်ကို နားလည်ပြီး ဆင်တူသဘောတူ ဥပမာများဖြင့် နားလည်မည်။

• **သင်၏ ပထမဆုံး MCP ဆာဗာ တည်ဆောက်ခြင်း** - သင်နှစ်သက်ရာ programming ဘာသာစကားဖြင့် လက်တွေ့ အလုပ်လုပ်သော MCP ဆာဗာတစ်ခုကို ရေးဆွဲနိုင်သည်။ ရိုးရှင်းသော ဥပမာများကနေ စတင်ပြီး တစ်ဆင့်ချင်းပိုမို ကျွမ်းကျင်မှုရရှိအောင် တိုးတတ်မည်။

• **AI မော်ဒယ်များကို အတည်တကျ ကိရိယာများနှင့် ချိတ်ဆက်ခြင်း** - AI မော်ဒယ်များနှင့် လက်တွေ့ ဝန်ဆောင်မှုများကို ချိတ်ဆက်ပေးခြင်း၏ နည်းလမ်းများ တီထွင်သိရှိမည်။

• **လုံခြုံရေး အကောင်းဆုံးနည်းလမ်းများ အကောင်အထည်ဖော်ခြင်း** - MCP ကို လုံခြုံစိတ်ချစွာ အသုံးပြုနိုင်ရန်၊ သုံးစွဲသူနှင့် အက်ပလီကေးရှင်းများကို ကာကွယ်ပေးရန် အကောင်းဆုံးနည်းလမ်းများ သင်ယူမည်။

• **ယုံကြည်စိတ်ချရသော တင်သွင်းမှုနည်းလမ်းများ** - MCP ပရောဂျက်များအား တိုးတက်မှုကနေ ထုတ်လုပ်မှုအဆင့်သို့ ရောက်ရှိအောင် နှင့် လက်တွေ့ အသုံးပြုနိုင်သည့် deployment စီမံကိန်းများ သိရှိရမည်။

• **MCP လူမှုအဖွဲ့တွင် ပူးပေါင်းခြင်း** - AI ဖွံ့ဖြိုးရေး ရှေ့ဆောင်များမှ ကြီးထွားလာနေသည့် လူမှုအဖွဲ့ အဖျော်ယမကာအဖွဲ့ဝင်ဖြစ်လာမည်။

## အရေးကြီးသော နောက်ခံ သိထားသင့်သည်များ

MCP အကြောင်း သွားရောက်သင်ကြားရန်မတိုင်ခင် အခြေခံအယူအဆအချို့မှာ နားလည်မှုရှိဖို့ လိုအပ်သည်။ ကျွမ်းကျင်သူ မဟုတ်သော်လည်း စိတ်ပူစရာ မလိုပါဘူး - လိုအပ်သည့် အချက်အလက်အားလုံးကို သေချာရှင်းပြပေးပါမည်။

### ပရိုတိုကောများ ကို နားလည်ခြင်း (အခြေခံအပ်)

ပရိုတိုကောကို စကားပြောညှိနှိုင်းမှု စည်းကမ်းများ၏ စည်းရုံးချက်အဖြစ် တွေးပါ။ မိတ်ဆွေတစ် ဦးကို ဖုန်းခေါ်လိုက်သော အခါ “ဟယ်လို” ဟု ပြောဆိုရန်လုပြီး တစ်ယောက်နဲ့တစ်ယောက် တဖြည်းဖြည်း ပြောဆိုခြင်း စသည်ဖြင့် ပြောဆိုမှု စည်းကမ်းများ ရှိသည်။ ကွန်ပျူတာအစီအစဉ်များလည်း ထိုသို့ စည်းကမ်းများလိုအပ်သည်။

MCP သည် ပရိုတိုကောတစ်ခုဖြစ်ပြီး AI မော်ဒယ်များနှင့် အက်ပလီကေးရှင်းများအတွက် ကိရိယာများနှင့် ဝန်ဆောင်မှုများနှင့် ထိရောက်သော “စကားပြော” ကို အောင်မြင်စေရန် သဘောတူစည်းကမ်းများ စုစည်းထားခြင်း ဖြစ်သည်။ လူသားများ၏ ဆက်သွယ်မှု ပိုမိုသေချာစေရန် စကားပြော စည်းကမ်းများလိုသည်ကဲ့သို့ AI အက်ပလီကေးရှင်းဆက်သွယ်မှုမှာ MCP က စည်းကမ်းများရှိခြင်းကြောင့် ပိုမိုယုံကြည်စိတ်ချရပြီး အင်အားကြီးစေသည်။

### Client-Server ဆက်ဆံရေးများ (အစီအစဉ်များ မည်ကဲ့သို့အတူတူ လုပ်ဆောင်ကြသည်)

သင် နေ့တိုင်း client-server ဆက်ဆံရေးကို အသုံးပြုနေပြီဖြစ်သည်! ဥပမာ၊ သင် ဝက်ဘ်ဘရောက်ဇာ (client) ဖြင့် ဝက်ဘ်ဆိုက်ကို ဝင်ကြည့်လျက်ရှိပါက ဝက်ဘ်ဆာဗာက စာမျက်နှာအကြောင်းအရာကို ပေးပို့သည်။ ဘရောက်ဇာက အချက်အလက် မေးမြန်းနည်းကို သိရှိပြီး ဆာဗာက စနစ်တကျ တုံ့ပြန်နိုင်ပါသည်။

MCP တွင်လည်း ယခုလို ဆက်ဆံရေးတူပါသည်။ AI မော်ဒယ်များသည် တောင်းဆိုသူ(client) အဖြစ် တောင်းဆိုမှုများထုတ်ပေးပြီး MCP ဆာဗာများက အဆိုပါလုပ်ဆောင်ချက်များကို ပေးဆောင်သည်။ AI သည် မိမိအကူအညီလိုသောအရာများကို ဆာဗာထံ မေးမြန်းနိုင်သော အကူအညီပေးသူတစ်ဦးကဲ့သို့ ဖြစ်သည်။

### စံသတ်မှတ်ခြင်း အရေးပါမှု (ပစ္စည်းများကို တစ်ဦးနှင့်တစ်ဦး လုပ်ဆောင်နိုင်စေရန်)

ကားထုတ်လုပ်သူတိုင်းက မတူညီသော ဂျက်(လောင်စာထောက်ပံ့ရေး) ပုံးပုံစံအသုံးပြုခဲ့လျှင် - ကားအသီးသီးအတွက် ကြွပ်ကွန်နက်တာ မတူကြမည်။ စံတမ်းတစ်ရပ် သက်သေသည်မှာ မည်သည့်ပစ္စည်းမျှ အပြန်အလှန်လိုက်ဖက် အလုပ်လုပ်နိုင်စေရန် စာချုပ်တစ်ရပ်သို့ သဘောတူခြင်း ဖြစ်သည်။

MCP သည် AI အက်ပလီကေးရှင်းများအတွက် ဒီလို စံချိန်စံညွှန်းကို ပေးသည်။ AI မော်ဒယ် စီစဉ်မှု တစ်ခုစီအတွက် အသီးသီးကုဒ်ရေးသားရန် မလိုပဲ MCP က AI မော်ဒယ်များနှင့် ကိရိယာများ ပေါင်းစည်းနိုင်ရန် စံချိန်စံညွှန်း တစ်ခုဖြစ်သည်။ ဒီအတွက် ဖန်တီးသူများသည် ကိရိယာတစ်ခု ဖန်တီးပြီး များပြားသော AI စနစ်များနဲ့ အသုံးပြုနိုင်မည် ဖြစ်သည်။

## 🧭 သင်ယူမှုလမ်းကြောင်းအကြမ်း

သင်၏ MCP ခရီးစဉ်ကို ကျွမ်းကျင်မှုနှင့် ယုံကြည်စိတ်ချမှု တိုးတက်အောင် သတိထားပြီး တိုးတက်စေသည်။ အဆင့်တိုင်းတွင် အသစ်သော သဘောတရားများကို မိတ်ဆက်ပြီး ယခင်တတ်ရပ်များကို ပြန်လည် တစ်ဖန်လေ့လာပေးသည်။

### 🌱 အခြေခံအဆင့်: အခြေခံအယူအဆ နားလည်ခြင်း (မော်ဂျူး 0-2)

ဤနေရာမှာ သင်၏ စွန့်စားခန်း စတင်သည်။ MCP အကြောင်း မိတ်ဆက်ပေးပြီး၊ သင်နှစ်သက်သည့် ဥပမာများနှင့် ရိုးရှင်းသော နှိုင်းယှဉ်ချက်များ အသုံးပြုကာ MCP ၏ အဓိကကို နားလည်စေသည်။

• **မော်ဂျူး 0 - MCP မိတ်ဆက်ခြင်း** - MCP ဆိုတာဘာလဲ၊ ယနေ့ AI အက်ပလီကေးရှင်းများအတွက် ဘာကြောင့် လိုအပ်တာလဲဆိုတာ ရှင်းပြပေးမယ်။ MCP သုံး၍ကြုံတွေ့ရသည့် အခက်အခဲများကို မဖြေရှင်းနိုင်မီနောက် MCP ကို သုံး၍ ပြုပြင်ဖြေရှင်းနည်းကို လေ့လာမယ်။

• **မော်ဂျူး 1 - အဓိက အယူအဆများ ရှင်းပြချက်** - MCP အတွက် အခြေခံတည်ဆောက်ပုံများကို သိရှိစေမည်။ နားလည်ရလွယ်ကူစေရန် နှိုင်းယှဉ်ချက်များနှင့် ရုပ်ပုံ ဥပမာများကို စုံလင်စွာ အသုံးချမည်။

• **မော်ဂျူး 2 - MCP တွင် လုံခြုံရေး** - လုံခြုံရေးသည် ရှုပ်ထွေးကြောင်းထင်နိုင်ပေမယ့် MCP တွင် ပါဝင်သော built-in လုံခြုံရေးနည်းလမ်းများနှင့် အကောင်းဆုံး လုပ်ထုံးလုပ်နည်းများကို သင်ယူမည်။

### 🔨 တည်ဆောက်ခြင်း အဆင့်: သင်၏ ပထမဆုံး အကောင်အထည်ဖော်မှုများ (မော်ဂျူး 3)
အခုမှ စပြီး စိတ်အားထက်သန်ဖွယ်ကောင်းတဲ့အချိန်ပါပဲ။ သင်ဟာ လက်တွေ့ MCP ဆာဗာတွေတည်ဆောက်ခြင်းနဲ့ client တွေ ဖန်တီးခြင်းကို လက်တွေ့လုပ်ဆောင်ဖို့ အတွေ့အကြုံ ရရှိမယ်။ စိတ်ပူစရာမလိုပါဘူး - ကျွန်တော်တို့က ရိုးရှင်းအဆင့်ကစပြီး လှမ်းလိုက်သွားမှာ ဖြစ်ပါတယ်။

ဒီမော်ဂျူးထဲမှာ သင်ကြိုက်သုံးတဲ့ programming language နဲ့ လေ့ကျင့်နိုင်ဖို့ လက်တွေ့ လမ်းညွှန်ချက်များစွာ ပါဝင်ပါတယ်။ သင်အရင်ဆုံး MCP ဆာဗာကို တည်ဆောက်ပြီး၊ အဲဒီဆာဗာထဲမှာ ချိတ်ဆက်ဖို့ client ကို ဖန်တီးပြီး၊ VS Code စတဲ့ လူကြိုက်များတဲ့ พัฒนา Tools တွေနဲ့လည်း ပေါင်းစည်း ပေးလိမ့်မယ်။

လမ်းညွှန်တိုင်းမှာ ကုဒ်နမူနာပြည့်စုံ၊ ပြဿနာဖြေရှင်းနည်းများနဲ့ ဒီဇိုင်းရွေးချယ်မှုများ၏ ရှင်းလင်းချက်တွေ ပါဝင်တာကြောင့် ဒီအဆင့်ပြီးရင် သင်မှာ MCP အကောင်အထည်ဖော်ထားတဲ့ အဆင်ပြေတဲ့ implementations တွေ ရရှိသွားမှာပါ။

### 🚀 ကြီးထွားမှု အဆင့်: အဆင့်မြင့် အတွေးအခေါ်များနှင့် လက်တွေ့ လျှောက်လွှာ (Modules 4-5)

 အခြေခံတွေ ကျွမ်းကျင်ပြီးသွားရင် MCP ရဲ့ ပိုမိုတိုးတက်အဆင့်မြင့် features တွေကို လေ့လာဖို့ အသင့် ဖြစ်ပါပြီ။ လက်တွေ့အသုံးပြုနည်း များ၊ debugging နည်းလမ်းများနဲ့ multi-modal AI ပေါင်းစပ်ခြင်း စတဲ့ အဆင့်မြင့် အကြောင်းအရာများကို ဖုံးကွယ်ယူလိမ့်မယ်။ 

ဆက်လက်ပြီး MCP ကို ထုတ်လုပ်မှုအတွက် စွမ်းဆောင်နိုင်အောင် ချဲ့ထွင်ခြင်းနဲ့ Azure အစရှိသည့် cloud platform များနှင့် ပေါင်းစည်းခြင်းနည်းလမ်းများကို သင်ယူရမည်။ ဤမော်ဂျူးများက သင့်အား လက်တွေ့လိုအပ်ချက်များကို ကိုင်တွယ်နိုင်တဲ့ MCP ဖြေရှင်းချက်များ တည်ဆောက်ရန် ပြင်ဆင်ပေးပါသည်။

### 🌟 ကျွမ်းကျင်မှု အဆင့်: လူမှု့အသိုင်းအဝိုင်းနှင့် အထူးပြုခြင်း (Modules 6-11)

နောက်ဆုံးအဆင့်က MCP လူမှု့အသိုင်းအဝိုင်းထဲ ဝင်ရောက်ပြီး ကိုယ်စိတ်ဝင်စားရာ နေရာများတွင် အထူးပြု ပါပဲ။ သင်ဤအဆင့်တွင် open-source MCP ပရောဂျက်များသို့ ပါဝင်ရန်၊ အဆင့်မြင့် authentication ပုံစံများ အကောင်အထည်ဖော်ခြင်းနှင့် database ပေါင်းစပ်ထားသော ဖြေရှင်းချက် အပြည့်အစုံ တည်ဆောက်ခြင်းများကို လေ့လာမည်မို့ဖြစ်သည်။

Module 11 လည်း အထူးကြားစရာကောင်းတယ် - PostgresSQL ပေါင်းစပ်ခြင်းဖြင့် production အသုံးပြုနိုင်သော MCP ဆာဗာများကို တည်ဆောက်ရန် လေးစားစရာ 13-လက်တွေ့ သင်ကြားမှု အခြေလမ်းကြောင်း တစ်ခုဖြစ်ပါတယ်။ သင်သင်ယူထားတဲ့ အရာတွေကို လက်တွဲပြီး အဆုံးသတ် သင်ကြားမှု Project တစ်ခုလို ဖြစ်မြောက်စေမှာ ဖြစ်ပါတယ်။

## 🎓 သင်၏ MCP ခရီးစတင်ပါပြီ

ဂုဏ်ယူပါတယ်! သင်ဟာ ပရိုဂရမ်မာများတိုးတက်မှုနဲ့ AI ဖွံ့ဖြိုးတိုးတက်မှု အကြမ်းဖျဉ်းသုံးသပ်မှုဆိုင်ရာ ခရီးထဲ က ဇာတ်လမ်းအစမှ အဆင့်တစ်ခုကို မျှတစွာ ရောက်ရှိခဲ့ပါပြီ။

သင်၏ MCP ခရီးစဥ် စတင်လိုက်ပါပြီ! Module 0 မှစ၍ သင်၏ပထမဆုံးလက်တွေ့ MCP အတွေ့အကြုံများကို စတင်ပါ၊ လည်းမဟုတ် နမူနာပရောဂျက်များကို စူးစမ်းကြည့်ပါ၊ သင်ဘာဆောက်မလဲဆိုတာကို ကြည့်ရှုနိုင်ပါတယ်။ မေတ္တာနဲ့ အလေ့အကျင့်များဖြင့် ဘယ်အဆင့်မှ ကျွမ်းကျင်သူ တစ်ယောက်တည်း ဖြစ်လာနိုင်ပါတယ်။

Model Context Protocol ဖွံ့ဖြိုးတိုးတက်မှုကမ္ဘာကြီးသို့ ကြိုဆိုပါတယ်။ အတူတကွ အံ့သြစရာတွေ ဖန်တီးကြပါစို့!

## 🤝 သင်ယူကြဖို့ အဖွဲ့ကို ပါဝင် ကူညီခြင်း

ဒီ အသိပညာအစီအစဉ်ဟာ သင်လိုသင်ယူသူတွေ ဆန္ဒရှိသူတွေ အဆင့်မြှင့် တိုးတက်လာပါတယ်။ အမှားစု၊ ထပ်ထည့်ချက် သို့မဟုတ် သဘောထင်အိုပေးမှု ပြုလုပ်ခြင်းကနေ မစှေးပဲ သင်၏ ပံ့ပိုးမှုတွေဟာ တခြားလူငယ်သစ်တွေ အောင်မြင်မှာ အထောက်အကူဖြစ်စေမှာပါ။

Microsoft Valued Professional [Shivam Goyal](https://www.linkedin.com/in/shivam2003/) ကို ကုဒ်နမူနာများ ပံ့ပိုးပေးမှုအတွက် ကျေးဇူးတင်ရှိပါသည်။

ပံ့ပိုးမှုလုပ်ငန်းစဉ်ဟာ ယုံကြည်စိတ်ချရပြီး ပံ့ပိုးမှုများအတွက် အဆင်ပြေစေစေ ဖြစ်အောင် တည်ဆောက်ထားပါသည်။ များသောပံ့ပိုးမှုများသည် Contributor License Agreement (CLA) လက်မှတ်တင်ရန် လိုအပ်ပါသည်၊ ဒါပေမယ့် အလိုအလျှောက်သော ကိရိယာများသည် ချောမွေ့စွာ ဦးတည်လမ်းပြပါမည်။

## 📜 အဖွင့် အရင်းအမြစ် သင်ယူခြင်း

ဒီ အစီအစဉ်တစ်ခုလုံးကို MIT [LICENSE](../../LICENSE) အောက်မှာဖြန့်ချိထားတာဖြစ်သည်၊ သင်သည် လွတ်လပ်စွာ အသုံးပြု၊ ပြင်ဆင် ပြန်ဖြန့်ဖြူးနိင်ပါသည်။ ဒါဟာ MCP အကြောင်းအသိပညာကို ကမ္ဘာလုံးဆိုင်ရာ သတ်မှတ်ထားသူများအတွက် ရရှိနိုင်စေမည့် লক্ষ্যကို ထောက်ပံ့ပေးနေပါသည်။
## 🤝 ပံ့ပိုးမှု လမ်းညွှန်ချက်များ

ဒီပရောဂျက်က ပံ့ပိုးမှုတွေကို ကြိုဆိုပါတယ်။ များသောပံ့ပိုးမှုများတွင် Contributor License Agreement (CLA) ကို သဘောတူညီရန် လိုအပ်ပါသည်၊ ၎င်းတွင် သင်၏ပံ့ပိုးမှုအတွက် ပိုင်ဆိုင်ခွင့် တူညီစေခြင်း နှင့် သက်ဆိုင်ရာ အခွင့်အရေးများကို ဖော်ပြထားသည်။ အသေးစိတ်အချက်အလက်များအတွက် <https://cla.opensource.microsoft.com> ကို သွားကြည့်ပါ။
