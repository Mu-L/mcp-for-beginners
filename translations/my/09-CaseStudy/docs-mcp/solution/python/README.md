# Chainlit နှင့် Microsoft Learn Docs MCP အသုံးပြုပြီး သင်ယူအစီအစဉ် ජනရေးကိရိယာ

## မလိုအပ်သော အချက်များ

- Python 3.8 သို့မဟုတ် အထက်
- pip (Python package မန်နေဂျာ)
- Microsoft Learn Docs MCP ဆာဗာနှင့် ချိတ်ဆက်ရန် အင်တာနက် ချိတ်ကိုင်မှု

## တပ်ဆင်ခြင်း

1. ဤ repository ကို clone ဆွဲပါ သို့မဟုတ် project ဖိုင်များကို ဒေါင်းလုဒ်လုပ်ပါ။
2. လိုအပ်သော dependencies များကို တပ်ဆင်ပါ။

   ```bash
   pip install -r requirements.txt
   ```

## အသုံးပြုနည်း

### အမှုကိစ္စ ၁: Docs MCP ကို ရိုးရှင်းသော မေးခွန်းတစ်ခု စစ်ဆေးခြင်း
Docs MCP ဆာဗာနှင့်ချိတ်ဆက်ပြီး မေးခွန်းတစ်ခု ပေးပို့ကာ ရလာဒ်ကို မွမ်းမံပေးသော command-line client ။

1. script ကို run ပါ။
   ```bash
   python scenario1.py
   ```
2. prompt တွင် သင်၏စာတမ်း မေးခွန်းကို ရိုက်ထည့်ပါ။

### အမှုကိစ္စ ၂: သင်ယူအစီအစဉ် စက် (Chainlit Web App)
Chainlit ကို အသုံးပြုထားသည့် ဝက်ဘ်အခြေခံ မျက်နှာပြင်ဖြင့် အသုံးပြုသူများအား နည်းပညာဆိုင်ရာဘာသာရပ်တစ်ခုအတွက် အပတ်စဉ် စီမှတ်ထားသော ကိုယ်ပိုင် သင်ယူအစီအစဉ် တစ်ခု ထုတ်ပေးနိုင်သည်။

1. Chainlit app ကို စတင်ပါ။
   ```bash
   chainlit run scenario2.py
   ```
2. terminal ရဲ့ local URL (ဥပမာ http://localhost:8000) ကို browser တွင် ဖွင့်ပါ။
3. chat ပြတင်းပေါ်တွင် သင်ယူလိုသော ဘာသာရပ်နှင့် သင်ယူမည့် အပတ်ရေ (ဥပမာ "AI-900 certification, 8 weeks") ကို ဖြည့်ပါ။
4. app မှ သင်ယူမှု အပတ်စဉ်အစီအစဉ်ကို ပြန်ပေးပြီး Microsoft Learn သင်ခန်းစာများဆီ ချိတ်ဆက်သောလင့်ခ်များပါဝင်သည်။

**လိုအပ်သော Environment Variables:**

အမှုကိစ္စ ၂ (Azure OpenAI ဖြင့် Chainlit web app) ကို အသုံးပြုရန် `python` ဖိုလ်ဒါအတွင်း `.env` ဖိုင်ထဲတွင် အောက်ပါ environment variables များကို သတ်မှတ်ရပါမည်။

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

app ကို run မလုပ်မီ သင့် Azure OpenAI resource အသေးစိတ်များဖြင့် ထည့်သွင်းပါ။

> [!TIP]
> သင့်ကိုယ်ပိုင် မော်ဒယ်များကို [Microsoft Foundry](https://ai.azure.com/) မှာ လွယ်ကူစွာ deploy လုပ်နိုင်သည်။

### အမှုကိစ္စ ၃: VS Code တွင် MCP ဆာဗာရဲ့ In-Editor Docs

Browser tab အသစ်သို့ ပြောင်းလဲ ရှာဖွေခြင်းမပြုဘဲကို Microsoft Learn Docs ကို VS Code ထဲသို ့ တိုက်ရိုက်သွင်းနိုင်ပြီး MCP ဆာဗာအား အသုံးပြုနိုင်သည်။ ၎င်းက စီစဉ်တတ်သောအရာများမှာ -
- VS Code မှာဘဲ documentation ရှာဖွေ ဖတ်ရှုနိုင်ပြီး coding ပတ်ဝန်းကျင်မှ ထွက်မနေပါ။
- documentation ကို ရည်ညွှန်းလိုသောနေရာတွင်လင့်ခ်ဖြည့်ရန် ခွင့်ပြုပါသည်။
- GitHub Copilot နှင့် MCP ကိုပေါင်းစပ် ပြီး AI စွမ်းအား ကူညီသော documentation workflow အတွက် အသုံးပြုနိုင်သည်။

**အသုံးပြုမှု နမူနာများ:**
- README ဖိုင်ထဲတွင် ရည်ညွှန်းလင့်ခ်များကို ချက်ချင်း ထည့်သွင်းလိုက်ပါ။
- Copilot က ကိုဒ်ထုတ်ပေးစဉ် MCP က အချက်အလက် ရှာဖွေ ဆွဲဆောင်ပါ။
- editor တွင် အာရုံစူးစိုက် ထားပြီး ထွက်ခွာမှုမရှိ စွမ်းဆောင်နိုင်မှု တိုးတက်စေပါသည်။

> [!IMPORTANT]
> သင်၏ workspace တွင် တရားဝင် [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) configuration မရှိမဖြစ် လိုအပ်သည် (တည်နေရာ `.vscode/mcp.json`)။

## အမှုကိစ္စ ၂ အတွက် Chainlit သုံးရတဲ့ အကြောင်းရင်း

Chainlit သည် စကားပြောဆိုနိုင်သော ဝက်ဘ်လ်အပလီကေးရှင်းများ ဖန်တီးရာတွင် အသုံးပြုနိုင်သော ခေတ်မီအသုံးအဆောင် framework ဖြစ်သည်။ Microsoft Learn Docs MCP ဆာဗာကဲ့သို့ ကျန် backend ဝန်ဆောင်မှုများနှင့် ချိတ်ဆက်နိုင်သော chat ပေါ် မျက်နှာပြင်များ ဖန်တီးရန် လွယ်ကူစေရန် ဒီ project မှာ Chainlit ကို အသုံးပြုထားသည်။ Chainlit က အချိန်နှင့်တပြေးညီ ကိုယ်ပိုင် သင်ယူအစီအစဉ်များကို စီမံရာတွင် အလွန် တော့လွယ်ကူပြီး အပြန်အလှန် ဆက်သွယ်မှု ပိုမိုရရှိစေသည်။ Chainlit ဖြင့် သင်သည် စိတ်တိုင်းကျ chat tools များ အလွယ်တကူ တည်ဆောက် deployment ဖြင့် ထုတ်လုပ်နိုင်မည်ဖြစ်ပြီး လေ့လာမှုနှင့် ထုတ်လုပ်မှုမြှင့်တင်မှုကို တိုးတက်စေမည်ဖြစ်သည်။

## ဒီកម្មវិធី၏ လုပ်ဆောင်ချက်

ဤ app သည် အသုံးပြုသူများအား ဘာသာရပ်နှင့် သတ်မှတ်သည့် အချိန်ကာလ တစ်ခုသာ ရိုက်ထည့်၍ ကိုယ်ပိုင် သင်ယူအစီအစဉ် ဖန်တီးရန် ခွင့်ပြုသည်။ app သည် သင့်ရိုက်ထည့်ချက်ကို စိစစ်ပြီး Microsoft Learn Docs MCP ဆာဗာသို့ မေးခွန်းတစ်ခု ပေးပို့စေကာ သက်ဆိုင်ရာအချက်အလက်များကို စုစည်း ပြီး အပတ်ပေါင်း၏ အစီအစဉ်တစ်ခုအဖြစ် စီမံခန့်ခွဲပေးသည်။ အပတ်စဉ် အကြံပြုချက်များကို chat တွင် ပြပါသည်၊ သင်၏ တိုးတက်မှုကို လိုက်လံကြည့်ရှုရန် လွယ်ကူစေသည်။ ပေါင်းစပ်မှုကြောင့် တစ်ခါတည်း နောက်ဆုံးပေါ် သင်ယူစရာ အရင်းအမြစ်များကို လက်လှမ်းမှီစေသည်။

ဒီမေးခွန်းများကို chat ပြတင်းပေါ်တွင် စမ်းသပ် ကြည့်ပါ။
