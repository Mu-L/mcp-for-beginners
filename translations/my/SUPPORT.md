# စွမ်းဆောင်မှု

## ပဋိညာဉ်တင်ခြင်းနှင့် အကူအညီရယူနည်း  

ဤပရောဂျက်သည် GitHub Issues ကို ပျက်ကွက်ချက်များနှင့် လုပ်ဆောင်ချက်တောင်းဆိုချက်များကို မှတ်တမ်းတင်ရန် အသုံးပြုသည်။ မသိမမှတ် မှတ်တမ်းတင်မှု မဖြစ်စေရန် ရှိပြီးသား 
ပဋိညာဉ်များကို ရှာဖွေကြည့်ပါ။ အသစ်တင်ရန်အတွက်တော့ သင်၏ ပျက်ကွက်ချက် သို့မဟုတ် 
လုပ်ဆောင်ချက်တောင်းဆိုချက်ကို အသစ်တင်ပါ။

ဤပရောဂျက်ကို အသုံးပြုပုံနှင့် မေးခွန်းများအတွက် ကျေးဇူးပြုပြီး-
- ကျွမ်းကျင်သူများနှင့် အတူ Developer များစုစည်းရာ [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ကို စစ်ဆေးပါ
- protocol နှင့် ပတ်သက်သော မေးခွန်းများအတွက် [Model Context Protocol Community](https://modelcontextprotocol.io/community/) အရင်းအမြစ်များကို သုံးပါ

## Microsoft အထောက်အပံ့ မူဝါဒ  

Model Context Protocol (MCP) for Beginners ပရောဂျက်အတွက် အထောက်အပံ့သည် အထက်ဖော်ပြပါ အရင်းအမြစ်များကို ကျန်ရှိသည်။
