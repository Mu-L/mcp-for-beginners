# అడ్వాన్సడ్ MCP సెక్యూరిటీ తో Azure కంటెంట్ సేఫ్టీ

> **OWASP MCP రిస్క్ అడ్రస్ చేయబడింది**: [MCP06 - ఉద్దేశ్య ప్రవాహ స్వభావప్రతిఘటన](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

Azure కంటెంట్ సేఫ్టీ మీ MCP అమలుల సెక్యూరిటీని మెరుగుపరచడానికి అనేక శక్తివంతమైన సాధనాలు అందిస్తుంది. ప్రాక్టికల్ అమలుకు, [MCP సెక్యూరిటీ సమ్మిట్ వర్క్షాప్ (షెర్పా)](https://azure-samples.github.io/sherpa/) క్యాంప్ 3: I/O సెక్యూరిటీని పరిశీలించండి.

## ప్రాంప్ట్ షీల్డ్స్

Microsoft యొక్క AI ప్రాంప్ట్ షీల్డ్స్ నేరుగా మరియు పరోక్షంగా ప్రాంప్ట్ ఇంజక్షన్ దాడులనుండి కఠినమైన రక్షణను ఇస్తాయి:

1. **అడ్వాన్స్డ్ డిటెక్షన్**: కంటెంట్‌లో దుష్టమైన సూచనలను గుర్తించడానికి మెషీన్ లెర్నింగ్ ఉపయోగిస్తుంది.
2. **స్పాట్లైట్ చేయడం**: ఇన్పుట్ టెక్ట్స్‌ని మార్చి AI సిస్టమ్‌లకు సరైన సూచనలు మరియు బాహ్య ఇన్పుట్‌లను వేరు చేసుకోవడంలో సహాయం చేస్తుంది.
3. **డెలిమిటర్స్ మరియు డేటామార్కింగ్**: నమ్మకమైన మరియు నమ్మకమేని డేటాకు మధ్య సరిహద్దులను గుర్తిస్తుంది.
4. **కంటెంట్ సేఫ్టీ ఇంటిగ్రేషన్**: Azure AI కంటెంట్ సేఫ్టీతో కలిసి జైల్ బ్రేక్ ప్రయత్నాలు మరియు హానికర కంటెంట్‌ను గుర్తిస్తుంది.
5. **నిరంతర నవీకరణలు**: Microsoft కొత్త ముప్పులపై రక్షణ వ్యవస్థలను తరచూ అప్‌డేట్ చేస్తుంది.

## Azure కంటెంట్ సేఫ్టీని MCP తో అమలు చేయడం

ఈ విధానం బహుళస్థాయి రక్షణను అందిస్తుంది:
- ప్రాసెసింగ్ ముందు ఇన్పుట్‌ల స్కానింగ్
- రిటర్న్ చేయడం ముందు అవుట్‌పుట్‌ల తనిఖీ
- తెలిసిన హానికర నమూనాల కోసం బ్లాక్‌లిస్ట్‌లు ఉపయోగించడం
- Azure యొక్క నిరంతర నవీకరణ పొందిన కంటెంట్ సేఫ్టీ మోడల్‌లను వినియోగించడం

## Azure కంటెంట్ సేఫ్టీ వనరులు

మీ MCP సర్వర్లతో Azure కంటెంట్ సేఫ్టీని అమలు చేయడం గురించి తెలుసుకోవడానికి, ఈ అధికారిక వనరులను పరిశీలించండి:

1. [Azure AI కంటెంట్ సేఫ్టీ డాక్యుమెంటేషన్](https://learn.microsoft.com/azure/ai-services/content-safety/) - Azure కంటెంట్ సేఫ్టీకి సంబంధించిన అధికారిక డాక్యుమెంటేషన్.
2. [ప్రాంప్ట్ షీల్డ్ డాక్యుమెంటేషన్](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/prompt-shield) - ప్రాంప్ట్ ఇంజక్షన్ దాడులను నివారించే విధానం తెలుసుకోండి.
3. [కంటెంట్ సేఫ్టీ API రిఫరెన్స్](https://learn.microsoft.com/rest/api/contentsafety/) - కంటెంట్ సేఫ్టీని అమలు కోసం వివరణాత్మక API సూచిక.
4. [క్విక్‌స్టార్ట్: C# తో Azure కంటెంట్ సేఫ్టీ](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-csharp) - C# ఉపయోగించి త్వరిత అమలు గైడ్.
5. [కంటెంట్ సేఫ్టీ క్లైంట్ లైబ్రరీలు](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-client-libraries-rest-api) - వివిధ ప్రోగ్రామింగ్ భాషల కోసం క్లైంట్ లైబ్రరీలు.
6. [జైల్ బ్రేక్ ప్రయత్నాలను గుర్తించడం](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/jailbreak-detection) - జైల్ బ్రేక్ ప్రయత్నాలను గుర్తించడం మరియు నివారించడంపై ప్రత్యేక మార్గదర్శకం.
7. [కంటెంట్ సేఫ్టీకి ఉత్తమ ఆచారాలు](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/best-practices) - కంటెంట్ సేఫ్టీని సమర్థవంతంగా అమలుచేయడానికి ఉత్తమ విధానాలు.

ఇంకా లోతైన అమలుకు, మా [Azure కంటెంట్ సేఫ్టీ అమలులు గైడ్](./azure-content-safety-implementation.md) చూడండి.
