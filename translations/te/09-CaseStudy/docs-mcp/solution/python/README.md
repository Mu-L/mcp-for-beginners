# Chainlit & Microsoft Learn Docs MCPతో స్టడీ ప్లాన్ జనరేటర్

## ప్రాధమిక అవసరాలు

- Python 3.8 లేదా అంతకంటే పైగా
- pip (Python ప్యాకేజ్ మేనేజర్)
- Microsoft Learn Docs MCP సర్వర్‌కు కనెక్ట్ కావడానికి ఇంటర్నెట్ యాక్సెస్

## ఇన్‌స్టాలేషన్

1. ఈ రిపాజిటరీని క్లోన్ చేయండి లేదా ప్రాజెక్ట్ ఫైళ్లను డౌన్లోడ్ చేయండి.
2. కావలసిన డిపెండెన్సీలను ఇన్‌స్టాల్ చేయండి:

   ```bash
   pip install -r requirements.txt
   ```

## ఉపయోగం

### సన్నివేశం 1: Docs MCP కు సింపుల్ క్వెరీ
Docs MCP సర్వర్‌కు కనెక్ట్ అయ్యే, ఒక క్వెరీ పంపించే, మరియు ఫలితాన్ని ప్రింట్ చేసే కమాండ్ లైన్ క్లయింట్.

1. స్క్రిప్ట్‌ను రన్ చేయండి:
   ```bash
   python scenario1.py
   ```
2. ప్రాంప్ట్ వద్ద మీ డాక్యుమెంటేషన్ ప్రశ్నను ఇవ్వండి.

### సన్నివేశం 2: స్టడీ ప్లాన్ జనరేటర్ (Chainlit వెబ్ యాప్)
వాడుకరులు ఏదైనా సాంకేతిక విషయానికి వ్యక్తిగత, వారం వారీ స్టడీ ప్లాన్‌ను రూపొందించుకునేందుకు అనుమతించే వెబ్ ఆధారిత ఇంటర్‌ఫేస్ (Chainlit ఉపయోగించి).

1. Chainlit యాప్‌ను ప్రారంభించండి:
   ```bash
   chainlit run scenario2.py
   ```
2. మీ టెర్మినల్లో ఇచ్చిన లోకల్ URL (ఉదాహరణకు, http://localhost:8000) ను బ్రౌజర్లో తెరవండి.
3. చాట్ విండోలో మీ స్టడీ టాపిక్ మరియు మీరు స్టడీ చేయాలనుకునే వారాల సంఖ్యను నమోదు చేయండి (ఉదా: "AI-900 సర్టిఫికేషన్, 8 వారం").
4. యాప్ సంబంధించిన Microsoft Learn డాక్యుమెంటేషన్ లింకులతో వారాల వారీ స్టడీ ప్లాన్‌తో స్పందిస్తుంది.

**అవసరమైన ఎన్విరాన్‌మెంట్ వేరియబుల్స్:**

సన్నివేశం 2 (Azure OpenAIతో Chainlit వెబ్ యాప్) ఉపయోగించడానికి, మీరు `python` డైరెక్టరీలో రచన `.env` ఫైల్లో ఈ ఎన్విరాన్‌మెంట్ వేరియబుల్స్‌ని సెటప్ చేయాలి:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

యాప్ ప్రారంభించడానికి ముందు ఈ విలువలను మీ Azure OpenAI వనరు వివరాలతో నింపండి.

> [!TIP]
> మీరు మీ సొంత మోడల్స్‌ను సులభంగా [Microsoft Foundry](https://ai.azure.com/) ఉపయోగించి డిప్లాయ్ చేయవచ్చు.

### సన్నివేశం 3: VS కోడ్‌లో MCP సర్వర్‌తో ఇన్-ఎడిటర్ డాక్స్

డాక్యుమెంటేషన్ వెతకడానికి బ్రౌజర్ టాబ్స్ మార్చుకోవడం కాకుండా, మీరు Microsoft Learn Docs ను నేరుగా VS కోడ్‌లో MCP సర్వర్ ఉపయోగించి తెచ్చుకోవచ్చు. ఇది మీకు:
- VS కోడ్ లోనే డాక్స్ శోధించి చదవడం కుడా, కోడింగ్ వాతావరణం విడిచి పోకుండా చేయగలదు.
- డాక్యుమెంటేషన్ సూచనలు ఇచ్చి README లేదా కోర్స్ ఫైల్స్‌లో నేరుగా లింకులు చేర్చి ఉపయోగించండి.
- GitHub Copilot మరియు MCP కలిసి సజావుగా పనిచేసే AI ఆధారిత డాక్యుమెంటేషన్ వర్క్‌ఫ్లో ని అందిస్తుంది.

**ఉదాహరణ ఉపయోగాలు:**
- కోర్స్ లేదా ప్రాజెక్ట్ డాక్యుమెంటేషన్ వ్రాయేటప్పుడు READMEకు త్వరగా రిఫరెన్స్ లింకులు జోడించండి.
- కోడ్ ఉత్పత్తి చేసేందుకు Copilot ఉపయోగించి, సంబంధిత డాక్స్‌ను తక్షణమే కనుగొనడానికి MCP ఉపయోగించండి.
- మీ ఎడిటర్‌లో концентрация గా ఉండి ఉత్పాదకత పెంచుకోండి.

> [!IMPORTANT]
> మీ వర్క్‌స్పేస్‌లో చెలామణిలో ఉన్న [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) కాన్ఫిగరేషన్ (స్థానం `.vscode/mcp.json`) ఉన్నదని ఖాయం చేసుకోండి.

## సన్నివేశం 2కి Chainlit ఎందుకు?

Chainlit అనేది సంభాషణాత్మక వెబ్ అప్లికేషన్లు సృష్టించడానికి ఆధునిక ఓపెన్ సోర్స్ ఫ్రేమ్‌వర్క్. ఇది Microsoft Learn Docs MCP సర్వర్ వంటి బ్యాక్‌ఎండ్ సర్వీసులతో కనెక్ట్ అయ్యే చాట్ ఆధారిత యూజర్ ఇంటర్‌ఫేస్‌లను సులభంగా సృష్టిస్తుంది. ఈ ప్రాజెక్ట్ Chainlit ఉపయోగించి వ్యక్తిగతీకరించిన స్టడీ ప్లాన్‌లను రియల్ టైమ్ లో సృష్టించడానికి సరళ, ఇంటరాక్టివ్ మార్గాన్ని అందిస్తుంది. Chainlit వలన మీరు త్వరగా చాట్ ఆధారిత టూల్స్‌ను నిర్మించి, ఉత్పాదకత మరియు అభ్యాసాన్ని మెరుగుజేస్తారు.

## ఇది ఏమి చేస్తుంది

ఈ యాప్ యూజర్లకు ఒక టాపిక్ మరియు వ్యవధిని మాత్రమే ఇవ్వడం ద్వారా వ్యక్తిగతీకరించిన స్టడీ ప్లాన్ సృష్టించడాన్ని అనుమతిస్తుంది. యాప్ మీ ఇన్‌పుట్‌ను పార్స్ చేసి, సంబంధిత విషయాలకు Microsoft Learn Docs MCP సర్వర్‌ను ఆడుతుంది, మరియు ఫలితాలను వారం వారీగా అమరిక చేసిన ప్లానులో నిర్వచిస్తుంది. ప్రతి వారం సిఫార్సులు చాట్ లో ప్రదర్శింపబడతాయి, తద్వారా మీరు సులభంగా అనుసరించగలరు మరియు మీ పురోగతిని ట్రాక్ చెయ్యగలరు. ఈ ఇంటిగ్రేషన్ మీకు ఎప్పుడూ తాజా మరియు అత్యంత సంబంధిత అభ్యాస వనరులు అందజేస్తుంది.

చాట్ విండోలో ఈ ప్రశ్నలను ప్రయత్నించి యాప్ ఎలా స్పందిస్తుంది అనేది చూడండి:

ఈ ఉదాహరణలు యాప్ వివిధ అభ్యాస లక్ష్యాలు మరియు సమయముల కోసం ఎంతవరకు సరిపోతుందో చూపిస్తాయి.
