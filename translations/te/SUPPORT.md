# మద్దతు

## సమస్యలను ఎలా నమోదు చేయాలి మరియు సహాయం పొందాలి

ఈ ప్రాజెక్ట్ బగ్స్ మరియు ఫీచర్ అభ్యర్థనలను ట్రాక్ చేయడానికి GitHub Issues ను ఉపయోగిస్తుంది. కొత్త సమస్యలను నమోదు చేసే ముందు దయచేసి ఇప్పటికే ఉన్న సమస్యలను వెతకండి, ఇది డూప్లికేట్లను నివారించడానికి. కొత్త సమస్యలకు, మీ బగ్ లేదా ఫీచర్ అభ్యర్థనను కొత్త Issue గా నమోదు చేయండి.

ఈ ప్రాజెక్ట్ ఉపయోగించే విషయంలో సహాయం మరియు ప్రశ్నల కోసం, దయచేసి:
- నిపుణులు మరియు ఇతర డెవలపర్‌లు చేరుకునే [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ను తనిఖీ చేయండి
- ప్రోటోకాల్-సంబంధిత ప్రశ్నల కోసం [Model Context Protocol Community](https://modelcontextprotocol.io/community/) వనరులను ఉపయోగించండి

## Microsoft మద్దతు విధానము

Beginners ప్రాజెక్టుకు సంబంధించిన Model Context Protocol (MCP) కి మద్దతు పైగా పేర్కొన్న వనరులకు మాత్రమే పరిమితం.
