## 🧠 Microsoft Foundry Toolkit పరిచయం

**Microsoft Foundry Toolkit Extension for VS Code** Microsoft యొక్క ప్రాథమిక ఎక్స్‌టెన్షన్, ఇది VS Code ను సమగ్ర AI అభివృద్ధి వాతావరణంగా మార్చుతుంది. ఇది AI పరిశోధన మరియు ప్రాక్టికల్ అప్లికేషన్ అభివృద్ధి మధ్య పొలాన్ని తొలగిస్తుంది, ప్రతి స్థాయి డెవలపర్లకు జనరేటివ్ AI ని అందిస్తుంది.

### 🎯 Microsoft Foundry Toolkit ముఖ్యత్యం

- **🚀 వేగవంతమైన అభివృద్ధి**: ఆలోచన నుంచి ప్రోటోటైప్ వరకు కొన్ని నిమిషాల్లో
- **🔄 ఏకీకృత పని విధానం**: బహుళ AI ప్రొవైడర్ల కోసం ఒకే ఇంటర్‌ఫేస్
- **🧪 సులభంగా ప్రయోగాలు**: సులభ సెటప్ లేకుండా మోడల్స్ ని తులనాత్మకంగా చూడండి
- **📈 ప్రొడక్షన్ రెడీ**: ప్రోటోటైప్ నుంచి డిప్లాయ్‌మెంట్ వరకు స్తిర మార్పు

## 🧪 ప్రాక్టికల్ వ్యాయామం 1: GitHub మోడల్స్ అన్వేషణ

**🎯 లక్ష్యం**: మోడల్ క్యాటలాగ్ లో నైపుణ్యం సాధించి మొదటి AI మోడల్ పరీక్ష చేయండి

### 📊 దశ 1: మోడల్ క్యాటలాగ్ నావిగేట్ చేయండి

మోడల్ క్యాటలాగ్ AI ఎకోసిస్టమ్ కి మీ ద్వారం. ఇది బహుళ ప్రొవైడర్ల నుండి మోడల్‌లను సమకూర్చుతుంది, అందువల్ల ఎంపికలను కనిపెట్టడం మరియు పోల్చడం సులభం అవుతుంది.

**🔍 నావిగేషన్ గైడ్:**

Microsoft Foundry Toolkit సైడ్‌బార్ లో **MODELS - Catalog** పై క్లిక్ చేయండి

![Model Catalog](../../../../translated_images/te/aimodel.263ed2be013d8fb0.webp)

**💡 ప్రో టిప్**: మీ వాడుక సందర్భానికి తగిన ప్రత్యేక సామర్ధ్యాలు ఉన్న మోడల్‌లను చూసుకోండి (ఉదా: కోడ్ సృష్టి, సృజనాత్మక రచన, విశ్లేషణ).

**⚠️ గమనిక**: GitHub-ఆధారిత మోడల్‌లు (అంటే GitHub మోడల్‌లు) ఉచితంగా ఉపయోగించవచ్చు, కానీ రిక్వెస్ట్‌లు మరియు టోకెన్లకు పరిమితులు ఉంటాయి. మీరు GitHub కాకుండా ఉన్న మోడల్‌లకు (అంటే Azure AI లేదా ఇతర ఎండ్‌పాయింట్లలో హోస్ట్ చేసిన మోడల్స్) యాక్సెస్ కోరుకుంటే, సంబంధిత API కీ లేదా ధృవపత్రం అవసరం.

### 🚀 దశ 2: మీ మొదటి మోడల్ జోడించి కాన్ఫిగర్ చేయండి

**మోడల్ ఎంపిక వ్యూహం:**
- **GPT-4.1**: సంక్లిష్ట తర్కం మరియు విశ్లేషణ కోసం ఉత్తమం
- **Phi-4-mini**: తక్కువ బరువు, సులభ పనుల కోసం వేగవంతమైన స్పందనలు

**🔧 కాన్ఫిగరేషన్ ప్రక్రియ:**
1. క్యాటలాగ్ నుండి **OpenAI GPT-4.1** ఎంచుకోండి
2. **Add to My Models** క్లిక్ చేయండి - ఇది మోడల్ వినియోగం కోసం నమోదు చేస్తుంది
3. **Try in Playground** ఎంచుకుని పరీక్ష వాతావరణాన్ని ప్రారంభించండి
4. మోడల్ ప్రారంభం అయ్యే వరకు వేచి ఉండండి (మొదటి సారి సెటప్ కొంత సమయం పడుతుంది)

![Playground Setup](../../../../translated_images/te/playground.dd6f5141344878ca.webp)

**⚙️ మోడల్ పరామితుల అర్థం చేద్దాం:**
- **Temperature**: సృజనాత్మకతను నియంత్రిస్తుంది (0 = నిర్దిష్టమైన, 1 = సృజనాత్మకమైన)
- **Max Tokens**: గరిష్ట స్పందన పొడవు
- **Top-p**: స్పందన వైవిధ్యానికి సంకల్ప నమూనా

### 🎯 దశ 3: ప్లేగ్రౌండ్ ఇంటర్‌ఫేస్ నైపుణ్యం

ప్లేగ్రౌండ్ మీ AI ప్రయోగాల ప్రయోగశాల. దీని సామర్థ్యాన్ని ఎలా ప్రొత్తినిపెట్టుకోవచ్చో చూడండి:

**🎨 ప్రాంప్ట్ ఇంజనీరింగ్ ఉత్తమ పద్ధతులు:**
1. **ప్రముఖంగా ఉండండి**: స్పష్టమైన, వివరమైన సూచనలు మెరుగైన ఫలితాలు ఇస్తాయి
2. **పరిశ్రావకులు ఇవ్వండి**: సంబంధిత నేపథ్య సమాచారం చేర్చండి
3. **ఉదాహరణలు ఇవ్వండి**: మోడల్ కి మీరు కోరేది చూపించండి
4. **పునరావృతం చేయండి**: మొదటి ఫలితాల ఆధారంగా ప్రాంప్ట్‌లను మెరుగుపరచండి

**🧪 పరీక్ష కన్సెప్ట్లు:**
```markdown
# Example 1: Code Generation
"Write a Python function that calculates the factorial of a number using recursion. Include error handling and docstrings."

# Example 2: Creative Writing
"Write a professional email to a client explaining a project delay, maintaining a positive tone while being transparent about challenges."

# Example 3: Data Analysis
"Analyze this sales data and provide insights: [paste your data]. Focus on trends, anomalies, and actionable recommendations."
```

## 🤖 ప్రాక్టికల్ వ్యాయామం 2: Agent Builder తో అనుకూల ఏజెంట్లు నిర్మాణం

**🎯 లక్ష్యం**: నిర్దిష్ట పనులు మరియు వర్క్‌ఫ్లోలకు తగిన ప్రత్యేక AI ఏజెంట్లను సృష్టించండి

### 🏗️ దశ 1: Agent Builder అర్థం చేసుకోవడం

Agent Builder Microsoft Foundry Toolkit లో నిజంగా ప్రతిభను ప్రదర్శించేది. ఇది పెద్ద భాషా మోడల్స్ శక్తిని అనుకూల సూచనలు, ప్రత్యేక పరామితులు మరియు ప్రత్యేక జ్ఞానం తో కలిపి లక్ష్యం ఉద్దేశ్య AI సహాయకులను సృష్టించటానికి అనుమతిస్తుంది.

**🧠 ఏజెంట్ నిర్మాణ భాగాలు:**
- **మూల మోడల్**: ప్రాథమిక LLM (GPT-4, Groks, Phi, మొదలైనవి)
- **సిస్టమ్ ప్రాంప్ట్**: ఏజెంట్ వ్యక్తిత్వం మరియు ప్రవర్తన నిర్వచిస్తుంది
- **పరామితులు**: ఉత్తమ పనితీరు కోసం సరిచేయబడిన సెట్టింగ్లు
- **సాధనాలు ఇంటిగ్రేషన్**: బాహ్య API మరియు MCP సేవలకు కనెక్షన్
- **మెమోరీ**: సంభాషణ సందర్భం మరియు సెషన్ నిలుపుదల
