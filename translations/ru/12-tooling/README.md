# 🚀 Инструменты MCP

## 🧠 Обзор учебного пути интеграции базы данных MCP

Здесь вы научитесь использовать инструменты, которые используют MCP. Ваш опыт использования этих инструментов улучшится по мере подключения к ним серверов MCP.

## 🔗 Официальные ресурсы MCP

- 📘 [Документация MCP](https://modelcontextprotocol.io/) – Подробные учебники и руководства пользователя
- 📜 [Спецификация MCP (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – Архитектура протокола и технические справочники
- 🧑‍💻 [Репозиторий MCP на GitHub](https://github.com/modelcontextprotocol) – Открытые SDK, инструменты и примеры кода
- 🌐 [Сообщество MCP](https://github.com/orgs/modelcontextprotocol/discussions) – Участвуйте в обсуждениях и вносите вклад в сообщество
- 🔒 [OWASP MCP Топ 10](https://microsoft.github.io/mcp-azure-security-guide/mcp/) – Лучшие практики безопасности и снижение рисков
