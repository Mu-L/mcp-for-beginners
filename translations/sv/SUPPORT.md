# Support

## Hur man rapporterar problem och får hjälp  

Detta projekt använder GitHub Issues för att spåra buggar och funktionsförfrågningar. Vänligen sök igenom befintliga 
issues innan du skapar nya för att undvika dubbletter. För nya issues, filera din bugg eller 
funktionsförfrågan som en ny Issue.

För hjälp och frågor om användning av detta projekt, vänligen:
- Kolla in [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) där experter och medutvecklare samlas
- Använd resurserna från [Model Context Protocol Community](https://modelcontextprotocol.io/community/) för protokollrelaterade frågor

## Microsoft Support Policy  

Support för Model Context Protocol (MCP) för nybörjarprojekt är begränsad till de resurser som listas ovan.
