# תרחיש 3: תיעוד בתוך העורך עם שרת MCP ב-VS Code

## סקירה כללית

בתרחיש זה, תלמד כיצד להכניס את מסמכי Microsoft Learn ישירות לסביבת Visual Studio Code שלך באמצעות שרת MCP. במקום להחליף כל הזמן בין טאבים בדפדפן כדי לחפש תיעוד, תוכל לגשת, לחפש ולהפנות למסמכים הרשמיים ישירות בתוך העורך שלך. גישה זו מייעלת את תהליך העבודה שלך, שומרת על הריכוז שלך ומאפשרת אינטגרציה חלקה עם כלים כמו GitHub Copilot.

- חפש וקרא מסמכים בתוך VS Code מבלי לעזוב את סביבת הקידוד שלך.
- הפנה לתיעוד והכנס קישורים ישירות לקובצי README או לקורס שלך.
- השתמש ב-GitHub Copilot וב-MCP יחד לתהליך תיעוד חלק ומונחה בינה מלאכותית.

## מטרות הלמידה

בסוף פרק זה, תבין כיצד להגדיר ולהשתמש בשרת MCP בתוך VS Code כדי לשפר את תהליך התיעוד והפיתוח שלך. תוכל:

- להגדיר את סביבת העבודה שלך להשתמש בשרת MCP לחיפוש תיעוד.
- לחפש ולהכניס תיעוד ישירות מתוך VS Code.
- לשלב את העוצמה של GitHub Copilot ו-MCP לתהליך עבודה פרודוקטיבי ומועשר בבינה מלאכותית.

כישורים אלו יעזרו לך להישאר מרוכז, לשפר את איכות התיעוד ולהגביר את הפרודוקטיביות שלך כמפתח או ככותב טכני.

## פתרון

כדי להשיג גישה לתיעוד בתוך העורך, תלך אחר סדרת שלבים שמשלבים את שרת MCP עם VS Code ו-GitHub Copilot. פתרון זה מתאים לסופרי קורסים, כותבי תיעוד ומפתחים שרוצים לשמור על הריכוז שלהם בתוך העורך בזמן העבודה עם התיעוד וה-Copilot.

- הוסף במהירות קישורי הפניה ל-README תוך כדי כתיבת קורס או תיעוד פרויקט.
- השתמש ב-Copilot לייצור קוד וב-MCP למציאת ציטוט מידי לתיעוד רלוונטי.
- הישאר מרוכז בעורך שלך והגבר פרודוקטיביות.

### מדריך שלב-אחר-שלב

כדי להתחיל, עקוב אחר השלבים הבאים. עבור כל שלב, תוכל להוסיף צילום מסך מתיקיית הנכסים כדי להמחיש את התהליך ויזואלית.

1. **הוסף את הגדרת ה-MCP:**
   בשורש הפרויקט שלך, צור קובץ `.vscode/mcp.json` והוסף את ההגדרות הבאות:
   ```json
   {
     "servers": {
       "LearnDocsMCP": {
         "url": "https://learn.microsoft.com/api/mcp"
       }
     }
   }
   ```
   הגדרה זו מורה ל-VS Code כיצד להתחבר ל-[`Microsoft Learn Docs MCP server`](https://github.com/MicrosoftDocs/mcp).
   
   ![שלב 1: הוסף mcp.json לתיקיית .vscode](../../../../../../translated_images/he/step1-mcp-json.c06a007fccc3edfa.webp)
    
2. **פתח את לוח הצ’אט של GitHub Copilot:**
   אם לא התקנת כבר את התוסף GitHub Copilot, עבור לתצוגת התוספים ב-VS Code והתקן אותו. ניתן להוריד אותו ישירות מ-[Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot-chat). לאחר מכן, פתח את לוח הצ’אט של Copilot מהסרגל הצדדי.

   ![שלב 2: פתח את לוח הצ’אט של Copilot](../../../../../../translated_images/he/step2-copilot-panel.f1cc86e9b9b8cd1a.webp)

3. **אפשר מצב סוכן ואמת את הכלים:**
   בלוח הצ’אט של Copilot, אפשר את מצב הסוכן.

   ![שלב 3: אפשר מצב סוכן ואמת את הכלים](../../../../../../translated_images/he/step3-agent-mode.cdc32520fd7dd1d1.webp)

   לאחר הפעלת מצב הסוכן, אמת שהשרת MCP מופיע כאחד מכלי העבודה הזמינים. זה מוודא שסוכן Copilot יכול לגשת לשרת התיעוד כדי לשלוף מידע רלוונטי.
   
   ![שלב 3: אמת את כלי שרת MCP](../../../../../../translated_images/he/step3-verify-mcp-tool.76096a6329cbfecd.webp)
4. **התחל שיחה חדשה ופנה לסוכן:**
   פתח שיחה חדשה בלוח הצ’אט של Copilot. כעת תוכל לפנות לסוכן עם שאילתות התיעוד שלך. הסוכן ישתמש בשרת MCP כדי לאחזר ולהציג תיעוד Microsoft Learn רלוונטי ישירות בעורך שלך.

   - *"אני מנסה לכתוב תוכנית לימוד לנושא X. אני מתכוון ללמוד אותו במשך 8 שבועות, עבור כל שבוע, הצע תוכן שעלי לקחת."*

   ![שלב 4: פנה לסוכן בצ’אט](../../../../../../translated_images/he/step4-prompt-chat.12187bb001605efc.webp)

5. **שאילתא חיה:**

   > בואו ניקח שאילתא חיה מתוך סקשן [#get-help](https://discord.gg/D6cRhjHWSC) בדיסקורד Microsoft Foundry ([צפה בהודעה המקורית](https://discord.com/channels/1113626258182504448/1385498306720829572)):
   
   *"אני מחפש תשובות כיצד לפרוס פתרון מרובה סוכנים עם סוכני AI שפותחו ב-Azure AI Foundry. אני רואה שאין שיטת פריסה ישירה, כמו ערוצי Copilot Studio. אז, מהן הדרכים השונות לבצע פריסה זו עבור משתמשי ארגונים כדי ליצור אינטראקציה ולהשלים את המשימה? ישנם מאמרים/בלוגים רבים שאומרים שניתן להשתמש בשירות Azure Bot כדי לבצע את העבודה הזו שיכול לשמש כגשר בין MS Teams ל-Azure AI Foundry Agents, האם זה יעבוד אם אני אגדיר Azure bot שמתחבר ל-Orchestrator Agent על Azure AI Foundry דרך פונקציית Azure לביצוע התזמור או שעלי ליצור פונקציית Azure עבור כל אחד מסוכני ה-AI החלקים בפתרון מרובה הסוכנים כדי לבצע את התזמור במסגרת Bot? כל הצעה נוספת תהיה מבורכת."*

   ![שלב 5: שאילתות חיות](../../../../../../translated_images/he/step5-live-queries.49db3e4a50bea273.webp)

   הסוכן יגיב עם קישורים לספרות רלוונטית וסיכומים, אותם תוכל להכניס ישירות לקבצי ה-markdown שלך או להשתמש בהם כהפניות בקוד שלך.
   
### שאילתות לדוגמה

להלן כמה שאילתות לדוגמה שתוכל לנסות. שאילתות אלו יציגו כיצד שרת MCP ו-Copilot יכולים לעבוד יחד כדי לספק תיעוד והפניות מיידיות, מותאמות להקשר, מבלי לעזוב את VS Code:

- "הראה לי כיצד להשתמש בטריגרים של Azure Functions."
- "הכנס קישור לתיעוד הרשמי של Azure Key Vault."
- "מהן שיטות העבודה המומלצות לאבטחת משאבי Azure?"
- "מצא מדריך התחלה מהירה לשירותי Azure AI."

שאילתות אלו ידגימו כיצד שרת MCP ו-Copilot יכולים לעבוד יחד כדי לספק תיעוד והפניות מיידיות, מותאמות להקשר, מבלי לצאת מ-VS Code.
