# מחולל תוכנית לימודים עם Chainlit ומסמכי Microsoft Learn Docs MCP

## דרישות מוקדמות

- Python 3.8 ומעלה
- pip (מנהל החבילות של Python)
- חיבור לאינטרנט להתחברות לשרת Microsoft Learn Docs MCP

## התקנה

1. שיבטו את המאגר או הורידו את קבצי הפרויקט.
2. התקינו את התלויות הנדרשות:

   ```bash
   pip install -r requirements.txt
   ```

## שימוש

### תרחיש 1: שאילתה פשוטה ל-Docs MCP
לקוח שורת פקודה שמתחבר לשרת Docs MCP, שולח שאילתה ומדפיס את התוצאה.

1. הריצו את הסקריפט:
   ```bash
   python scenario1.py
   ```
2. הכניסו את שאלת התיעוד שלכם בהפנייה.

### תרחיש 2: מחולל תוכנית לימודים (אפליקציית רשת Chainlit)
ממשק מבוסס רשת (באמצעות Chainlit) המאפשר למשתמשים ליצור תוכנית לימודים מותאמת אישית, שבוע אחר שבוע, לכל נושא טכני.

1. התחילו את אפליקציית Chainlit:
   ```bash
   chainlit run scenario2.py
   ```
2. פתחו את כתובת ה-URL המקומית המוצגת בטרמינל שלכם (למשל, http://localhost:8000) בדפדפן.
3. בחלון הצ’אט, הכניסו את נושא הלימוד שלכם ואת מספר השבועות שברצונכם ללמוד (למשל, "AI-900 certification, 8 weeks").
4. האפליקציה תענה בתוכנית לימודים שבוע-אחרי-שבוע, כולל קישורים לתיעוד הרלוונטי של Microsoft Learn.

**משתני סביבה נדרשים:**

כדי להשתמש בתרחיש 2 (אפליקציית הרשת של Chainlit עם Azure OpenAI), עליכם להגדיר את משתני הסביבה הבאים בקובץ `.env` בתיקיית `python`:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

מלאו ערכים אלה בפרטי משאב Azure OpenAI שלכם לפני הרצת האפליקציה.

> [!TIP]
> ניתן לפרוס דגמים משלכם בקלות באמצעות [Microsoft Foundry](https://ai.azure.com/).

### תרחיש 3: מסמכי Docs בתוך העורך עם שרת MCP ב-VS Code

במקום לעבור בין לשוניות בדפדפן כדי לחפש תיעוד, תוכלו להביא את Microsoft Learn Docs ישירות ל-VS Code שלכם באמצעות שרת MCP. זה מאפשר לכם:
- לחפש ולקרוא תיעוד מתוך VS Code מבלי לעזוב את סביבת הקידוד.
- להפנות לתיעוד ולהכניס קישורים ישירות לקבצי README או קורס.
- להשתמש ב-GitHub Copilot ו-MCP יחד עבור תהליך תיעוד חלק ומונע בינה מלאכותית.

**דוגמאות לשימוש:**
- להוסיף במהירות קישורים להפניות ב-README תוך כדי כתיבת קורס או תיעוד פרויקט.
- להשתמש ב-Copilot כדי ליצור קוד ו-MCP כדי למצוא ולהצטט מסמכים רלוונטיים בצורה מיידית.
- לשמור על ריכוז בעורך ולשפר את הפרודוקטיביות.

> [!IMPORTANT]
> ודאו שיש לכם קובץ הגדרות תקף [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) בסביבת העבודה שלכם (המיקום הוא `.vscode/mcp.json`).

## למה Chainlit לתרחיש 2?

Chainlit היא מסגרת קוד פתוח מודרנית לבניית אפליקציות רשת שיחתיות. היא מקילה על יצירת ממשקי משתמש מבוססי צ’אט שמתחברים לשירותי backend כמו שרת Microsoft Learn Docs MCP. פרויקט זה משתמש ב-Chainlit כדי לספק דרך פשוטה ואינטראקטיבית ליצירת תוכניות לימודים מותאמות אישית בזמן אמת. באמצעות Chainlit, תוכלו לבנות ולפרוס במהירות כלים מבוססי צ’אט שמגבירים פרודוקטיביות ולמידה.

## מה זה עושה

האפליקציה מאפשרת למשתמשים ליצור תוכנית לימודים מותאמת אישית על ידי הזנת נושא ומשך זמן בלבד. האפליקציה מנתחת את הקלט שלכם, שואלת את שרת Microsoft Learn Docs MCP על תוכן רלוונטי, ומארגנת את התוצאות לתוכנית מובנית, שבוע אחר שבוע. ההמלצות של כל שבוע מוצגות בצ’אט, מה שמקל על המעקב וההתקדמות. האינטגרציה מבטיחה שתמיד תקבלו את המשאבים העדכניים והרלוונטיים ביותר ללימוד.

## שאילתות לדוגמה

נסו את השאילתות האלה בחלון הצ’אט כדי לראות כיצד האפליקציה מגיבה:

- `AI-900 certification, 8 weeks`
- `Learn Azure Functions, 4 weeks`
- `Azure DevOps, 6 weeks`
- `Data engineering on Azure, 10 weeks`
- `Microsoft security fundamentals, 5 weeks`
- `Power Platform, 7 weeks`
- `Azure AI services, 12 weeks`
- `Cloud architecture, 9 weeks`

דוגמאות אלו מדגימות את הגמישות של האפליקציה למטרות לימוד וזמני מחקר שונים.
