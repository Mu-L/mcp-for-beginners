# 🚀 כלים של MCP

## 🧠 סקירה כללית של מסלול הלמידה לשילוב מסד הנתונים של MCP

כאן תלמד כיצד להשתמש בכלים שמשתמשים ב-MCP. הניסיון שלך בשימוש בכלים אלו ישתפר ככל שתחבר שרתי MCP אליהם.

## 🔗 משאבי MCP רשמיים

- 📘 [תיעוד MCP](https://modelcontextprotocol.io/) – מדריכים מפורטים ומדריכי משתמש
- 📜 [מפרט MCP (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – ארכיטקטורת הפרוטוקול והפניות טכניות
- 🧑‍💻 [מאגר GitHub של MCP](https://github.com/modelcontextprotocol) – SDKs, כלים ודוגמאות קוד קוד פתוח
- 🌐 [קהילת MCP](https://github.com/orgs/modelcontextprotocol/discussions) – הצטרף לדיונים ותתרום לקהילה
- 🔒 [OWASP MCP עשרת המובילים](https://microsoft.github.io/mcp-azure-security-guide/mcp/) – שיטות מיטב אבטחה והפחתת סיכונים
