# תמיכה

## כיצד לדווח על בעיות ולקבל עזרה

הפרויקט הזה משתמש ב-GitHub Issues למעקב אחר באגים ובקשות לתכונות. אנא חפשו את הבעיות הקיימות לפני פתיחת בעיות חדשות כדי למנוע כפילויות. עבור בעיות חדשות, פתחו את הבאג או בקשת התכונה שלכם כבעיה חדשה.

לקבלת עזרה ושאלות בנוגע לשימוש בפרויקט זה, אנא:
- בדקו את [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) שם מתכנסים מומחים ומפתחים
- השתמשו במשאבים של [Model Context Protocol Community](https://modelcontextprotocol.io/community/) לשאלות הקשורות לפרוטוקול

## מדיניות התמיכה של מיקרוסופט

התמיכה בפרויקט Model Context Protocol (MCP) למתחילים מוגבלת למשאבים המפורטים למעלה.
