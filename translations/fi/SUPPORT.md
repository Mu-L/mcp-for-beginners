# Tuki

## Kuinka tehdä ilmoituksia ja saada apua  

Tämä projekti käyttää GitHub Issues -järjestelmää virheiden ja ominaisuuspyyntöjen seurantaan. Etsi olemassa olevat 
ilmoitukset ennen uusien tekemistä, jotta vältät päällekkäisyyksiä. Uusien ongelmien kohdalla tee virhe- tai 
ominaisuuspyyntö uutena Issue-ilmoituksena.

Apua ja kysymyksiä tästä projektista varten:
- Tarkista [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4), jossa asiantuntijat ja muut kehittäjät kokoontuvat
- Käytä [Model Context Protocol Community](https://modelcontextprotocol.io/community/) -resursseja protokollaan liittyvissä kysymyksissä

## Microsoftin tukipolitiikka  

Model Context Protocol (MCP) for Beginners -projektin tuki rajoittuu yllä lueteltuihin resursseihin.
