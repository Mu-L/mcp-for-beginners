# 支援

## 如何提交問題及獲取幫助

本專案使用 GitHub 問題追蹤錯誤和功能請求。請在提交新問題前先搜尋現有問題，以避免重覆。對於新問題，請以新 Issue 的形式提交您的錯誤或功能請求。

有關本專案的使用問題及查詢，請：
- 查看 [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4)，這裡聚集了專家和開發者同行
- 使用 [Model Context Protocol Community](https://modelcontextprotocol.io/community/) 的資源，查詢協議相關問題
