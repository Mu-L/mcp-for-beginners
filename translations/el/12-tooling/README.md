# 🚀 Εργαλεία MCP

## 🧠 Επισκόπηση της Διαδρομής Μάθησης Ενσωμάτωσης Βάσης Δεδομένων MCP

Εδώ θα μάθετε πώς να χρησιμοποιείτε εργαλεία που χρησιμοποιούν το MCP. Η εμπειρία σας με αυτά τα εργαλεία θα βελτιωθεί καθώς συνδέετε τους MCP διακομιστές με αυτά.

## 🔗 Επίσημοι Πόροι MCP

- 📘 [Τεκμηρίωση MCP](https://modelcontextprotocol.io/) – Αναλυτικά σεμινάρια και οδηγούς χρήστη
- 📜 [Προδιαγραφή MCP (2025-11-25)](https://spec.modelcontextprotocol.io/specification/2025-11-25/) – Αρχιτεκτονική πρωτοκόλλου και τεχνικές αναφορές
- 🧑‍💻 [Αποθετήριο GitHub MCP](https://github.com/modelcontextprotocol) – Ανοιχτού κώδικα SDKs, εργαλεία και παραδείγματα κώδικα
- 🌐 [Κοινότητα MCP](https://github.com/orgs/modelcontextprotocol/discussions) – Συμμετοχή σε συζητήσεις και συμβολή στην κοινότητα
- 🔒 [OWASP MCP Top 10](https://microsoft.github.io/mcp-azure-security-guide/mcp/) – Καλύτερες πρακτικές ασφάλειας και μετριάσεις κινδύνων
