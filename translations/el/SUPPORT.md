# Υποστήριξη

## Πώς να υποβάλετε θέματα και να λάβετε βοήθεια  

Αυτό το έργο χρησιμοποιεί τα GitHub Issues για την παρακολούθηση σφαλμάτων και αιτημάτων λειτουργιών. Παρακαλούμε αναζητήστε τα υπάρχοντα θέματα πριν υποβάλετε νέα, για να αποφύγετε διπλότυπα. Για νέα θέματα, υποβάλετε το σφάλμα ή το αίτημα λειτουργίας σας ως νέο Θέμα.

Για βοήθεια και ερωτήσεις σχετικά με τη χρήση αυτού του έργου, παρακαλούμε:
- Ελέγξτε το [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) όπου συγκεντρώνονται ειδικοί και συνεταιριστές προγραμματιστές
- Χρησιμοποιήστε τους πόρους της [Model Context Protocol Community](https://modelcontextprotocol.io/community/) για ερωτήσεις σχετικά με το πρωτόκολλο

## Πολιτική Υποστήριξης Microsoft  

Η υποστήριξη για το έργο Model Context Protocol (MCP) για αρχάριους περιορίζεται στους πόρους που αναφέρονται παραπάνω.
