# Δημιουργός Σχέδιου Μελέτης με Chainlit & Microsoft Learn Docs MCP

## Απαιτήσεις

- Python 3.8 ή νεότερη έκδοση
- pip (διαχειριστής πακέτων Python)
- Πρόσβαση στο Internet για σύνδεση με τον διακομιστή Microsoft Learn Docs MCP

## Εγκατάσταση

1. Κλωνοποιήστε αυτό το αποθετήριο ή κατεβάστε τα αρχεία του έργου.
2. Εγκαταστήστε τις απαιτούμενες εξαρτήσεις:

   ```bash
   pip install -r requirements.txt
   ```

## Χρήση

### Σενάριο 1: Απλό Ερώτημα προς Docs MCP
Ένας πελάτης γραμμής εντολών που συνδέεται με τον διακομιστή Docs MCP, στέλνει ένα ερώτημα και εμφανίζει το αποτέλεσμα.

1. Τρέξτε το script:
   ```bash
   python scenario1.py
   ```
2. Πληκτρολογήστε την ερώτησή σας για την τεκμηρίωση στην προτροπή.

### Σενάριο 2: Δημιουργός Σχέδιου Μελέτης (Εφαρμογή Ιστού Chainlit)
Ένα περιβάλλον ιστού (με χρήση Chainlit) που επιτρέπει στους χρήστες να δημιουργούν ένα εξατομικευμένο, εβδομαδιαίο σχέδιο μελέτης για οποιοδήποτε τεχνικό θέμα.

1. Ξεκινήστε την εφαρμογή Chainlit:
   ```bash
   chainlit run scenario2.py
   ```
2. Ανοίξτε το τοπικό URL που παρέχεται στο τερματικό σας (π.χ. http://localhost:8000) στον φυλλομετρητή σας.
3. Στο παράθυρο συνομιλίας, εισάγετε το θέμα μελέτης και τον αριθμό εβδομάδων που θέλετε να μελετήσετε (π.χ., "AI-900 certification, 8 weeks").
4. Η εφαρμογή θα απαντήσει με ένα εβδομαδιαίο σχέδιο μελέτης, συμπεριλαμβάνοντας συνδέσμους στην αντίστοιχη τεκμηρίωση Microsoft Learn.

**Απαιτούμενες Μεταβλητές Περιβάλλοντος:**

Για να χρησιμοποιήσετε το Σενάριο 2 (την εφαρμογή ιστού Chainlit με Azure OpenAI), πρέπει να ρυθμίσετε τις ακόλουθες μεταβλητές περιβάλλοντος σε αρχείο `.env` στον φάκελο `python`:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

Συμπληρώστε αυτές τις τιμές με τα στοιχεία του πόρου Azure OpenAI πριν τρέξετε την εφαρμογή.

> [!TIP]
> Μπορείτε εύκολα να αναπτύξετε τα δικά σας μοντέλα χρησιμοποιώντας το [Microsoft Foundry](https://ai.azure.com/).

### Σενάριο 3: Τεκμηρίωση μέσα στον Επεξεργαστή με MCP Server στο VS Code

Αντί να αλλάζετε καρτέλες στον φυλλομετρητή για αναζήτηση τεκμηρίωσης, μπορείτε να φέρετε το Microsoft Learn Docs απευθείας στο VS Code χρησιμοποιώντας τον διακομιστή MCP. Αυτό σας επιτρέπει να:
- Αναζητάτε και να διαβάζετε τεκμηρίωση μέσα στο VS Code χωρίς να αφήσετε το περιβάλλον κωδικοποίησης.
- Αναφέρετε τεκμηρίωση και εισάγετε συνδέσμους απευθείας σε README ή αρχεία μαθήματος.
- Χρησιμοποιείτε το GitHub Copilot και το MCP μαζί για μια ομαλή ροή εργασίας με τεχνητή νοημοσύνη και τεκμηρίωση.

**Παραδείγματα Χρήσης:**
- Προσθέστε γρήγορα συνδέσμους αναφοράς σε README όσο γράφετε τεκμηρίωση μαθήματος ή έργου.
- Χρησιμοποιήστε το Copilot για να δημιουργήσετε κώδικα και το MCP για να βρείτε και να παραθέσετε σχετική τεκμηρίωση αμέσως.
- Μείνετε συγκεντρωμένοι στον επεξεργαστή σας και αυξήστε την παραγωγικότητα.

> [!IMPORTANT]
> Βεβαιωθείτε ότι έχετε μια έγκυρη ρύθμιση [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) στον χώρο εργασίας σας (η θέση είναι `.vscode/mcp.json`).

## Γιατί Chainlit για το Σενάριο 2;

Το Chainlit είναι ένα σύγχρονο open-source πλαίσιο για την κατασκευή συνομιλιακών εφαρμογών ιστού. Διευκολύνει τη δημιουργία διεπαφών χρήστη βασισμένων σε συνομιλία που συνδέονται με υπηρεσίες στο παρασκήνιο όπως ο διακομιστής Microsoft Learn Docs MCP. Αυτό το έργο χρησιμοποιεί το Chainlit για να παρέχει έναν απλό, διαδραστικό τρόπο δημιουργίας εξατομικευμένων σχεδίων μελέτης σε πραγματικό χρόνο. Με τη χρήση του Chainlit, μπορείτε γρήγορα να δημιουργήσετε και να αναπτύξετε εργαλεία συνομιλίας που αυξάνουν την παραγωγικότητα και τη μάθηση.

## Τι Κάνει Αυτό

Αυτή η εφαρμογή επιτρέπει στους χρήστες να δημιουργούν ένα εξατομικευμένο σχέδιο μελέτης απλώς εισάγοντας ένα θέμα και μια διάρκεια. Η εφαρμογή αναλύει την είσοδό σας, στέλνει ερώτημα στον διακομιστή Microsoft Learn Docs MCP για σχετικό περιεχόμενο και οργανώνει τα αποτελέσματα σε μια δομημένη, εβδομαδιαία διάταξη. Οι προτάσεις κάθε εβδομάδας εμφανίζονται στη συνομιλία, καθιστώντας εύκολη την παρακολούθηση και την εξέλιξη. Η ενσωμάτωση εξασφαλίζει ότι λαμβάνετε πάντα τους πιο πρόσφατους και συναφείς εκπαιδευτικούς πόρους.
