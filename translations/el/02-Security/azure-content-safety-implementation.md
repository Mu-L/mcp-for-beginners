# Εφαρμογή Azure Content Safety με MCP

> **OWASP MCP Κίνδυνος που Αντιμετωπίζεται**: [MCP06 - Παραβίαση Ροής Σκοπού](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

Για να ενισχύσετε την ασφάλεια του MCP έναντι της ένεσης προτροπής, της δηλητηρίασης εργαλείων και άλλων ευπαθειών ειδικών για AI, συνιστάται ιδιαίτερα η ενσωμάτωση του Azure Content Safety. Αυτός ο οδηγός υλοποίησης ευθυγραμμίζεται με το [MCP Security Summit Workshop (Sherpa)](https://azure-samples.github.io/sherpa/) Camp 3: Ασφάλεια Εισόδου/Εξόδου.

## Ενσωμάτωση με τον MCP Server

Για να ενσωματώσετε το Azure Content Safety με τον MCP server σας, προσθέστε το φίλτρο content safety ως middleware στη ροή επεξεργασίας αιτημάτων σας:

1. Αρχικοποιήστε το φίλτρο κατά την εκκίνηση του server  
2. Επαληθεύστε όλα τα εισερχόμενα αιτήματα εργαλείων πριν την επεξεργασία  
3. Ελέγξτε όλες τις εξερχόμενες απαντήσεις πριν τις επιστρέψετε στους πελάτες  
4. Καταγράψτε και ειδοποιήστε για παραβιάσεις ασφαλείας  
5. Υλοποιήστε κατάλληλη διαχείριση σφαλμάτων για αποτυχημένους ελέγχους content safety  

Αυτό παρέχει μια ισχυρή άμυνα έναντι:  
- Επιθέσεων ένεσης προτροπής  
- Προσπαθειών δηλητηρίασης εργαλείων  
- Εξαγωγής δεδομένων μέσω κακόβουλων εισόδων  
- Γεννήτριας επιβλαβούς περιεχομένου  

## Καλές Πρακτικές για Ενσωμάτωση Azure Content Safety

1. **Προσαρμοσμένες Μαύρες Λίστες**: Δημιουργήστε προσαρμοσμένες μαύρες λίστες ειδικά για μοτίβα ένεσης MCP  
2. **Προσαρμογή Βαρύτητας**: Ρυθμίστε τα όρια σοβαρότητας με βάση τη συγκεκριμένη χρήση και την ανοχή κινδύνου σας  
3. **Πλήρης Κάλυψη**: Εφαρμόστε ελέγχους content safety σε όλες τις εισόδους και εξόδους  
4. **Βελτιστοποίηση Απόδοσης**: Σκεφτείτε την υλοποίηση caching για επαναλαμβανόμενους ελέγχους content safety  
5. **Μηχανισμοί Εφεδρείας**: Ορίστε σαφείς συμπεριφορές εφεδρείας όταν οι υπηρεσίες content safety δεν είναι διαθέσιμες  
6. **Ανατροφοδότηση Χρήστη**: Παρέχετε σαφή ανατροφοδότηση στους χρήστες όταν το περιεχόμενο μπλοκάρεται λόγω ανησυχιών ασφαλείας  
7. **Συνεχής Βελτίωση**: Ενημερώνετε τακτικά τις μαύρες λίστες και τα μοτίβα με βάση τις αναδυόμενες απειλές
