## 🧠 Εισαγωγή στο Microsoft Foundry Toolkit

Η **Microsoft Foundry Toolkit Extension για VS Code** είναι η ναυαρχίδα της Microsoft που μετατρέπει το VS Code σε ένα ολοκληρωμένο περιβάλλον ανάπτυξης AI. Γεφυρώνει το χάσμα μεταξύ της έρευνας AI και της πρακτικής ανάπτυξης εφαρμογών, καθιστώντας τη γενετική AI προσβάσιμη σε προγραμματιστές όλων των επιπέδων δεξιοτήτων.

### 🌟 Κύριες Δυνατότητες

| Δυνατότητα | Περιγραφή | Περίπτωση Χρήσης |
|---------|-------------|----------|
| **🗂️ Κατάλογος Μοντέλων** | Πρόσβαση σε 100+ μοντέλα από GitHub, ONNX, OpenAI, Anthropic, Google | Ανακάλυψη και επιλογή μοντέλων |
| **🔌 Υποστήριξη BYOM** | Ενσωμάτωση δικών σας μοντέλων (τοπικά/απομακρυσμένα) | Ανάπτυξη προσαρμοσμένων μοντέλων |
| **🎮 Διαδραστικό Playground** | Δοκιμή μοντέλων σε πραγματικό χρόνο με διεπαφή chat | Γρήγορο πρωτότυπο και δοκιμή |
| **📎 Υποστήριξη Πολλαπλών Τύπων** | Διαχείριση κειμένου, εικόνων και συνημμένων | Σύνθετες εφαρμογές AI |
| **⚡ Μαζική Επεξεργασία** | Εκτέλεση πολλαπλών προτροπών ταυτόχρονα | Αποτελεσματικές ροές εργασίας δοκιμών |
| **📊 Αξιολόγηση Μοντέλων** | Ενσωματωμένα μετρικά (F1, συνάφεια, ομοιότητα, συνοχή) | Αξιολόγηση απόδοσης |

### 🎯 Γιατί το Microsoft Foundry Toolkit είναι σημαντικό

- **🚀 Επιταχυνόμενη Ανάπτυξη**: Από την ιδέα στο πρωτότυπο σε λίγα λεπτά
- **🔄 Ενιαία Ροή Εργασίας**: Μια διεπαφή για πολλούς παρόχους AI
- **🧪 Εύκολος Πειραματισμός**: Σύγκριση μοντέλων χωρίς πολύπλοκη ρύθμιση
- **📈 Έτοιμο για Παραγωγή**: Αδιάλειπτη μετάβαση από πρωτότυπο σε ανάπτυξη

## 🧪 Πρακτική Άσκηση 1: Εξερεύνηση Μοντέλων GitHub

**🎯 Στόχος**: Κατακτήστε τον Κατάλογο Μοντέλων και δοκιμάστε το πρώτο AI μοντέλο σας

### 📊 Βήμα 1: Πλοηγηθείτε στον Κατάλογο Μοντέλων

Ο Κατάλογος Μοντέλων είναι η πύλη σας στο οικοσύστημα AI. Συγκεντρώνει μοντέλα από πολλούς παρόχους, κάνοντας εύκολη την ανακάλυψη και τη σύγκριση επιλογών.

**🔍 Οδηγός Πλοήγησης:**

Κάντε κλικ στο **MODELS - Catalog** στη πλαϊνή μπάρα του Microsoft Foundry Toolkit

![Model Catalog](../../../../translated_images/el/aimodel.263ed2be013d8fb0.webp)

**💡 Συμβουλή**: Αναζητήστε μοντέλα με συγκεκριμένες δυνατότητες που ταιριάζουν στη χρήση σας (π.χ. δημιουργία κώδικα, δημιουργική γραφή, ανάλυση).

**⚠️ Σημείωση**: Τα μοντέλα φιλοξενούμενα στο GitHub (δηλαδή GitHub Models) είναι δωρεάν στη χρήση αλλά υπόκεινται σε όρια ρυθμού στα αιτήματα και τα tokens. Αν θέλετε να έχετε πρόσβαση σε μη-GitHub μοντέλα (δηλαδή εξωτερικά μοντέλα που φιλοξενούνται μέσω Azure AI ή άλλων endpoints), θα πρέπει να παρέχετε το κατάλληλο κλειδί API ή πιστοποίηση.

### 🚀 Βήμα 2: Προσθέστε και Ρυθμίστε το Πρώτο σας Μοντέλο

**Στρατηγική Επιλογής Μοντέλου:**
- **GPT-4.1**: Ιδανικό για σύνθετη συλλογιστική και ανάλυση
- **Phi-4-mini**: Ελαφρύ, γρήγορες απαντήσεις για απλές εργασίες

**🔧 Διαδικασία Ρυθμίσεων:**
1. Επιλέξτε **OpenAI GPT-4.1** από τον κατάλογο
2. Κάντε κλικ **Add to My Models** - αυτό καταχωρεί το μοντέλο για χρήση
3. Επιλέξτε **Try in Playground** για να ανοίξετε το περιβάλλον δοκιμών
4. Περιμένετε για την αρχικοποίηση του μοντέλου (η πρώτη φορά μπορεί να διαρκέσει λίγο)

![Playground Setup](../../../../translated_images/el/playground.dd6f5141344878ca.webp)

**⚙️ Κατανόηση Παραμέτρων Μοντέλου:**
- **Temperature**: Ελέγχει τη δημιουργικότητα (0 = καθοριστικό, 1 = δημιουργικό)
- **Max Tokens**: Μέγιστο μήκος απάντησης
- **Top-p**: Δειγματοληψία nucleus για ποικιλία απαντήσεων

### 🎯 Βήμα 3: Εξοικείωση με τη Διεπαφή Playground

Το Playground είναι το εργαστήρι πειραματισμού σας με AI. Να πώς να αξιοποιήσετε τη μέγιστη δυναμική του:

**🎨 Βέλτιστες Πρακτικές Μηχανικής Προτροπών:**
1. **Να είστε συγκεκριμένοι**: Καθαρές, λεπτομερείς οδηγίες αποδίδουν καλύτερα αποτελέσματα
2. **Παρέχετε πλαίσιο**: Συμπεριλάβετε σχετικό υπόβαθρο
3. **Χρησιμοποιήστε παραδείγματα**: Δείξτε στο μοντέλο τι θέλετε με παραδείγματα
4. **Επαναλάβετε**: Βελτιώστε τις προτροπές βάσει των πρώτων αποτελεσμάτων

**🧪 Σενάρια Δοκιμών:**
```markdown
# Example 1: Code Generation
"Write a Python function that calculates the factorial of a number using recursion. Include error handling and docstrings."

# Example 2: Creative Writing
"Write a professional email to a client explaining a project delay, maintaining a positive tone while being transparent about challenges."

# Example 3: Data Analysis
"Analyze this sales data and provide insights: [paste your data]. Focus on trends, anomalies, and actionable recommendations."
```

## 🤖 Πρακτική Άσκηση 2: Δημιουργία Προσαρμοσμένων Πρακτόρων με το Agent Builder

**🎯 Στόχος**: Δημιουργήστε εξειδικευμένους πράκτορες AI προσαρμοσμένους σε συγκεκριμένες εργασίες και ροές εργασίας

### 🏗️ Βήμα 1: Κατανόηση του Agent Builder

Το Agent Builder είναι όπου το Microsoft Foundry Toolkit λάμπει πραγματικά. Σας επιτρέπει να δημιουργήσετε βοηθούς AI ειδικού σκοπού που συνδυάζουν τη δύναμη μεγάλων γλωσσικών μοντέλων με προσαρμοσμένες οδηγίες, συγκεκριμένες παραμέτρους και εξειδικευμένη γνώση.

**🧠 Συνιστώσες Αρχιτεκτονικής Πράκτορα:**
- **Core Model**: Το βασικό LLM (GPT-4, Groks, Phi κ.ά.)
- **System Prompt**: Ορίζει την προσωπικότητα και συμπεριφορά του πράκτορα
- **Parameters**: Ρυθμίσεις βελτιστοποίησης απόδοσης
- **Tools Integration**: Σύνδεση με εξωτερικά API και υπηρεσίες MCP
- **Memory**: Πλαίσιο συνομιλίας και διατήρηση συνεδρίας

![Agent Builder Interface](../../../../translated_images/el/agentbuilder.25895b2d2f8c02e7.webp)

### ⚙️ Βήμα 2: Εμβάθυνση στη Ρύθμιση Πράκτορα

**🎨 Δημιουργία Αποτελεσματικών System Prompts:**
```markdown
# Template Structure:
## Role Definition
You are a [specific role] with expertise in [domain].

## Capabilities
- List specific abilities
- Define scope of knowledge
- Clarify limitations

## Behavior Guidelines
- Response style (formal, casual, technical)
- Output format preferences
- Error handling approach

## Examples
Provide 2-3 examples of ideal interactions
```

*Φυσικά, μπορείτε επίσης να χρησιμοποιήσετε τη λειτουργία Generate System Prompt για να βοηθηθείτε από την AI στη δημιουργία και βελτιστοποίηση προτροπών*

**🔧 Βελτιστοποίηση Παραμέτρων:**
| Παράμετρος | Προτεινόμενο Εύρος | Περίπτωση Χρήσης |
|-----------|------------------|----------|
| **Temperature** | 0.1-0.3 | Τεχνικές/ακριβείς απαντήσεις |
| **Temperature** | 0.7-0.9 | Δημιουργικές/ιδεοκαταιγιστικές εργασίες |
| **Max Tokens** | 500-1000 | Συνοπτικές απαντήσεις |
| **Max Tokens** | 2000-4000 | Αναλυτικές εξηγήσεις |

### 🐍 Βήμα 3: Πρακτική Άσκηση - Πράκτορας Προγραμματισμού Python

**🎯 Αποστολή**: Δημιουργήστε έναν εξειδικευμένο βοηθό κώδικα Python

**📋 Βήματα Ρύθμισης:**

1. **Επιλογή Μοντέλου**: Επιλέξτε το **Claude 3.5 Sonnet** (εξαιρετικό για κώδικα)

2. **Σχεδίαση System Prompt**:
```markdown
# Python Programming Expert Agent

## Role
You are a senior Python developer with 10+ years of experience. You excel at writing clean, efficient, and well-documented Python code.

## Capabilities
- Write production-ready Python code
- Debug complex issues
- Explain code concepts clearly
- Suggest best practices and optimizations
- Provide complete working examples

## Response Format
- Always include docstrings
- Add inline comments for complex logic
- Suggest testing approaches
- Mention relevant libraries when applicable

## Code Quality Standards
- Follow PEP 8 style guidelines
- Use type hints where appropriate
- Handle exceptions gracefully
- Write readable, maintainable code
```

3. **Ρυθμίσεις Παραμέτρων**:
   - Temperature: 0.2 (για συνεπή, αξιόπιστο κώδικα)
   - Max Tokens: 2000 (αναλυτικές εξηγήσεις)
   - Top-p: 0.9 (ισορροπημένη δημιουργικότητα)

![Python Agent Configuration](../../../../translated_images/el/pythonagent.5e51b406401c165f.webp)

### 🧪 Βήμα 4: Δοκιμή του Python Πράκτορά σας
