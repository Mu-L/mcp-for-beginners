## 🏗️ Τεχνολογικό Στοίβαγμα

### 🔌 Προτυπο Πλαισίου Μοντέλου (MCP)

Το MCP είναι το **"USB-C για το AI"** – ένα παγκόσμιο πρότυπο που συνδέει μοντέλα AI με εξωτερικά εργαλεία και πηγές δεδομένων.

**✨ Βασικά Χαρακτηριστικά:**

- 🔄 **Τυποποιημένη Ενσωμάτωση**: Οικουμενική διεπαφή για συνδέσεις εργαλείων AI  
- 🏛️ **Ευέλικτη Αρχιτεκτονική**: Τοπικοί και απομακρυσμένοι εξυπηρετητές μέσω stdio/SSE μεταφοράς  
- 🧰 **Πλούσιο Οικοσύστημα**: Εργαλεία, προτροπές και πόροι σε ένα πρωτόκολλο  
- 🔒 **Έτοιμο για Επιχειρήσεις**: Ενσωματωμένη ασφάλεια και αξιοπιστία

**🎯 Γιατί έχει σημασία το MCP:**  
Όπως το USB-C εξάλειψε το χάος των καλωδίων, έτσι το MCP απλοποιεί τις πολυπλοκότητες των ενσωματώσεων AI. Ένα πρωτόκολλο, άπειρες δυνατότητες.

### 🤖 Επέκταση Microsoft Foundry Toolkit για VS Code

Η κορυφαία επέκταση ανάπτυξης AI της Microsoft που μετατρέπει το VS Code σε κέντρο ισχύος AI.

**🚀 Κύριες Ικανότητες:**

- 📦 **Κατάλογος Μοντέλων**: Πρόσβαση σε μοντέλα από Azure AI, GitHub, Hugging Face, Ollama  
- ⚡ **Τοπική Εκτέλεση**: Βελτιστοποίηση ONNX σε CPU/GPU/NPU  
- 🏗️ **Κατασκευαστής Πρακτόρων (Agent Builder)**: Οπτική ανάπτυξη πρακτόρων AI με ενσωμάτωση MCP  
- 🎭 **Πολυμορφική Υποστήριξη**: Κείμενο, όραση και δομημένη έξοδος

**💡 Οφέλη για την Ανάπτυξη:**

- Ανάπτυξη μοντέλων χωρίς ρυθμίσεις  
- Οπτικός σχεδιασμός προτροπών  
- Πραγματικό χρόνο περιβάλλον ελέγχου  
- Αδιάλειπτη ενσωμάτωση MCP server

## 🛠️ Απαιτήσεις & Ρύθμιση

### 💻 Απαιτήσεις Συστήματος

| Συνιστώσα | Απαίτηση | Σημειώσεις |
|-----------|----------|------------|
| **Λειτουργικό Σύστημα** | Windows 10+, macOS 10.15+, Linux | Οποιοδήποτε σύγχρονο OS |
| **Visual Studio Code** | Τελευταία σταθερή έκδοση | Απαραίτητο για Microsoft Foundry Toolkit |
| **Node.js** | v18.0+ και npm | Για ανάπτυξη MCP server |
| **Python** | 3.10+ | Προαιρετικό για Python MCP servers |
| **Μνήμη** | Τουλάχιστον 8GB RAM | 16GB συνιστάται για τοπικά μοντέλα |

### 🔧 Περιβάλλον Ανάπτυξης

#### Συνιστώμενες Επεκτάσεις VS Code

- **Microsoft Foundry Toolkit** (ms-windows-ai-studio.windows-ai-studio)  
- **Python** (ms-python.python)  
- **Python Debugger** (ms-python.debugpy)  
- **GitHub Copilot** (GitHub.copilot) - Προαιρετικό αλλά χρήσιμο

#### Προαιρετικά Εργαλεία

- **uv**: Σύγχρονος διαχειριστής πακέτων Python  
- **MCP Inspector**: Οπτικό εργαλείο αποσφαλμάτωσης MCP servers  
- **Playwright**: Για παραδείγματα αυτοματισμού web
