  - Οι Hosts ορίζονται πλέον σαφώς ως εφαρμογές AI που συντονίζουν πολλαπλές συνδέσεις πελατών MCP  
  - Οι Clients περιγράφονται ως συνδετήρες πρωτοκόλλου που διατηρούν σχέσεις ένα προς ένα με servers  
  - Οι Servers βελτιώθηκαν με σενάρια τοπικής έναντι απομακρυσμένης ανάπτυξης  
- **Αναδιάρθρωση Πρωτόγονων (Primitives)**: Πλήρης αναμόρφωση των πρωτόγονων για servers και clients  
  - Πρωτόγονα Server: Πόροι (πηγές δεδομένων), Προτροπές (πρότυπα), Εργαλεία (εκτελέσιμες λειτουργίες) με λεπτομερείς εξηγήσεις και παραδείγματα

  - Ενημερωμένα με τρέχοντα πρότυπα μεθόδων ανακάλυψης (`*/list`), ανάκτησης (`*/get`), και εκτέλεσης (`*/call`)

  - Επίπεδο Δεδομένων: Βάση JSON-RPC 2.0 με διαχείριση κύκλου ζωής και πρωτόγονα  
  - Επίπεδο Μεταφοράς: STDIO (τοπικό) και Streamable HTTP με μηχανισμούς μεταφοράς SSE (απομακρυσμένο)

- **Πρότυπα Επικοινωνίας**: Ενημερωμένα μηνύματα πρωτοκόλλου για εμφάνιση ροών αρχικοποίησης, ανακάλυψης, εκτέλεσης, και ειδοποιήσεων  
- **Παραδείγματα Κώδικα**: Ανανεωμένα παραδείγματα πολλαπλών γλωσσών (.NET, Java, Python, JavaScript) για να αντανακλούν τα τρέχοντα πρότυπα MCP SDK
