  - Οι Hosts ορίζονται πλέον σαφώς ως εφαρμογές AI που συντονίζουν πολλαπλές συνδέσεις πελατών MCP  
  - Οι Clients περιγράφονται ως συνδεδεμένοι μέσω πρωτοκόλλου με διατήρηση σχέσεων ένας-προς-έναν με servers

- **Αναδιάρθρωση Πρωτόγονων Στοιχείων**: Πλήρης αναθεώρηση των πρωτόγονων στοιχείων σε server και client  
  - Πρωτόγονα Server: Πόροι (πηγές δεδομένων), Προτροπές (πρότυπα), Εργαλεία (εκτελέσιμες λειτουργίες) με λεπτομερείς εξηγήσεις και παραδείγματα  
  - Πρωτόγονα Client: Δειγματοληψία (ολοκληρώσεις LLM), Εξαγωγή (εισροή χρήστη), Καταγραφή (αποσφαλμάτωση/παρακολούθηση)

  - Επίπεδο Δεδομένων: Βάση JSON-RPC 2.0 με διαχείριση κύκλου ζωής και πρωτόγονα στοιχεία  
  - Επίπεδο Μεταφοράς: STDIO (τοπικό) και HTTP με δυνατότητα ροής και SSE (απομακρυσμένη μεταφορά)

- **Πρότυπα Επικοινωνίας**: Ενημερωμένα μηνύματα πρωτοκόλλου για εμφάνιση ροών εκκίνησης, ανακάλυψης, εκτέλεσης και ειδοποίησης  
- **Παραδείγματα Κώδικα**: Ανανέωση παραδειγμάτων σε πολλές γλώσσες (.NET, Java, Python, JavaScript) για να αντικατοπτρίζουν τα τρέχοντα πρότυπα MCP SDK
