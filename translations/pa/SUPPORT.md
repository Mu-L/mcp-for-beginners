# Support

## ਮਸਲੇ ਦਰਜ ਕਰਨ ਅਤੇ ਮਦਦ ਪ੍ਰਾਪਤ ਕਰਨ ਦਾ ਤਰੀਕਾ  

ਇਹ ਪ੍ਰੋਜੈਕਟ ਬਗ ਅਤੇ ਫੀਚਰ ਦੀ ਬੇਨਤੀ ਨੂੰ ਟ੍ਰੈਕ ਕਰਨ ਲਈ GitHub Issues ਦਾ ਇਸਤੇਮਾਲ ਕਰਦਾ ਹੈ। ਕਿਰਪਾ ਕਰਕੇ ਨਵੇਂ ਮਸਲੇ ਦਰਜ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਮੌਜੂਦਾ ਮਸਲਿਆਂ ਨੂੰ ਖੋਜੋ ਤਾਂ ਜੋ ਨਕਲ ਤੋਂ ਬਚਿਆ ਜਾ ਸਕੇ। ਨਵੇਂ ਮਸਲਿਆਂ ਲਈ, ਆਪਣਾ ਬਗ ਜਾਂ ਫੀਚਰ ਬੇਨਤੀ ਨਵੇਂ Issue ਵਜੋਂ ਦਰਜ ਕਰੋ।

ਇਸ ਪ੍ਰੋਜੈਕਟ ਦੀ ਵਰਤੋਂ ਲਈ ਮਦਦ ਅਤੇ ਸਵਾਲਾਂ ਲਈ, ਕਿਰਪਾ ਕਰਕੇ:
- [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ਨੂੰ ਚੈੱਕ ਕਰੋ ਜਿੱਥੇ ਵਿਸ਼ੇਸ਼ਜ્ઞ ਅਤੇ ਸਾਥੀ ਡਿਵੈਲਪਰ ਇਕੱਠੇ ਹੁੰਦੇ ਹਨ
- ਪ੍ਰੋਟੋਕੋਲ-ਸੰਬੰਧੀ ਸਵਾਲਾਂ ਲਈ [Model Context Protocol Community](https://modelcontextprotocol.io/community/) ਦੇ ਸਰੋਤ ਵਰਤੋਂ

## Microsoft Support Policy  

Model Context Protocol (MCP) for Beginners ਪ੍ਰੋਜੈਕਟ ਲਈ ਮਦਦ ਸਿਰਫ਼ ਉਪਰੋਕਤ ਸਰੋਤਾਂ ਤੱਕ ਸੀਮਤ ਹੈ।
