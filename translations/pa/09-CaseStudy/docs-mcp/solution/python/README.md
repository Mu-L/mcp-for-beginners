# ਸਟਡੀ ਪਲੈਨ ਜਨਰੇਟਰ ਚੇਨਲਿਟ ਅਤੇ ਮਾਇਕ੍ਰੋਸੌਫਟ ਲਰਨ ਡੌਕਸ MCP ਨਾਲ

## ਪਹਿਲਾਂ ਤੋਂ ਲੋੜੀਂਦੀਆਂ ਚੀਜ਼ਾਂ

- ਪਾਇਥਨ 3.8 ਜਾਂ ਉੱਪਰ
- ਪਿਪ (ਪਾਇਥਨ ਪੈਕੇਜ ਮੈਨੇਜਰ)
- ਮਾਇਕ੍ਰੋਸੌਫਟ ਲਰਨ ਡੌਕਸ MCP ਸਰਵਰ ਨਾਲ ਕਨੈਕਟ ਕਰਨ ਲਈ ਇੰਟਰਨੈੱਟ ਐਕਸੈੱਸ

## ਇੰਸਟਾਲੇਸ਼ਨ

1. ਇਸ ਰਿਪੋਜ਼ੀਟਰੀ ਨੂੰ ਕਲੋਨ ਕਰੋ ਜਾਂ ਪ੍ਰੋਜੈਕਟ ਫਾਇਲਾਂ ਡਾਊਨਲੋਡ ਕਰੋ।
2. ਲੋੜੀਂਦੇ ਡਿਪੈਂਡੇਨਸੀਜ਼ ਇੰਸਟਾਲ ਕਰੋ:

   ```bash
   pip install -r requirements.txt
   ```

## ਵਰਤੋਂ

### ਦ੍ਰਿਸ਼ 1: ਡੌਕਸ MCP ਲਈ ਸਧਾਰਣ ਕੁਐਰੀ
ਇੱਕ ਕਮਾਂਡ-ਲਾਈਨ ਕਲਾਇੰਟ ਜੋ ਡੌਕਸ MCP ਸਰਵਰ ਨਾਲ ਕੁਨੈਕਟ ਕਰਦਾ ਹੈ, ਕੁਐਰੀ ਭੇਜਦਾ ਹੈ ਅਤੇ ਨਤੀਜਾ ਛਾਪਦਾ ਹੈ।

1. ਸਕ੍ਰਿਪਟ ਚਲਾਓ:
   ```bash
   python scenario1.py
   ```
2. ਪ੍ਰਾਂਪਟ 'ਤੇ ਆਪਣੇ ਦਸਤਾਵੇਜ਼ੀ ਸਵਾਲ ਦਰਜ ਕਰੋ।

### ਦ੍ਰਿਸ਼ 2: ਸਟਡੀ ਪਲੈਨ ਜਨਰੇਟਰ (ਚੇਨਲਿਟ ਵੈੱਬ ਐਪ)
ਇੱਕ ਵੈੱਬ-ਆਧਾਰਤ ਇੰਟਰਫੇਸ (ਚੇਨਲਿਟ ਵਰਤ ਕੇ) ਜੋ ਵਰਤੋਂਕਾਰਾਂ ਨੂੰ ਕਿਸੇ ਵੀ ਤਕਨੀਕੀ ਵਿਸ਼ੇ ਲਈ ਹਫ਼ਤਾਵਾਰ ਪੜ੍ਹਾਈ ਯੋਜਨਾ ਬਣਾਉਣ ਦਾ ਮੌਕਾ ਦਿੰਦਾ ਹੈ।

1. ਚੇਨਲਿਟ ਐਪ ਸ਼ੁਰੂ ਕਰੋ:
   ```bash
   chainlit run scenario2.py
   ```
2. ਟਰਮੀਨਲ ਵਿੱਚ ਦਿੱਤਾ ਹੋਇਆ ਲੋਕਲ ਯੂਆਰਐਲ (ਜਿਵੇਂ http://localhost:8000) ਆਪਣੇ ਬ੍ਰਾਊਜ਼ਰ ਵਿੱਚ ਖੋਲ੍ਹੋ।
3. ਚੈਟ ਵਿੰਡੋ ਵਿੱਚ ਆਪਣਾ ਅਧਿਐਨ ਵਿਸ਼ਾ ਅਤੇ ਕਿੰਨੇ ਹਫਤੇ ਪੜ੍ਹਾਈ ਕਰਨੀ ਹੈ ਦਰਜ ਕਰੋ (ਜਿਵੇਂ "AI-900 ਸਰਟੀਫਿਕੇਸ਼ਨ, 8 ਹਫਤੇ")।
4. ਐਪ ਤੁਹਾਡੇ ਲਈ ਹਫ਼ਤੇ ਦਰ ਹਫ਼ਤਾ ਪੜ੍ਹਾਈ ਯੋਜਨਾ ਮੁਹੱਈਆ ਕਰੇਗਾ, ਜਿਸ ਵਿੱਚ ਸੰਬੰਧਤ ਮਾਇਕ੍ਰੋਸੌਫਟ ਲਰਨ ਡੌਕਸ ਦੇ ਲਿੰਕ ਸ਼ਾਮਲ ਹਨ।

**ਲੋੜੀਂਦੇ ਐਨਵਾਇਰਨਮੈਂਟ ਵੈਰੀਏਬਲ:**

ਦ੍ਰਿਸ਼ 2 (ਚੇਨਲਿਟ ਵੈੱਬ ਐਪ સાથે ਏਜ਼ਯੂਰ ਓਪਨ ਏਆਈ) ਵਰਤਣ ਲਈ, ਤੁਹਾਨੂੰ `python` ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ `.env` ਫਾਇਲ ਵਿਚ ਹੇਠ ਲਿਖੇ ਐਨਵਾਇਰਨਮੈਂਟ ਵੈਰੀਏਬਲ ਸੈੱਟ ਕਰਨੇ ਹਨ:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

ਐਪ ਚਲਾਉਣ ਤੋਂ ਪਹਿਲਾਂ ਇਹ ਵੈਲੂੰਜ਼ ਆਪਣੇ ਏਜ਼ਯੂਰ ਓਪਨ ਏਆਈ ਰਿਸੋਰਸ ਜਾਣਕਾਰੀ ਨਾਲ ਭਰੋ।

> [!TIP]
> ਤੁਸੀਂ ਆਪਣੀਆਂ ਮਾਡਲਜ਼ ਨੂੰ ਆਸਾਨੀ ਨਾਲ [Microsoft Foundry](https://ai.azure.com/) ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਡਿਪਲਾਇ ਕਰ ਸਕਦੇ ਹੋ।

### ਦ੍ਰਿਸ਼ 3: VS ਕੋਡ ਵਿੱਚ ਟੈਕਸਟ ਐਡੀਟਰ ਨਾਲ MCP ਸਰਵਰ ਡੌਕਸ

ਬ੍ਰਾਊਜ਼ਰ ਟੈਬ ਬਦਲ ਕੇ ਡੌਕਸ ਖੋਜਣ ਦੀ ਥਾਂ, ਤੁਸੀਂ ਮਾਇਕ੍ਰੋਸੌਫਟ ਲਰਨ ਡੌਕਸ ਨੂੰ ਸਿੱਧਾ VS ਕੋਡ ਵਿੱਚ ਲੈ ਆ ਸਕਦੇ ਹੋ MCP ਸਰਵਰ ਦੀ ਵਰਤੋਂ ਕਰਕੇ। ਇਸ ਨਾਲ ਤੁਸੀਂ:
- VS ਕੋਡ ਦੇ ਅੰਦਰ ਬਿਨਾਂ ਕੋਡਿੰਗ ਮਾਹੌਲ ਛੱਡੇ ਡੌਕਸ ਨੂੰ ਖੋਜ ਅਤੇ ਪੜ੍ਹ ਸਕਦੇ ਹੋ।
- ਡੌਕਸ ਦੀਆਂ ਸੰਦਰਭ ਲਿੰਕਾਂ ਆਪਣੇ README ਜਾਂ ਕੋਰਸ ਫਾਇਲਾਂ ਵਿੱਚ ਸਿੱਧਾ ਸ਼ਾਮਲ ਕਰ ਸਕਦੇ ਹੋ।
- GitHub Copilot ਅਤੇ MCP ਨੂੰ ਮਿਲਾ ਕੇ ਸਿਮਲੈੱਸ, ਏਆਈ-ਚਲਿਤ ਡੌਕਸ ਵਰਕਫਲੋ ਪਾਉਗੇ।

**ਉਦਾਹਰਨ ਵਰਤੋ:**
- ਇੱਕ README ਵਿੱਚ ਤੇਜ਼ੀ ਨਾਲ ਸੰਦਰਭ ਲਿੰਕਾਂ ਸ਼ਾਮਲ ਕਰੋ ਜਦੋਂ ਤੁਸੀਂ ਕੋਰਸ ਜਾਂ ਪ੍ਰੋਜੈਕਟ ਡੌਕਸ ਲਿਖ ਰਹੇ ਹੋ।
- ਕੋਡ ਜਨਰੇਟ ਕਰਨ ਲਈ Copilot ਵਰਤੋ ਅਤੇ MCP ਨਾਲ ਤੁਰੰਤ ਸੰਬੰਧਤ ਡੌਕਸ ਲੱਭੋ ਤੇ ਹਵਾਲੇ ਦਿਓ।
- ਆਪਣੇ ਐਡੀਟਰ ਵਿੱਚ ਧਿਆਨ ਕੇਂਦਰਿਤ ਰੱਖੋ ਅਤੇ ਉਤਪਾਦਕਤਾ ਵਧਾਓ।

> [!IMPORTANT]
> ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡੇ ਵਰਕਸਪੇਸ ਵਿੱਚ ਇੱਕ ਵਿਧਾਨਿਕ [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) ਫਾਈਲ ਮੌਜੂਦ ਹੈ (ਥਾਂ `.vscode/mcp.json` ਹੈ)।

## ਦ੍ਰਿਸ਼ 2 ਲਈ ਚੇਨਲਿਟ ਕਿਉਂ?

ਚੇਨਲਿਟ ਇੱਕ ਆਧੁਨਿਕ ਓਪਨ-ਸੋਰਸ ਫਰੇਮਵਰਕ ਹੈ ਜੋ ਗੱਲਬਾਤੀ ਵੈੱਬ ਐਪਲੀਕੇਸ਼ਨਾਂ ਬਣਾਉਣ ਲਈ ਹੈ। ਇਹ ਆਸਾਨ ਬਣਾਉਂਦਾ ਹੈ ਚੈਟ-ਆਧਾਰਿਤ ਯੂਜ਼ਰ ਇੰਟਰਫੇਸ ਜੋ ਮਾਇਕ੍ਰੋਸੌਫਟ ਲਰਨ ਡੌਕਸ MCP ਸਰਵਰ ਵਰਗੀਆਂ ਬੈਕਐਂਡ ਸੇਵਾਵਾਂ ਨਾਲ ਜੁੜਦੇ ਹਨ। ਇਹ ਪ੍ਰੋਜੈਕਟ ਚੇਨਲਿਟ ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ ਕਿ ਸਧਾਰਣ ਅਤੇ ਇੰਟਰਐਕਟਿਵ ਤਰੀਕੇ ਨਾਲ ਨਿੱਜੀਕृत ਸਟਡੀ ਪਲੈਨ ਤਿਆਰ ਕੀਤੇ ਜਾ ਸਕਣ। ਚੇਨਲਿਟ ਦੀ ਵਰਤੋਂ ਨਾਲ ਤੁਸੀਂ ਤੇਜ਼ੀ ਨਾਲ ਚੈਟ-ਆਧਾਰਿਤ ਟੂਲ ਤਿਆਰ ਕਰ ਸਕਦੇ ਹੋ ਜੋ ਉਤਪਾਦਕਤਾ ਅਤੇ ਸਿੱਖਣ ਵਿੱਚ ਸਭ ਤੋਂ ਵਧੀਆ ਹੈ।

## ਇਹ ਕੀ ਕਰਦਾ ਹੈ

ਇਹ ਐਪ ਵਰਤੋਂਕਾਰਾਂ ਨੂੰ ਸਿਰਫ਼ ਇੱਕ ਵਿਸ਼ਾ ਅਤੇ ਸਮਾਂ ਦਰਜ ਕਰਕੇ ਨਿੱਜੀਕ੍ਰਿਤ ਸਟਡੀ ਪਲੈਨ ਬਣਾਉਣ ਦੀ ਆਗਿਆ ਦਿੰਦਾ ਹੈ। ਐਪ ਤੁਹਾਡੇ ਇਨਪੁੱਟ ਨੂੰ ਪਾਰਸ ਕਰਦਾ ਹੈ, ਮਾਇਕ੍ਰੋਸੌਫਟ ਲਰਨ ਡੌਕਸ MCP ਸਰਵਰ ਨੂੰ ਸੰਬੰਧਤ ਸਮੱਗਰੀ ਲਈ ਕੁਐਰੀ ਕਰਦਾ ਹੈ, ਅਤੇ ਨਤੀਜਿਆਂ ਨੂੰ ਵਿਵਸਥਿਤ ਹਫ਼ਤੇ ਦਰ ਹਫ਼ਤੇ ਯੋਜਨਾ ਵਿੱਚ ਰਚਦਾ ਹੈ। ਹਰ ਹਫ਼ਤੇ ਦੀ ਸਿਫਾਰਸ਼ਾਂ ਚੈਟ ਵਿੱਚ ਦਰਸਾਈਆਂ ਜਾਂਦੀਆਂ ਹਨ, ਜਿਸ ਨਾਲ ਪੜ੍ਹਾਈ ਨੂੰ ਅਸਾਨੀ ਨਾਲ ਫਾਲੋ ਅਤੇ ਟ੍ਰੈਕ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ। ਇਹ ਇੰਟਿਗ੍ਰੇਸ਼ਨ ਤੁਹਾਨੂੰ ਹਰ ਵਕਤ ਤਾਜ਼ਾ ਅਤੇ ਸਭ ਤੋਂ ਸੰਬੰਧਤ ਸਿੱਖਣ ਵਾਲਾ ਸਮੱਗਰੀ ਦਿੰਦਾ ਹੈ।

## ਉਦਾਹਰਨ ਕੁਐਰੀਜ਼

ਇਹ ਕੁਐਰੀਜ਼ ਚੈਟ ਵਿੰਡੋ ਵਿੱਚ ਅਜਮਾਓ ਅਤੇ ਵੇਖੋ ਕਿ ਐਪ ਕਿਵੇਂ ਜਵਾਬ ਦਿੰਦਾ ਹੈ:

- `Learn Azure Functions, 4 weeks`
- `Azure DevOps, 6 weeks`
- `Data engineering on Azure, 10 weeks`
- `Microsoft security fundamentals, 5 weeks`
- `Power Platform, 7 weeks`
- `Azure AI services, 12 weeks`
- `Cloud architecture, 9 weeks`

ਇਹ ਉਦਾਹਰਨ ਐਪ ਦੀ ਬਹੁਪੱਖੀਤਾ ਨੂੰ ਵੱਖ-ਵੱਖ ਸਿੱਖਣ ਵਾਲੇ ਲਕੜੀ ਅਤੇ ਸਮੇਂ ਲਈ ਦਰਸਾਉਂਦੀਆਂ ਹਨ।
