# AGENTS.md

## ਪ੍ਰੋਜੈਕਟ ਓਵਰਵਿਊ

**ਮੁਲਾਂਕਣ ਕਾਰਜ ਪ੍ਰोटੋਕੋਲ ਲਈ ਨਵਾਂ ਸਿਖਿਆਰਥੀਆਂ ਲਈ MCP** ਇੱਕ ਖੁੱਲ੍ਹਾ ਸਰੋਤ ਸਿੱਖਿਆਤਮਕ ਕੋਰਸ ਹੈ ਜੋ ਮਾਡਲ ਸੰਦਰਭ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਸਿੱਖਣ ਲਈ ਹੈ - ਜੋ AI ਮਾਡਲਾਂ ਅਤੇ ਕਲਾਇੰਟ ਐਪਲੀਕੇਸ਼ਨਾਂ ਵਿਚਕਾਰ ਪਰਸਪਰਕਿਰਿਆਵਾਂ ਲਈ ਇੱਕ ਮਿਆਰੀ ਪ੍ਰਣਾਲੀ ਹੈ। ਇਹ ਰਿਪੋਜ਼ਿਟਰੀ ਕਈ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ ਵਿੱਚ ਹੱਥ-ਵਰਗੀ ਕੋਡ ਉਦਾਹਰਣਾਂ ਸਮੇਤ ਵਿਸਤ੍ਰਿਤ ਸਿੱਖਣ ਸਮੱਗਰੀ ਮੁਹੱਈਆ ਕਰਦੀ ਹੈ।

### ਮੁੱਖ ਤਕਨਾਲੋਜੀਆਂ

- **ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ**: C#, Java, JavaScript, TypeScript, Python, Rust  
- **ਫਰੇਮਵਰਕ ਅਤੇ SDKs**:  
  - MCP SDK (`@modelcontextprotocol/sdk`)  
  - Spring Boot (Java)  
  - FastMCP (Python)  
  - LangChain4j (Java)  
- **ਡੇਟਾਬੇਸ**: PostgreSQL ਪੀਜੀਵੇਕਟਰ ਐਕਸਟੈਂਸ਼ਨ ਨਾਲ  
- **ਕਲਾਉਡ ਪਲੇਟਫਾਰਮ**: Azure (ਕੰਟੇਨਰ ਐਪਸ, OpenAI, ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ, ਐਪਲੀ케ਸ਼ਨ ਇਨਸਾਈਟਸ)  
- **ਬਿਲਡ ਟੂਲਜ਼**: npm, Maven, pip, Cargo  
- **ਡੌਕਯੂਮੈਂਟੇਸ਼ਨ**: ਮਾਰਕਡਾਊਨ ਸਥਿਰਮੂਲ ਅਨੇਕ ਭਾਸ਼ਾਵਾਂ ਲਈ ਆਟੋਮੇਟਿਕ ਅਨੁਵਾਦ (48+ ਭਾਸ਼ਾਵਾਂ)  

### ਆਰਕੀਟੈਕਚਰ

- **11 ਕੋਰ ਮਾਡਿਊਲ (00-11)**: ਮੂਲਤਤਵੀ ਤੋਂ ਉੱਚ ਪੱਧਰ ਦੇ ਵਿਸ਼ਿਆਂ ਤੱਕ ਕ੍ਰਮਵਾਰ ਸਿੱਖਣ ਦਾ ਰਸਤਾ  
- **ਹੈਂਡਸ-ਆਨ ਲੈਬਸ**: ਕਈ ਭਾਸ਼ਾਵਾਂ ਵਿੱਚ ਪੂਰਾ ਹੱਲ ਕੋਡ ਵਾਲੇ ਕਿਰਿਆਸ਼ੀਲ ਅਭਿਆਸ  
- **ਨਮੂਨਾ ਪ੍ਰੋਜੈਕਟ**: ਕੰਮ ਕਰਨ ਵਾਲੇ MCP ਸਰਵਰ ਅਤੇ ਕਲਾਇੰਟ ਸਮਾਧਾਨ  
- **ਅਨੁਵਾਦ ਪ੍ਰਣਾਲੀ**: ਬਹੁਭਾਸ਼ੀ ਸਹਾਇਤਾ ਲਈ ਆਟੋਮੈਟਿਕ GitHub Action ਵਰਕਫ਼ਲੋ  
- **ਚਿੱਤਰ ਐਸੈਟਸ**: ਕੇਂਦਰੀਕ੍ਰਿਤ ਚਿੱਤਰ ਡਾਇਰੈਕਟਰੀ ਜਿਸ ਵਿੱਚ ਅਨੁਵਾਦਿਤ ਵਰਜਨ ਮੌਜੂਦ  

## ਸੈਟਅੱਪ ਕਮਾਂਡ

ਇਹ ਡੌਕਯੂਮੈਂਟੇਸ਼ਨ ਕੇਂਦਰਤ ਰਿਪੋਜ਼ਿਟਰੀ ਹੈ। ਜ਼ਿਆਦਾਤਰ ਸੈਟਅੱਪ ਨਮੂਨਾ ਪ੍ਰੋਜੈਕਟ ਅਤੇ ਲੈਬਾਂ ਵਿੱਚ ਹੋਦਾ ਹੈ।

### ਰਿਪੋਜ਼ਿਟਰੀ ਸੈਟਅੱਪ

```bash
# ਰਿਪੋਜ਼ਿਟਰੀ ਨੂੰ ਕਲੋਨ ਕਰੋ
git clone https://github.com/microsoft/mcp-for-beginners.git
cd mcp-for-beginners
```

### ਨਮੂਨਾ ਪ੍ਰੋਜੈਕਟਾਂ ਨਾਲ ਕੰਮ ਕਰਨਾ

ਨਮੂਨਾ ਪ੍ਰੋਜੈਕਟ ਸਥਿਤ ਹਨ:  
- `03-GettingStarted/samples/` - ਭਾਸ਼ਾ-ਨਿਰਧਾਰਤ ਉਦਾਹਰਣ  
- `03-GettingStarted/01-first-server/solution/` - ਪਹਿਲਾ ਸਰਵਰ ਸਮਾਧਾਨ  
- `03-GettingStarted/02-client/solution/` - ਕਲਾਇੰਟ ਸਮਾਧਾਨ  
- `11-MCPServerHandsOnLabs/` - ਵਿਸਤ੍ਰਿਤ ਡੇਟਾਬੇਸ ਇੰਟਿਗ੍ਰੇਸ਼ਨ ਲੈਬਸ  

ਹਰ ਨਮੂਨਾ ਪ੍ਰੋਜੈਕਟ ਦੀ ਆਪਣੀ ਸੈਟਅੱਪ ਹਦਾਇਤਾਂ ਹਨ:

#### ਟਾਈਪਸਕ੍ਰਿਪਟ/ਜਾਵਾਸਕ੍ਰਿਪਟ ਪ੍ਰੋਜੈਕਟ  
```bash
cd <project-directory>
npm install
npm start
```
  
#### ਪਾਇਥਨ ਪ੍ਰੋਜੈਕਟ  
```bash
cd <project-directory>
pip install -r requirements.txt
# ਜਾਂ
pip install -e .
python main.py
```
  
#### ਜਾਵਾ ਪ੍ਰੋਜੈਕਟ  
```bash
cd <project-directory>
mvn clean install
mvn spring-boot:run
```
  
## ਵਿਕਾਸ ਕਾਰਜ ਪ੍ਰਵਾਹ

### ਡੌਕਯੂਮੈਂਟੇਸ਼ਨ ਸਰਚਨਾ

- **ਮਾਡਿਊਲ 00-11**: ਕ੍ਰਮਵਾਰ ਮੁੱਖ ਕੋਰਸ ਸਮੱਗਰੀ  
- **translations/**: ਭਾਸ਼ਾ-ਨਿਰਧਾਰਤ ਵਰਜਨ (ਆਟੋ-ਬਣਾਏ ਗਏ, ਸਿੱਧਾ ਸੰਪਾਦਨ ਨਾ ਕਰੋ)  
- **translated_images/**: ਸਥਾਨਕীক੍ਰਿਤ ਚਿੱਤਰ (ਆਟੋ-ਬਣਾਏ ਗਏ)  
- **images/**: ਮੂਲ ਚਿੱਤਰ ਅਤੇ ਡਾਇਗ੍ਰਾਮ  

### ਡੌਕਯੂਮੈਂਟੇਸ਼ਨ ਬਦਲਾਵ ਕਰਨ ਦਾ ਤਰੀਕਾ

1. ਕੇਵਲ ਮੂਲ ਮਾਡਿਊਲ ਡਾਇਰੈਕਟਰੀਆਂ (00-11) ਵਿੱਚ ਅੰਗਰੇਜ਼ੀ ਮਾਰਕਡਾਊਨ ਫਾਈਲਾਂ ਸੰਪਾਦਿਤ ਕਰੋ  
2. ਜੇ ਜਰੂਰੀ ਹੋਵੇ ਤਾਂ `images/` ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਚਿੱਤਰਾਂ ਨੂੰ ਅਪਡੇਟ ਕਰੋ  
3. co-op-translator GitHub Action ਆਟੋਮੈਟਿਕ ਤੌਰ ਤੇ ਅਨੁਵਾਦ ਬਣਾਏਗਾ  
4. ਅਨੁਵਾਦ ਮੱਖੀ ਸ਼ਾਖਾ ‘main’ ‘ਤੇ ਪੁਸ਼ ਕਰਨ ‘ਤੇ ਦੁਬਾਰਾ ਬਣਾਏ ਜਾਂਦੇ ਹਨ  

### ਅਨੁਵਾਦਾਂ ਨਾਲ ਕੰਮ ਕਰਨਾ

- **ਆਟੋਮੈਟਿਕ ਅਨੁਵਾਦ**: ਸਾਰੇ ਅਨੁਵਾਦ GitHub Actions ਵਰਕਫਲੋ ਦੁਆਰਾ ਪ੍ਰਬੰਧਿਤ ਹੁੰਦੇ ਹਨ  
- `translations/` ਡਾਇਰੈਕਟਰੀ ਦੀਆਂ ਫਾਈਲਾਂ ਹੱਥੋਂ ਸੰਪਾਦਨ ਨਾ ਕਰੋ  
- ਹਰ ਅਨੁਵਾਦਿਤ ਫਾਈਲ ਵਿੱਚ ਮੈਟਾਡੇਟਾ ਐम्बੈੱਡ ਕੀਤਾ ਹੁੰਦਾ ਹੈ  
- ਸਹਾਇਤਾ ਪ੍ਰਾਪਤ ਭਾਸ਼ਾਵਾਂ: 48+ ਭਾਸ਼ਾਵਾਂ ਜਿਵੇਂ ਕਿ ਅਰਬੀ, ਚੀਨੀ, ਫ੍ਰੈਂਚ, ਜਰਮਨ, ਹਿੰਦੀ, ਜਾਪਾਨੀ, ਕੋਰੀਅਨ, ਪੋਰਟੁਗੀਜ਼, ਰੂਸੀ, ਸਪੇਨੀ ਅਤੇ ਹੋਰ ਬਹੁਤ ਕੁਝ  

## ਟੈਸਟਿੰਗ ਨਿਰਦੇਸ਼

### ਡੌਕਯੂਮੈਂਟੇਸ਼ਨ ਵੈਰਿਫਿਕੇਸ਼ਨ

ਇਹ ਮੁੱਖ ਰੂਪ ਵਿੱਚ ਡੌਕਯੂਮੈਂਟੇਸ਼ਨ ਰਿਪੋਜ਼ਿਟਰੀ ਹੈ, ਟੈਸਟਿੰਗ ਇਸ ‘ਤੇ ਧਿਆਨ ਲਗਾਉਂਦੀ ਹੈ:

1. **ਲਿੰਕ ਵੈਰੀਫਿਕੇਸ਼ਨ**: ਸਭ ਅੰਦਰੂਨੀ ਲਿੰਕ ਠੀਕ ਕੰਮ ਕਰ ਰਹੇ ਹਨ ਇਹ ਯਕੀਨੀ ਬਣਾਓ  
```bash
# ਟੁੱਟੇ ਹੋਏ ਮਾਰਕਡਾਊਨ ਲਿੰਕਾਂ ਦੀ ਜਾਂਚ ਕਰੋ
find . -name "*.md" -type f | xargs grep -n "\[.*\](../../.*)"
```
  
2. **ਕੋਡ ਉਧਾਰਣ ਦੀ ਪਰਖ**: ਕੋਡ ਉਦਾਹਰਣ ਕੰਪਾਈਲ/ਚਲਾਓ ਜਾਂਦੇ ਹਨ ਇਹ ਚੈੱਕ ਕਰੋ  
```bash
# ਕਿਸੇ ਵਿਸ਼ੇਸ਼ ਨਮੂਨੇ ਤੇ ਜਾਓ ਅਤੇ ਉਸਦੇ ਟੈਸਟ ਚਲਾਓ
cd 03-GettingStarted/samples/typescript
npm install && npm test
```
  
3. **ਮਾਰਕਡਾਊਨ ਲਿੰਟਿੰਗ**: ਫਾਰਮੈਟਿੰਗ ਸਥਿਰਤਾ ਜਾਂਚੋ  
```bash
# ਜੇ ਲੋੜ ਹੋਵੇ ਤਾਂ markdownlint ਦੀ ਵਰਤੋਂ ਕਰੋ
npx markdownlint-cli2 "**/*.md" "#node_modules"
```
