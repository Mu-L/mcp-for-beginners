- **05-AdvancedTopics/mcp-security/README.md**: "Current Standard" ਬੈਨਰ, ਤਾਰੂਫ, ਕੋਰ ਸੁਰੱਖਿਆ ਸਿਧਾਂਤ ਸਿਰਲੇਖ, ਜ਼ਰੂਰੀ ਆਵਸ਼ਕਤਾਵਾਂ ਸਿਰਲੇਖ, Microsoft Entra ID ਭਾਗ, ਸੰਦਰਭ ਅਤੇ ਸਰੋਤ ਲਿੰਕ, ਅਤੇ ਬੰਦ ਕਰਨ ਵਾਲਾ ਸੁਰੱਖਿਆ ਨੋਟਿਸ (8 ਹਵਾਲੇ) ਨੂੰ 2025-11-25 ਲਈ ਅਪਡੇਟ ਕੀਤਾ
- **05-AdvancedTopics/mcp-transport/README.md**: ਅਤਿਰਿਕਤ ਸਰੋਤ ਸਪੈਸ਼ੀਫਿਕੇਸ਼ਨ ਲਿੰਕ ਅਤੇ "Current Standard" ਬੈਨਰ ਨੂੰ 2025-11-25 ਲਈ ਅਪਡੇਟ ਕੀਤਾ
- **05-AdvancedTopics/mcp-realtimesearch/README.md**: ਪੁਰਾਣਾ `2025-03-26` ਸੁਰੱਖਿਆ-ਅਤੇ-ਭਰੋਸਾ ਲਿੰਕ ਬਦਲ ਕੇ 2025-11-25 ਸੁਰੱਖਿਆ ਦੀਆਂ ਸਰਬੋਤਮ ਪ੍ਰਥਾਵਾਂ ਵਾਲੇ ਫ਼ਰਸ਼ੇ ਵਿੱਚ ਰਖਿਆ
- **03-GettingStarted/14-sampling/README.md**: ਅਧਿਕਾਰਿਕ ਸੈਂਪਲਿੰਗ ਡੌਕਸ ਲਿੰਕ ਨੂੰ 2025-11-25 ਲਈ ਅਪਡੇਟ ਕੀਤਾ
- **03-GettingStarted/05-stdio-server/README.md**: ਵਰਤਮਾਨ ਸਮੇਂ ਵਾਲਾ "ਮੌਜੂਦਾ MCP ਵਿਸ਼ੇਸ਼ਤਾ" ਹਵਾਲਾ ਅਤੇ ਅਤਿਰਿਕਤ ਸਰੋਤ ਸਪੈਸ਼ੀਫਿਕੇਸ਼ਨ ਲਿੰਕ ਨੂੰ 2025-11-25 ਲਈ ਅਪਡੇਟ ਕੀਤਾ (ਈਤਿਹਾਸਕ SSE-ਡਿਪ੍ਰਿਕੇਸ਼ਨ ਨੋਟਸ ਸਹੀ ਸਮਝ ਲਈ ਅਸਲ ਵਾਂਗ ਰਹੇ)

#### ਵਰਤਮਾਨ SDKs ਨਾਲ ਨਮੂਨਾ ਸਹੀਤਾ

- **ਟਾਈਪਸਕ੍ਰਿਪਟ (03-GettingStarted/01-first-server/solution/typescript)**: `npm install` ਨੇ `@modelcontextprotocol/sdk@1.29.0` ਹੱਲ ਕੀਤਾ; `tsc --noEmit` ਕੋਈ ਟਾਈਪ ਐਰਰ ਨਾ ਦਿਖਾਈਏ — ਮੌਜੂਦਾ `McpServer`/`StdioServerTransport` APIs ਵੈਧ ਹਨ

- ਸਾਰੇ ਨਮੂਨਾ `@modelcontextprotocol/sdk` ਵਰਜਨ ਸੀਮਾਂ (`>=1.26.0` / `^1.26.0` / `^1.27.0`) ਸਹੀ ਤਰੀਕੇ ਨਾਲ ਵਰਤਮਾਨ `1.29.0` ਵਿੱਚ ਹੱਲ ਹੋ ਰਹੇ ਹਨ ਬਿਨਾਂ ਕੋਈ API ਬ੍ਰੇਕਟ ਨੁਕਸਾਨ

#### ਡੀਪੈਂਡੈਂਸੀ ਪਿਨ ਸੰਰੇਖਣ (ਵਰਜਨ ਗੈਪ ਬੰਦ ਕਰਨਾ)

### ਸੁਰੱਖਿਆ ਮਜ਼ਬੂਤੀ ਅਤੇ ਡੀਪੈਂਡੈਂਸੀ ਖਾਮੀਆਂ ਦਾ ਸੁਧਾਰ

ਹਰੇਕ ਡੀਪੈਂਡੈਂਸੀ ਮੈਨਿਫੈਸਟ ਅਤੇ ਨਮੂਨਾ ਸੋਰਸ ਕੋਡ ਦੀ ਪੂਰੀ ਸੁਰੱਖਿਆ ਜਾਂਚ ਕੀਤੀ, ਫਿਰ ਸਾਰੇ ਨਪਮ ਸਲਾਹਾਂ ਤੇ ਇੱਕ ਕੋਡ-ਸਤਰ ਤੇ ਮਿਲੀ ਖ਼ਾਮੀ ਦੇ ਸੁਧਾਰ ਕੀਤੇ। ਸੁਧਾਰ ਤੋਂ ਬਾਅਦ, `npm audit` ਹਰ ਇਕ ਜਾਂਚੀ ਗਈ ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ **0 ਖਤਰੇ** ਦਰਜ ਕਰਦਾ ਹੈ।

#### npm ਡੀਪੈਂਡੈਂਸੀ ਖਤਰੇ (ਟ੍ਰਾਂਜ਼ਿਟਿਵ) — ਠੀਕ ਕੀਤੇ

- **10-StreamliningAIWorkflows.../lab4/code/github_mcp_server/inspector** ਅਤੇ **lab3/code/weather_mcp/inspector**: `@modelcontextprotocol/inspector` (`0.16.6` / `0.14.1` → `0.22.0`) ਨੂੰ ਬੰਪ ਕੀਤਾ, ਜਿਸ ਨਾਲ ਜੁੜੇ ਹੋਏ `ajv`, `brace-expansion`, `diff`, `path-to-regexp` ਅਤੇ `ws` ਸਲਾਹਾਂ ਹਟ ਗਈਆਂ। npm `overrides` ਵਰਤ ਕੇ `shell-quote@1.8.4` ਨੂੰ ਬਾਂਧਿਆ ਜੋ ਕਾਂਕਰਨਟਲੀ ਵਿਚਲੀ ਮਹੱਤਵਪੂਰਨ ਸਲਾਹ ਨੂੰ ਖਤਮ ਕਰਦਾ ਹੈ; ਦੋਨੋ ਲਾਕਫਾਈਲਜ਼ ਦੁਬਾਰਾ ਬਣਾਈਆਂ (ਹੁਣ 0 ਖਤਰੇ)

### MCP Specification 2025-11-25 ਅਨੁਕੂਲਤਾ ਸਮੀਖਿਆ

#### ਕੋਰ ਸੰਕਲਪ ਉਨ੍ਹਣਾ (01-CoreConcepts/)
- **ਨਵਾਂ ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵ - ਰੂਟਸ**: ਰੂਟਸ ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵ ਬਾਰੇ ਵਿਆਪਕ ਦਸਤਾਵੇਜ਼ ਸ਼ਾਮਿਲ ਕੀਤਾ, ਜੋ ਸਰਵਰਾਂ ਨੂੰ ਫਾਇਲਸਿਸਟਮ ਸੀਮਾਵਾਂ ਅਤੇ ਐਕਸੈੱਸ ਅਧਿਕਾਰ ਸਮਝਣਯੋਗ ਬਣਾਂਦਾ ਹੈ
- **ਟੂਲ ਐਨੋਟੇਸ਼ਨਜ਼**: ਟੂਲ ਬਿਹੇਵਿਓਰਲ ਐਨੋਟੇਸ਼ਨਜ਼ (`readOnlyHint`, `destructiveHint`) ਬਾਰੇ ਦਸਤਾਵੇਜ਼ ਸ਼ਾਮਿਲ ਕੀਤਾ ਜੋ ਟੂਲ ਨਿਰਵਾਹ ਫੈਸਲੇ ਨੂੰ ਸੁਧਾਰਦਾ ਹੈ
- **ਨਮੂਨਾ ਵਿੱਚ ਟੂਲ ਕਾਲਿੰਗ**: ਨਮੂਨਾ ਦਸਤਾਵੇਜ਼ ਵਿੱਚ `tools` ਅਤੇ `toolChoice` ਪੈਰਾਮੀਟਰ ਸ਼ਾਮਿਲ ਕੀਤੇ, ਜੋ ਮਾਡਲ-ਚਲਿਤ ਟੂਲ ਕਾਲਿੰਗ ਲਈ ਸੈਂਪਲਿੰਗ ਅਨੁਰੋਧਾਂ ਦੌਰਾਨ ਵਰਤੇ ਜਾਂਦੇ ਹਨ
- **URL ਮੋਡ ਇਲੀਸੀਟੇਸ਼ਨ**: ਸਰਵਰ-ਸ਼ੁਰੂਆਤ ਕੀਤੀਆਂ ਬਾਹਰੀ ਵੈੱਬ ਇੰਟਰੈਕਸ਼ਨਾਂ ਲਈ URL-ਆਧਾਰਿਤ ਇਲੀਸੀਟੇਸ਼ਨ ਬਾਰੇ ਦਸਤਾਵੇਜ਼ ਸ਼ਾਮਿਲ ਕੀਤਾ
- **ਟਾਸਕ (ਪਰਿਗਣਤ)**: ਟਾਸਕ ਫੀਚਰ ਬਾਰੇ ਨਵਾਂ ਅਨੁਭਵੀ ਭਾਗ ਦਸਤਾਵੇਜ਼ ਕੀਤਾ ਜੋ ਟਿਕਾਊ ਨਿਰਵਾਹ ਰੈਪਰ ਅਤੇ ਮੁਲਤਵੀ ਨਤੀਜਾ ਪ੍ਰਾਪਤੀ ਲਈ ਹੈ
- **ਆਈਕਾਨ ਸਹਾਇਤਾ**: ਹੁਣ ਟੂਲਜ਼, ਸਾਧਨ, ਰਿਸੋਰਸ ਟੈਮਪਲੇਟ ਅਤੇ ਪ੍ਰੋਂਪਟ ਵਿੱਚ ਵਾਧੂ ਮੈਟਾਡੇਟਾ ਵਜੋਂ ਆਈਕਾਨ ਸ਼ਾਮਿਲ ਕੀਤੇ ਜਾ ਸਕਦੇ ਹਨ

#### ਦਸਤਾਵੇਜ਼ ਅੱਪਡੇਟਸ
- **README.md**: MCP Specification 2025-11-25 ਵਰਜਨ ਸੰਦਰਭ ਅਤੇ ਤਾਰੀਖ ਅਧਾਰਿਤ ਵਰਜਨਦਾਰੀ ਸਮਝਾਇਆ
- **study_guide.md**: ਕੋਰ ਕੌਂਸੈਪਟ ਸੈਕਸ਼ਨ ਵਿੱਚ ਟਾਸਕ ਅਤੇ ਟੂਲ ਐਨੋਟੇਸ਼ਨਜ਼ ਸ਼ਾਮਿਲ ਕੀਤੇ; ਦਸਤਾਵੇਜ਼ ਦਿਨਾਂਕ ਅੱਪਡੇਟ ਕੀਤਾ

#### ਅਨੁਕੂਲਤਾ ਸਮੀਖਿਆ
- **ਪ੍ਰੋਟੋਕੋਲ ਵਰਜਨ**: ਸਾਰੇ ਦਸਤਾਵੇਜ਼ MCP Specification 2025-11-25 ਦੇ ਨਾਲ ਮੈਚ ਕੀਤੇ
- **ਆਰਕੀਟੈਕਚਰ ਅਨੁਕੂਲਤਾ**: ਦੁ-ਸਤਰੀਆ (ਡਾਟਾ ਲੇਅਰ + ਟ੍ਰਾਂਸਪੋਰਟ ਲੇਅਰ) ਦੀ ਦਸਤਾਵੇਜ਼ੀ ਸਹੀਤਾ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ
- **ਪ੍ਰਿਮਿਟਿਵ ਦਸਤਾਵੇਜ਼ੀ**: ਸਰਵਰ ਪ੍ਰਿਮਿਟਿਵ (ਰਿਸੋਰਸ, ਪ੍ਰੋਂਪਟ, ਟੂਲ) ਅਤੇ ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵ (ਸੈਂਪਲਿੰਗ, ਇਲੀਸੀਟੇਸ਼ਨ, ਲੌਗਿੰਗ, ਰੂਟਸ) ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ
- **ਟ੍ਰਾਂਸਪੋਰਟ ਮਕੈਨੀਜ਼ਮ**: STDIO ਅਤੇ ਸਟ੍ਰੀਮਏਬਲ HTTP ਟ੍ਰਾਂਸਪੋਰਟ ਦਸਤਾਵੇਜ਼ੀ ਦੀ ਪੁਸ਼ਟੀ
- **ਸੁਰੱਖਿਆ ਦਿਸ਼ਾ-ਨਿਰਦੇਸ਼**: ਮੌਜੂਦਾ MCP ਸੁਰੱਖਿਆ ਦੀਆਂ ਚੰਗੀਆਂ ਅਭਿਆਸਾਂ ਨਾਲ সਹਿਯੋਜਿਤ ਦੀ ਪੁਸ਼ਟੀ ਕੀਤੀ

### ਸ਼ੁਰੂਆਤੀ ਭਾਗ ਦਾ ਵਿਸਥਾਰ – ਉन्नਤ ਸਰਵਰ ਵਰਤੋਂ ਅਤੇ ਸਧਾਰਣ ਪ੍ਰਮਾਣੂਕਰਨ

#### ਉन्नਤ ਸਰਵਰ ਵਰਤੋਂ (03-GettingStarted/10-advanced)
- **ਨਵਾਂ ਅਧਿਆਇ ਸ਼ਾਮਿਲ ਕੀਤਾ**: ਉੱਚ ਪੱਧਰ ਅਤੇ ਤਲ-ਪੱਧਰ ਸਰਵਰ ਆਰਕੀਟੈਕਚਰਸ ਸਮੇਤ MCP ਸਰਵਰ ਦੀ ਵਿਸਥਾਰਿਤ ਵਰਤੋਂ ਸਬੰਧੀ ਮਾਰਗਦਰਸ਼ਨ ਸ਼ਾਮਿਲ ਕੀਤਾ ਗਿਆ ਹੈ।
  - **ਨਿਯਮਿਤ ਬਨਾਮ ਲੋਅ-ਲੇਵਲ ਸਰਵਰ**: ਦੋਹਾਂ ਪਹੁੰਚਾਂ ਲਈ ਪਾਇਥਨ ਅਤੇ ਟਾਈਪਸਕ੍ਰਿਪਟ ਵਿੱਚ ਵਿਸਥਾਰ ਨਾਲ ਤੁਲਨਾ ਅਤੇ ਕੋਡ ਉਦਾਹਰਨਾਂ।
  - **ਹੈਂਡਲਰ-ਅਧਾਰਿਤ ਡਿਜ਼ਾਈਨ**: ਸਕੇਲਬਲ, ਲਚਕੀਲੇ ਸਰਵਰ ਅਮਲਾਂ ਲਈ ਹੈਂਡਲਰ-ਅਧਾਰਿਤ ਟੂਲ/ਸਰੋਤ/ਪ੍ਰੌਂਪਟ ਪ੍ਰਬੰਧਨ ਦੀ ਵਿਆਖਿਆ।
  - **ਵਿਹਾਰਕ ਪੈਟਰਨ**: ਅਸਲ ਜ਼ਿੰਦਗੀ ਦੇ ਦ੍ਰਿਸ਼ਾਂਤ ਜਿੱਥੇ ਲੋਅ-ਲੇਵਲ ਸਰਵਰ ਪੈਟਰਨ ਉन्नਤ ਫੀਚਰਾਂ ਅਤੇ ਆਰਕੀਟੈਕਚਰ ਲਈ ਲਾਭਦਾਇਕ ਹਨ।

#### ਸਧਾਰਣ ਪ੍ਰਮਾਣਿਕਤਾ (03-GettingStarted/11-simple-auth)
- **ਨਵਾਂ ਅਧਿਆਇ ਸ਼ਾਮਿਲ ਕੀਤਾ ਗਿਆ**: MCP ਸਰਵਰਾਂ ਵਿੱਚ ਸਧਾਰਣ ਪ੍ਰਮਾਣਿਕਤਾ ਲਾਗੂ ਕਰਨ ਲਈ ਕਦਮ-ਦਰ-कਦਮ ਗਾਈਡ।
  - **ਪ੍ਰਮਾਣਿਕਤਾ ਸੰਕਲਪ**: ਪ੍ਰਮਾਣਿਕਤਾ ਵਿਰੁੱਧ ਪ੍ਰਮਾਣਿਕਰਨ ਅਤੇ ਪ੍ਰਮਾਣ ਪੱਤਰ ਸੰਭਾਲਣ ਦੀ ਸਪਸ਼ਟ ਵਿਆਖਿਆ।
  - **ਮੂਲ ਭੁਗਤਾਨ ਅਮਲ**: ਪਾਇਥਨ (ਸਟਾਰਲੇਟ) ਅਤੇ ਟਾਈਪਸਕ੍ਰਿਪਟ (ਐਕਸਪ੍ਰੈਸ) ਵਿੱਚ ਮਿਡਲਵੇਅਰ ਅਧਾਰਿਤ ਪ੍ਰਮਾਣਿਕਤਾ ਪੈਟਰਨ, ਕੋਡ ਉਦਾਹਰਨਾਂ ਸਮੇਤ।
  - **ਉन्नਤ ਸੁਰੱਖਿਆ ਵੱਲ ਪ੍ਰਗਤੀ**: ਸਧਾਰਣ ਪ੍ਰਮਾਣਿਕਤਾ ਨਾਲ ਸ਼ੁਰੂਆਤ ਕਰਨ ਅਤੇ OAuth 2.1 ਅਤੇ RBAC ਵੱਲ ਵਧਨ ਦੀ ਰਹਿਨੁਮਾ, ਉन्नਤ ਸੁਰੱਖਿਆ ਮੋਡੀਊਲਾਂ ਦੇ ਸੰਦਰਭਾਂ ਨਾਲ।

ਇਹ ਵਾਧੇ MCP ਸਰਵਰ ਅਮਲਾਂ ਨੂੰ ਹੋਰ ਮਜ਼ਬੂਤ, ਸੁਰੱਖਿਅਤ, ਅਤੇ ਲਚਕੀਲਾ ਬਣਾਉਣ ਲਈ ਪ੍ਰਯੋਗਿਕ, ਹੱਥ-ਵਾਲੀ ਮਦਦ ਪ੍ਰਦਾਨ ਕਰਦੇ ਹਨ, ਬੁਨਿਆਦੀ ਸੰਕਲਪਾਂ ਨੂੰ ਉन्नਤ ਉਤਪਾਦਨ ਪੈਟਰਨਾਂ ਨਾਲ ਜੋੜਦੇ ਹਨ।

  - **ਉੱਚ ਤਰ੍ਹਾਂ ਦੀ ਧਮਕੀ ਰੋਕਥਾਮ**: ਵਿਸਤ੍ਰਿਤ ਅਮਲ ਉਦਾਹਰਨਾਂ ਲਈ
    - PKCE ਅਤੇ ਉਪਭੋਗਤਾ ਸਹਿਮਤੀ ਪ੍ਰਮਾਣੀਕਰਨ ਨਾਲ Confused Deputy ਹਮਲਾ ਰੋਕਥਾਮ
    - ਦਰਸ਼ਕ ਪ੍ਰਮਾਣੀਕਰਨ ਅਤੇ ਸੁਰੱਖਿਅਤ ਟੋਕਨ ਪ੍ਰਬੰਧਨ ਨਾਲ ਟੋਕਨ ਪਾਸਥਰੂ ਰੋਕਥਾਮ
    - ਸੈਸ਼ਨ ਹਾਈਜੈਕਿੰਗ ਰੋਕਥਾਮ ਨਾਲ ਕ੍ਰਿਪਟੋਗ੍ਰਾਫਿਕ ਬਾਈਂਡਿੰਗ ਅਤੇ ਵਿਵਹਾਰਕ ਵਿਸ਼ਲੇਸ਼ਣ
  - **ਐਂਟਰਪ੍ਰਾਈਜ਼ ਸੁਰੱਖਿਆ ਇੰਟੀਗ੍ਰੇਸ਼ਨ**: Azurੇ ਐਪਲੀਕੇਸ਼ਨ ਇੰਸਾਈਟਸ ਮਾਨੀਟਰਨਗ, ਧਮਕੀ ਪਛਾਣ ਪਾਈਪਲਾਈਨਾਂ, ਅਤੇ ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ

- **ਦ੍ਰਸ਼ਟੀਗਤ ਸੰਕੇਤਕ**: ਜ਼ਰੂਰੀ ਮੰਗਾਂ ਅਤੇ ਸੁਪਾਰਸ਼ ਕੀਤੀਆਂ ਪ੍ਰਥਾਵਾਂ ਦੀ ਸਪਸ਼ਟ ਚਿਨ੍ਹਤਾਈ

#### ਮੁੱਖ ਧਾਰਾ (01-CoreConcepts/) - ਪੂਰੀ ਤਰ੍ਹਾਂ ਨਵੀਨੀਕਰਨ
- **ਪ੍ਰੋਟੋਕੋਲ ਵਰਜ਼ਨ ਅੱਪਡੇਟ**: ਮੌਜੂਦਾ MCP ਸੁਜ਼ਾਇਦਗੀ 2025-06-18 ਨੂੰ ਹਵਾਲੇ ਨਾਲ ਅੱਪਡੇਟ ਕੀਤਾ ਗਿਆ, ਤਾਰੀਖ ਅਧਾਰਤ ਵਰਜ਼ਨ (YYYY-MM-DD ਦੇ ਫਾਰਮੈਟ) ਨਾਲ
- **ਆਰਕੀਟੈਕਚਰ ਸੁਧਾਰ**: ਮੌਜੂਦਾ MCP ਆਰਕੀਟੈਕਚਰ ਪੈਟਰਨਾਂ ਨੂੰ ਦਰਸਾਉਂਦੇ ਹੋਏ ਹੋਸਟਸ, ਕਲਾਇੰਟਸ ਅਤੇ ਸਰਵਰਾਂ ਦਾ ਵਧੀਆ ਵਰਣਨ
  - ਹੋਸਟਸ ਹੁਣ ਸਾਫ਼ ਤੌਰ 'ਤੇ AI ਐਪਲੀਕੇਸ਼ਨਾਂ ਵਜੋਂ ਪਰਿਭਾਸ਼ਿਤ ਜੋ ਕਈ MCP ਕਲਾਇੰਟ ਕਨੈਕਸ਼ਨਾਂ ਦਾ ਸੰਚਾਲਨ ਕਰਦੀਆਂ ਹਨ
  - ਕਲਾਇੰਟ ਦਰਸਾਏ ਗਏ ਹਨ ਜਿਵੇਂ ਪ੍ਰੋਟੋਕੋਲ ਕਨੈਕਟਰ ਜੋ ਇਕ-ਇੱਕ ਸਰਵਰ ਸੰਬੰਧਾਂ ਨੂੰ ਬਣਾਏ ਰੱਖਦੇ ਹਨ
  - ਸਰਵਰਾਂ ਨੂੰ ਸਥਾਨਕ ਅਤੇ ਦੂਰੇ ਪਰिनਿਆਮਾਂ ਨਾਲ ਮਜ਼ਬੂਤ ਕੀਤਾ ਗਿਆ
- **ਪ੍ਰਿਮਿਟਿਵ ਪੁਨਰਸੰਰਚਨਾ**: ਸਰਵਰ ਅਤੇ ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵ ਦੀ ਪੂਰੀ ਤਰ੍ਹਾਂ ਨਵੀਨਤਮ ਬਦਲਾਅ
  - ਸਰਵਰ ਪ੍ਰਿਮਿਟਿਵ: ਸਰੋਤਾਂ (ਡਾਟਾ ਸਰੋਤ), ਪ੍ਰਾਂਪਟ (ਟੈਮਪਲੇਟ), ਸੰਦ (ਕਾਰਜਯੋਗ ਫੰਕਸ਼ਨ) ਵਿਸਥਾਰ ਨਾਲ ਸਮਝਾਏ ਅਤੇ ਉਦਾਹਰਣ ਸਮੇਤ
  - ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵ: ਸਮਪਲਿੰਗ (LLM ਮੁਕੰਮਲ), ਪ੍ਰੇਰਣਾ (ਉਪਭੋਗਤਾ ਇਨਪੁਟ), ਲੋਗਿੰਗ (ਡਿਬੱਗ/ਮਾਨੀਟਰਨਗ)
  - ਮੌਜੂਦਾ ਖੋਜ (`*/list`), ਪ੍ਰਾਪਤੀ (`*/get`), ਅਤੇ ਕਾਰਜਨਵਾਈ (`*/call`) ਵਿਧੀ ਪੈਟਰਨਾਂ ਨਾਲ ਅੱਪਡੇਟ ਕੀਤਾ
- **ਪ੍ਰੋਟੋਕੋਲ ਆਰਕੀਟੈਕਚਰ**: ਦੋ-ਪਹਿਰਾ ਆਰਕੀਟੈਕਚਰ ਮਾਡਲ ਛੱਡਿਆ
  - ਡਾਟਾ ਲੇਅਰ: JSON-RPC 2.0 ਬੁਨਿਆਦ ਨਾਲ ਜੀਵਨਚੱਕਰ ਪਰਬੰਧਨ ਅਤੇ ਪ੍ਰਿਮਿਟਿਵ
  - ਟ੍ਰਾਂਸਪੋਰਟ ਲੇਅਰ: STDIO (ਸਥਾਨਕ) ਅਤੇ ਸਟਰੀਮਬਲ HTTP ਨਾਲ SSE (ਦੂਰੀ) ਟ੍ਰਾਂਸਪੋਰਟ ਮਕੈਨਿਜ਼ਮ
- **ਸੁਰੱਖਿਆ ਢਾਂਚਾ**: ਵਿਸਥਾਰਿਕ ਸੁਰੱਖਿਆ ਸਿਧਾਂਤ ਜਿਵੇਂ ਸਪਸ਼ਟ ਉਪਭੋਗਤਾ ਸਹਿਮਤੀ, ਡਾਟਾ ਗੋਪਨੀਯਤਾ ਸੁਰੱਖਿਆ, ਸੰਦਾਂ ਦੀ ਸੁਰੱਖਿਆ, ਅਤੇ ਟ੍ਰਾਂਸਪੋਰਟ ਲੇਅਰ ਸੁਰੱਖਿਆ
- **ਸੰਚਾਰ ਪੈਟਰਨ**: ਸ਼ੁਰੂਆਤ, ਖੋਜ, ਕਾਰਜਨਵਾਈ ਅਤੇ ਸੂਚਨਾ ਪ੍ਰਵਾਹਾਂ ਦਿਖਾਉਂਦੇ ਪ੍ਰੋਟੋਕੋਲ ਸੁਨੇਹੇ ਅੱਪਡੇਟ ਕੀਤੇ
- **ਕੋਡ ਉਦਾਹਰਣ**: ਕੁਝ-ਭਾਸ਼ਾ ਉਦਾਹਰਣ (.NET, ਜਾਵਾ, ਪਾਇਥਾਨ, ਜਾਵਾਸਕ੍ਰਿਪਟ) ਨੂੰ ਮੌਜੂਦਾ MCP SDK ਪੈਟਰਨਾਂ ਨਾਲ ਤਾਜ਼ਾ ਕੀਤਾ

#### ਸੁਰੱਖਿਆ (02-Security/) - ਵਿਸਥਾਰਿਕ ਸੁਰੱਖਿਆ ਨਵੀਨੀਕਰਨ  
- **ਮਿਆਰੀਆਂ ਦਾ ਸੰਗਤੀਬੱਧਤਾ**: MCP ਸੁਜ਼ਾਇਦਗੀ 2025-06-18 ਦੀਆਂ ਸੁਰੱਖਿਆ ਜ਼ਰੂਰਤਾਂ ਨਾਲ ਪੂਰੀ ਸੰਗਤੀ
- **ਪਹਚਾਣ ਵਿਕਾਸ**: ਕਸਟਮ OAuth ਸਰਵਰਾਂ ਤੋਂ ਬਾਹਰੀ ਪਹਚਾਣ ਪ੍ਰਦਾਤਾ ਡੈਲਿਗੇਸ਼ਨ (Microsoft Entra ID) ਤੱਕ ਵਿਕਾਸ ਦਰਜ
- **AI ਵਿਸ਼ੇਸ਼ ਧਮਕੀ ਵਿਸ਼ਲੇਸ਼ਣ**: ਆਧੁਨਿਕ AI ਹਮਲੇ ਦੇ ਵੇਕਟਰਾਂ ਦੀ ਵਧੀਕ ਕਵਰੇਜ
  - ਵਿਸਥਾਰ ਨਾਲ ਪ੍ਰਾਂਪਟ ਇੰਜੈਕਸ਼ਨ ਹਮਲੇ ਦੀਆਂ ਸਥਿਤੀਆਂ ਅਸਲੀ ਉਦਾਹਰਣਾਂ ਨਾਲ
  - ਸੰਦ ਬੇਮਾਰ ਕਰਨ ਵਾਲੀਆਂ ਤਕਨੀਕਾਂ ਅਤੇ "ਰੱਗ ਪੁਲ" ਹਮਲੇ ਦੇ ਪੈਟਰਨ
  - ਸੰਦਰਭ ਵਿਂਡੋ ਬੇਮਾਰ ਕਰਨ ਅਤੇ ਮਾਡਲ ਗਲਤਫਹਿਮੀ ਹਮਲੇ
- **ਮਾਈਕਰੋਸੌਫਟ AI ਸੁਰੱਖਿਆ ਹੱਲ**: ਮਾਈਕਰੋਸੌਫਟ ਸੁਰੱਖਿਆ ਇਕੋਸਿਸਟਮ ਦੀ ਵਿਸਥਾਰਿਕ ਕਵਰੇਜ
  - AI ਪ੍ਰਾਂਪਟ ਸ਼ੀਲਡਸ ਵਧੀਆ ਪਛਾਣ, ਸਪੌਟਲਾਈਟਿੰਗ ਅਤੇ ਡੈਲੀਮੀਟਰ ਤਕਨੀਕਾਂ ਨਾਲ
  - Azure ਕੰਟੈਂਟ ਸੇਫਟੀ ਇੰਟੀਗ੍ਰੇਸ਼ਨ ਪੈਟਰਨ
  - GitHub ਐਡਵਾਂਸਡ ਸੁਰੱਖਿਆ ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ ਲਈ
- **ਉੱਚ ਪੱਧਰ ਦੀ ਧਮਕੀ ਨਿਵਾਰਣ**: ਵਿਸਥਾਰਿਤ ਸੁਰੱਖਿਆ ਨਿਯੰਤਰਣ ਲਈ
  - MCP ਵਿਸ਼ੇਸ਼ ਹਮਲੇ ਸਥਿਤੀਆਂ ਅਤੇ ਕ੍ਰਿਪਟੋਗ੍ਰਾਫਿਕ ਸੈਸ਼ਨ ID ਲੋੜਾਂ ਨਾਲ ਸੈਸ਼ਨ ਹਾਈਜੈਕਿੰਗ
  - MCP ਪ੍ਰਾਕਸੀ ਸਥਿਤੀਆਂ ਵਿੱਚ ਗੁੰਝਲਦਾਰ ਡਿਊਟੀ ਹੱਲਾਂ ਲਈ ਸਪਸ਼ਟ ਸਹਿਮਤੀ ਜ਼ਰੂਰੀਆਂ
  - ਟੋਕਨ ਪਾਸਥਰੂ ਕਮਜ਼ੋਰੀਆਂ ਲਈ ਮੈਚੂਰੀ ਵੈਧਤਾ ਨਿਯੰਤਰਣ
- **ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ**: ਆਈ ਸਪਲਾਈ ਚੇਨ ਦੀ ਵਿਆਪਕ ਕਵਰੇਜ ਜਿੱਥੇ ਬੁਨਿਆਦੀ ਮਾਡਲ, ਐਂਬੈੱਡਿੰਗ ਸੇਵਾਵਾਂ, ਸੰਦਰਭ ਪ੍ਰਦਾਤਾ, ਅਤੇ ਤੀਜੀ ਪਾਰਟੀ ਏਪੀਆਈਜ਼ ਸ਼ਾਮਲ ਹਨ
- **ਬੁਨਿਆਦੀ ਸੁਰੱਖਿਆ**: ਐਂਟਰਪ੍ਰਾਈਜ਼ ਸੁਰੱਖਿਆ ਪੈਟਰਨ ਸਮੇਤ ਜ਼ੀਰੋ ਟਰੱਸਟ ਆਰਕੀਟੈਕਚਰ ਅਤੇ ਮਾਈਕਰੋਸੌਫਟ ਸੁਰੱਖਿਆ ਇਕੋਸਿਸਟਮ ਨਾਲ ਵਧੀਆ ਇੰਟੀਗ੍ਰੇਸ਼ਨ
- **ਸੰਸਾਧਨ ਦਾ ਸੰਗਠਨ**: ਕਿਸਮ-ਵਾਰ (ਅਧਿਕਾਰਿਕ ਦਸਤਾਵੇਜ਼, ਮਿਆਰ, ਖੋਜ, ਮਾਈਕਰੋਸੌਫਟ ਹੱਲ, ਲਾਗੂ ਕਰਨ ਵਾਲੇ ਗਾਇਡ) ਵਾਂਗ ਸੰਸਾਧਨਾਂ ਨੂੰ ਵਰਗੀਕ੍ਰਿਤ ਕੀਤਾ
