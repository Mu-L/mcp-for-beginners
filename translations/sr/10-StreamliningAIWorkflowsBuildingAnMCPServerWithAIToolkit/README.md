## 🏗️ Технолошки Стек

### 🔌 Model Context Protocol (MCP)

MCP је **„USB-C за AI“** - универзални стандард који повезује AI моделе са спољним алатима и изворима података.

**✨ Кључне Карактеристике:**

- 🔄 **Стандардизована Интеграција**: Универзални интерфејс за повезивање AI алата
- 🏛️ **Флексибилна Архитектура**: Локални и удаљени сервери преко stdio/SSE транспорта
- 🧰 **Богат Екосистем**: Алатке, упити и ресурси у једном протоколу
- 🔒 **Спремно за Предузећа**: Уграђена безбедност и поузданост

**🎯 Зашто је MCP важан:**
Као што је USB-C елиминисао неред каблова, MCP уклања сложеност AI интеграција. Један протокол, бескрајне могућности.

### 🤖 Microsoft Foundry Toolkit екстензија за VS Code

Заступничка Microsoft екстензија за развој AI која претвара VS Code у AI центар моћи.

**🚀 Основне Могућности:**

- 📦 **Каталог Модела**: Приступ моделима са Azure AI, GitHub, Hugging Face, Ollama
- ⚡ **Локално Инференцирање**: ONNX оптимизовано извршавање на CPU/GPU/NPU
- 🏗️ **Agent Builder**: Визуелни развој AI агената са MCP интеграцијом
- 🎭 **Више-Модалности**: Подршка за текст, визуелне и структуиране излазе

**💡 Предности развоја:**

- Деплој без конфигурације модела
- Визуелно креирање упита
- Песковник за тестирање у реалном времену
- Беспрекорна интеграција MCP сервера

## 📚 Пут учења

### [🚀 Модул 1: Основе Microsoft Foundry Toolkit-а](./lab1/README.md)

**Трајање**: 15 минута

- 🛠️ Инсталирајте и конфигуришите Microsoft Foundry Toolkit за VS Code
- 🗂️ Истражите Каталог Модела (100+ модела са GitHub, ONNX, OpenAI, Anthropic, Google)
- 🎮 Савладајте Interactive Playground за тестирање модела у реалном времену
- 🤖 Креирајте свог првог AI агента помоћу Agent Builder-а
- 📊 Процените перформансе модела са уграђеним мерљивим показатељима (F1, релевантност, сличност, кохеренција)
- ⚡ Упознајте се с обрадом у серијама и подршком за више модалитета

**🎯 Резултат учења**: Направити функционалног AI агента са свеобухватним разумевањем могућности Microsoft Foundry Toolkit-а

### [🌐 Модул 2: MCP са Microsoft Foundry Toolkit Основама](./lab2/README.md)

**Трајање**: 20 минута

- 🧠 Савладајте Model Context Protocol (MCP) архитектуру и концепте
- 🌐 Истражите Microsoft MCP сервер екосистем
- 🤖 Направите агента за аутоматизацију прегледача користећи Playwright MCP сервер
- 🔧 Интегришите MCP сервере са Microsoft Foundry Toolkit Agent Builder-ом
- 📊 Конфигуришите и тестирате MCP алатке у вашим агентима
- 🚀 Извезите и деплоy-ујте AI агенте покретане MCP-ом за продукцију

**🎯 Резултат учења**: Деплоy AI агента ојачаног спољним алатима преко MCP-а

### [🔧 Модул 3: Напредни MCP развој са Microsoft Foundry Toolkit-ом](./lab3/README.md)

**Трајање**: 20 минута

- 💻 Креирајте прилагођене MCP сервере користећи Microsoft Foundry Toolkit
- 🐍 Конфигуришите и користите најновији MCP Python SDK (в1.9.3)
- 🔍 Подесите и користите MCP Inspector за дебаговање
- 🛠️ Направите Weather MCP Server са професионалним дебаг радним токовима
- 🧪 Отстрањујте проблеме MCP сервера у Agent Builder-у и Inspector окружењима

**🎯 Резултат учења**: Развијање и дебаговање прилагођених MCP сервера савременим алатима

### [🐙 Модул 4: Практични MCP развој - прилагођени GitHub Clone Server](./lab4/README.md)

**Трајање**: 30 минута

- 🏗️ Направите реалан GitHub Clone MCP Server за развојне радне токове
- 🔄 Имплементирајте интелигентно клонирање репозиторијума са валидацијом и обрадом грешака
- 📁 Креирајте интелигентно управљање директоријумима и интеграцију са VS Code
- 🤖 Користите GitHub Copilot Agent Mode са прилагођеним MCP алаткама
- 🛡️ Примените поузданост спремну за продукцију и компатибилност на више платформи

**🎯 Резултат учења**: Деплоy MCP сервер спреман за продукцију који оптимизује стварне развојне радне токове

## 💡 Практичне примене и утицај

### 🏢 Пословни случајеви

#### 🔄 Аутоматизација DevOps

Трансформишите свој развојни ток интелигентном аутоматизацијом:

- **Интелигентно управљање репозиторијумом**: AI вођена ревизија кода и одлуке о спајању
- **Интелигентан CI/CD**: Аутоматска оптимизација цевовода на основу измена у коду
- **Трирање проблематика**: Аутоматска класификација и додела грешака

#### 🧪 Револуција у контроли квалитета

Подигните тестирање AI-ом подржаном аутоматизацијом:

- **Интелигентна генерација тестова**: Аутоматско креирање свеобухватних тестова
- **Визуелно регресионo тестирање**: AI препознавање промена у UI-у
- **Надгледање перформанси**: Проактивно идентификовање и решавање проблема

#### 📊 Интелигенција у обради података

Креирајте паметније токове обраде података:

- **Адаптивни ETL процеси**: Самооптимизујуће трансформације података
- **Детекција аномалија**: Надгледање квалитета података у реалном времену
- **Интелигентно руковање подацима**: Управљање протоком података

#### 🎧 Побољшање корисничког искуства

Остварите изузетне корисничке интеракције:

- **Подршка свесна контекста**: AI агенти са приступом историји корисника
- **Проактивно решавање проблема**: Прогностичка корисничка служба
- **Интеграција на више канала**: Јединствено AI искуство на разним платформама

## 🛠️ Захтеви и Подешавање

### 💻 Системски Захтеви

| Компонента | Захтев | Напомена |
|-----------|-------------|-------|
| **Оперативни систем** | Windows 10+, macOS 10.15+, Linux | Било који савремени OS |
| **Visual Studio Code** | Најновија стабилна верзија | Потребно за Microsoft Foundry Toolkit |
| **Node.js** | v18.0+ и npm | За развој MCP сервера |
| **Python** | 3.10+ | Опционо за Python MCP сервере |
| **Рам Меморија** | Минимум 8GB RAM | 16GB препоручено за локалне моделе |

### 🔧 Развојно Окружење

#### Препоручене VS Code екстензије

- **Microsoft Foundry Toolkit** (ms-windows-ai-studio.windows-ai-studio)
- **Python** (ms-python.python)
- **Python Debugger** (ms-python.debugpy)
- **GitHub Copilot** (GitHub.copilot) - Опционо али корисно

#### Опционални Алати

- **uv**: Модерни менаџер Python пакета
- **MCP Inspector**: Визуелни алат за дебаг MCP сервера
- **Playwright**: За примере веб аутоматизације
