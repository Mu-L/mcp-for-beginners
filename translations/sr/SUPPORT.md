# Подршка

## Како пријавити проблеме и добити помоћ  

Овај пројекат користи GitHub Issues за праћење грешака и захтева за нове функције. Молимо вас да прегледате постојеће 
проблеме пре него што пријавите нове како бисте избегли дупликате. За нове проблеме, пријавите грешку или 
захтев за функцију као нови Issue.

За помоћ и питања у вези коришћења овог пројекта, молимо вас да:
- Проверите [Microsoft Foundry Discord](https://discord.com/invite/ByRwuEEgH4) где се састају стручњаци и други програмери
- Користите ресурсе [Model Context Protocol Community](https://modelcontextprotocol.io/community/) за питања везана за протокол

## Microsoft политика подршке  

Подршка за пројекат Model Context Protocol (MCP) за почетнике ограничена је на горе наведене ресурсе.
